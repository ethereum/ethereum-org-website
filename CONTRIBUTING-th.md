# วิธีการมีส่วนร่วมในโปรเจกต์ (ภาษาไทย)

สวัสดีครับ! ขอบคุณที่สนใจช่วยพัฒนาโปรเจกต์ Ethereum.org คุณสามารถช่วยเราได้ง่ายๆ แม้จะมีแค่มือถือเครื่องเดียว

## สิ่งที่คุณช่วยเราได้
* **การแปลภาษา:** ช่วยแปลเนื้อหาจากภาษาอังกฤษเป็นภาษาไทยผ่านระบบ Crowdin
* **รายงานข้อผิดพลาด:** หากพบจุดไหนบนเว็บไซต์ที่แสดงผลผิดปกติ สามารถแจ้งได้ที่หน้า Issues
* **เสนอแนะไอเดีย:** หากมีไอเดียใหม่ๆ ในการปรับปรุงเว็บไซต์ให้คนไทยใช้งานง่ายขึ้น สามารถพูดคุยกับเราได้เสมอ

## ขั้นตอนการส่งงาน (Contribution)
1. **Fork** โปรเจกต์นี้ไปยังบัญชีของคุณ
2. **แก้ไขหรือเพิ่มเนื้อหา** ในสาขา (Branch) ที่คุณต้องการ
3. **Commit** การเปลี่ยนแปลงและส่ง **Pull Request (PR)** มาให้เราตรวจสอบ

มาร่วมเป็นส่วนหนึ่งของการสร้างสังคม Web3 ในไทยไปด้วยกันครับ!
