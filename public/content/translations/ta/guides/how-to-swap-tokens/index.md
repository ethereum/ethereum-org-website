# டோக்கன்களை மாற்றுவது எப்படி

உங்களுக்குப் பிடித்தமான டோக்கன்கள் அனைத்தையும் பட்டியலிடும் ஒரு பரிமாற்றத்தைத் தேடி சோர்வடைந்துவிட்டீர்களா? [பரவலாக்கப்பட்ட பரிமாற்றங்களைப்](/glossary/#dex) பயன்படுத்தி பெரும்பாலான டோக்கன்களை நீங்கள் மாற்றிக்கொள்ளலாம்.

டோக்கன் மாற்றம் என்பது எத்தேரியம் நெட்வொர்க்கில் இருக்கும் இரண்டு வெவ்வேறு சொத்துக்களின் பரிமாற்றத்தை உள்ளடக்கியது, எடுத்துக்காட்டாக ETH-ஐ DAI-க்காக மாற்றுவது (ஒரு [ERC-20](/glossary/#erc-20) டோக்கன்). இந்தச் செயல்முறை மிகவும் வேகமானது மற்றும் மலிவானது. டோக்கன்களை மாற்றுவதற்கு உங்களுக்கு ஒரு கிரிப்டோ பணப்பை தேவைப்படும்.

**முன்நிபந்தனை:**

- ஒரு [கிரிப்டோ பணப்பை](/glossary/#wallet)-ஐ வைத்திருக்க வேண்டும்; உங்களிடம் இல்லையென்றால், [எத்தேரியம் கணக்கை எவ்வாறு உருவாக்குவது](/guides/how-to-create-an-ethereum-account/) என்பது குறித்த இந்த வழிகாட்டியைப் பின்பற்றலாம்.
- உங்கள் பணப்பையில் நிதிகளைச் சேர்க்கவும்

## 1. உங்களுக்கு விருப்பமான பரவலாக்கப்பட்ட பரிமாற்றத்துடன் (DEX) உங்கள் பணப்பையை இணைக்கவும்

சில பிரபலமான பரிமாற்றங்கள்:

- [Uniswap](https://app.uniswap.org/#/swap)
- [Sushiswap](https://www.sushi.com/swap)
- [1Inch](https://app.1inch.io/#/1/unified/swap/ETH/DAI)
- [Curve](https://www.curve.finance/dex/ethereum/swap/)

சுவாரஸ்யமாக உள்ளதா? [பரவலாக்கப்பட்ட நிதி (DeFi)](/defi/) என்றால் என்ன, இந்த புதிய வகை பரிமாற்றங்கள் எவ்வாறு செயல்படுகின்றன என்பதைப் பற்றி மேலும் அறியுங்கள்.

## 2. நீங்கள் மாற்ற விரும்பும் டோக்கன்களின் ஜோடியைத் தேர்ந்தெடுக்கவும்

எடுத்துக்காட்டாக, ETH மற்றும் DAI. இரண்டு டோக்கன்களில் ஒன்றில் உங்களிடம் நிதி உள்ளதா என்பதை உறுதிப்படுத்திக்கொள்ளவும்.
![மாற்றுவதற்கான பொதுவான இடைமுகம்](./swap1.png)

## ஒரு நல்ல கதை. நீங்கள் வர்த்தகம் செய்ய விரும்பும் டோக்கன்களின் அளவை உள்ளிட்டு, swap என்பதைக் கிளிக் செய்யவும்

நீங்கள் எத்தனை டோக்கன்களைப் பெறுவீர்கள் என்பதைப் பரிமாற்றம் தானாகவே கணக்கிடும்.

![மாற்றுவதற்கான பொதுவான இடைமுகம்](./swap2.png)

## 4. பரிவர்த்தனையை உறுதிப்படுத்தவும்

பரிவர்த்தனையின் விவரங்களை மதிப்பாய்வு செய்யவும். விரும்பத்தகாத ஆச்சரியங்களைத் தவிர்க்க, பரிமாற்ற விகிதம் மற்றும் வேறு ஏதேனும் கட்டணங்களைச் சரிபார்க்கவும்.

![பரிவர்த்தனையை மதிப்பாய்வு செய்வதற்கான பொதுவான இடைமுகம்](./swap3.png)

## Visual Direction • உள்ளடக்கம். பரிவர்த்தனை செயல்படுத்தப்படும் வரை காத்திருக்கவும்

எந்தவொரு பிளாக்செயின் எக்ஸ்புளோரரிலும் பரிவர்த்தனையின் முன்னேற்றத்தை நீங்கள் காணலாம். இந்தச் செயல்முறைக்கு 10 நிமிடங்களுக்கு மேல் ஆகக்கூடாது.

பரிவர்த்தனை செயல்படுத்தப்பட்டவுடன், மாற்றப்பட்ட டோக்கன்களை உங்கள் பணப்பையில் தானாகவே பெறுவீர்கள். <br />

## அடிக்கடி கேட்கப்படும் கேள்விகள் (Faqs)

### எனது பணப்பையிலிருந்து ETH-ஐ BTC-க்காக மாற்ற முடியுமா?

இல்லை, எத்தேரியம் நெட்வொர்க்கிற்கு உரிய டோக்கன்களை மட்டுமே நீங்கள் மாற்ற முடியும், அதாவது ETH, ERC-20 டோக்கன்கள் அல்லது NFTகள் போன்றவை. எத்தேரியத்தில் இருக்கும் பிட்காயினின் "சுற்றப்பட்ட" வடிவங்களை மட்டுமே நீங்கள் மாற்ற முடியும்.

### நழுவல் என்றால் என்ன?

இது நீங்கள் எதிர்பார்க்கும் பரிமாற்ற விகிதத்திற்கும் உண்மையான விகிதத்திற்கும் உள்ள வித்தியாசம்.
