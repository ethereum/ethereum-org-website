Ethereum சுற்றுச்சூழல் அமைப்பில் மிகவும் பிரபலமான மொழிகளில் ஜாவாஸ்கிரிப்ட் ஒன்றாகும். உண்மையில், Ethereum-ஐ முடிந்தவரை ஜாவாஸ்கிரிப்டுக்குக் கொண்டுவருவதற்காக அர்ப்பணிக்கப்பட்ட ஒரு [குழு](https://github.com/ethereumjs) உள்ளது.

[அடுக்கின் அனைத்து நிலைகளிலும்](/developers/docs/ethereum-stack/) ஜாவாஸ்கிரிப்ட் (அல்லது அதற்கு நெருக்கமான ஒன்றை) எழுத வாய்ப்புகள் உள்ளன.

## Ethereum உடன் தொடர்புகொள்ளுங்கள் {#interact-with-ethereum}

### ஜாவாஸ்கிரிப்ட் API நூலகங்கள் {#javascript-api-libraries}

பிளாக்செயினை வினவவும், பரிவர்த்தனைகளை அனுப்பவும் மற்றும் பலவற்றைச் செய்யவும் நீங்கள் ஜாவாஸ்கிரிப்ட் எழுத விரும்பினால், இதைச் செய்வதற்கான மிகவும் வசதியான வழி [ஜாவாஸ்கிரிப்ட் API நூலகத்தைப்](/developers/docs/apis/javascript/) பயன்படுத்துவதாகும். இந்த API-கள் டெவலப்பர்களை [Ethereum நெட்வொர்க்கில் உள்ள நோடுகளுடன்](/developers/docs/nodes-and-clients/) எளிதாகத் தொடர்புகொள்ள அனுமதிக்கின்றன.

Ethereum-இல் உள்ள ஸ்மார்ட் ஒப்பந்தங்களுடன் தொடர்புகொள்ள இந்த நூலகங்களை நீங்கள் பயன்படுத்தலாம், எனவே முன்பே இருக்கும் ஒப்பந்தங்களுடன் தொடர்புகொள்ள ஜாவாஸ்கிரிப்டை மட்டும் பயன்படுத்தும் ஒரு dapp-ஐ உருவாக்க முடியும்.

**இவற்றைப் பாருங்கள்**

- [Web3.js](https://web3js.readthedocs.io)
- [Ethers.js](https://ethers.org) – _ஜாவாஸ்கிரிப்ட் மற்றும் டைப்ஸ்கிரிப்ட்டில் (TypeScript) Ethereum வாலட் செயலாக்கம் மற்றும் பயன்பாடுகளை உள்ளடக்கியது._
- [viem](https://viem.sh) – _Ethereum-உடன் தொடர்புகொள்வதற்கான குறைந்த-நிலை நிலையற்ற (stateless) அடிப்படைகளை வழங்கும் Ethereum-க்கான டைப்ஸ்கிரிப்ட் இடைமுகம்._
- [Drift](https://ryangoree.github.io/drift/) – _web3 நூலகங்கள் முழுவதும் சிரமமற்ற Ethereum மேம்பாட்டிற்காக உள்ளமைக்கப்பட்ட கேச்சிங் (caching), ஹூக்குகள் (hooks) மற்றும் சோதனை மாக்குகளைக் (test mocks) கொண்ட டைப்ஸ்கிரிப்ட் மெட்டா-நூலகம்._

### ஸ்மார்ட் ஒப்பந்தங்கள் {#smart-contracts}

நீங்கள் ஒரு ஜாவாஸ்கிரிப்ட் டெவலப்பராக இருந்து, உங்கள் சொந்த ஸ்மார்ட் ஒப்பந்தத்தை எழுத விரும்பினால், நீங்கள் [Solidity](https://solidity.readthedocs.io)-ஐப் பற்றி தெரிந்துகொள்ள விரும்பலாம். இது மிகவும் பிரபலமான ஸ்மார்ட் ஒப்பந்த மொழியாகும், மேலும் இது தொடரியல் ரீதியாக ஜாவாஸ்கிரிப்டைப் போலவே உள்ளது, இது கற்றுக்கொள்வதை எளிதாக்கும்.

[ஸ்மார்ட் ஒப்பந்தங்கள்](/developers/docs/smart-contracts/) பற்றி மேலும் அறிக.

## நெறிமுறையைப் புரிந்துகொள்ளுங்கள் {#understand-the-protocol}

### Ethereum மெய்நிகர் இயந்திரம் (EVM) {#the-ethereum-virtual-machine}

[Ethereum-இன் மெய்நிகர் இயந்திரத்தின்](/developers/docs/evm/) ஜாவாஸ்கிரிப்ட் செயலாக்கம் உள்ளது. இது சமீபத்திய ஃபோர்க் (fork) விதிகளை ஆதரிக்கிறது. ஃபோர்க் விதிகள் என்பது திட்டமிடப்பட்ட மேம்படுத்தல்களின் விளைவாக EVM-இல் செய்யப்பட்ட மாற்றங்களைக் குறிக்கின்றன.

இது பல்வேறு ஜாவாஸ்கிரிப்ட் தொகுப்புகளாகப் பிரிக்கப்பட்டுள்ளது, இவற்றை நீங்கள் நன்கு புரிந்துகொள்ளப் பார்க்கலாம்:

- கணக்குகள் (Accounts)
- தொகுதிகள் (Blocks)
- பிளாக்செயின் (The blockchain itself)
- பரிவர்த்தனைகள் (Transactions)
- மேலும் பல...

"ஒரு கணக்கின் தரவு அமைப்பு என்ன?" போன்ற விஷயங்களைப் புரிந்துகொள்ள இது உங்களுக்கு உதவும்.

நீங்கள் குறியீட்டைப் படிக்க விரும்பினால், எங்கள் ஆவணங்களைப் படிப்பதற்கு இந்த ஜாவாஸ்கிரிப்ட் ஒரு சிறந்த மாற்றாக இருக்கும்.

**EVM-ஐப் பாருங்கள்**  
[`@ethereumjs/evm`](https://github.com/ethereumjs/ethereumjs-monorepo/tree/master/packages/evm)

### நோடுகள் மற்றும் கிளையண்டுகள் {#nodes-and-clients}

Ethereum கிளையண்டுகள் எவ்வாறு செயல்படுகின்றன என்பதை நீங்கள் புரிந்துகொள்ளும் மொழியான ஜாவாஸ்கிரிப்ட்டில் ஆராய அனுமதிக்கும் ஒரு Ethereumjs கிளையண்ட் செயலில் உள்ள மேம்பாட்டில் உள்ளது!

## பிற திட்டங்கள் {#other-projects}

Ethereum ஜாவாஸ்கிரிப்ட் உலகில் இன்னும் பல விஷயங்கள் நடந்து கொண்டிருக்கின்றன, அவற்றுள்:

- வாலட் பயன்பாடுகளின் நூலகங்கள்.
- Ethereum திறவுகோல்களை (keys) உருவாக்க, இறக்குமதி செய்ய மற்றும் ஏற்றுமதி செய்வதற்கான கருவிகள்.
- `merkle-patricia-tree`-இன் செயலாக்கம் – இது Ethereum மஞ்சள் தாளில் (yellow paper) கோடிட்டுக் காட்டப்பட்டுள்ள ஒரு தரவு அமைப்பாகும்.

[EthereumJS ரெப்போவில் (repo)](https://github.com/ethereumjs) உங்களுக்கு மிகவும் விருப்பமானவற்றை ஆராயுங்கள்.
