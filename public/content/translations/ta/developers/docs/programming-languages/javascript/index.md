எத்திரியம் சுற்றுச்சூழல் அமைப்பில் மிகவும் பிரபலமான மொழிகளில் JavaScript ஒன்றாகும். உண்மையில், எத்திரியத்தை முடிந்தவரை JavaScript-க்கு கொண்டு வர அர்ப்பணிக்கப்பட்ட ஒரு [குழு](https://github.com/ethereumjs) உள்ளது.

[அடுக்கின் அனைத்து நிலைகளிலும்](/developers/docs/ethereum-stack/) JavaScript-ஐ (அல்லது அதற்கு நெருக்கமான ஒன்றை) எழுத வாய்ப்புகள் உள்ளன.

## எத்திரியத்துடன் தொடர்புகொள்ளுங்கள் {#interact-with-ethereum}

### JavaScript API நிரலகங்கள் {#javascript-api-libraries}

தொகுதிச்சங்கிலியை வினவவும், பரிவர்த்தனைகளை அனுப்பவும் மற்றும் பலவற்றைச் செய்யவும் நீங்கள் JavaScript-ஐ எழுத விரும்பினால், இதைச் செய்வதற்கான மிகவும் வசதியான வழி [JavaScript API நிரலகத்தைப்](/developers/docs/apis/javascript/) பயன்படுத்துவதாகும். இந்த API-கள் டெவலப்பர்களை [எத்திரியம் பிணையத்தில் உள்ள கணுக்களுடன்](/developers/docs/nodes-and-clients/) எளிதாகத் தொடர்புகொள்ள அனுமதிக்கின்றன.

எத்திரியத்தில் உள்ள திறன் ஒப்பந்தங்களுடன் தொடர்புகொள்ள இந்த நிரலகங்களை நீங்கள் பயன்படுத்தலாம், எனவே முன்பே இருக்கும் ஒப்பந்தங்களுடன் தொடர்புகொள்ள JavaScript-ஐ மட்டுமே பயன்படுத்தும் ஒரு பரவலாக்கப்பட்ட செயலியை (dapp) உருவாக்க முடியும்.

**இவற்றைப் பாருங்கள்**

- [Web3.js](https://web3js.readthedocs.io)
- [Ethers.js](https://ethers.org) – _JavaScript மற்றும் TypeScript-இல் எத்திரியம் பணப்பை செயலாக்கம் மற்றும் பயன்பாடுகளை உள்ளடக்கியது._
- [viem](https://viem.sh) – _எத்திரியத்துடன் தொடர்புகொள்வதற்கான குறைந்த-நிலை நிலையற்ற அடிப்படைகளை வழங்கும் எத்திரியத்திற்கான ஒரு TypeScript இடைமுகம்._
- [Drift](https://ryangoree.github.io/drift/) – _Web3 நிரலகங்கள் முழுவதும் சிரமமற்ற எத்திரியம் மேம்பாட்டிற்காக உள்ளமைக்கப்பட்ட தேக்ககம், ஹூக்குகள் மற்றும் சோதனை மாதிரிகளைக் கொண்ட ஒரு TypeScript மெட்டா-நிரலகம்._

### திறன் ஒப்பந்தங்கள் {#smart-contracts}

நீங்கள் ஒரு JavaScript டெவலப்பராக இருந்து, உங்கள் சொந்த திறன் ஒப்பந்தத்தை எழுத விரும்பினால், நீங்கள் [Solidity](https://solidity.readthedocs.io)-ஐப் பற்றி நன்கு தெரிந்துகொள்ள விரும்பலாம். இது மிகவும் பிரபலமான திறன் ஒப்பந்த மொழியாகும், மேலும் இது தொடரியல் ரீதியாக JavaScript-ஐப் போலவே உள்ளது, இது கற்றுக்கொள்வதை எளிதாக்கலாம்.

[திறன் ஒப்பந்தங்கள்](/developers/docs/smart-contracts/) பற்றி மேலும் அறிக.

## நெறிமுறையைப் புரிந்துகொள்ளுங்கள் {#understand-the-protocol}

### எத்திரியம் மெய்நிகர் இயந்திரம் {#the-ethereum-virtual-machine}

[எத்திரியத்தின் மெய்நிகர் இயந்திரத்தின்](/developers/docs/evm/) JavaScript செயலாக்கம் ஒன்று உள்ளது. இது சமீபத்திய கவை விதிகளை ஆதரிக்கிறது. கவை விதிகள் என்பது திட்டமிடப்பட்ட மேம்படுத்தல்களின் விளைவாக EVM-இல் செய்யப்பட்ட மாற்றங்களைக் குறிக்கின்றன.

இது பல்வேறு JavaScript தொகுப்புகளாகப் பிரிக்கப்பட்டுள்ளது, இவற்றை நீங்கள் நன்கு புரிந்துகொள்ளப் பார்க்கலாம்:

- கணக்குகள்
- தொகுதிகள்
- தொகுதிச்சங்கிலி
- பரிவர்த்தனைகள்
- மற்றும் பல...

"ஒரு கணக்கின் தரவு கட்டமைப்பு என்ன?" போன்ற விஷயங்களைப் புரிந்துகொள்ள இது உங்களுக்கு உதவும்.

நீங்கள் குறியீட்டைப் படிக்க விரும்பினால், எங்கள் ஆவணங்களைப் படிப்பதற்கு இந்த JavaScript ஒரு சிறந்த மாற்றாக இருக்கும்.

**EVM-ஐப் பாருங்கள்**  
[`@ethereumjs/evm`](https://github.com/ethereumjs/ethereumjs-monorepo/tree/master/packages/evm)

### கணுக்கள் மற்றும் கிளையண்டுகள் {#nodes-and-clients}

உங்களுக்குப் புரிந்த மொழியான JavaScript-இல் எத்திரியம் கிளையண்டுகள் எவ்வாறு செயல்படுகின்றன என்பதை ஆராய அனுமதிக்கும் ஒரு EthereumJS கிளையண்ட் செயலில் உள்ள மேம்பாட்டில் உள்ளது!

## பிற திட்டங்கள் {#other-projects}

எத்திரியம் JavaScript உலகில் இன்னும் பல விஷயங்கள் நடக்கின்றன, அவற்றுள் சில:

- பணப்பைப் பயன்பாடுகளின் நிரலகங்கள்.
- எத்திரியம் திறவுகோல்களை உருவாக்க, இறக்குமதி செய்ய மற்றும் ஏற்றுமதி செய்வதற்கான கருவிகள்.
- `merkle-patricia-tree`-இன் செயலாக்கம் – இது எத்திரியம் மஞ்சள் அறிக்கையில் கோடிட்டுக் காட்டப்பட்டுள்ள ஒரு தரவு கட்டமைப்பு.

[EthereumJS களஞ்சியத்தில்](https://github.com/ethereumjs) உங்களுக்கு மிகவும் விருப்பமானவற்றை ஆராயுங்கள்
