எத்தேரியம் சூழியலமைப்பில் மிகவும் பிரபலமான மொழிகளில் ஜாவாஸ்கிரிப்ட்டும் ஒன்றாகும். உண்மையில், எத்தேரியத்தை முடிந்தவரை ஜாவாஸ்கிரிப்டிற்குக் கொண்டு வருவதற்கென அர்ப்பணிக்கப்பட்ட ஒரு [குழு](https://github.com/ethereumjs) உள்ளது.

ஸ்டேக்கின் [அனைத்து மட்டங்களிலும்](/developers/docs/ethereum-stack/) ஜாவாஸ்கிரிப்ட் (அல்லது அதற்கு நெருக்கமான ஒன்றை) எழுத வாய்ப்புகள் உள்ளன.

## எத்தேரியத்துடன் ஊடாடுதல் {#interact-with-ethereum}

### ஜாவாஸ்கிரிப்ட் API நூலகங்கள் {#javascript-api-libraries}

பிளாக்செயினை வினவவும், பரிவர்த்தனைகளை அனுப்பவும் மற்றும் பலவற்றிற்கும் நீங்கள் ஜாவாஸ்கிரிப்ட் எழுத விரும்பினால், இதைச் செய்வதற்கான மிகவும் வசதியான வழி [ஜாவாஸ்கிரிப்ட் API நூலகத்தைப்](/developers/docs/apis/javascript/) பயன்படுத்துவதாகும். இந்த APIகள் டெவலப்பர்களை [எத்தேரியம் நெட்வொர்க்கில் உள்ள முனைகளுடன்](/developers/docs/nodes-and-clients/) எளிதாக ஊடாட அனுமதிக்கின்றன.

எத்தேரியத்தில் ஸ்மார்ட் ஒப்பந்தங்களுடன் ஊடாட இந்த நூலகங்களைப் பயன்படுத்தலாம், எனவே முன்பே இருக்கும் ஒப்பந்தங்களுடன் ஊடாடுவதற்கு ஜாவாஸ்கிரிப்ட்டை மட்டும் பயன்படுத்தும் ஒரு dapp-ஐ உருவாக்க முடியும்.

**பாருங்கள்**

- [Web3.js](https://web3js.readthedocs.io)
- [Ethers.js](https://ethers.org) – _ஜாவாஸ்கிரிப்ட் மற்றும் டைப்ஸ்கிரிப்டில் எத்தேரியம் பணப்பைச் செயலாக்கம் மற்றும் பயன்பாடுகளை உள்ளடக்கியது._
- [viem](https://viem.sh) – _எத்தேரியத்துடன் ஊடாடுவதற்கு குறைந்த-நிலை நிலை இல்லாத அடிப்படைக் கூறுகளை வழங்கும் எத்தேரியத்திற்கான ஒரு டைப்ஸ்கிரிப்ட் இடைமுகம்._
- [Drift](https://ryangoree.github.io/drift/) – _web3 நூலகங்கள் முழுவதும் சிரமமற்ற எத்தேரியம் உருவாக்கத்திற்கான உள்ளமைக்கப்பட்ட தற்காலிகச் சேமிப்பு, ஹூக்குகள் மற்றும் சோதனை மாக்குகளுடன் கூடிய ஒரு டைப்ஸ்கிரிப்ட் மெட்டா-நூலகம்._

### ஸ்மார்ட் ஒப்பந்தங்கள் {#smart-contracts}

நீங்கள் ஒரு ஜாவாஸ்கிரிப்ட் டெவலப்பராக இருந்து, உங்கள் சொந்த ஸ்மார்ட் ஒப்பந்தத்தை எழுத விரும்பினால், நீங்கள் [Solidity](https://solidity.readthedocs.io) பற்றி அறிந்துகொள்ள விரும்பலாம். இது மிகவும் பிரபலமான ஸ்மார்ட் ஒப்பந்த மொழியாகும், மேலும் இது தொடரியல் ரீதியாக ஜாவாஸ்கிரிப்டைப் போன்றது, இது கற்றுக்கொள்வதை எளிதாக்கலாம்.

[ஸ்மார்ட் ஒப்பந்தங்கள்](/developers/docs/smart-contracts/) பற்றிய கூடுதல் தகவல்.

## நெறிமுறையைப் புரிந்து கொள்ளுங்கள் {#understand-the-protocol}

### எத்தேரியம் மெய்நிகர் இயந்திரம் {#the-ethereum-virtual-machine}

[எத்தேரியத்தின் மெய்நிகர் இயந்திரத்தின்](/developers/docs/evm/) ஒரு ஜாவாஸ்கிரிப்ட் செயலாக்கம் உள்ளது. இது சமீபத்திய கிளைப்பிரிப்பு விதிகளை ஆதரிக்கிறது. கிளைப்பிரிப்பு விதிகள் என்பது திட்டமிடப்பட்ட மேம்படுத்தல்களின் விளைவாக EVM-இல் செய்யப்படும் மாற்றங்களைக் குறிக்கின்றன.

இதை நன்கு புரிந்துகொள்ள, நீங்கள் சரிபார்க்கக்கூடிய பல்வேறு ஜாவாஸ்கிரிப்ட் தொகுப்புகளாக இது பிரிக்கப்பட்டுள்ளது:

- கணக்குகள்
- தொகுதிகள்
- பிளாக்செயின் அதுவே
- பரிவர்த்தனைகள்
- மேலும் பல...

"ஒரு கணக்கின் தரவு அமைப்பு என்ன?" போன்ற விஷயங்களைப் புரிந்துகொள்ள இது உங்களுக்கு உதவும்.

நீங்கள் குறியீட்டைப் படிக்க விரும்பினால், எங்கள் ஆவணங்களைப் படிப்பதற்கு இந்த ஜாவாஸ்கிரிப்ட் ஒரு சிறந்த மாற்றாக இருக்கும்.

**EVM-ஐப் பாருங்கள்**\n[`@ethereumjs/evm`](https://github.com/ethereumjs/ethereumjs-monorepo/tree/master/packages/evm)

### முனைகள் மற்றும் கிளையன்ட்கள் {#nodes-and-clients}

ஒரு Ethereumjs கிளையன்ட் செயலில் வளர்ச்சியில் உள்ளது, இது உங்களுக்குப் புரியும் மொழியான ஜாவாஸ்கிரிப்டில் எத்தேரியம் கிளையன்ட்கள் எவ்வாறு செயல்படுகின்றன என்பதை ஆராய உதவுகிறது!

**கிளையன்ட்டைப் பாருங்கள்**\n[`@ethereumjs/client`](https://github.com/ethereumjs/ethereumjs-monorepo/tree/master/packages/client)

## பிற திட்டங்கள் {#other-projects}

எத்தேரியம் ஜாவாஸ்கிரிப்ட் உலகில் இன்னும் பல விஷயங்கள் நடந்து கொண்டிருக்கின்றன, அவற்றுள் அடங்குபவை:

- பணப்பை பயன்பாடுகளின் நூலகங்கள்.
- எத்தேரியம் சாவிகளை உருவாக்க, இறக்குமதி செய்ய மற்றும் ஏற்றுமதி செய்வதற்கான கருவிகள்.
- `merkle-patricia-tree`-இன் ஒரு செயலாக்கம் – எத்தேரியம் மஞ்சள் தாளில் கோடிட்டுக் காட்டப்பட்டுள்ள ஒரு தரவு அமைப்பு.

[EthereumJS ரெப்போவில்](https://github.com/ethereumjs) உங்களுக்கு மிகவும் விருப்பமானதை ஆராயுங்கள்
