<FeaturedText>எலிக்சர் அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி எத்தேரியத்திற்கு எவ்வாறு உருவாக்குவது என்பதை அறியுங்கள்.</FeaturedText>

கிரிப்டோகரன்சி மற்றும் பிளாக்செயின் தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட பயன்பாடுகளை (அல்லது "டாப்ஸ்") உருவாக்க Ethereum ஐப் பயன்படுத்தவும். இந்த டாப்கள் (dapps) நம்பிக்கை தேவைப்படாததாக இருக்கலாம், அதாவது அவை எத்தேரியத்தில் பயன்படுத்தப்பட்டவுடன், அவை எப்போதும் திட்டமிட்டபடி இயங்கும். புதிய வகையான நிதி பயன்பாடுகளை உருவாக்க அவர்கள் டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்படலாம், அதாவது எந்த ஒரு நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்துவதில்லை மற்றும் தணிக்கை செய்ய இயலாது.

## எலிக்ஸர் திட்டங்கள் மற்றும் கருவிகள் {#elixir-projects-and-tools}

### செயலில் உள்ளவை {#active}

- [block_keys](https://github.com/ExWeb3/block_keys) - _எலிக்ஸரில் BIP32 & BIP44 செயல்படுத்தல் (தீர்மானிக்கப்பட்ட வாலெட்டுகளுக்கான பல-கணக்கு படிநிலை)_
- [ethereumex](https://github.com/mana-ethereum/ethereumex) - _எத்தேரியம் பிளாக்செயினுக்கான எலிக்ஸர் JSON-RPC கிளையன்ட்_
- [ethers](https://github.com/ExWeb3/elixir_ethers) - _எலிக்ஸரைப் பயன்படுத்தி எத்தேரியத்தில் உள்ள ஸ்மார்ட் ஒப்பந்தங்களுடன் ஊடாடுவதற்கான ஒரு விரிவான Web3 லைப்ரரி_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) - _Ethers-க்கான ஒரு KMS கையொப்பமிடும் லைப்ரரி (AWS KMS உடன் பரிவர்த்தனைகளில் கையொப்பமிடுதல்)_
- [ex_abi](https://github.com/poanetwork/ex_abi) - _எலிக்ஸரில் எத்தேரியம் ABI பாகுபடுத்தி/குறிவிலக்கி/குறிமுறைப்படுத்தி செயல்படுத்தல்_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) - _ஒரு NIF உருவாக்கப்பட்ட டைனி-கெக்காக் ரஸ்ட் கிரேட்டைப் பயன்படுத்தி கெக்காக் SHA3-256 ஹாஷ்களைக் கணக்கிடுவதற்கான எலிக்ஸர் லைப்ரரி_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) - _எத்தேரியத்தின் RLP (மீண்டும் வரும் நீள முன்னொட்டு) குறியாக்கத்தின் எலிக்ஸர் செயல்படுத்தல்_

### காப்பகப்படுத்தப்பட்டவை / இனி பராமரிக்கப்படாதவை {#archived--no-longer-maintained}
