<FeaturedText>ரூபி அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி எத்தேரியத்திற்கு எவ்வாறு உருவாக்குவது என்பதை அறிக.</FeaturedText>

கிரிப்டோகரன்சி மற்றும் பிளாக்செயின் தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட பயன்பாடுகளை (அல்லது "டாப்ஸ்") உருவாக்க Ethereum ஐப் பயன்படுத்தவும். இந்த டாப்கள் (dapps) நம்பிக்கை தேவைப்படாததாக இருக்கலாம், அதாவது அவை எத்தேரியத்தில் பயன்படுத்தப்பட்டவுடன், அவை எப்போதும் திட்டமிட்டபடி இயங்கும். புதிய வகையான நிதி பயன்பாடுகளை உருவாக்க அவர்கள் டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்படலாம், அதாவது எந்த ஒரு நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்துவதில்லை மற்றும் தணிக்கை செய்ய இயலாது.

## ரூபி திட்டங்கள் மற்றும் கருவிகள் {#ruby-projects-and-tools}

### செயலில் உள்ளவை {#active}

- [eth.rb](https://github.com/q9f/eth.rb) - _எத்தேரியம் கணக்குகள், செய்திகள் மற்றும் பரிவர்த்தனைகளைக் கையாள ரூபி நூலகம் மற்றும் RPC-கிளையன்ட்_
- [keccak.rb](https://github.com/q9f/keccak.rb) - _எத்தேரியம் பயன்படுத்தும் கெக்காக் (SHA3) ஹாஷ்_
- [siwe-ruby](https://github.com/signinwithethereum/siwe-ruby) - _எத்தேரியம் மூலம் உள்நுழைவதற்கான ரூபி செயலாக்கம்_
- [siwe-rails](https://github.com/signinwithethereum/siwe-rails) - _SIWE லோக்கல் சைன் இன் ரூட்களைச் சேர்க்கும் ஒரு ரயில்ஸ் ஜெம்_
- [siwe-rails-examples](https://github.com/signinwithethereum/siwe-rails-examples) - _தனிப்பயன் கன்ட்ரோலருடன் ரூபி ஆன் ரயில்ஸைப் பயன்படுத்தும் SIWE எடுத்துக்காட்டு_
- [omniauth-siwe](https://github.com/signinwithethereum/omniauth-siwe) - _எத்தேரியம் மூலம் உள்நுழைவதற்கான (SIWE) ஆம்னிஆத் உத்தி_
- [omniauth-nft](https://github.com/valthon/omniauth-nft) - _NFT உரிமை வழியாக அங்கீகரிப்பதற்கான ஆம்னிஆத் உத்தி_
- [ethereum-on-rails](https://github.com/q9f/ethereum-on-rails) - _MetaMask-ஐ ரூபி ஆன் ரயில்ஸுடன் இணைக்க அனுமதிக்கும் எத்தேரியம் ஆன் ரயில்ஸ் டெம்ப்ளேட்_

### காப்பகப்படுத்தப்பட்டவை / இனி பராமரிக்கப்படாதவை {#archived--no-longer-maintained}

- [web3-eth](https://github.com/spikewilliams/vtada-ethereum) - _ரூபியுடன் எத்தேரியம் முனையின் RPC முறைகளை அழைத்தல்_
- [ethereum_tree](https://github.com/longhoangwkm/ethereum_tree) - _BIP32 தரநிலையின்படி ஒரு படிநிலை நிர்ணய பணப்பையிலிருந்து ETH முகவரிகளை உருவாக்க உதவும் ரூபி நூலகம்_
- [etherlite](https://github.com/budacom/etherlite) - _ரூபி ஆன் ரயில்ஸிற்கான எத்தேரியம் ஒருங்கிணைப்பு_
- [ethereum.rb](https://github.com/EthWorks/ethereum.rb) - _பரிவர்த்தனைகளை அனுப்புதல், ஒப்பந்தங்களை உருவாக்குதல் மற்றும் ஊடாடுதல் மற்றும் எத்தேரியம் முனையுடன் வேலை செய்வதற்கான பயனுள்ள கருவித்தொகுப்பு ஆகியவற்றிற்காக JSON-RPC இடைமுகத்தைப் பயன்படுத்தும் ரூபி எத்தேரியம் கிளையன்ட்_
- [omniauth-ethereum.rb](https://github.com/q9f/omniauth-ethereum.rb) - _OmniAuth-க்கான எத்தேரியம் வழங்குநர் உத்தியைச் செயல்படுத்துகிறது_
