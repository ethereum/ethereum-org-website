<FeaturedText>ரூபி அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி Ethereum-க்கு எவ்வாறு உருவாக்குவது என்பதை அறிக.</FeaturedText>

கிரிப்டோகரன்சி மற்றும் பிளாக்செயின் தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட பயன்பாடுகளை (அல்லது "dapps") உருவாக்க Ethereum-ஐப் பயன்படுத்தவும். இந்த dapps நம்பகத்தன்மையற்றதாக இருக்கலாம், அதாவது அவை Ethereum-ல் பயன்படுத்தப்பட்டவுடன், அவை எப்போதும் நிரல்படுத்தப்பட்டபடியே இயங்கும். புதிய வகையான நிதிப் பயன்பாடுகளை உருவாக்க அவை டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்பட்டதாக இருக்கலாம், அதாவது எந்தவொரு தனி நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்த முடியாது மற்றும் தணிக்கை செய்வது கிட்டத்தட்ட சாத்தியமற்றது.

## ரூபி திட்டங்கள் மற்றும் கருவிகள் {#ruby-projects-and-tools}

### செயலில் உள்ளவை {#active}

- [eth.rb](https://github.com/q9f/eth.rb) - _Ethereum கணக்குகள், செய்திகள் மற்றும் பரிவர்த்தனைகளைக் கையாள ரூபி நூலகம் மற்றும் RPC-கிளையண்ட்_
- [keccak.rb](https://github.com/q9f/keccak.rb) - _Ethereum பயன்படுத்தும் Keccak (SHA3) ஹாஷ்_
- [siwe-ruby](https://github.com/signinwithethereum/siwe-ruby) - _Ethereum உடன் உள்நுழைவதற்கான ரூபி செயலாக்கம்_
- [siwe-rails](https://github.com/signinwithethereum/siwe-rails) - _SIWE உள்ளூர் உள்நுழைவு வழிகளைச் சேர்க்கும் Rails ஜெம்_
- [siwe-rails-examples](https://github.com/signinwithethereum/siwe-rails-examples) - _தனிப்பயன் கன்ட்ரோலருடன் Ruby on Rails-ஐப் பயன்படுத்தும் SIWE உதாரணம்_
- [omniauth-siwe](https://github.com/signinwithethereum/omniauth-siwe) - _Ethereum உடன் உள்நுழைவதற்கான (SIWE) OmniAuth உத்தி_
- [omniauth-nft](https://github.com/valthon/omniauth-nft) - _NFT உரிமை மூலம் அங்கீகரிப்பதற்கான OmniAuth உத்தி_
- [ethereum-on-rails](https://github.com/q9f/ethereum-on-rails) - _MetaMask-ஐ Ruby on Rails உடன் இணைக்க அனுமதிக்கும் Ethereum on Rails டெம்ப்ளேட்_

### காப்பகப்படுத்தப்பட்டவை / இனி பராமரிக்கப்படாதவை {#archived--no-longer-maintained}

- [ethereum_tree](https://github.com/longhoangwkm/ethereum_tree) - _BIP32 தரநிலையின்படி படிநிலை தீர்மானிக்கும் வாலட்டிலிருந்து ETH முகவரிகளை உருவாக்க ரூபி நூலகம்_
- [etherlite](https://github.com/budacom/etherlite) - _Ruby on Rails-க்கான Ethereum ஒருங்கிணைப்பு_
- [ethereum.rb](https://github.com/EthWorks/ethereum.rb) - _பரிவர்த்தனைகளை அனுப்ப, ஒப்பந்தங்களை உருவாக்க மற்றும் தொடர்புகொள்ள JSON-RPC இடைமுகத்தைப் பயன்படுத்தும் ரூபி Ethereum கிளையண்ட் மற்றும் Ethereum நோடுடன் வேலை செய்வதற்கான பயனுள்ள கருவித்தொகுப்பு_
- [omniauth-ethereum.rb](https://github.com/q9f/omniauth-ethereum.rb) - _OmniAuth-க்கான Ethereum வழங்குநர் உத்தியைச் செயல்படுத்துகிறது_
