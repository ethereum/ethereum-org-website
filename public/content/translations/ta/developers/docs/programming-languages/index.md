எத்திரியத்தில் உருவாக்க டெவலப்பர்கள் [ஸ்மார்ட் ஒப்பந்தங்களை](/developers/docs/smart-contracts/) எழுத வேண்டும் என்பது ஒரு பொதுவான தவறான கருத்தாகும். இது தவறானது.
எத்திரியம் பிணையம் மற்றும் சமூகத்தின் சிறப்பம்சங்களில் ஒன்று, நீங்கள் எந்தவொரு நிரலாக்க மொழியிலும் [பங்கேற்க](/community/) முடியும் என்பதாகும்.

எத்திரியம் மற்றும் அதன் சமூகம் திறந்த மூலத்தை (open source) வரவேற்கின்றன. கிளையண்ட் செயலாக்கங்கள், APIகள், மேம்பாட்டு கட்டமைப்புகள், சோதனை கருவிகள் போன்ற சமூகத் திட்டங்களை நீங்கள் பல்வேறு மொழிகளில் காணலாம்.

### எனது மொழி ஆதரிக்கப்படவில்லை என்றால் என்ன செய்வது {#other-lang}

கூடுதல் நிரலாக்க மொழிக்கான வளங்களை இணைக்க அல்லது மெய்நிகர் சமூகத்தைச் சுட்டிக்காட்ட நீங்கள் விரும்பினால், [ஒரு சிக்கலை உருவாக்குவதன்](https://github.com/ethereum/ethereum-org-website/issues/new/choose) மூலம் புதிய பக்கத்தைக் கோரலாம்.

தற்போது ஆதரிக்கப்படாத மொழியைப் பயன்படுத்தி தொகுதிச்சங்கிலியுடன் இடைமுகமாகச் செயல்பட நீங்கள் குறியீட்டை எழுத விரும்பினால், எத்திரியம் பிணையத்துடன் இணைக்க [ஜேசன்-ஆர்பிசி இடைமுகத்தைப்](/developers/docs/apis/json-rpc/) பயன்படுத்தலாம். TCP/IP ஐப் பயன்படுத்தக்கூடிய எந்தவொரு நிரலாக்க மொழியும் இந்த இடைமுகத்தைப் பயன்படுத்தலாம்.
