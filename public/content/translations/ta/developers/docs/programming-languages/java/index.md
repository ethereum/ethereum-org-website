<FeaturedText>Java அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி எத்திரியத்திற்கு எவ்வாறு உருவாக்குவது என்பதை அறிக</FeaturedText>

மறைக்குறியீட்டு நாணயம் மற்றும் தொகுதிச்சங்கிலி தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட செயலிகளை (அல்லது "dapps") உருவாக்க எத்திரியத்தைப் பயன்படுத்தவும். இந்த dapp-கள் நம்பகமானவையாக இருக்கலாம், அதாவது அவை எத்திரியத்தில் நிலைநிறுத்தப்பட்டவுடன், அவை எப்போதும் நிரல்படுத்தப்பட்டபடியே இயங்கும். புதிய வகையான நிதிச் செயலிகளை உருவாக்க அவை டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்பட்டவையாக இருக்கலாம், அதாவது எந்தவொரு தனி நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்த முடியாது மற்றும் தணிக்கை செய்வது கிட்டத்தட்ட சாத்தியமற்றது.
