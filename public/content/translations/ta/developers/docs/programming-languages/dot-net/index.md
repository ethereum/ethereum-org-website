<FeaturedText>.NET-அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி எத்தேரியத்திற்கு எவ்வாறு உருவாக்குவது என்பதை அறிக</FeaturedText>

கிரிப்டோகரன்சி மற்றும் பிளாக்செயின் தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட பயன்பாடுகளை (அல்லது "டாப்ஸ்") உருவாக்க Ethereum ஐப் பயன்படுத்தவும். இந்த டாப்ஸ் நம்பகமானதாக இருக்கக்கூடும், அதாவது அவை எத்தேரியத்திற்கு அனுப்பப்பட்டவுடன், அவை எப்போதும் திட்டமிடப்பட்டபடி இயங்கும். புதிய வகையான நிதி பயன்பாடுகளை உருவாக்க அவர்கள் டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்படலாம், அதாவது எந்த ஒரு நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்துவதில்லை மற்றும் தணிக்கை செய்ய இயலாது.

மைக்ரோசாஃப்ட் தொழில்நுட்ப அடுக்கிலிருந்து கருவிகள் மற்றும் மொழிகளைப் பயன்படுத்தி ஸ்மார்ட் ஒப்பந்தங்களுடன் தொடர்பு கொள்ளுங்கள் - நெட் ஃபிரேம்வொர்க் / .நெட் கோர் முழுவதும் வி.எஸ்.கோட் மற்றும் விஷுவல் ஸ்டுடியோ போன்ற கருவிகளில் சி #, # விஷுவல் பேசிக் .நெட், எஃப் # ஐ ஆதரிக்கிறது. / நெட் தரநிலை. மைக்ரோசாஃப்ட் அஸூர் பிளாக்செயினைப் பயன்படுத்தி அஸூரில் ஒரு எத்தேரியம் பிளாக்செயினை நிமிடங்களில் பயன்படுத்தவும். .NET-இன் மீதான அன்பை எத்தேரியத்திற்கு கொண்டு வாருங்கள்!

## .NET திட்டங்கள், கருவிகள் மற்றும் பிற வேடிக்கையான விஷயங்கள் {#dot-net-projects-tools-and-other-fun-stuff}

- [Nethereum Playground](http://playground.nethereum.com/) - _உலாவியில் Nethereum குறியீட்டுத் துணுக்குகளைத் தொகுக்கவும், உருவாக்கவும் மற்றும் இயக்கவும்_
- [Nethereum Codegen Blazor](https://github.com/Nethereum/Nethereum.CodeGen.Blazor) - _Blazor-இல் UI உடன் Nethereum codegen_
- [Nethereum Blazor](https://github.com/Nethereum/NethereumBlazor) - _ஒரு .NET Wasm SPA லைட் பிளாக்செயின் எக்ஸ்புளோரர் மற்றும் எளிய வாலெட்_
- [Wonka வணிக விதிகள் இயந்திரம்](https://docs.nethereum.com/en/latest/wonka/) - _ஒரு வணிக விதிகள் இயந்திரம் (.NET தளம் மற்றும் எத்தேரியம் தளம் ஆகிய இரண்டிற்கும்) உள்ளார்ந்த முறையில் மெட்டாடேட்டாவால் இயக்கப்படுகிறது_
- [Nethermind](https://github.com/NethermindEth/nethermind) - _Linux, Windows, MacOS-க்கான ஒரு .NET Core எத்தேரியம் கிளையன்ட்_
- [eth-utils](https://github.com/ethereum/eth-utils/) - _Ethereum தொடர்பான குறியீட்டுத் தளங்களுடன் வேலை செய்வதற்கான பயன்பாட்டுச் செயல்பாடுகள்_
- [TestChains](https://github.com/Nethereum/TestChains) - _விரைவான பதிலுக்காக முன்-கட்டமைக்கப்பட்ட .NET devchains (PoA)_

மேலும் ஆதாரங்களைத் தேடுகிறீர்களா? [ethereum.org/developers](/developers/) பக்கத்தைப் பாருங்கள்.

## .NET சமூகப் பங்களிப்பாளர்கள் {#dot-net-community-contributors}

Nethereum-இல், நாங்கள் பெரும்பாலும் [Gitter](https://gitter.im/Nethereum/Nethereum)-இல் நேரத்தைச் செலவிடுவோம். அங்கு கேள்விகளைக் கேட்கவும்/பதிலளிக்கவும், உதவி பெறவும் அல்லது சாதாரணமாக இருக்கவும் அனைவரும் வரவேற்கப்படுகிறார்கள். [Nethereum GitHub ரெபாசிட்டரியில்](https://github.com/Nethereum) ஒரு PR செய்யவும் அல்லது ஒரு சிக்கலைத் திறக்கவும், அல்லது எங்களிடம் உள்ள பல பக்க/மாதிரி திட்டங்களைப் பார்க்கவும். நீங்கள் எங்களை [Discord](https://discord.gg/jQPrR58FxX)-இலும் காணலாம்!

நீங்கள் Nethermind-க்கு புதியவர் மற்றும் தொடங்குவதற்கு உதவி தேவைப்பட்டால், எங்கள் [Discord](http://discord.gg/PaCMRFdvWT)-இல் சேருங்கள். உங்கள் கேள்விகளுக்குப் பதிலளிக்க எங்கள் டெவலப்பர்கள் தயாராக உள்ளனர். [Nethermind GitHub ரெபாசிட்டரியில்](https://github.com/NethermindEth/nethermind) ஒரு PR-ஐத் திறக்கவோ அல்லது ஏதேனும் சிக்கல்களை எழுப்பவோ தயங்க வேண்டாம்.
