<FeaturedText>ரஸ்ட் அடிப்படையிலான திட்டங்கள் மற்றும் கருவிகளைப் பயன்படுத்தி எத்தேரியத்திற்கு எவ்வாறு உருவாக்குவது என்பதை அறிக</FeaturedText>

கிரிப்டோகரன்சி மற்றும் பிளாக்செயின் தொழில்நுட்பத்தின் நன்மைகளைப் பயன்படுத்தும் பரவலாக்கப்பட்ட பயன்பாடுகளை (அல்லது "டாப்ஸ்") உருவாக்க Ethereum ஐப் பயன்படுத்தவும். இந்த டாப்ஸ் நம்பகமானதாக இருக்கக்கூடும், அதாவது அவை எத்தேரியத்திற்கு அனுப்பப்பட்டவுடன், அவை எப்போதும் திட்டமிடப்பட்டபடி இயங்கும். புதிய வகையான நிதி பயன்பாடுகளை உருவாக்க அவர்கள் டிஜிட்டல் சொத்துக்களைக் கட்டுப்படுத்தலாம். அவை பரவலாக்கப்படலாம், அதாவது எந்த ஒரு நிறுவனமோ அல்லது நபரோ அவற்றைக் கட்டுப்படுத்துவதில்லை மற்றும் தணிக்கை செய்ய இயலாது.

## ரஸ்ட் திட்டங்கள் மற்றும் கருவிகள் {#rust-projects-and-tools}

- [pwasm-ethereum](https://github.com/paritytech/pwasm-ethereum) - _எத்தேரியம் போன்ற நெட்வொர்க்குடன் தொடர்பு கொள்ளும் externsகளின் தொகுப்பு_
- [Lighthouse](https://github.com/sigp/lighthouse) - _வேகமான எத்தேரியம் ஒருமித்த கருத்து அடுக்கு கிளையன்ட்_
- [Ethereum WebAssembly](https://ewasm.readthedocs.io/en/mkdocs/) - _WebAssembly இன் ஒரு தீர்மானகரமான துணைத் தொகுப்பைப் பயன்படுத்தி எத்தேரியம் ஸ்மார்ட் ஒப்பந்த செயலாக்க அடுக்கின் முன்மொழியப்பட்ட மறுவடிவமைப்பு_
- [oasis_std](https://docs.rs/oasis-std/latest/oasis_std/index.html) - _OASIS API குறிப்பு_
- [Solaris](https://github.com/paritytech/sol-rs) - _நேட்டிவ் பேரிட்டி கிளையன்ட் EVM ஐப் பயன்படுத்தும் சாலிடிட்டி ஸ்மார்ட் ஒப்பந்தங்கள் யூனிட் டெஸ்ட் ஹார்னஸ்._
- [SputnikVM](https://github.com/rust-blockchain/evm) - _ரஸ்ட் எத்தேரியம் மெய்நிகர் இயந்திர செயலாக்கம்_
- [Wavelet](https://wavelet.perlin.net/docs/smart-contracts) - _ரஸ்டில் வேவ்லெட் ஸ்மார்ட் ஒப்பந்தம்_
- [Foundry](https://github.com/foundry-rs/foundry) - _எத்தேரியம் பயன்பாட்டு உருவாக்கத்திற்கான கருவித்தொகுப்பு_
- [Alloy](https://alloy.rs) - _எத்தேரியம் மற்றும் பிற EVM-அடிப்படையிலான சங்கிலிகளுடன் தொடர்பு கொள்வதற்கான உயர் செயல்திறன், நன்கு சோதிக்கப்பட்ட மற்றும் ஆவணப்படுத்தப்பட்ட நூலகங்கள்._
- [Ethers_rs](https://github.com/gakonst/ethers-rs) - _எத்தேரியம் நூலகம் மற்றும் பணப்பை செயலாக்கம்_
- [SewUp](https://github.com/second-state/SewUp) - _ரஸ்ட் கொண்டு உங்கள் எத்தேரியம் வெப்அசெம்பிளி ஒப்பந்தத்தை ஒரு பொதுவான பின்தளத்தில் உருவாக்குவது போலவே உருவாக்க உதவும் ஒரு நூலகம்_
- [Substreams](https://github.com/streamingfast/substreams) - _இணையான பிளாக்செயின் தரவு அட்டவணைப்படுத்தல் தொழில்நுட்பம்_
- [Reth](https://github.com/paradigmxyz/reth) Reth (ரஸ்ட் எத்தேரியத்தின் சுருக்கம்) என்பது ஒரு புதிய எத்தேரியம் முழு-நோட் செயலாக்கமாகும்
- [Awesome Ethereum Rust](https://github.com/Vid201/awesome-ethereum-rust) - _எத்தேரியம் சூழல் அமைப்பில் ரஸ்டில் எழுதப்பட்ட தேர்ந்தெடுக்கப்பட்ட திட்டங்களின் தொகுப்பு_
