இந்த ஆவணம் Ethereum உடன் உருவாக்க உதவ வடிவமைக்கப்பட்டுள்ளது. இது Ethereum ஐ ஒரு கருத்து (concept) என்ற வகையில் விளக்குகிறது, Ethereum தொழில்நுட்ப stack ஐ விளக்குகிறது, மேலும் கூடுதலான செயலிகள் மற்றும் பயன்பாட்டு நிலைகளுக்கான முன்னேற்றமான தலைப்புகளை ஆவணப்படுத்துகிறது.

இது ஒரு திறந்த மூல (open-source) சமூக முயற்சி ஆகும், எனவே புதிய தலைப்புகளை பரிந்துரைக்க, புதிய உள்ளடக்கத்தைச் சேர்க்க, மற்றும் உதவியாக இருக்கும் என்று நினைக்கப்படும் இடங்களில் எடுத்துக்காட்டுகளை வழங்கவும் தயங்க வேண்டாம்.  அனைத்து ஆவணங்களையும் GitHub வழியாகத் திருத்தலாம் – எப்படி என்று உங்களுக்குத் தெரியாவிட்டால், [இந்த வழிமுறைகளைப் பின்பற்றவும்](https://github.com/ethereum/ethereum-org-website/blob/dev/docs/editing-markdown.md).
