இந்த ஆவணங்கள் [Ethereum](/)-ஐப் பயன்படுத்தி நீங்கள் உருவாக்குவதற்கு உதவும் வகையில் வடிவமைக்கப்பட்டுள்ளன. இது Ethereum-ஐ ஒரு கருத்தாக உள்ளடக்கியுள்ளது, Ethereum தொழில்நுட்ப அடுக்கை (tech stack) விளக்குகிறது, மேலும் மிகவும் சிக்கலான பயன்பாடுகள் மற்றும் பயன்பாட்டு நிகழ்வுகளுக்கான மேம்பட்ட தலைப்புகளை ஆவணப்படுத்துகிறது.

இது ஒரு ஓப்பன் சோர்ஸ் சமூக முயற்சியாகும், எனவே புதிய தலைப்புகளைப் பரிந்துரைக்கவும், புதிய உள்ளடக்கத்தைச் சேர்க்கவும், மேலும் பயனுள்ளதாக இருக்கும் என நீங்கள் நினைக்கும் இடங்களில் எடுத்துக்காட்டுகளை வழங்கவும் தயங்க வேண்டாம். அனைத்து ஆவணங்களையும் GitHub மூலம் திருத்தலாம் – எப்படி என்று உங்களுக்குத் தெரியாவிட்டால், [இந்த வழிமுறைகளைப் பின்பற்றவும்](https://github.com/ethereum/ethereum-org-website/blob/dev/docs/editing-markdown.md).

## மேம்பாட்டுத் தொகுதிகள் {#development-modules}

Ethereum மேம்பாட்டில் இது உங்களின் முதல் முயற்சி என்றால், தொடக்கத்திலிருந்தே ஆரம்பித்து ஒரு புத்தகத்தைப் படிப்பது போலப் படிப்படியாகப் படித்துப் புரிந்துகொள்ளப் பரிந்துரைக்கிறோம்.
