## அறிமுகம் {#introduction}

பல [எத்திரியம்](/) மேம்பாட்டுத் தரநிலைகள் வில்லை இடைமுகங்களில் கவனம் செலுத்துகின்றன. இந்தத் தரநிலைகள் திறன் ஒப்பந்தங்கள் தொகுக்கக்கூடியதாக இருப்பதை உறுதிசெய்ய உதவுகின்றன, எனவே ஒரு புதிய திட்டம் ஒரு வில்லையை வெளியிடும்போது, அது ஏற்கனவே உள்ள பரவலாக்கப்பட்ட பரிமாற்றங்கள் மற்றும் பயன்பாடுகளுடன் இணக்கமாக இருக்கும்.

எத்திரியம் சுற்றுச்சூழல் அமைப்பு முழுவதும் வில்லைகள் எவ்வாறு செயல்படுகின்றன மற்றும் தொடர்பு கொள்கின்றன என்பதை வில்லை தரநிலைகள் வரையறுக்கின்றன. அவை டெவலப்பர்கள் ஏற்கனவே உள்ளதை மீண்டும் உருவாக்காமல் எளிதாக உருவாக்குவதை சாத்தியமாக்குகின்றன, மேலும் வாலெட்டுகள், பரிமாற்றங்கள் மற்றும் பரவலாக்கப்பட்ட நிதி (DeFi) தளங்களுடன் வில்லைகள் தடையின்றி செயல்படுவதை உறுதிசெய்கின்றன. கேமிங், ஆளுகை அல்லது பிற பயன்பாட்டு நிகழ்வுகளாக இருந்தாலும், இந்தத் தரநிலைகள் நிலைத்தன்மையை வழங்குகின்றன மற்றும் எத்திரியத்தை மேலும் ஒன்றோடொன்று இணைக்கப்பட்டதாக மாற்றுகின்றன.

## வில்லை தரநிலைகள் {#token-standards}

எத்திரியத்தில் உள்ள மிகவும் பிரபலமான சில வில்லை தரநிலைகள் இங்கே:

- [ERC-20](/developers/docs/standards/tokens/erc-20/) - வாக்கு வில்லைகள், பங்குவைத்தல் வில்லைகள் அல்லது மெய்நிகர் நாணயங்கள் போன்ற பூஞ்சைய (பரிமாற்றக்கூடிய) வில்லைகளுக்கான நிலையான இடைமுகம்.

### NFT தரநிலைகள் {#nft-standards}

- [ERC-721](/developers/docs/standards/tokens/erc-721/) - கலைப்படைப்பு அல்லது பாடலுக்கான பத்திரம் போன்ற பூஞ்சையற்ற வில்லைகளுக்கான நிலையான இடைமுகம்.
- [ERC-1155](/developers/docs/standards/tokens/erc-1155/) - ERC-1155 மிகவும் திறமையான வர்த்தகங்கள் மற்றும் பரிவர்த்தனைகளை ஒன்றிணைக்க அனுமதிக்கிறது – இதன் மூலம் செலவுகளைச் சேமிக்கிறது. இந்த வில்லை தரநிலையானது பயன்பாட்டு வில்லைகள் ($BNB அல்லது $BAT போன்றவை) மற்றும் CryptoPunks போன்ற பூஞ்சையற்ற வில்லைகள் இரண்டையும் உருவாக்க அனுமதிக்கிறது.

[ERC](https://eips.ethereum.org/erc) முன்மொழிவுகளின் முழுப் பட்டியல்.
