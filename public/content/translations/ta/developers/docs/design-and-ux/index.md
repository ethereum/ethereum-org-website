Ethereum-உடன் வடிவமைப்பதற்கு நீங்கள் புதியவரா? இது உங்களுக்கான சரியான இடம். web3 வடிவமைப்பு மற்றும் ஆராய்ச்சியின் அடிப்படைகளை உங்களுக்கு அறிமுகப்படுத்த Ethereum சமூகம் வளங்களை எழுதியுள்ளது. நீங்கள் நன்கு அறிந்த பிற பயன்பாட்டு வடிவமைப்புகளிலிருந்து வேறுபடக்கூடிய முக்கிய கருத்துகளைப் பற்றி நீங்கள் அறிந்து கொள்வீர்கள்.

முதலில் web3 பற்றிய இன்னும் அடிப்படையான புரிதல் தேவையா? [**கற்றல் மையத்தைப்**](/learn/) பார்க்கவும்.

## பயனர் ஆராய்ச்சியிலிருந்து தொடங்குங்கள் {#start-with-user-research}

பயனுள்ள வடிவமைப்பு என்பது பார்வைக்கு ஈர்க்கும் பயனர் இடைமுகங்களை உருவாக்குவதைத் தாண்டிச் செல்கிறது. இது பயனரின் தேவைகள், நோக்கங்கள் மற்றும் உந்துதல் காரணிகளைப் பற்றிய ஆழமான புரிதலைப் பெறுவதை உள்ளடக்கியது. எனவே, அனைத்து வடிவமைப்பாளர்களும் தங்கள் பணி திட்டமிட்டதாகவும் நோக்கத்துடனும் இருப்பதை உறுதிசெய்ய, [**இரட்டை வைர செயல்முறை (double diamond process)**](<https://en.wikipedia.org/wiki/Double_Diamond_(design_process_model)>) போன்ற வடிவமைப்பு செயல்முறையைப் பின்பற்றுமாறு நாங்கள் மிகவும் பரிந்துரைக்கிறோம்.

- [Web3-க்கு அதிக UX ஆராய்ச்சியாளர்கள் மற்றும் வடிவமைப்பாளர்கள் தேவை](https://blog.akasha.org/akasha-conversations-9-web3-needs-more-ux-researchers-and-designers) - தற்போதைய வடிவமைப்பு முதிர்ச்சியின் மேலோட்டப் பார்வை
- [web3-இல் UX ஆராய்ச்சிக்கான எளிய வழிகாட்டி](https://uxplanet.org/a-complete-guide-to-ux-research-for-web-3-0-products-d6bead20ebb1) - ஆராய்ச்சி செய்வது எப்படி என்பதற்கான எளிய வழிகாட்டி
- [Web3-இல் UX முடிவுகளை எவ்வாறு அணுகுவது](https://archive.devcon.org/archive/watch/6/data-empathy-how-to-approach-ux-decisions-in-web3/) - அளவு மற்றும் தரமான ஆராய்ச்சி மற்றும் இரண்டிற்கும் இடையிலான வேறுபாடுகள் பற்றிய சுருக்கமான கண்ணோட்டம் (காணொளி, 6 நிமிடம்)
- [web3-இல் ஒரு UX ஆராய்ச்சியாளராக இருப்பது](https://medium.com/@georgia.rakusen/what-its-like-being-a-user-researcher-in-web3-6a4bcc096849) - web3-இல் ஒரு UX ஆராய்ச்சியாளராக இருப்பது எப்படி இருக்கும் என்பது பற்றிய தனிப்பட்ட பார்வை

## web3-இல் ஆராய்ச்சி ஆய்வுகள் {#research-in-web3}

## web3-க்கான வடிவமைப்பு {#design-for-web3}

- [Web3 வடிவமைப்பு பிளேபுக்](https://learnweb3.design/) - வடிவமைப்பாளர்கள் மற்றும் நிறுவனர்களுக்கான Web3 UX கொள்கைகள், DeFi வடிவங்கள், ஆளுமை வடிவமைப்பு, வாலட் UX மற்றும் நெறிமுறை-நிலை சிந்தனை பற்றிய கட்டமைப்புகள் மற்றும் குறிப்புகளின் விரிவான தொகுப்பு
- [Web3 UX வடிவமைப்பு கையேடு](https://web3ux.design/) - Web3 பயன்பாடுகளை வடிவமைப்பதற்கான நடைமுறை வழிகாட்டி
- [Web3 வடிவமைப்பு கோட்பாடுகள்](https://medium.com/@lyricalpolymath/web3-design-principles-f21db2f240c1) - பிளாக்செயின் அடிப்படையிலான dapps-க்கான UX விதிகளின் கட்டமைப்பு
- [பிளாக்செயின் வடிவமைப்பு கோட்பாடுகள்](https://medium.com/design-ibm/blockchain-design-principles-599c5c067b6e) - IBM-இல் உள்ள பிளாக்செயின் வடிவமைப்பு குழு கற்றுக்கொண்ட பாடங்கள்
- [Neueux.com](https://neueux.com/apps) - பல்வேறு வடிகட்டுதல் விருப்பங்களைக் கொண்ட பயனர் ஓட்டங்களின் UI நூலகம்
- [Web3-இன் பயன்பாட்டினை நெருக்கடி: நீங்கள் தெரிந்து கொள்ள வேண்டியவை!](https://www.youtube.com/watch?v=oBSXT_6YDzg) - டெவலப்பர் மையப்படுத்தப்பட்ட திட்ட உருவாக்கத்தின் ஆபத்துகள் பற்றிய குழு விவாதம் (காணொளி, 34 நிமிடம்)
