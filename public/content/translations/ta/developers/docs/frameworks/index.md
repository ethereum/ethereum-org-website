## கட்டமைப்புகளுக்கான அறிமுகம் {#introduction-to-frameworks}

ஒரு முழுமையான dapp-ஐ உருவாக்குவதற்கு
பல்வேறு தொழில்நுட்பங்கள் தேவைப்படுகின்றன. மென்பொருள் கட்டமைப்புகள் தேவையான பல அம்சங்களை
உள்ளடக்கியுள்ளன அல்லது நீங்கள் விரும்பும் கருவிகளைத் தேர்ந்தெடுக்க
எளிதான செருகுநிரல் அமைப்புகளை வழங்குகின்றன.

கட்டமைப்புகள் பயன்படுத்த தயாராக இருக்கும் பல செயல்பாடுகளுடன்
வருகின்றன, உதாரணமாக:

- ஒரு உள்ளூர் பிளாக்செயின் நிகழ்வை உருவாக்குவதற்கான அம்சங்கள்.
- உங்கள் ஸ்மார்ட் ஒப்பந்தங்களைத் தொகுத்து சோதிப்பதற்கான பயன்பாடுகள்.
- ஒரே திட்டம்/களஞ்சியத்திற்குள் உங்கள் பயனர் எதிர்கொள்ளும்
  பயன்பாட்டை உருவாக்க கிளையன்ட் மேம்பாட்டு துணை நிரல்கள்.
- எத்தீரியம் நெட்வொர்க்குகளுடன் இணைப்பதற்கும் ஒப்பந்தங்களை
  வரிசைப்படுத்துவதற்கும் உள்ளமைவு, அது உள்ளூரில் இயங்கும் ஒரு நிகழ்வாக இருந்தாலும் சரி,
  அல்லது எத்தீரியத்தின் பொது நெட்வொர்க்குகளில் ஒன்றாக இருந்தாலும் சரி.
- பரவலாக்கப்பட்ட செயலி விநியோகம் - IPFS போன்ற சேமிப்பக
  விருப்பங்களுடன் ஒருங்கிணைப்புகள்.

## முன்னேற்றக் கட்டுரை {#prerequisites}

கட்டமைப்புகளில் மூழ்குவதற்கு முன், எங்கள் [dapps](/developers/docs/dapps/) மற்றும் [எத்தீரியம் ஸ்டேக்](/developers/docs/ethereum-stack/) பற்றிய அறிமுகத்தை நீங்கள் முதலில் படிக்குமாறு பரிந்துரைக்கிறோம்.
