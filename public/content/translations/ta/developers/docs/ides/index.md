ஒரு [ஒருங்கிணைந்த மேம்பாட்டுச் சூழலை (IDE)](https://wikipedia.org/wiki/Integrated_development_environment) அமைக்கும் போது, எத்தேரியத்தில் நிரலாக்கப் பயன்பாடுகள் வேறு எந்த மென்பொருள் திட்டத்தையும் நிரலாக்குவதைப் போன்றது. தேர்வு செய்ய பல விருப்பங்கள் உள்ளன, எனவே இறுதியில், உங்கள் விருப்பங்களுக்கு மிகவும் பொருத்தமான IDE அல்லது குறியீடு எடிட்டரைத் தேர்ந்தெடுக்கவும். பெரும்பாலும் உங்கள் எத்தேரியம் மேம்பாட்டிற்கான சிறந்த IDE தேர்வு என்பது பாரம்பரிய மென்பொருள் மேம்பாட்டிற்கு நீங்கள் ஏற்கனவே பயன்படுத்தும் IDE ஆகும்.

## வலை அடிப்படையிலான IDEகள் {#web-based-ides}

நீங்கள் ஒரு [உள்ளூர் மேம்பாட்டு சூழலை அமைப்பதற்கு](/developers/local-environment/) முன் குறியீட்டுடன் பரிசோதனை செய்ய விரும்பினால், இந்த வலைப் பயன்பாடுகள் எத்தேரியம் ஸ்மார்ட் ஒப்பந்த மேம்பாட்டிற்காக பிரத்தியேகமாக உருவாக்கப்பட்டவை.

**[ரீமிக்ஸ்](https://remix.ethereum.org/)** - **_உள்ளமைக்கப்பட்ட நிலையான பகுப்பாய்வு மற்றும் ஒரு சோதனை பிளாக்செயின் மெய்நிகர் இயந்திரத்துடன் கூடிய வலை அடிப்படையிலான IDE_**

- [ஆவணங்கள்](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[செயின்IDE](https://chainide.com/)** - **_ஒரு கிளவுட் அடிப்படையிலான மல்டி-செயின் IDE_**

- [ஆவணங்கள்](https://chainide.gitbook.io/chainide-english-1/)
- [உதவி மன்றம்](https://forum.chainide.com/)

**[Replit (Solidity Starter - Beta)](https://replit.com/@replit/Solidity-starter-beta)** - **_ஹாட் ரீலோடிங், பிழை சரிபார்ப்பு மற்றும் முதல்-தர டெஸ்ட்நெட் ஆதரவுடன் எத்தேரியத்திற்கான தனிப்பயனாக்கக்கூடிய மேம்பாட்டுச் சூழல்_**

- [ஆவணங்கள்](https://docs.replit.com/)

**[டெண்டர்லி சாண்ட்பாக்ஸ்](https://sandbox.tenderly.co/)** - **_சொலிடிட்டி மற்றும் ஜாவாஸ்கிரிப்ட்டைப் பயன்படுத்தி உலாவியில் ஸ்மார்ட் ஒப்பந்தங்களை எழுத, செயல்படுத்த மற்றும் பிழைத்திருத்தம் செய்யக்கூடிய ஒரு வேகமான முன்மாதிரி சூழல்_**

**[EthFiddle](https://ethfiddle.com/)** - **_உங்கள் ஸ்மார்ட் ஒப்பந்தத்தை எழுத, தொகுக்க மற்றும் பிழைத்திருத்தம் செய்ய அனுமதிக்கும் வலை அடிப்படையிலான IDE_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## டெஸ்க்டாப் IDEகள் {#desktop-ides}

பெரும்பாலான நிறுவப்பட்ட IDEகள் எத்தேரியம் மேம்பாட்டு அனுபவத்தை மேம்படுத்துவதற்காகச் செருகுநிரல்களை உருவாக்கியுள்ளன. குறைந்தபட்சம், அவை [ஸ்மார்ட் ஒப்பந்த மொழிகளுக்கான](/developers/docs/smart-contracts/languages/) தொடரியல் சிறப்பம்சத்தை வழங்குகின்றன.

**விஷுவல் ஸ்டுடியோ குறியீடு -** **_அதிகாரப்பூர்வ எத்தேரியம் ஆதரவுடன் தொழில்முறை கிராஸ்-பிளாட்ஃபார்ம் IDE_**

- [விஷுவல் ஸ்டுடியோ குறியீடு](https://code.visualstudio.com/)
- [குறியீடு மாதிரிகள்](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**JetBrains IDEகள் (IntelliJ IDEA, போன்றவை) -** **_மென்பொருள் உருவாக்குநர்கள் (டெவலப்பர்கள்) மற்றும் குழுக்களுக்கான அத்தியாவசியக் கருவிகள்_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ சொலிடிட்டி](https://github.com/intellij-solidity/intellij-solidity/)

**ரீமிக்ஸ் டெஸ்க்டாப் -** **_உங்கள் உள்ளூர் கணினியில் ரீமிக்ஸ் IDE ஐ அனுபவியுங்கள்_**

- [பதிவிறக்கம்](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)
