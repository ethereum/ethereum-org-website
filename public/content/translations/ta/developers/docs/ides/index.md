[ஒருங்கிணைந்த மேம்பாட்டுச் சூழலை (IDE)](https://wikipedia.org/wiki/Integrated_development_environment) அமைக்கும் போது, Ethereum-இல் பயன்பாடுகளை நிரலாக்கம் செய்வது வேறு எந்த மென்பொருள் திட்டத்தையும் நிரலாக்கம் செய்வதைப் போன்றதே. தேர்வு செய்ய பல விருப்பங்கள் உள்ளன, எனவே இறுதியில், உங்கள் விருப்பங்களுக்கு மிகவும் பொருத்தமான IDE அல்லது குறியீட்டு திருத்தியைத் (code editor) தேர்ந்தெடுக்கவும். பெரும்பாலும் உங்கள் Ethereum மேம்பாட்டிற்கான சிறந்த IDE தேர்வு, பாரம்பரிய மென்பொருள் மேம்பாட்டிற்கு நீங்கள் ஏற்கனவே பயன்படுத்தும் IDE ஆகத்தான் இருக்கும்.

## இணைய அடிப்படையிலான IDEகள் {#web-based-ides}

நீங்கள் [உள்ளூர் மேம்பாட்டுச் சூழலை அமைப்பதற்கு](/developers/local-environment/) முன் குறியீட்டைப் பரிசோதிக்க விரும்பினால், இந்த இணையப் பயன்பாடுகள் Ethereum ஸ்மார்ட் ஒப்பந்த மேம்பாட்டிற்காகவே தனிப்பயனாக்கப்பட்டவை.

**[Remix](https://remix.ethereum.org/)** - **_உள்ளமைக்கப்பட்ட நிலையான பகுப்பாய்வு மற்றும் சோதனை பிளாக்செயின் மெய்நிகர் இயந்திரத்துடன் கூடிய இணைய அடிப்படையிலான IDE_**

- [ஆவணங்கள்](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[ChainIDE](https://chainide.com/)** - **_கிளவுட் அடிப்படையிலான மல்டி-செயின் IDE_**

- [ஆவணங்கள்](https://chainide.gitbook.io/chainide-english-1/)
- [உதவி மன்றம்](https://forum.chainide.com/)

**[Replit (Solidity Starter - Beta)](https://replit.com/@replit/Solidity-starter-beta)** - **_ஹாட் ரீலோடிங், பிழை சரிபார்த்தல் மற்றும் முதல் தர டெஸ்ட்நெட் ஆதரவுடன் Ethereum-க்கான தனிப்பயனாக்கக்கூடிய மேம்பாட்டுச் சூழல்_**

- [ஆவணங்கள்](https://docs.replit.com/)

**[Tenderly Sandbox](https://sandbox.tenderly.co/)** - **_Solidity மற்றும் JavaScript-ஐப் பயன்படுத்தி உலாவியிலேயே ஸ்மார்ட் ஒப்பந்தங்களை எழுத, இயக்க மற்றும் பிழைத்திருத்தம் செய்யக்கூடிய வேகமான முன்மாதிரிச் சூழல்_**

**[EthFiddle](https://ethfiddle.com/)** - **_உங்கள் ஸ்மார்ட் ஒப்பந்தத்தை எழுத, தொகுக்க மற்றும் பிழைத்திருத்தம் செய்ய அனுமதிக்கும் இணைய அடிப்படையிலான IDE_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## டெஸ்க்டாப் IDEகள் {#desktop-ides}

பெரும்பாலான நிறுவப்பட்ட IDEகள் Ethereum மேம்பாட்டு அனுபவத்தை மேம்படுத்த செருகுநிரல்களை உருவாக்கியுள்ளன. குறைந்தபட்சம், அவை [ஸ்மார்ட் ஒப்பந்த மொழிகளுக்கான](/developers/docs/smart-contracts/languages/) தொடரியல் சிறப்பம்சத்தை (syntax highlighting) வழங்குகின்றன.

**Visual Studio Code -** **_அதிகாரப்பூர்வ Ethereum ஆதரவுடன் கூடிய தொழில்முறை குறுக்கு-தள IDE_**

- [Visual Studio Code](https://code.visualstudio.com/)
- [குறியீட்டு மாதிரிகள்](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**JetBrains IDEs (IntelliJ IDEA, போன்றவை.) -** **_மென்பொருள் உருவாக்குநர்கள் மற்றும் குழுக்களுக்கான அத்தியாவசியக் கருவிகள்_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ Solidity](https://github.com/intellij-solidity/intellij-solidity/)

**Remix Desktop -** **_உங்கள் உள்ளூர் கணினியில் Remix IDE-ஐ அனுபவியுங்கள்_**

- [பதிவிறக்கம்](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)
