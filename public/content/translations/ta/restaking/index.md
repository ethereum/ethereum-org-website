Ethereum நெட்வொர்க் 24/7, 365 நாட்களும் பில்லியன் கணக்கான டாலர்கள் மதிப்பைப் பாதுகாக்கிறது. எப்படி?

உலகெங்கிலும் உள்ள மக்கள் Ethereum பரிவர்த்தனைகளைச் செயலாக்கும் மற்றும் Ethereum நெட்வொர்க்கைப் பாதுகாக்கும் மென்பொருளை இயக்குவதற்காக ஸ்மார்ட் ஒப்பந்தங்களில் [ether (ETH)](/what-is-ether/)-ஐப் பூட்டி வைக்கிறார்கள் (அல்லது "ஸ்டேக்" செய்கிறார்கள்). இதற்குப் பதிலாக, அவர்களுக்கு அதிக ETH வெகுமதியாகக் கிடைக்கிறது.

ரீஸ்டேக்கிங் என்பது [ஸ்டேக்கர்கள்](/staking/) இந்தப் பாதுகாப்பை மற்ற சேவைகள், பயன்பாடுகள் அல்லது நெட்வொர்க்குகளுக்கு விரிவுபடுத்துவதற்காக உருவாக்கப்பட்ட ஒரு தொழில்நுட்பமாகும். இதற்குப் பதிலாக, அவர்கள் கூடுதல் ரீஸ்டேக்கிங் வெகுமதிகளைப் பெறுகிறார்கள். இருப்பினும், அவர்கள் தங்களின் ஸ்டேக் செய்யப்பட்ட ETH-ஐ அதிக ஆபத்தில் வைக்கிறார்கள்.

**ரீஸ்டேக்கிங் 18 நிமிடங்களில் விளக்கப்பட்டுள்ளது**

## ரீஸ்டேக்கிங் என்றால் என்ன? {#what-is-restaking}

ரீஸ்டேக்கிங் என்பது ஸ்டேக்கர்கள் தங்களின் ஏற்கனவே ஸ்டேக் செய்யப்பட்ட ETH-ஐப் பயன்படுத்தி மற்ற பரவலாக்கப்பட்ட சேவைகளைப் பாதுகாப்பதாகும். இதற்குப் பதிலாக, ரீஸ்டேக்கர்கள் தங்களின் வழக்கமான ETH ஸ்டேக்கிங் வெகுமதிகளுக்கு மேல் அந்த மற்ற சேவைகளிலிருந்து கூடுதல் வெகுமதிகளைப் பெறலாம்.

ரீஸ்டேக்கிங் மூலம் பாதுகாக்கப்படும் பரவலாக்கப்பட்ட சேவைகள் "செயல்பாட்டுடன் சரிபார்க்கப்பட்ட சேவைகள்" (Actively Validated Services - AVSs) என்று அழைக்கப்படுகின்றன.
பல ETH ஸ்டேக்கர்கள் Ethereum சரிபார்ப்பு மென்பொருளை இயக்குவதைப் போலவே, பல ரீஸ்டேக்கர்கள் சிறப்பு AVS மென்பொருளை இயக்குகிறார்கள்.

## ஸ்டேக்கிங் vs ரீஸ்டேக்கிங் {#staking-vs-restaking}

| ஸ்டேக்கிங் | ரீஸ்டேக்கிங் |
| ------------------------------ | ------------------------------------------------- |
| ETH வெகுமதிகளைப் பெறுதல் | ETH வெகுமதிகள் + AVS வெகுமதிகளைப் பெறுதல் |
| Ethereum நெட்வொர்க்கைப் பாதுகாக்கிறது | Ethereum நெட்வொர்க் + AVS-களைப் பாதுகாக்கிறது |
| குறைந்தபட்ச ETH இல்லை | குறைந்தபட்ச ETH இல்லை |
| குறைந்த ஆபத்து நிலை | குறைந்த முதல் அதிக ஆபத்து நிலை |
| திரும்பப் பெறும் நேரம் வரிசையைப் பொறுத்தது | திரும்பப் பெறும் நேரம் வரிசை + அன்பாண்டிங் காலத்தைப் பொறுத்தது |

## நமக்கு ஏன் ரீஸ்டேக்கிங் தேவை? {#why-do-we-need-restaking}

இரண்டு உலகங்களை கற்பனை செய்து பாருங்கள்; ஒன்று ரீஸ்டேக்கிங் உள்ளது, மற்றொன்று இல்லை.

 <TabbedSection />

ரீஸ்டேக்கிங் உள்ள இந்த உலகில், AVS மற்றும் ஸ்டேக்கர் இருவரும் ஒருவரையொருவர் கண்டுபிடித்து, கூடுதல் வெகுமதிகளுக்காகப் பாதுகாப்பைப் பரிமாறிக்கொள்வதன் மூலம் பயனடைகிறார்கள்.

## ரீஸ்டேக்கிங் எப்படி வேலை செய்கிறது? {#how-does-restaking-work}

ரீஸ்டேக்கிங்கில் பல நிறுவனங்கள் ஈடுபட்டுள்ளன — அவை ஒவ்வொன்றும் ஒரு முக்கியப் பங்கை வகிக்கின்றன.

| **சொல்** | **விளக்கம்** |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ரீஸ்டேக்கிங் தளங்கள் (Restaking platforms)** | ரீஸ்டேக்கிங் தளம் என்பது AVS-கள், ETH ஸ்டேக்கர்கள் மற்றும் ஆபரேட்டர்களை இணைக்கும் ஒரு சேவையாகும். ஸ்டேக்கர்கள் தங்களின் ETH-ஐ ரீஸ்டேக் செய்வதற்கான பரவலாக்கப்பட்ட பயன்பாடுகளையும், ஸ்டேக்கர்கள், AVS-கள் மற்றும் ஆபரேட்டர்கள் ஒருவரையொருவர் கண்டறியக்கூடிய சந்தைகளையும் அவை உருவாக்குகின்றன. |
| **நேட்டிவ் ரீஸ்டேக்கர்கள் (Native restakers)** | தங்களின் சொந்த Ethereum வேலிடேட்டர்களை இயக்குவதன் மூலம் தங்களின் ETH-ஐ ஸ்டேக் செய்யும் நபர்கள், ETH வேலிடேட்டர் வெகுமதிகளுக்கு மேல் ரீஸ்டேக்கிங் வெகுமதிகளைப் பெற, EigenLayer மற்றும் பிற ரீஸ்டேக்கிங் தளங்களுடன் தங்களின் ஸ்டேக் செய்யப்பட்ட ETH-ஐ இணைக்கலாம். |
| **லிக்விட் ரீஸ்டேக்கர்கள் (Liquid restakers)** | Lido அல்லது Rocket Pool போன்ற மூன்றாம் தரப்பு லிக்விட் ஸ்டேக்கிங் வழங்குநர் மூலம் தங்களின் ETH-ஐ ஸ்டேக் செய்யும் நபர்கள், தங்களின் ஸ்டேக் செய்யப்பட்ட ETH-ஐக் குறிக்கும் லிக்விட் ஸ்டேக்கிங் டோக்கன்களை (LSTs) பெறுகிறார்கள். தங்களின் அசல் ETH-ஐ ஸ்டேக் செய்தபடியே வைத்துக்கொண்டு, ரீஸ்டேக்கிங் வெகுமதிகளைப் பெற இந்த LST-களை அவர்கள் ரீஸ்டேக் செய்யலாம். |
| **ஆபரேட்டர்கள் (Operators)** | ஆபரேட்டர்கள் AVS-களின் ரீஸ்டேக்கிங் மென்பொருளை இயக்குகிறார்கள், ஒவ்வொரு AVS-க்கும் தேவையான சரிபார்ப்புப் பணிகளைச் செய்கிறார்கள். ஆபரேட்டர்கள் பொதுவாக அப்டைம் மற்றும் செயல்திறன் போன்றவற்றுக்கு உத்தரவாதம் அளிக்கும் தொழில்முறை சேவை வழங்குநர்களாகும். ஆபரேட்டர் அல்லாத ரீஸ்டேக்கர்களைப் போலவே, ஆபரேட்டர்களும் AVS-களைப் பாதுகாக்க ஸ்டேக் செய்யப்பட்ட ETH-ஐப் பயன்படுத்துகிறார்கள், ஆனால் ஆபரேட்டர்கள் தங்களின் வேலைக்கு ஈடாகக் கூடுதல் வெகுமதிகளையும் பெறுகிறார்கள். |
| **AVS-கள்** | இவை பரவலாக்கப்பட்ட சேவைகளாகும் — விலை ஆரக்கிள்கள், டோக்கன் பிரிட்ஜ்கள் மற்றும் தரவு அமைப்புகள் போன்றவை — இவை ரீஸ்டேக்கர்களிடமிருந்து பாதுகாப்பைப் பெற்று, அதற்குப் பதிலாக டோக்கன் வெகுமதிகளை வழங்குகின்றன. |

## ரீஸ்டேக்கிங் மூலம் நான் எவ்வளவு சம்பாதிக்க முடியும்? {#how-much-can-i-make-from-restaking}

AVS-கள் வெவ்வேறு விகிதங்களை வழங்கினாலும், eETH போன்ற லிக்விட் ரீஸ்டேக்கிங் டோக்கன்கள் (LRTs) நீங்கள் எவ்வளவு சம்பாதிக்கலாம் என்பது பற்றிய ஒரு யோசனையை உங்களுக்குத் தருகின்றன. உங்கள் ETH-ஐ ஸ்டேக் செய்வதற்கு stETH போன்ற LST-களைப் பெறுவதைப் போலவே, stETH-ஐ ரீஸ்டேக் செய்வதற்கு eETH போன்ற LRT-களைப் பெறலாம். இந்த டோக்கன்கள் ETH ஸ்டேக்கிங் மற்றும் ரீஸ்டேக்கிங் வெகுமதிகளைப் பெறுகின்றன.

**ரீஸ்டேக்கிங்கில் உள்ள அபாயங்களை ஒப்புக்கொள்வது முக்கியம். சாத்தியமான வெகுமதிகள் கவர்ச்சிகரமானதாக இருக்கலாம், ஆனால் அவை ஆபத்து இல்லாதவை அல்ல.**

## ரீஸ்டேக்கிங்கின் அபாயங்கள் யாவை? {#what-are-the-risks-of-restaking}

| **அபாயங்கள்** | **விளக்கம்** |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **அபராதங்கள் (அல்லது “ஸ்லாஷிங்”)** | ETH ஸ்டேக்கிங்கைப் போலவே, ரீஸ்டேக்கர்கள்/ஆபரேட்டர்கள் ஆஃப்லைனுக்குச் சென்றால், செய்திகளைத் தணிக்கை செய்தால் அல்லது நெட்வொர்க்கைச் சிதைக்க முயன்றால், அவர்களின் ஸ்டேக் பகுதியளவு அல்லது முழுமையாக ஸ்லாஷ் (எரிக்கப்படலாம்) செய்யப்படலாம். |
| **மையப்படுத்தல் (Centralization)** | சில ஆபரேட்டர்கள் பெரும்பாலான ரீஸ்டேக்கிங்கில் ஆதிக்கம் செலுத்தினால், அவர்கள் ரீஸ்டேக்கர்கள், AVS-கள் மற்றும் ரீஸ்டேக்கிங் தளங்களில் கூடப் பெரிய தாக்கத்தை ஏற்படுத்தலாம். |
| **சங்கிலி எதிர்வினைகள் (Chain reactions)** | பல AVS-களைப் பாதுகாக்கும் போது ஒரு ரீஸ்டேக்கர் ஸ்லாஷ் செய்யப்பட்டால், இது மற்ற AVS-களுக்கான பாதுகாப்பைக் குறைத்து, அவற்றைப் பாதிக்கப்படக்கூடியதாக மாற்றும். |
| **நிதிக்கான உடனடி அணுகல்** | ரீஸ்டேக் செய்யப்பட்ட ETH-ஐத் திரும்பப் பெறுவதற்குக் காத்திருப்பு நேரம் (அல்லது “அன்பாண்டிங் காலம்”) உள்ளது, எனவே நீங்கள் எப்போதும் உடனடியாக அணுகலைப் பெற முடியாது. |
