எத்திரியம் பிணையம் 24/7, 365 நாட்களும் பில்லியன் கணக்கான டாலர்கள் மதிப்பைப் பாதுகாக்கிறது. எப்படி?

எத்திரியம் பரிவர்த்தனைகளைச் செயலாக்கும் மற்றும் எத்திரியம் பிணையத்தைப் பாதுகாக்கும் மென்பொருளை இயக்குவதற்காக, உலகெங்கிலும் உள்ள மக்கள் திறன் ஒப்பந்தங்களில் [ஈதரை (ETH)](/what-is-ether/) பூட்டி வைக்கிறார்கள் (அல்லது "பங்குவைக்கிறார்கள்"). இதற்குப் பதிலாக, அவர்களுக்கு அதிக ETH வெகுமதியாகக் கிடைக்கிறது.

மறுபங்குவைப்பு என்பது [பங்குவைப்பாளர்கள்](/staking/) இந்தப் பாதுகாப்பைப் பிற சேவைகள், பயன்பாடுகள் அல்லது பிணையங்களுக்கு விரிவுபடுத்துவதற்காக உருவாக்கப்பட்ட ஒரு தொழில்நுட்பமாகும். இதற்குப் பதிலாக, அவர்கள் கூடுதல் மறுபங்குவைப்பு வெகுமதிகளைப் பெறுகிறார்கள். இருப்பினும், அவர்கள் தங்கள் பங்குவைக்கப்பட்ட ETH ஐ அதிக ஆபத்தில் வைக்கிறார்கள்.

**மறுபங்குவைப்பு 18 நிமிடங்களில் விளக்கப்பட்டுள்ளது**

## மறுபங்குவைப்பு என்றால் என்ன? {#what-is-restaking}

பங்குவைப்பாளர்கள் தங்களின் ஏற்கனவே பங்குவைக்கப்பட்ட ETH ஐப் பயன்படுத்திப் பிற பரவலாக்கப்பட்ட சேவைகளைப் பாதுகாப்பதே மறுபங்குவைப்பு ஆகும். இதற்குப் பதிலாக, மறுபங்குவைப்பாளர்கள் தங்களின் வழக்கமான ETH பங்குவைத்தல் வெகுமதிகளுடன் அந்தப் பிற சேவைகளிலிருந்து கூடுதல் வெகுமதிகளைப் பெறலாம்.

மறுபங்குவைப்பு மூலம் பாதுகாக்கப்படும் பரவலாக்கப்பட்ட சேவைகள் "செயல்பாட்டுடன் சரிபார்க்கப்பட்ட சேவைகள்" (Actively Validated Services - AVSs) என்று அழைக்கப்படுகின்றன.
பல ETH பங்குவைப்பாளர்கள் எத்திரியம் சரிபார்ப்பு மென்பொருளை இயக்குவதைப் போலவே, பல மறுபங்குவைப்பாளர்கள் சிறப்பு AVS மென்பொருளை இயக்குகிறார்கள்.

## பங்குவைத்தல் vs மறுபங்குவைப்பு {#staking-vs-restaking}

| பங்குவைத்தல்                        | மறுபங்குவைப்பு                                         |
| ------------------------------ | ------------------------------------------------- |
| ETH வெகுமதிகளைப் பெறுதல்               | ETH வெகுமதிகள் + AVS வெகுமதிகளைப் பெறுதல்                    |
| எத்திரியம் பிணையத்தைப் பாதுகாக்கிறது   | எத்திரியம் பிணையம் + AVSகளைப் பாதுகாக்கிறது               |
| குறைந்தபட்ச ETH இல்லை                 | குறைந்தபட்ச ETH இல்லை                                    |
| குறைந்த ஆபத்து நிலை                 | குறைந்த முதல் அதிக ஆபத்து நிலை                            |
| திரும்பப் பெறும் நேரம் வரிசையைப் பொறுத்தது | திரும்பப் பெறும் நேரம் வரிசை + பிணைப்பு நீக்கக் காலத்தைப் பொறுத்தது |

## நமக்கு ஏன் மறுபங்குவைப்பு தேவை? {#why-do-we-need-restaking}

இரண்டு உலகங்களை கற்பனை செய்து பாருங்கள்; ஒன்று மறுபங்குவைப்புடன், மற்றொன்று அது இல்லாமல்.

மறுபங்குவைப்பு உள்ள இந்த உலகில், AVS மற்றும் பங்குவைப்பாளர் இருவருமே ஒருவரையொருவர் கண்டுபிடித்து, கூடுதல் வெகுமதிகளுக்காகப் பாதுகாப்பைப் பரிமாறிக்கொள்வதன் மூலம் பயனடைகிறார்கள்.

## மறுபங்குவைப்பு எவ்வாறு வேலை செய்கிறது? {#how-does-restaking-work}

மறுபங்குவைப்பில் பல நிறுவனங்கள் ஈடுபட்டுள்ளன — அவை ஒவ்வொன்றும் ஒரு முக்கியப் பங்கை வகிக்கின்றன.

| **சொல்**                | **விளக்கம்**                                                                                                                                                                                                                                                                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **மறுபங்குவைப்பு தளங்கள்** | மறுபங்குவைப்பு தளம் என்பது AVSகள், ETH பங்குவைப்பாளர்கள் மற்றும் இயக்குபவர்களை இணைக்கும் ஒரு சேவையாகும். பங்குவைப்பாளர்கள் தங்கள் ETH ஐ மறுபங்குவைப்பு செய்வதற்கான பரவலாக்கப்பட்ட பயன்பாடுகளையும், பங்குவைப்பாளர்கள், AVSகள் மற்றும் இயக்குபவர்கள் ஒருவரையொருவர் கண்டறியக்கூடிய சந்தைகளையும் அவை உருவாக்குகின்றன.                                                                                                                |
| **நேட்டிவ் மறுபங்குவைப்பாளர்கள்**    | தங்களின் சொந்த எத்திரியம் சரிபார்ப்பான்களை இயக்குவதன் மூலம் தங்கள் ETH ஐப் பங்குவைக்கும் நபர்கள், ETH சரிபார்ப்பான் வெகுமதிகளுடன் மறுபங்குவைப்பு வெகுமதிகளைப் பெற, EigenLayer மற்றும் பிற மறுபங்குவைப்பு தளங்களுடன் தங்கள் பங்குவைக்கப்பட்ட ETH ஐ இணைக்கலாம்.                                                                                                                             |
| **நீர்ம மறுபங்குவைப்பாளர்கள்**    | லிடோ அல்லது Rocket Pool போன்ற மூன்றாம் தரப்பு நீர்மப் பங்குவைப்பு வழங்குநர் மூலம் தங்கள் ETH ஐப் பங்குவைக்கும் நபர்கள், தங்கள் பங்குவைக்கப்பட்ட ETH ஐக் குறிக்கும் திரவ ஸ்டேக்கிங் டோக்கன் (LST)களைப் பெறுகிறார்கள். அவர்கள் தங்களின் அசல் ETH ஐப் பங்குவைக்கப்பட்ட நிலையிலேயே வைத்துக்கொண்டு, மறுபங்குவைப்பு வெகுமதிகளைப் பெற இந்த LSTகளை மறுபங்குவைப்பு செய்யலாம்.                                                                                  |
| **இயக்குபவர்கள்**           | இயக்குபவர்கள் AVSகளின் மறுபங்குவைப்பு மென்பொருளை இயக்குகிறார்கள், ஒவ்வொரு AVSக்கும் தேவையான சரிபார்ப்புப் பணிகளைச் செய்கிறார்கள். இயக்குபவர்கள் பொதுவாகச் செயல்பாட்டு நேரம் மற்றும் செயல்திறன் போன்றவற்றுக்கு உத்தரவாதம் அளிக்கும் தொழில்முறை சேவை வழங்குநர்களாக இருப்பார்கள். இயக்குபவர்கள் அல்லாத மறுபங்குவைப்பாளர்களைப் போலவே, இயக்குபவர்களும் AVSகளைப் பாதுகாக்கப் பங்குவைக்கப்பட்ட ETH ஐப் பயன்படுத்துகிறார்கள், ஆனால் இயக்குபவர்கள் தங்கள் வேலைக்கு ஈடாகக் கூடுதல் வெகுமதிகளையும் பெறுகிறார்கள். |
| **AVSகள்**                | இவை விலை ஆரக்கிள்கள், வில்லைப் பாலங்கள் மற்றும் தரவு அமைப்புகள் போன்ற பரவலாக்கப்பட்ட சேவைகளாகும் — இவை மறுபங்குவைப்பாளர்களிடமிருந்து பாதுகாப்பைப் பெறுகின்றன மற்றும் அதற்குப் பதிலாக வில்லை வெகுமதிகளை வழங்குகின்றன.                                                                                                                                                                              |

## மறுபங்குவைப்பு மூலம் நான் எவ்வளவு சம்பாதிக்க முடியும்? {#how-much-can-i-make-from-restaking}

AVSகள் வெவ்வேறு விகிதங்களை வழங்கினாலும், eETH போன்ற நீர்ம மறுபங்குவைப்பு வில்லைகள் (LRTs) நீங்கள் எவ்வளவு சம்பாதிக்கலாம் என்பது பற்றிய ஒரு யோசனையை உங்களுக்கு வழங்குகின்றன. உங்கள் ETH ஐப் பங்குவைப்பதற்காக stETH போன்ற LSTகளைப் பெறுவதைப் போலவே, stETH ஐ மறுபங்குவைப்பு செய்வதற்காக eETH போன்ற LRTகளைப் பெறலாம். இந்த வில்லைகள் ETH பங்குவைத்தல் மற்றும் மறுபங்குவைப்பு வெகுமதிகளைப் பெறுகின்றன.

**மறுபங்குவைப்பில் உள்ள ஆபத்துகளை ஒப்புக்கொள்வது முக்கியம். சாத்தியமான வெகுமதிகள் கவர்ச்சிகரமானதாக இருக்கலாம், ஆனால் அவை ஆபத்து இல்லாதவை அல்ல.**

## மறுபங்குவைப்பின் ஆபத்துகள் யாவை? {#what-are-the-risks-of-restaking}

| **ஆபத்துகள்**                     | **விளக்கம்**                                                                                                                                                |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **அபராதங்கள் (அல்லது "வெட்டுதல்")** | ETH பங்குவைப்பைப் போலவே, மறுபங்குவைப்பாளர்கள்/இயக்குபவர்கள் ஆஃப்லைனுக்குச் சென்றாலோ, செய்திகளைத் தணிக்கை செய்தாலோ அல்லது பிணையத்தைச் சீர்குலைக்க முயன்றாலோ, அவர்களின் பங்குவைப்பு பகுதியளவு அல்லது முழுமையாக வெட்டப்படலாம் (எரிக்கப்படலாம்). |
| **மையப்படுத்தல்**            | சில இயக்குபவர்கள் பெரும்பாலான மறுபங்குவைப்பில் ஆதிக்கம் செலுத்தினால், அவர்கள் மறுபங்குவைப்பாளர்கள், AVSகள் மற்றும் மறுபங்குவைப்பு தளங்களில் கூடப் பெரும் செல்வாக்கைக் கொண்டிருக்கலாம்.                             |
| **சங்கிலி எதிர்வினைகள்**           | பல AVSகளைப் பாதுகாக்கும் போது ஒரு மறுபங்குவைப்பாளர் வெட்டப்பட்டால், இது மற்ற AVSகளுக்கான பாதுகாப்பைக் குறைத்து, அவற்றைப் பாதிக்கப்படக்கூடியதாக மாற்றும்.                             |
| **நிதிகளுக்கான உடனடி அணுகல்** | மறுபங்குவைப்பு செய்யப்பட்ட ETH ஐத் திரும்பப் பெறுவதற்குக் காத்திருப்பு நேரம் (அல்லது "பிணைப்பு நீக்கக் காலம்") உள்ளது, எனவே நீங்கள் எப்போதும் உடனடியாக அணுகலைப் பெற முடியாது.                                       |
