ethereum.org என்பது **12 000+** பங்களிப்பாளர்களைக் கொண்ட ஒரு திறந்த மூல திட்டமாகும், இவர்கள் இணையதளத்தை மொழிபெயர்க்கவும், எழுதவும், வடிவமைக்கவும் மற்றும் பராமரிக்கவும் உதவுகிறார்கள்.

நாங்கள் உங்களை வரவேற்கும் ஒரு சமூகமாக இருக்கிறோம், இது [எத்திரியம்](/) சுற்றுச்சூழல் அமைப்பில் நீங்கள் வளரவும் கற்கவும் உதவும், அதே நேரத்தில் அர்த்தமுள்ள வகையில் பங்களித்து தொடர்புடைய நடைமுறை அனுபவத்தைப் பெறவும் உதவும்!

## பங்களிக்கத் தொடங்குவதற்கான சிறந்த முதல் பணிகள் {#good-first-tasks-to-start-contributing}

இவை நீங்கள் எங்களுக்குத் தீர்க்க உதவக்கூடிய மற்றும் பொறுப்பேற்கக்கூடிய சில தற்போதைய பணிகளாகும். இணையதளத்தில் பெரும்பாலான மாற்றங்கள் GitHub மூலம் செய்யப்படுவதால், பெரும்பாலானவற்றிற்கு உங்களுக்கு GitHub கணக்கு தேவைப்படும்.

## ethereum.org-இல் எவ்வாறு வேலை செய்வது {#how-to-update-content}

[மொழிபெயர்ப்புத் திட்டத்தில்](/contributing/translation-program/) பங்களிக்க நீங்கள் விரும்பினால், [Crowdin](https://crowdin.com/project/ethereum-org)-இல் ஒரு கணக்கை உருவாக்குமாறு கேட்டுக்கொள்கிறோம். மற்ற அனைத்திற்கும் – இணையதளத்தில் உள்ளடக்கம் அல்லது காட்சிகளைச் சேர்ப்பது அல்லது திருத்துவது, பிழைகளைச் சரிசெய்வது, திறந்த பணிகளில் வேலை செய்வது – உங்களுக்கு ஒரு [GitHub](https://github.com/) கணக்கு தேவைப்படும்.

அனைத்துப் புதுப்பிப்புகளும் GitHub PR செயல்முறை வழியாகவே செய்யப்படுகின்றன. அதாவது நீங்கள் இணையதளத்தின் உள்ளூர் நகலை (local copy) உருவாக்கி, உங்கள் மாற்றங்களைச் செய்து, உங்கள் மாற்றங்களை ஒன்றிணைக்கக் கோருகிறீர்கள். இதற்கு முன் நீங்கள் இதைச் செய்ததில்லை என்றால், எங்கள் [GitHub களஞ்சியத்தின் (repository)](https://github.com/ethereum/ethereum-org-website) கீழே உள்ள வழிமுறைகளைப் பின்பற்றவும்.

எதிலும் வேலை செய்ய உங்களுக்கு அனுமதி தேவையில்லை, ஆனால் நீங்கள் என்ன செய்யத் திட்டமிட்டுள்ளீர்கள் என்பதை எங்களுக்குத் தெரியப்படுத்துவது எப்போதும் சிறந்தது. இதை நீங்கள் பின்வருமாறு செய்யலாம்:

- [GitHub](https://github.com/ethereum/ethereum-org-website)-இல் உள்ள ஒரு சிக்கல் (issue) அல்லது PR-இல் கருத்துத் தெரிவிப்பதன் மூலம்
- எங்கள் [டிஸ்கார்ட் சேவையகத்தில்](https://discord.gg/ethereum-org) செய்தி அனுப்புவதன் மூலம்

பங்களிப்பதற்கு முன், பின்வருவனவற்றை நீங்கள் நன்கு அறிந்திருப்பதை உறுதிசெய்து கொள்ளவும்:

- வளர்ந்து வரும் [ethereum.org-இன் தொலைநோக்கு பார்வை](/about/)
- எங்கள் [வடிவமைப்புக் கோட்பாடுகள்](/contributing/design-principles/)
- எங்கள் [நடை வழிகாட்டி (style guide)](/contributing/style-guide/)
- எங்கள் [நடத்தை விதிகள்](/community/code-of-conduct)

## தளத்தைப் பற்றிய முடிவுகள் எவ்வாறு எடுக்கப்படுகின்றன {#how-decisions-about-the-site-are-made}

தனிப்பட்ட PR-கள், வடிவமைப்புப் பரிணாமம் மற்றும் முக்கிய மேம்படுத்தல்கள் பற்றிய முடிவுகள் எத்திரியம் சுற்றுச்சூழல் அமைப்பு முழுவதிலுமிருந்து வரும் ஒரு குழுவால் எடுக்கப்படுகின்றன. இந்தக் குழுவில் திட்ட மேலாளர்கள், டெவலப்பர்கள், வடிவமைப்பாளர்கள், சந்தைப்படுத்தல் மற்றும் தகவல் தொடர்பு நிபுணர்கள் மற்றும் துறைசார் வல்லுநர்கள் அடங்குவர். சமூகத்தின் உள்ளீடுகள் ஒவ்வொரு முடிவையும் வழிநடத்துகின்றன: எனவே தயவுசெய்து சிக்கல்களில் (issues) கேள்விகளை எழுப்புங்கள், PR-களைச் சமர்ப்பியுங்கள் அல்லது குழுவைத் தொடர்புகொள்ளுங்கள்:

- [website@ethereum.org](mailto:website@ethereum.org)
- [@ethdotorg](https://twitter.com/ethdotorg)
- [டிஸ்கார்ட் சேவையகம்](https://discord.gg/ethereum-org)

### கருத்துத் திருட்டு (plagiarism) பற்றிய ஒரு குறிப்பு {#plagiarism}

ethereum.org-க்கு ஏதேனும் உள்ளடக்கம் அல்லது கலைப்பொருளைப் பங்களிக்கும்போது, உங்கள் அசல் படைப்பு அல்லது பயன்படுத்த உங்களுக்கு அனுமதி உள்ள உள்ளடக்கத்தை மட்டுமே பயன்படுத்தவும். எத்திரியம் சுற்றுச்சூழல் அமைப்பிற்குள் உள்ள பல திட்டங்கள் தகவல்களை இலவசமாகப் பகிர்வதை அனுமதிக்கும் திறந்த மூல உரிமங்களைப் பயன்படுத்துகின்றன. இருப்பினும், இந்தத் தகவலை உங்களால் கண்டுபிடிக்க முடியவில்லை என்றால், அதை ethereum.org-இல் சேர்க்க முயற்சிக்க வேண்டாம். கருத்துத் திருட்டு எனக் கருதப்படும் எந்தவொரு இழுப்பு கோரிக்கைகளும் (pull requests) நிராகரிக்கப்படும்.

## திறந்த மூலத்திற்குப் புதியவரா? {#new-to-open-source}

திறந்த மூலத்திற்குப் புதிய டெவலப்பர்களுக்காகவே பிரத்யேகமாக வடிவமைக்கப்பட்ட, நுழைவதற்கு எளிதான சிக்கல்கள் எங்கள் GitHub களஞ்சியத்தில் உள்ளன, அவை [good first issue](https://github.com/ethereum/ethereum-org-website/issues?q=is%3Aopen+is%3Aissue+label%3A%22good+first+issue%22) எனப் பெயரிடப்பட்டுள்ளன.

## உங்கள் சங்கிலிசார் சாதனை வில்லையை (OAT) உரிமைக்கோரல் {#oat}

உங்கள் பங்களிப்பு ethereum.org-இல் ஒன்றிணைக்கப்பட்டால், [Galxe](https://app.galxe.com/quest/ethereumorg)-இல் ஒரு சிறப்பு பேட்ஜை உரிமைக்கோர உங்களுக்கு வாய்ப்பு கிடைக்கும். சங்கிலிசார் சாதனை வில்லை (OAT) என்பது சுற்றுச்சூழல் அமைப்பைச் சற்றே சிறப்பானதாக மாற்ற நீங்கள் உதவினீர்கள் என்பதற்கான சான்றாகும்.

[OAT-கள் பற்றி மேலும் அறிய](https://help.galxe.com/en/articles/9645630-create-quest-rewards#h_1c5d63ba03)

### உரிமைக்கோருவது எப்படி {#how-to-claim}
1. எங்கள் [டிஸ்கார்ட் சேவையகத்தில்](https://discord.gg/ethereum-org) சேரவும்.
2. `#🥇 | proof-of-contribution` சேனலில் உங்கள் பங்களிப்பிற்கான இணைப்பை ஒட்டவும்.
3. உங்கள் OAT-க்கான இணைப்பை எங்கள் குழு உறுப்பினர் ஒருவர் உங்களுக்கு அனுப்பும் வரை காத்திருக்கவும்.
4. உங்கள் OAT-ஐ உரிமைக்கோருங்கள்!

OAT-களை உரிமைக்கோர நீங்கள் சுய-பாதுகாப்பு பணப்பைகளை மட்டுமே பயன்படுத்த வேண்டும். பரிமாற்றக் கணக்குகள் அல்லது தனிப்பட்ட திறவுகோல்கள் (private keys) உங்களிடம் இல்லாத பிற கணக்குகளைப் பயன்படுத்த வேண்டாம், ஏனெனில் இவை உங்கள் OAT-களை அணுகவும் நிர்வகிக்கவும் உங்களை அனுமதிக்காது.

## உங்கள் GitPOAP-ஐ உரிமைக்கோரல் {#claim-gitpoap}

GitPOAP தானாகவே உங்கள் ஒன்றிணைக்கப்பட்ட பங்களிப்பை அங்கீகரித்து, அவர்களின் தளத்திலேயே ஒரு தனித்துவமான பங்களிப்பாளர் POAP-ஐ அச்சிட உங்களை அனுமதிக்கும்!


### உரிமைக்கோருவது எப்படி {#how-to-claim-2}

1. [GitPOAP](https://www.gitpoap.io)-ஐப் பார்வையிடவும்.
2. உங்கள் பணப்பையுடன் அல்லது உள்நுழைவு விருப்பத்தின் மூலம் உங்கள் மின்னஞ்சலுடன் இணைக்கவும்.
3. நீங்கள் தகுதியானவரா என்பதைச் சரிபார்க்க உங்கள் GitHub பயனர் பெயர், ETH முகவரி, ENS பெயர்கள் அல்லது ஏதேனும் GitPOAP-ஐத் தேடவும்.
4. உங்கள் GitHub கணக்கு தகுதியானதாக இருந்தால், உங்களால் ஒரு GitPOAP-ஐ அச்சிட முடியும்!
