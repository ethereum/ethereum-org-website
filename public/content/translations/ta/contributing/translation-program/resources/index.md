# ஆதாரங்கள் {#resources}

ethereum.org மொழிபெயர்ப்பாளர்களுக்கான சில பயனுள்ள வழிகாட்டிகள் மற்றும் கருவிகள், அத்துடன் மொழிபெயர்ப்பு சமூகங்கள் மற்றும் புதுப்பிப்புகளைக் கீழே காணலாம்.

- [மொழிபெயர்ப்பு அடிக்கடி கேட்கப்படும் கேள்விகள் (FAQs)](/contributing/translation-program/faq/) _– ethereum.org மொழிபெயர்ப்புத் திட்டம் பற்றிய அடிக்கடி கேட்கப்படும் கேள்விகள் மற்றும் பதில்கள்_

## மொழிபெயர்ப்பாளர்களுக்கான அலுவலக நேரங்கள் {#office-hours}

ஒவ்வொரு மாதத்தின் இரண்டாவது புதன்கிழமையிலும் மொழிபெயர்ப்பாளர்களுக்கான அலுவலக நேரங்கள் எங்களிடம் உள்ளன. இவை [ethereum.org Discord](https://discord.gg/ethereum-org) இல் உள்ள #office-hours குரல் சேனலில் நடைபெறுகின்றன, அங்கு நீங்கள் சரியான நேரங்கள் மற்றும் கூடுதல் விவரங்களையும் காணலாம்.

அலுவலக நேரங்கள் எங்கள் மொழிபெயர்ப்பாளர்களை மொழிபெயர்ப்பு செயல்முறை பற்றி கேள்விகளைக் கேட்கவும், திட்டத்தைப் பற்றிய கருத்துக்களை வழங்கவும், அவர்களின் யோசனைகளைப் பகிரவும் அல்லது முக்கிய ethereum.org குழுவுடன் அரட்டையடிக்கவும் அனுமதிக்கின்றன.
இறுதியாக, மொழிபெயர்ப்புத் திட்டத்தின் சமீபத்திய முன்னேற்றங்களைத் தொடர்புகொள்வதற்கும், எங்கள் பங்களிப்பாளர்களுடன் முக்கிய குறிப்புகள் மற்றும் வழிமுறைகளைப் பகிர்ந்துகொள்வதற்கும் இந்த அழைப்புகளைப் பயன்படுத்த விரும்புகிறோம்.

நீங்கள் ஒரு ethereum.org மொழிபெயர்ப்பாளராக இருந்தால் அல்லது ஒருவராக மாற விரும்பினால், இந்த அமர்வுகளில் ஒன்றின் போது எங்களுடன் சேர தயங்க வேண்டாம்.
