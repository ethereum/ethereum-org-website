# டெவலப்பர் கருவிகளைச் சேர்த்தல் {#contributing-to-ethereumorg-}

மக்கள் நம்பிக்கையுடன் உருவாக்குவதற்கும் அவர்களுக்குத் தேவையான ஆதரவைப் பெறுவதற்கும், சிறந்த டெவலப்பர் ஆதாரங்களை நாங்கள் பட்டியலிடுவதை உறுதிசெய்ய விரும்புகிறோம்.

நாங்கள் தவறவிட்ட பயனுள்ள டெவலப்பர் கருவி ஏதேனும் இருந்தால், பொருத்தமான இடத்தில் அதைப் பரிந்துரைக்க தயங்க வேண்டாம்.

நாங்கள் தற்போது எங்கள் [டெவலப்பர் போர்ட்டல்](/developers/) முழுவதும் டெவலப்பர் கருவிகளைப் பட்டியலிடுகிறோம்.

**பொருத்தமான பக்கங்களில் புதிய சேர்த்தல்களைப் பரிந்துரைக்க தயங்க வேண்டாம்.**

## நாங்கள் எவ்வாறு முடிவு செய்கிறோம் {#ways-to-contribute}

டெவலப்பர் கருவி சமர்ப்பிப்புகள் பின்வரும் அளவுகோல்களால் மதிப்பிடப்படும்:

**ஏற்கனவே பட்டியலிடப்பட்டுள்ள கருவிகளிலிருந்து இது அர்த்தமுள்ள வகையில் வேறுபடுகிறதா?**

- புதிய பிரிவுகள் அல்லது கருவிகளின் வகைகள்
- தற்போதுள்ள ஒத்த கருவிகளுடன் ஒப்பிடும்போது புதிய அம்சங்கள்
- தற்போதுள்ள ஒத்த கருவிகளால் உள்ளடக்கப்படாத ஒரு தனித்துவமான பயன்பாட்டு-வழக்கை இலக்காகக் கொண்டது

**கருவி நன்கு ஆவணப்படுத்தப்பட்டுள்ளதா?**

- ஆவணங்கள் உள்ளதா?
- கருவியைப் பயன்படுத்த இது போதுமானதா?
- இது சமீபத்தில் புதுப்பிக்கப்பட்டதா?

**கருவி பரவலாகப் பயன்படுத்தப்படுகிறதா?**

- GitHub நட்சத்திரங்கள், பதிவிறக்க புள்ளிவிவரங்கள் மற்றும் அறியப்பட்ட நிறுவனங்கள் அல்லது திட்டங்களால் இது பயன்படுத்தப்படுகிறதா போன்ற அளவீடுகளை நாங்கள் கருத்தில் கொள்வோம்

**கருவி போதுமான தரத்தில் உள்ளதா?**

- மீண்டும் மீண்டும் வரும் பிழைகள் உள்ளதா?
- கருவி நம்பகமானதா?
- கருவி தீவிரமாகப் பராமரிக்கப்படுகிறதா?

**கருவி ஓப்பன் சோர்ஸா?**

Ethereum வெளியில் உள்ள பல திட்டங்கள் ஓப்பன் சோர்ஸ் ஆகும். சமூக டெவலப்பர்கள் குறியீட்டை ஆய்வு செய்யவும் அதற்குப் பங்களிக்கவும் அனுமதிக்கும் ஓப்பன் சோர்ஸ் திட்டங்களை நாங்கள் பட்டியலிட அதிக வாய்ப்புள்ளது.

## தயாரிப்பு வரிசைப்படுத்தல் {#product-ordering}

தயாரிப்புகள் அகரவரிசைப்படி போன்ற வேறுவிதமாகக் குறிப்பாக வரிசைப்படுத்தப்படாவிட்டால், தயாரிப்புகள் பக்கத்தில் சேர்க்கப்பட்டதிலிருந்து மிகச் சமீபத்தியது வரை காட்டப்படும். வேறு வார்த்தைகளில் கூறுவதானால், புதிய தயாரிப்புகள் பட்டியலின் கீழே சேர்க்கப்படும்.
