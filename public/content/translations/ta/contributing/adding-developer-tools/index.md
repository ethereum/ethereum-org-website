# டெவலப்பர் கருவிகளைச் சேர்த்தல் {#contributing-to-ethereumorg-}

நாங்கள் மக்கள் நம்பிக்கையுடன் கட்டமைக்க மற்றும் தேவையான ஆதரவைப் பெறுவதற்காக சிறந்த மேம்பாட்டாளர் வளங்களை பட்டியலிடுவது உறுதி செய்ய விரும்புகிறோம்.

எங்களால் கவனிக்கப்படாத பயனுள்ள மேம்பாட்டாளர் கருவி இருந்தால், அதை ஏற்ற இடத்தில் முன்மொழியலாம்.

நாங்கள் தற்போது எங்கள் [டெவலப்பர் போர்டல்](/developers/) முழுவதும் டெவலப்பர் கருவிகளைப் பட்டியலிடுகிறோம்.

**பொருத்தமான பக்கங்களுக்குப் புதிய சேர்த்தல்களைப் பரிந்துரைக்க தயங்க வேண்டாம்.**

## நாங்கள் எப்படி முடிவு செய்கிறோம் {#ways-to-contribute}

மேம்பாட்டாளர் கருவி சமர்ப்பணங்கள் கீழ்காணும் அளவுகோல்களால் மதிப்பீடு செய்யப்படும்:

**ஏற்கனவே பட்டியலிடப்பட்டுள்ள கருவிகளிலிருந்து இது அர்த்தமுள்ள வகையில் வேறுபடுகிறதா?**

- புதிய வகைகள் அல்லது கருவிகளின் வகைகள்
- ஏற்கனவே உள்ள ஒத்த கருவிகளுடன் ஒப்பிடுகையில் புதிய அம்சங்கள்
- ஏற்கனவே உள்ள ஒத்த கருவிகளால் உள்ளடக்கப்படாத ஒரு தனித்துப் பயன்பாட்டிற்கு இலக்கு வைக்கும்

**கருவி நன்கு ஆவணப்படுத்தப்பட்டுள்ளதா?**

- ஆவணங்கள் உள்ளதா?
- கருவியைப் பயன்படுத்துவதற்கு போதுமானதா?
- சமீபத்தில் புதுப்பிக்கப்பட்டதா?

**இந்தக் கருவி பரவலாகப் பயன்படுத்தப்படுகிறதா?**

- GitHub நட்சத்திரங்கள், பதிவிறக்கம் புள்ளிவிவரங்கள் மற்றும் இது அறியப்பட்ட நிறுவனங்கள் அல்லது திட்டங்களால் பயன்படுத்தப்படுகிறதா என்பதற்கான அளவுகோல்களை நாங்கள் கருத்தில் கொள்வோம்

**இந்தக் கருவி போதுமான தரம் வாய்ந்ததா?**

- மீண்டும் மீண்டும் பிழைகள் உள்ளனவா?
- கருவி நம்பகமானதா?
- கருவி செயலில் பராமரிக்கப்படுகிறதா?

**இந்தக் கருவி திறந்த மூலநிரலா?**

எத்தீரியத்தில் பல திட்டங்கள் திறந்த மூலமாக உள்ளன. சமூக மேம்பாட்டாளர்கள் குறியீட்டை ஆய்வு செய்து அதில் பங்களிக்க அனுமதிக்கும் திறந்த மூல திட்டங்களை பட்டியலிடுவதற்கான வாய்ப்பு அதிகமாக உள்ளது.

## தயாரிப்பு ஒழுங்கு {#product-ordering}

தயாரிப்புகள் குறிப்பிட்ட முறையில் ஒழுங்குபடுத்தப்படாவிட்டால், உதாரணமாக அகரவரிசையாக, தயாரிப்புகள் பக்கத்தில் குறைந்தது முதல் அதிகமாக சேர்க்கப்பட்டவரை அடிப்படையாகக் கொண்டு காட்சிப்படுத்தப்படும். மற்ற சொற்களில், புதிய தயாரிப்புகள் பட்டியலின் கீழே சேர்க்கப்படும்.
