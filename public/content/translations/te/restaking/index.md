Ethereum నెట్‌వర్క్ 24/7, 365 రోజుల పాటు బిలియన్ల డాలర్ల విలువను భద్రపరుస్తుంది. ఎలా?

ప్రపంచవ్యాప్తంగా ప్రజలు Ethereum లావాదేవీలను ప్రాసెస్ చేసే మరియు Ethereum నెట్‌వర్క్‌ను భద్రపరిచే సాఫ్ట్‌వేర్‌ను అమలు చేయడానికి స్మార్ట్ కాంట్రాక్టులలో [ఈథర్ (ETH)](/what-is-ether/) ని లాక్ చేస్తారు (లేదా “స్టేక్” చేస్తారు). దానికి బదులుగా, వారు మరిన్ని ETHతో రివార్డ్ పొందుతారు.

రీస్టేకింగ్ అనేది [స్టేకర్లు](/staking/) కోసం ఈ భద్రతను ఇతర సేవలు, అప్లికేషన్‌లు లేదా నెట్‌వర్క్‌లకు విస్తరించడానికి నిర్మించబడిన ఒక టెక్నాలజీ. దానికి బదులుగా, వారు అదనపు రీస్టేకింగ్ రివార్డులను సంపాదిస్తారు. అయితే, వారు తమ స్టేక్ చేసిన ETHని మరింత ప్రమాదంలో పడేస్తారు.

**18 నిమిషాల్లో రీస్టేకింగ్ వివరించబడింది**

## రీస్టేకింగ్ అంటే ఏమిటి? {#what-is-restaking}

స్టేకర్లు తమ ఇప్పటికే స్టేక్ చేయబడిన ETHని ఇతర వికేంద్రీకృత సేవలను భద్రపరచడానికి ఉపయోగించినప్పుడు దానిని రీస్టేకింగ్ అంటారు. దానికి బదులుగా, రీస్టేకర్లు వారి సాధారణ ETH స్టేకింగ్ రివార్డులతో పాటు ఆ ఇతర సేవల నుండి అదనపు రివార్డులను పొందవచ్చు.

రీస్టేకింగ్ ద్వారా భద్రపరచబడిన వికేంద్రీకృత సేవలను "యాక్టివ్‌లీ వాలిడేటెడ్ సర్వీసెస్" (AVSలు) అని అంటారు.
చాలా మంది ETH స్టేకర్లు Ethereum వాలిడేషన్ సాఫ్ట్‌వేర్‌ను నడుపుతున్నట్లే, చాలా మంది రీస్టేకర్లు ప్రత్యేకమైన AVS సాఫ్ట్‌వేర్‌ను నడుపుతారు.

## స్టేకింగ్ వర్సెస్ రీస్టేకింగ్ {#staking-vs-restaking}

| స్టేకింగ్                             | రీస్టేకింగ్                                                 |
| ------------------------------------- | ----------------------------------------------------------- |
| ETH రివార్డులను సంపాదించండి           | ETH రివార్డులు + AVS రివార్డులను సంపాదించండి                |
| Ethereum నెట్‌వర్క్‌ను భద్రపరుస్తుంది | Ethereum నెట్‌వర్క్ + AVSలను భద్రపరుస్తుంది                 |
| కనీస ETH లేదు                         | కనీస ETH లేదు                                               |
| తక్కువ ప్రమాద స్థాయి                  | తక్కువ-నుండి-అధిక ప్రమాద స్థాయి                             |
| విత్‌డ్రా సమయం క్యూపై ఆధారపడి ఉంటుంది | విత్‌డ్రా సమయం క్యూ + అన్‌బాండింగ్ వ్యవధిపై ఆధారపడి ఉంటుంది |

## మనకు రీస్టేకింగ్ ఎందుకు అవసరం? {#why-do-we-need-restaking}

రెండు ప్రపంచాలను చిత్రించండి; ఒకటి రీస్టేకింగ్‌తో మరియు మరొకటి లేకుండా.

## రీస్టేకింగ్ ఎలా పనిచేస్తుంది? {#how-does-restaking-work}

రీస్టేకింగ్‌లో అనేక సంస్థలు పాల్గొంటాయి — వాటిలో ప్రతి ఒక్కటి ఒక ముఖ్యమైన పాత్ర పోషిస్తుంది.

| **పదం**                         | **వివరణ**                                                                                                                                                                                                                                                                                                                                                                                                                   |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **రీస్టేకింగ్ ప్లాట్‌ఫారమ్‌లు** | రీస్టేకింగ్ ప్లాట్‌ఫారమ్ అనేది AVSలు, ETH స్టేకర్‌లు మరియు ఆపరేటర్‌లను కనెక్ట్ చేసే ఒక సేవ. వారు స్టేకర్‌ల కోసం తమ ETHని రీస్టేక్ చేయడానికి వికేంద్రీకృత అప్లికేషన్‌లను మరియు స్టేకర్‌లు, AVSలు మరియు ఆపరేటర్‌లు ఒకరినొకరు కనుగొనగల మార్కెట్‌ప్లేస్‌లను నిర్మిస్తారు.                                                                                                                       |
| **స్థానిక రీస్టేకర్లు**         | తమ స్వంత Ethereum వాలిడేటర్‌లను నడపడం ద్వారా తమ ETHని స్టేక్ చేసే వ్యక్తులు, EigenLayer మరియు ఇతరులతో సహా, ETH వాలిడేటర్ రివార్డులతో పాటు రీస్టేకింగ్ రివార్డులను సంపాదించడానికి తమ స్టేక్ చేసిన ETHని రీస్టేకింగ్ ప్లాట్‌ఫారమ్‌కు కనెక్ట్ చేయవచ్చు.                                                                                                                                                        |
|                                 |                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **లిక్విడ్ రీస్టేకర్లు**        | Lido లేదా Rocket Pool వంటి థర్డ్-పార్టీ లిక్విడ్ స్టేకింగ్ ప్రొవైడర్ ద్వారా తమ ETHని స్టేక్ చేసే వ్యక్తులు తమ స్టేక్ చేసిన ETHని సూచించే లిక్విడ్ స్టేకింగ్ టోకెన్‌లను (LSTలు) పొందుతారు. వారు తమ అసలు ETHని స్టేక్ చేసి ఉంచుకుంటూనే, రీస్టేకింగ్ రివార్డులను సంపాదించడానికి ఈ LSTలను రీస్టేక్ చేయవచ్చు.                                                                 |
|                                 |                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **ఆపరేటర్లు**                   | ఆపరేటర్లు AVSల రీస్టేకింగ్ సాఫ్ట్‌వేర్‌ను నడుపుతారు, ప్రతి AVSకి అవసరమైన వాలిడేషన్ పనులను నిర్వహిస్తారు. ఆపరేటర్లు సాధారణంగా అప్‌టైమ్ మరియు పనితీరు వంటి విషయాలకు హామీ ఇచ్చే వృత్తిపరమైన సేవా ప్రదాతలు. నాన్-ఆపరేటర్ రీస్టేకర్‌ల వలె, ఆపరేటర్‌లు AVSలను భద్రపరచడానికి స్టేక్ చేసిన ETHని ఉపయోగిస్తారు, కానీ ఆపరేటర్‌లు వారి పనికి బదులుగా అదనపు రివార్డులను కూడా పొందుతారు. |
|                                 |                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **AVSలు**                       | ఇవి ధర ఒరాకిల్స్, టోకెన్ బ్రిడ్జ్‌లు మరియు డేటా సిస్టమ్‌ల వంటి వికేంద్రీకృత సేవలు — ఇవి రీస్టేకర్‌ల నుండి భద్రతను పొందుతాయి మరియు బదులుగా టోకెన్ రివార్డులను అందిస్తాయి.                                                                                                                                                                                                                                    |

## రీస్టేకింగ్ నుండి నేను ఎంత సంపాదించగలను? {#how-much-can-i-make-from-restaking}

AVSలు వేర్వేరు రేట్లను అందిస్తున్నప్పటికీ, eETH వంటి లిక్విడ్ రీస్టేకింగ్ టోకెన్‌లు (LRTలు) మీరు ఎంత సంపాదించవచ్చనే దానిపై మీకు ఒక ఆలోచన ఇస్తాయి. మీరు మీ ETHని స్టేక్ చేసినందుకు stETH వంటి LSTలను పొందినట్లే, stETHని రీస్టేక్ చేసినందుకు మీరు eETH వంటి LRTలను పొందవచ్చు. ఈ టోకెన్లు ETH స్టేకింగ్ మరియు రీస్టేకింగ్ రివార్డులను సంపాదిస్తాయి.

**రీస్టేకింగ్‌తో ఉన్న ప్రమాదాలను గుర్తించడం ముఖ్యం. సంభావ్య రివార్డులు ఆకర్షణీయంగా ఉండవచ్చు, కానీ అవి ప్రమాద రహితం కావు.**

## రీస్టేకింగ్ యొక్క ప్రమాదాలు ఏమిటి? {#what-are-the-risks-of-restaking}

| **కేంద్రీకరణ**                                      | కొద్దిమంది ఆపరేటర్లు చాలా రీస్టేకింగ్‌పై ఆధిపత్యం చెలాయిస్తే, వారు రీస్టేకర్లు, AVSలు మరియు రీస్టేకింగ్ ప్లాట్‌ఫారమ్‌లపై కూడా గొప్ప ప్రభావాన్ని చూపగలరు.                                                                             |
