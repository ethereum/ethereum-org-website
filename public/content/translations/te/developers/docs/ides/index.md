[ఇంటిగ్రేటెడ్ డెవలప్‌మెంట్ ఎన్విరాన్‌మెంట్ (IDE)](https://wikipedia.org/wiki/Integrated_development_environment)ని సెటప్ చేసే విషయానికి వస్తే, ఎథీరియంపై అప్లికేషన్‌లను ప్రోగ్రామింగ్ చేయడం అనేది మరే ఇతర సాఫ్ట్‌వేర్ ప్రాజెక్ట్‌ను ప్రోగ్రామింగ్ చేసినట్లే ఉంటుంది. ఎంచుకోవడానికి అనేక ఎంపికలు ఉన్నాయి, కాబట్టి అంతిమంగా, మీ ప్రాధాన్యతలకు బాగా సరిపోయే IDE లేదా కోడ్ ఎడిటర్‌ను ఎంచుకోండి. బహుశా మీ ఎథీరియం డెవలప్‌మెంట్ కోసం ఉత్తమమైన IDE ఎంపిక, మీరు ఇప్పటికే సాంప్రదాయ సాఫ్ట్‌వేర్ డెవలప్‌మెంట్ కోసం ఉపయోగిస్తున్న IDE కావచ్చు.

## వెబ్-ఆధారిత IDEలు {#web-based-ides}

మీరు [లోకల్ డెవలప్‌మెంట్ ఎన్విరాన్‌మెంట్‌ను సెటప్ చేయడానికి](/developers/local-environment/) ముందు కోడ్‌తో ప్రయోగాలు చేయాలని చూస్తున్నట్లయితే, ఈ వెబ్ యాప్‌లు ఎథీరియం స్మార్ట్ కాంట్రాక్ట్ డెవలప్‌మెంట్ కోసం ప్రత్యేకంగా రూపొందించబడ్డాయి.

**[Remix](https://remix.ethereum.org/)** - **_అంతర్నిర్మిత స్టాటిక్ అనాలిసిస్ మరియు టెస్ట్ బ్లాక్‌చైన్ వర్చువల్ మెషీన్‌తో కూడిన వెబ్-ఆధారిత IDE_**

- [డాక్స్](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[ChainIDE](https://chainide.com/)** - **_క్లౌడ్-ఆధారిత మల్టీ-చైన్ IDE_**

- [డాక్స్](https://chainide.gitbook.io/chainide-english-1/)
- [సహాయ ఫోరమ్](https://forum.chainide.com/)

**[Replit (Solidity స్టార్టర్ - బీటా)](https://replit.com/@replit/Solidity-starter-beta)** - **_హాట్ రీలోడింగ్, ఎర్రర్ చెకింగ్ మరియు ఫస్ట్-క్లాస్ టెస్ట్‌నెట్ మద్దతుతో ఎథీరియం కోసం అనుకూలీకరించదగిన డెవలప్‌మెంట్ ఎన్విరాన్‌మెంట్_**

- [డాక్స్](https://docs.replit.com/)

**[Tenderly శాండ్‌బాక్స్](https://sandbox.tenderly.co/)** - **_Solidity మరియు JavaScript ఉపయోగించి బ్రౌజర్‌లోనే స్మార్ట్ కాంట్రాక్ట్‌లను వ్రాయడానికి, అమలు చేయడానికి మరియు డీబగ్ చేయడానికి వీలు కల్పించే వేగవంతమైన ప్రోటోటైపింగ్ ఎన్విరాన్‌మెంట్_**

**[EthFiddle](https://ethfiddle.com/)** - **_మీ స్మార్ట్ కాంట్రాక్ట్‌ను వ్రాయడానికి, కంపైల్ చేయడానికి మరియు డీబగ్ చేయడానికి మిమ్మల్ని అనుమతించే వెబ్-ఆధారిత IDE_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## డెస్క్‌టాప్ IDEలు {#desktop-ides}

చాలా వరకు స్థిరపడిన IDEలు ఎథీరియం డెవలప్‌మెంట్ అనుభవాన్ని మెరుగుపరచడానికి ప్లగిన్‌లను నిర్మించాయి. కనీసం, అవి [స్మార్ట్ కాంట్రాక్ట్ భాషల](/developers/docs/smart-contracts/languages/) కోసం సింటాక్స్ హైలైటింగ్‌ను అందిస్తాయి.

**Visual Studio Code -** **_అధికారిక ఎథీరియం మద్దతుతో ప్రొఫెషనల్ క్రాస్-ప్లాట్‌ఫారమ్ IDE_**

- [Visual Studio Code](https://code.visualstudio.com/)
- [కోడ్ నమూనాలు](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**JetBrains IDEలు (IntelliJ IDEA, మొదలైనవి) -** **_సాఫ్ట్‌వేర్ డెవలపర్‌లు మరియు బృందాలకు అవసరమైన సాధనాలు_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ Solidity](https://github.com/intellij-solidity/intellij-solidity/)

**Remix డెస్క్‌టాప్ -** **_మీ లోకల్ మెషీన్‌లో Remix IDEని అనుభవించండి_**

- [డౌన్‌లోడ్](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)
