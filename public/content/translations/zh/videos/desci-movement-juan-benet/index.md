#### 科学与进步简介 (0:10) {#introduction-to-science-and-progress-010}

大家好。我是胡安 (Juan)。我今天来谈谈去中心化科学 (DeSci)。我想谈谈我们如何利用去中心化科学 (DeSci) 为科学提供资金、组织和开放科学。首先，这是我们将要讨论的内容：我将简要介绍一下一般意义上的科学，然后我将讨论什么是去中心化科学 (DeSci) 运动，接着讨论我们如何为科学公共物品提供资金。然后是去中心化科学 (DeSci) 如何组织围绕科学的人员、项目和工作。接下来，我想谈谈开放获取和可重复科学，最后我将以行动号召结束。所以这会非常快。我们有很多内容要讲，所以我讲得会比较快。

首先，我想说的是，在过去的几个世纪里，我们取得了巨大的进步。几乎所有的人类指标都在改善。无论用我们能想到的任何标准来衡量，人类的生存状况都在显著改善，而实现这种进步的一个重要部分就是科学事业。通过扩展我们的认知，通过将我们所知道的知识转化为解决各种问题的技术和方案，我们已经能够使世界上很大一部分人摆脱贫困。我们能够养活无数人，为每个人提供住所，治愈各种疾病等等。多亏了科学，我们取得了巨大的进步。

科学是一项庞大的事业，包含许多不同的子领域和许多不同的知识领域。你可以想到任何特定的领域和任何研究领域，科学都是其中的重要组成部分。归根结底，科学就是发现事物的过程。创造新知识和结合新概念的过程。想想科学方法。费曼 (Feynman) 有一句名言：“如果它与实验不符，那它就是错的。”这就是科学的关键。

你可以把科学看作是一项涉及全球人类的大规模事业。这里有各种各样的努力和系统。你可以想到地球上各种大学、各种研究小组、不同领域和期刊。围绕综合我们所知、提出新想法、将这些想法转化为研究项目、将其转化为对假设的实际测试，以及收集数据以测试假设是否正确的过程，存在着许多不同的活动。一直到将这些结果写成某种论文，然后由科学界进行同行评审，添加到知识树中，进而扩展我们的认知。

也许故事到此为止，或者也许后来发现，实际上，那是不可重复的，我们必须推翻它。或者实际上，那是正确的，但它为大量其他新知识打开了大门。所以这是一个高度动态的领域，有着许多不同的活动。

现在，科学面临着很多问题。科学事业存在各种各样的问题。尽管它一直是进步的巨大引擎，但它也出现了各种各样的问题。特别是，各个领域都缺乏资金。与此同时，尽管缺乏资金，但总体上仍有大量资金投入科学。人们感觉资金不如以前那么耐用了，科学的投入产出比不再像以前那么高。在许多领域，获取资助的竞争都过于激烈。

一旦研究完成并发表，只有一小部分能够被重复。因此，有各种各样的科学研究被发表、被接受并被认为是正确的，结果后来却发现其中很大一部分实际上无法被重复。所以存在着巨大的可重复性危机。甚至有些科学发现的产物正在丢失。想想与某些结果相关的实际论文、代码或数据从我们的知识库中丢失。因此，围绕科学存在着各种需要解决的问题，而这正是去中心化科学 (DeSci) 的部分意义所在。它正在解决一系列此类问题，虽然不是全部，也不是彻底解决，但去中心化科学 (DeSci) 社区正在努力解决其中的许多问题。

#### 去中心化科学 (DeSci) 运动 (5:11) {#the-desci-movement-511}

那么什么是去中心化科学 (DeSci)？去中心化科学 (DeSci) 是一项利用 Web3 技术和工具来改善科学的运动。想象一下，能够利用哈希链接、区块链和智能合约的所有魔力，创建能够改善全球各个领域科学研究方式的系统和结构。

这里有许多不同的重点领域。想象一下，能够拥有开放获取的论文和数据公共物品，拥有更好的可重复实验，并且能够更好地组织实验室和团队。想象一下，创建像 DAO 这样的结构，使研究小组能够组建和组织、筹集资金，并向参与者分配奖励。还有全新的资金结构，比如 IPNFT。还有带有奖励的同行评审协议。从历史上看，同行评审一直是一种掠夺性的情况，学者们投入大量的时间和精力对所有工作进行同行评审，而期刊实际上并不为这些劳动支付任何报酬。现在有各种各样的新激励结构正在被试验。

这是一个相当新的运动。它已经存在了一段时间。当我创立 IPFS 时，在去中心化科学 (DeSci) 这个概念出现之前，它就算是一种去中心化科学 (DeSci) 运动。我创立 IPFS 的目标是让人们能够更好地分发数据，以用于科学研究。所以这些想法中的很多都是该项目核心的一部分。然而，在过去的一两年里，这场运动势头强劲，出现了许多新组织。这张版图在过去一年里扩大了两三倍，这真是太棒了。

现在有几个团体在做去中心化生物技术融资，比如 VitaDAO、Molecule 等团体。有许多组织试图提出新的科学资助结构。有几个 DAO 本身就是试图进行研发的科学组织。有几个基金会和机构正在支持大量的去中心化科学 (DeSci) 工作，或者以某种方式将自己与去中心化科学 (DeSci) 联系起来。有许多团体在探索不同的出版方式、许多科学 NFT 等等。这个社区在过去一两年里发展迅速。

现在也有很多不同的聚会和会议将这些社区聚集在一起。比如 DeSci Day、DeSci 柏林 (Berlin)、Gitcoin 社区的 Schelling Point 以及 Funding the Commons。这些会议汇集了许多关于去中心化科学 (DeSci) 的讨论。

#### 为公共物品提供资金 (10:40) {#funding-the-commons-1040}

让我们谈谈为公共物品提供资金。也许你们中的一些人见过我过去用过的关于创新鸿沟的图表。在科学到技术的转化中，去中心化科学 (DeSci) 部分主要集中在左侧——仅仅是科学部分——试图想出更好的激励结构和更好的协调团队的方法，以产生更好的科学成果。值得注意的是，全球研发资金总额从一个角度来看是巨大的，但从另一个角度来看，并没有那么大，而且在过去几十年里也没有发生太大变化，尽管我们正在构建的技术的吞吐量和成果已经取得了巨大的增长。

这些资金规模并非区块链无法企及。想想美国非国防研发，每年大约在 700 亿美元左右。这确实很多，但并非庞大到不可思议。单独看美国国家科学基金会 (NSF)，每年大约 100 亿美元，这完全可以通过区块链实现。想想加密货币领域，根据你观察的时间点，其规模大约在 1 到 3 万亿美元之间。

想象一下，如果区块链每年将其供应量的一部分用于研发。想象一下，每年拿出 1% 的 Filecoin、以太坊或比特币投入研发。你就会开始达到在民族国家层面资助科学的数字范围。如果加密货币再增长一两个数量级，加密货币将能够以民族国家的规模资助研发和科学，想想都觉得疯狂。因此，在我们达到那个阶段之前，弄清楚结构并找到良好的资金途径将是非常棒的。

当你开始分解这些机构的资金时，你会遇到各种各样的问题。某些领域受到的关注太少，或者项目本身存在反向激励，或者竞争过于激烈，导致科学家们花费大量时间仅仅为了撰写资助申请。在新冠疫情期间，有一项名为 Fast Grants 的计划，同样的效果在 Impetus Grants 中得到了复制，这些计划构建了一个非常快速的资助项目。他们能够以科学家投入时间的一小部分，发放 2 万到 20 万美元不等的资助。

在一项针对申请这些资助的科学家的调查中，他们指出了他们通常花多少时间申请资助。想想看，一个科学家 25% 到 50% 的时间仅仅用于说明他们在做什么以及申请各种资助。这有点疯狂。理想情况下，你希望科学家把绝大部分时间花在思考他们的工作、提出新想法和分析工作上。还有一种影响是，资助项目限制了人们最终探索的内容。许多科学家有更具雄心的研究想要追求，但他们最终却陷入了追求其他影响力远不及此的工作中，因为他们要符合资助项目的限制。

Web3 公共物品来拯救！有很多不同的团体。当然，这仍然很小；与全球科学研发资金相比，Web3 运动非常小，但如果我们能把结构弄对，把激励机制调整好，并证明它是有效的，那么我们就可以随着加密货币将其扩大几个数量级。我们应该探索许多不同种类的科学过程资金：不同的资助项目、影响力证书、影响力市场等等。Funding the Commons 社区一直在尝试各种不同的机制。

例如，像 VitaDAO 这样的团体正在创建一种数据结构，向团体发放资助以换取数据、知识和知识产权 (IP)。然后，他们将这些 IP 捆绑成具有法律效力的 IPNFT，将 IP 权利授予生物技术公司，并资助这些生物技术公司，目的是通过它们的成功获得投资回报。我倾向于称之为基础发展基金，通过本身不是公司的实验室开展重要工作，产生 IP，然后再资助公司。像 Molecule 这样的团体正在为这项工作的开展创建市场。

影响力证书是另一种代表追溯性资金的迷人结构。它们使参与者一旦取得某种影响力，就可以围绕该影响力铸造证书，并在市场上出售给任何想要申领该影响力的人。这使得投机市场得以出现，跨越时间形成闭环，从而追溯性地资助极其重要的工作。这至关重要，因为很多时候，只有在工作完成很久之后，你才会意识到某件事有多么宝贵。

#### 组织人员与数据 DAO (15:28) {#organizing-people-and-data-daos-1528}

现在简要谈谈关于组织人员的想法。过去，GitHub 在帮助组织科学发现方面取得了巨大成功。整本教科书和整个领域都是通过 GitHub 发展起来的。许多团体利用 GitHub 围绕问题、代码协作和版本控制的基本原语来组织实践和科学社区。但那里不完整的是，你没有办法创建进行研究、处理资金或向贡献者支付报酬的组织。

有一些有趣的实验，比如 LabDAO，创建实验室团队，团体可以在其中组建、筹集资金并进行分配。你能够对参与者的不同贡献水平进行编码，以公平地奖励他们。还有一些更具雄心的项目，围绕在更大网络中的参与者之间进行信用分配，在不同的耦合团队之间传播奖励。

有些团体正在试验同行评审协议，观察同行评审系统的经济学和动态，以激励工作并适当地奖励其发生。一个名为 Ants Review 的协议已经在做这件事，你可以将其与梅塔马斯克 (MetaMask) 一起使用。Gitcoin Grants 开创了大量可在此处使用的工作，并且已经为希望以这些方式组织的参与者提供工具支持。

这里真正关键的组成部分之一是通过哈希链接内容。你可以冻结一捆信息，获取内容寻址的哈希链接，并引用事物。这是你在文献中想要的核心原语。当你有一篇论文对另一篇论文的引用，或者从一篇论文对其数据或代码的引用时，CID 正是你想要的东西。想象一下，能够通过版本控制冻结整个文献，并冻结再次运行这些实验所需的所有重要数据集和代码。许多团体正在探索这一点，提出了通过 IPFS 进行同行评审和科学发展的不同方法。

你可以考虑将这种活动和数据生成与一种称为数据 DAO 的东西捆绑在一起。与我之前提到的已经开始运作的 DAO 不同，数据 DAO 是非常新的事物。想象一个能够收集、策划、转换和计算数据，并管理这些数据随着时间的推移如何被使用、如何被货币化以及如何被共享的团体。

最后谈谈关于开放获取和可重复科学的一些注意事项。IPFS 已经被大量用于许多种类的开放科学工作。它已经实现了开放大量科学访问的梦想，支持分布式维基百科副本、海量论文档案和数据集。

#### 开放获取、可重复科学与行动号召 (20:40) {#open-access-reproducible-science-and-call-to-action-2040}

我们在完全可重复性方面还没有完全达到目标。这是一个需要更多工作的领域，但许多人已经进行了思考。围绕使用 IPFS 的标准可重复性来冻结所有资产并构建完全可重复的管道，有非常好的规范和想法。你可以召唤回过去特定的实验，带回完全冻结的虚拟机或容器，重新运行所有数据管道，并验证实验是否正确。

还有一个全新的角度，即以去中心化科学 (DeSci) 为导向的方式进行数据科学本身，其中笔记本、数据分析和工件使用由 Web3 驱动的应用程序。像 Jupyter 笔记本、IPython 笔记本和 Wolfram 笔记本等已经与 CID 结合。我认为随着 Filecoin 网络的巨大增长，这在未来将得到极大的增强。Filecoin 网络拥有大量与计算相结合的存储——存储提供商在数据旁边拥有大量的 GPU。这些将在明年连接起来，具备围绕这些数据发布计算管道的能力。想象一下，为科学家生成一个大规模进行数据科学的平台，利用 Web3 计算平台进行信息的寻址和存储以及计算，创建完整的数据科学端到端管道。

最后，一个简短的行动号召。科学是进步的引擎。通过扩展我们的认知，我们能够生产更多的技术并改善我们的生活。如果我们能改善科学家的生活，使他们的工作更容易，加速他们的发展，降低他们的成本，并使他们能够花更多的时间解决问题而不是撰写资助申请，那么我们所有人都能以独特的方式更快地推动社会进步。

去中心化科学 (DeSci) 运动需要你。考虑尝试新的资金机制，构建开放获取和开放科学工具，或者研究公共数据集。考虑加入去中心化科学 (DeSci) 团队或 DAO。探索这些社区，我希望在这场运动中看到你。非常感谢，回见。
