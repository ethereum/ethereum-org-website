**Linda Xie** 在 Devconnect 上关于去中心化社交媒体的演讲。Linda 探讨了为什么用户和创作者能从开放社交图谱中受益，内嵌钱包和小程序如何在 Farcaster 上创造全新体验，采用过程中面临的真实挑战，并与观众进行了问答互动。

#### 中心化社交的问题 (0:00) {#the-problem-with-centralized-social-000}

我已经在加密货币领域工作了 11 年，目前在 Farcaster 团队。在此之前，我一直在 Farcaster 上进行构建。因此，我对加密货币和去中心化社交及其能为社会带来的改变充满热情。

我认为首先梳理一下当前传统社交媒体存在的问题会很有帮助。我想我们都经历过类似的问题：你在一个平台上起步，然后不得不在另一个平台上创建账户，一切完全从零开始。在这种情况下，你的受众实际上属于平台本身。当你被平台踢出，或者平台最终被某个国家封禁时，你将失去所有受众的分发渠道。对于用户和内容创作者来说，能够触达自己的受众是非常重要的。

最后，如果账户和数据归平台所有，这意味着一旦公司倒闭，你将无法再访问这些信息。我们已经屡次看到这种情况发生。在美国，TikTok 面临被封禁的可能，许多创作者对此感到担忧。在许多国家，这些平台甚至会自我封禁。所以这是人们在日常生活中切实经历的问题。

#### 开放社交图谱 (1:30) {#open-social-graphs-130}

去中心化社交的好处之一是，你可以随时访问社交图谱。使用这些数据无需任何许可。最棒的是，你可以看到底部这里有一个开放社交图谱。所有这些数据都可以被任何客户端、任何开发者、任何平台提取，你可以利用这些数据在该平台之上进行构建。

一个很好的例子——你有 Farcaster 应用，同时也有 Coinbase 构建的 Base 应用。你的粉丝数量差不多，可能会有细微差异，因为每个客户端都有自己的垃圾信息过滤机制。但我能够在 Farcaster 上积累粉丝，然后立即开始使用 Base 应用，而无需从头开始。当你体验到这一点时，那真是一个神奇的时刻。对于所有其他 Farcaster 客户端（如 Zapper、Uno 等）也是如此。

当你决定使用这个平台时，你不再需要从零开始。你能够保留自己的受众。你只需发布一次，就能在网络上的多个客户端获得分发。你可以是一名 Farcaster 用户，同时随着 Base 应用引导更多用户和创作者加入，你也能通过它获得更多分发。这终究是最好的用户体验。现在客户端正在争夺这些用户，因为切换成本非常低——你不需要从头开始，你能保留你的受众，因此客户端必须通过竞争来打造更好的用户体验。

#### 对构建者的益处 (3:13) {#benefits-for-builders-313}

然后是构建者方面，构建者经常被平台踢出。我们在 Twitter 上看到了这一点——它踢出了许多加密应用和构建者，这对他们的公司来说是一个单点故障。Bankr 最近在同一天被 Twitter 和 Telegram 封禁。幸运的是，几天后他们得以恢复，但这确实表明存在这些单点故障。如果你是一名构建者，投入了所有时间和资源来构建你的应用，那么分发渠道被瞬间切断将是一次极其痛苦的经历。

我们过去也看到过，像 Reddit 这样的应用更改了其 API 定价，突然之间你的业务运营成本变得极其高昂。过去有许多应用因为无法承担 Reddit 新的 API 定价而被迫关闭。这在社区中引起了轩然大波。这种事情在这些平台上屡次发生。他们可以出于任意理由将你踢出。他们甚至不需要说明踢出你的确切原因。如果他们仅仅视你为竞争对手，就可以将你从平台上移除。我亲自与几个直接经历过这种情况的团队交谈过。

去中心化社交真正令人惊叹的一点是，就像你作为用户可以在网络上发布一次内容一样，你也可以在无需许可的情况下在网络上进行构建。开发者能够无需许可地进行构建，而不必去每个平台询问：“嘿，我可以在你们的平台上构建吗？”今天早些时候我刚在 Farcaster 的展位上，人们问如何将自己加入白名单以便在 Farcaster 上进行构建，我可以直接告诉他们，这是无需许可的——你可以使用任何这些数据，直接将其整合到你的应用中，而且你不需要与团队沟通。这就是未来事物应有的构建方式。

你可以在这里看到——我发布了一个名为 Miniword 的填字游戏。每天人们都在上面竞争，就像玩《纽约时报》的填字游戏一样。我在 Farcaster 上发布了一次，它也会同步到 Base 应用和所有其他客户端。作为构建者，你能够获得这种分发量的提升。如果我点击打开这个小程序，我就可以开始与它互动，而且因为社交数据是可以无需许可地利用的，你可以拉取网络上不同人的用时，人们可以为了最高分而竞争。这变成了一种社交体验，而不仅仅是你一个人在玩拼图，你可以通过利用社交图谱获得更多的病毒式传播。

#### 加密货币与内嵌钱包 (6:30) {#crypto-and-embedded-wallets-630}

去中心化社交已经存在——我们在 Bluesky 上看到了这一点，随着平台在不同国家被封禁，他们在吸引用户方面做得非常出色。然而，真正特别的是当你将加密货币与去中心化社交结合起来时，因为你现在正在创造在传统平台上永远无法获得的体验。这正是我们团队正在大力投入的方向。

Farcaster 上的每个用户默认都有一个加密地址。在 Farcaster 应用内，你有一个内嵌钱包，用户可以轻松地向网络上的其他人发送和接收资金。显然，这是加密货币，所以你只需粘贴一个地址，就可以向世界各地的任何人发送资金。这就是加密货币如此特别和独特的原因。通过将其与开放社交图谱相结合，我们认为这将引领去中心化社交领域的下一波增长。

用户还可以通过他们的内嵌钱包在链上与小程序互动。你可以在这里看到一个在 ETH Global 上构建的 eSIM 小程序——在阿根廷或任何其他国家，你可以购买 eSIM，使用你的内嵌钱包支付 USDC，这是一种非常无缝的体验。你现在可以构建以前从未存在过的应用。

你还可以让代理（agents）在信息流中互动。如果你真的相信这个充满代理的未来，即许多代理相互互动，那么加密货币自然是它们用来相互发送资金的媒介。你不需要协调地址或某人如何获得报酬——你自动知道用户的地址，你拥有所有信息，无论他们住在世界上的哪个地方都无妨。你能够直接与他们互动。我坚信加密货币、代理和开放社交图谱将发挥更大的作用。

#### 微打赏 (8:36) {#micro-tipping-836}

关于加密货币中开放社交图谱的独特性，我最喜欢的例子之一就是微打赏。在加密货币的极早期，人们总是谈论微打赏会有多棒——如果你是一个发布优质内容的创作者，能够向创作者发送极少量的资金。我曾使用过这些早期的比特币打赏应用。但早期的问题在于，发送极少量的资金实际上非常缓慢且昂贵。

现在，得益于 L2（二层网络）的低廉和快速，你有能力发送一美分甚至更低金额的交易。像 Tipin 和 Noise 这样的应用使你能够为每一次互动自动打赏——关注、转发（recast）、点赞帖子、回复。你可以为你想要给出的金额设置一个授权额度。就我而言，我设置为每次点赞别人的帖子就打赏一美分，这在后台是自动进行的。每次我在网络上点赞内容时，我都会自动向该用户发送一美分。

你可以看到我的钱包活动历史——我正在向网络上的其他用户发送资金，同时我也在接收其他设置了这些授权额度的用户发来的资金。甚至还有低于一美分的交易，这是一个相当疯狂的概念。这些是生活在世界各地的人们。如果你是加密货币的新手并加入了 Farcaster，你默认就拥有一个钱包。一旦你开始发布高质量的内容，人们就可以开始自动打赏你，你无需购买加密货币就能积累余额。我与许多用户交谈过，他们说这实际上是他们通过创作内容或与小程序互动赚取的第一笔加密货币。

#### 小程序与创意工具 (11:01) {#mini-apps-and-creative-tools-1101}

我最喜欢的小程序之一是 Atown 及其团队构建的 Emerge，它很好地展示了内嵌钱包、加密交易和开放社交图谱的使用。它使创作者能够组合一个提示词（prompt），将你的个人资料图片转换为该提示词的某种版本。这里有一个例子——网络上非常受欢迎的创作者 Peachy 写了一个提示词，将个人资料图片转换为塔罗牌。我能够使用我的个人资料图片生成一张图像，并分享这个小程序，以便其他感兴趣的人也能生成他们自己的图像。这里有排行榜，显示哪些提示词表现得非常好。如果你是一个有创意的人，你可以创建自己的提示词并开始赚钱——我为这次提示词交易支付了 25 美分，其他 418 名用户也生成了它，所以 Peachy 因为创作这个内容而获得了收入，而且它可以在信息流中病毒式传播。

#### Warplet 现象 (12:41) {#the-warplet-phenomenon-1241}

由于最近发生的一次病毒式传播，我们实际上迎来了 Farcaster 上日活跃用户数最高的一天。首先，我们有一个名为 Warplet 的 Farcaster 非官方吉祥物。Farcaster 的联合创始人 Dan 创造了它。团队——包括我在内——可能觉得它作为官方吉祥物不够可爱，所以他把它发布到了社区，并说人们可以随意使用它。

一位名叫 Angel 的开发者采纳了这个想法，并决定构建一个小程序，你可以在其中铸造一个混合了你自己个人资料图片 DNA 的 Warplet 生物版本。NishProf 分享了她用个人资料图片定制的 Warplet——随后有超过 49,000 个账户生成了他们的 Warplet。它彻底火了。Angel 在短短几天内就通过创建这个小程序赚了将近 100,000 美元。这确实表明，当你拥有一个任何人都可以轻松进行交易的内嵌钱包时，作为构建者，你可以通过构建在社交信息流中病毒式传播的应用来赚钱。

同样非常酷的是看到了不同的混音（remix）文化。构建者们说：“好吧，现在你们每个人都有了自己的 Warplet，所以让我们在上面构建游戏和有趣的体验吧。”出现了“最丑的 Warplet 是什么？”以及“如果你把这两个结合起来，Warplet 宝宝会是什么样子？”的排名。所有这些有趣的游戏都是在网络上创建的，这表明当你利用社交图谱时，作为应用开发者，你可以打造出更具吸引力的体验。

#### 用于社会公益的小程序 (15:02) {#mini-apps-for-social-good-1502}

这是一个小程序的例子——实际上是团队中的 Horsefax 构建的。我认为这非常重要——你可以支持 Roman Storm 的法律辩护基金。我们使用了 Daimo Pay，你可以使用任何代币支付并为法律辩护基金捐款。因为它是一个内嵌钱包，你永远不会被踢出去使用另一个应用。你可以立即在你的信息流中分享，告诉你的粉丝你捐款了。然后他们可以发现这个应用并继续捐款。这是一种在每个人自己建立的粉丝和受众中进一步分享应用的机制。

这是另一个类似的例子——Seed Club 的 Nicholas 构建的一个众筹项目，就像 GoFundMe 或 Kickstarter 平台一样。你可以看到捐款的不同用户。你可以点击你的内嵌钱包，直接通过 Farcaster 应用捐献资金。你永远不需要跳出应用。创作者筹集的资金远远超过了他们的目标，你可以在你的信息流中分享，这样你所有的粉丝也能发现这个小程序。

#### 去中心化社交的挑战 (16:41) {#challenges-of-decentralized-social-1641}

所有这些都非常棒，去中心化社交听起来很好，但我想非常坦诚地谈谈面临的挑战。需要注意的一点是，它的规模很小——相对于所有这些拥有数十亿和数亿用户的大型传统社交平台来说，非常小。

然而，我真的想强调——我已经在加密货币领域工作了 11 年，从比特币的早期就开始了。你可以拿黄金之类的东西来比较——看看比特币已经发展到了什么程度。你也可以看看去中心化金融 (DeFi) 以及所有的总锁仓量 (TVL) 和兑换交易量，看看它增长了多少。这一切在社区、构建者和用户的共同努力下都是非常有可能实现的。我坚信社交将在加密货币领域经历另一个增长阶段，因为这完全合乎逻辑。这对用户来说是更好的体验，对构建者来说也是更好的体验，而且你现在可以创造以前根本不存在的东西。随着加密货币变得更加主流，我们希望能与之共同成长。

具体到 Farcaster，我们发现人们对内嵌钱包产生了强烈的共鸣，所以我们正在大力投入这些功能。引导非加密货币参与者是任何加密应用都在经历的事情——如果你是一个完全的新手，必须了解什么是钱包以及如何保证安全。与早期相比，我们已经改进了很多，但与传统的消费者体验相比，仍有很大的提升空间。

垃圾信息也是一个问题。这在中心化社交平台上也存在——你可能在 Twitter 上体验过，辨别什么是真实账户有多难。在 Farcaster 层面，你可以拥有在协议层面帮助客户端进行过滤的垃圾信息服务，或者客户端可以进行自己的垃圾信息过滤。

然后是内容审核——这也是传统社交平台面临的一个问题和挑战，即确保用户获得良好的体验并发现他们真正感兴趣的内容。这些都是我们一直在持续努力解决的问题。

#### 为什么是现在？ (19:48) {#why-now-1948}

去中心化社交已经存在了一段时间，但为什么是现在爆发？我认为这是所有这些基础组件汇聚在一起的结果。首先，你拥有任何人都可以使用和利用的开放社交图谱。我经常与以前甚至不认识的团队交谈——他们只是在构建 Farcaster 小程序，因为他们能够在没有我们的情况下利用这些数据。

L2 是其中的关键部分，使交易变得非常便宜和快速——你可以看到像我们之前分享的微打赏这样的体验。内嵌钱包也是关键——我真的想强调，在使用消费者应用时，它们的体验有多好。我们很多人都经历过被踢出到钱包应用，然后再被带回来，有时甚至会失败。有了内嵌钱包，你就可以留在应用体验中，流失率会大大降低。

像小程序和代理这样的新构建模块是你能够使用的全新事物，我们以前未必大规模地见过。而在加密货币之外发生的一个大趋势是人工智能变得有多好——你现在可以使用提示词来创建应用。这确实降低了人们实际构建的成本和门槛。我们有像 Neynar 这样的服务，使你无需成为开发者就能创建 Farcaster 小程序。

这一切将如何发展——我认为电子邮件是一个很好的类比。当你发送电子邮件时，你不需要创建一个 AOL 账户来与拥有 AOL 邮箱的人交谈。你可以使用 Gmail、Yahoo、Outlook 或任何你想要的邮箱，你们都能够在协议层面相互通信。我认为这就是去中心化社交应该呈现的方式——所有这些网络数据、所有这些社交图谱都在协议层面，每个客户端都可以在此基础上构建针对其用户群的特定功能。他们可以根据用户真正喜欢的内容，拥有不同的算法和不同的功能。

Farcaster 越来越多地将时间花在交易功能上；如果你对此不感兴趣，Base 应用和 Uno 在展示社交功能方面非常出色。从长远来看，这才是社交应有的样子。我们还有很长的路要走，但我真的认为这就是社交未来应该发展的方向。

#### 问答 (23:47) {#qa-2347}

**主持人：** 非常感谢你，Linda。我们有一些来自观众的问题。我认为真正酷的是，在现实世界中，我可以走到我的社区外面，有很多事情我可以自由地做。数字社区并不总是如此，这就是为什么去中心化社交的论点非常引人注目，尤其是随着我们越来越多的生活转移到线上。目前存在一种转变，即中心化社交媒体平台上的人们需要转移到去中心化社交媒体平台。根据你的经验，我们如何才能简化这种迁移？

**Linda Xie：** 这是一个好问题。我认为我们有责任创造比现有传统体验更好的体验。加密货币将实现你在传统社交平台上根本无法做到的事情。你来这里开始赚取加密货币，你体验所有这些构建在加密社交轨道上的加密应用。所以这就是我们吸引人的方式——如果你是一个非常优秀的构建者并且你想要分发，加密货币将是你赚钱的方式，而社交将是你的应用获得分发的方式。作为用户，你能够因为发布好内容而收到大量的打赏，这是你在传统平台上得不到的。我只是认为你需要创造更好的体验。

**主持人：** Farcaster 最近收购了 Clanker，这提供了一种替代的收入来源。你如何看待像 Farcaster 这样的去中心化社交网络中的广告？

**Linda Xie：** 每个客户端都可以自由决定他们的变现方式。你实际上可以作为一个客户端并决定使用广告，但其他客户端可以决定那不是他们想走的路线，并以不同的方式变现。具体到 Farcaster 应用，我们有 Farcaster Pro——人们可以每年支付 120 美元以获得额外功能。但任何客户端都可以完全倾向于广告。Base 应用甚至有一些广告格式，特别是随着对 Spindle 的收购。它的美妙之处在于每个客户端都可以做他们想做的事，但我们没有在协议层面内置广告——客户端可以决定是否开启它。

**主持人：** 我们现在在以太坊基金会 (EF) 经常提到的一个关键术语是反身性（reflexivity）——我们如何对最终用户的反馈做出反应。Farcaster 具有最强的反身性，因为你拥有真正的日常社交媒体参与者、进行钱包交易的用户和构建者。在从社区获取反馈时，你遵循的原则是什么？
