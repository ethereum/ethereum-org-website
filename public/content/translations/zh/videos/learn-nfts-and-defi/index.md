#### 同质化与非同质化 (0:00) {#fungible-vs-non-fungible-000}

让我们从“同质化 (fungible)”这个词开始。同质化意味着资产的各个单位是可以互换且彼此无法区分的。同质化资产的一个好例子是货币。一张五美元钞票的价值总是等于任何其他五美元钞票。你并不真正关心你收到的是哪一张特定的五美元钞票，因为它们的价值都是一样的。

然而，当谈到非同质化资产时，每个单位都是独一无二的，不能直接被另一个单位替换。一个很好的例子是机票。尽管机票看起来可能很相似，但每张机票都带有不同的乘客姓名、目的地、出发时间和座位号。试图将一张机票兑换成另一张机票可能会导致一些严重的问题。

另一个例子是集换式卡牌。尽管它们看起来可能很相似，但每张卡牌都有不同的属性。生产年份或卡牌的保存方式等因素都会产生影响。非同质化物品的一个极端例子是艺术品——例如，一幅画通常只创作一份原件。

#### NFT 的属性 (2:13) {#properties-of-nfts-213}

既然我们知道了“非同质化”的含义，让我们来看看 NFT 最常见的属性。

- **独一无二** — 每个 NFT 都有不同的属性，这些属性通常存储在代币的元数据中
- **可证明的稀缺性** — NFT 的数量通常是有限的，极端的例子是只有一份副本；代币的数量可以在区块链上进行验证
- **不可分割** — 大多数 NFT 不能分割成更小的面额，因此你不能购买或转账 NFT 的一小部分

与标准代币类似，NFT 也保证了资产的所有权，易于转账，并且具有防伪性。

#### 代币标准：ERC-20、ERC-721 和 ERC-1155 (3:17) {#token-standards-erc-20-erc-721-and-erc-1155-317}

尽管 NFT 可以在任何支持智能合约编程的区块链上实现，但最著名的标准是以太坊上的 ERC-721 和 ERC-1155。在我们深入探讨 NFT 标准之前，让我们快速回顾一下 ERC-20，因为这有助于进行比较。

**ERC-20** 是在以太坊区块链上创建代币的著名标准。例子包括 USDT 或 DAI 等稳定币，以及 LEND、YFI、SNX 和 UNI 等去中心化金融 (DeFi) 代币。ERC-20 允许创建同质化代币——在该标准下创建的所有代币都是完全无法区分的。无论你是从朋友那里还是从交易所收到 USDT 都没关系；每个代币的价值都是相同的。

**ERC-721** 是创建非同质化代币的标准。它允许创建生成具有不同属性的可区分代币的合约。一个常见的例子是著名的 CryptoKitties（迷恋猫）——一款允许收集和繁殖虚拟小猫的游戏。

**ERC-1155** 是非同质化代币创建的下一步。该标准允许创建同时支持同质化和非同质化代币的合约。它是由专注于基于区块链的游戏项目 Enjin 创建的。在《魔兽世界》等许多游戏中，玩家可以同时持有非同质化物品（剑、盾牌、盔甲）和同质化物品（如金币或箭矢）。ERC-1155 允许开发者定义同质化和非同质化代币，并决定每种代币应该存在多少个。

#### NFT 用例 (5:28) {#nft-use-cases-528}

除了 CryptoKitties 之外，还有其他几款利用 NFT 的热门游戏，例如 Gods Unchained 和 Decentraland。Decentraland 是一个有趣的例子，因为玩家可以购买数字土地，这些土地以后可以转售，甚至可以用作游戏内的广告空间。

其他例子包括数字艺术市场，如 Rarible 和 SuperRare，甚至像 OpenSea 这样的市场聚合器。另一个可以表示为 NFT 的稀缺物品的例子是域名——例如，带有 .eth 后缀的以太坊域名服务 (Ethereum Name Service) 和带有 .crypto 后缀的 Unstoppable Domains。

某些 NFT 可能极其昂贵。最昂贵的 CryptoKitty（名为 Dragon）在 2017 年底以 600 个以太币的价格售出——当时价值约 17 万美元。像 exchange.eth 这样稀缺的域名价值可能高达 50 万美元以上。

#### NFT 作为 DeFi 中的抵押品 (6:48) {#nfts-as-collateral-in-defi-648}

谈到 DeFi，NFT 可以为去中心化金融释放更多潜力。目前，绝大多数 DeFi 借贷协议都是有抵押的。最有趣的想法之一是使用 NFT 作为抵押品。这意味着你将能够提供代表艺术品、数字土地甚至代币化房地产的 NFT 作为抵押品，并以此借款。

这听起来很有希望，但存在一个问题。在 Compound 或 Aave 等标准 DeFi 借贷平台上，所提供抵押品的价值可以通过集成价格预言机轻松衡量。这些预言机汇总了来自多个流动性来源（如中心化和去中心化交易所）的价格。当涉及到 NFT 时，特定代币的市场通常缺乏流动性，这使得价格发现过程变得棘手。

为了更好地理解这个问题，想象一下有人以 10 个以太币的价格购买了一只稀有的 CryptoKitty。这个 NFT 后来被用作抵押品，借款人提取了 1,700 DAI——假设 10 个以太币价值 3,500 美元，并且这个特定的 NFT 具有 50% 的贷款价值比。在此之后，如果没有其他人愿意购买这只特定的 CryptoKitty，那么这个 NFT 的市场就缺乏流动性，甚至根本不存在。唯一的假设是该 NFT 的价值仍然与上次售出时的金额相同——这不是一个安全的假设，因为 NFT 的价值可能会发生非常剧烈的变化。

这就是为什么一些提供 NFT 抵押贷款的项目使用了一种略有不同的模型：点对点贷款。在这个市场模型中，借款人可以提供他们的 NFT 作为抵押品，而贷款人可以在发起贷款之前选择他们愿意接受哪个 NFT。用作抵押品的 NFT 保存在托管合约中，如果借款人违约，没有按时偿还借款金额加上利息，该 NFT 就会转账给贷款人。这个领域是全新的，但使用这种模型的公司之一是 NFTfi。

#### NFT 作为金融产品 (9:32) {#nfts-as-financial-products-932}

除了用作抵押品外，NFT 还可以代表更复杂的金融产品，如保险、债券或期权。Yearn Finance 的 Yinsure 是 NFT 在保险领域应用的一个很好的例子。在 Yinsure 中，每份保险合约都表示为一个 NFT，也可以在 Rarible 等二级市场上进行交易。

我们最近也开始看到 NFT 项目使用流动性挖矿等 DeFi 原生概念。例如，Rarible 开始用 RARI 治理代币奖励在其平台上创建、购买和销售 NFT 的用户。

#### 不断增长的 NFT 市场 (10:30) {#the-growing-nft-market-1030}

随着价值超过 1 亿美元的 NFT 被交易，仅在最近一个月就达到了 600 万美元，NFT 领域是加密货币中增长最快的细分市场之一。从数字小猫到复杂的金融产品，它具有巨大的潜力。
