这个领域每天都在发生变化。新术语不断进入以太坊用户的词典，我们需要你的帮助，为所有与以太坊相关的事物提供准确、最新的参考。查看当前的[术语表](/glossary/)，如果你想提供帮助，请参阅下文！

## 标准 {#criteria}

新的术语表词汇将根据以下标准进行评估：

- 术语/定义是否最新且与当前相关？
- 字典中是否已有类似的术语？（如果有，请考虑添加新术语与更新现有术语的利弊）
- 术语/定义是否不含产品广告或其他促销内容？
- 术语/定义是否与以太坊直接相关？
- 定义是否客观、准确，且不含主观判断或意见？
- 来源是否可靠？是否引用了其来源？

---

## 添加你的术语 {#how-decisions-about-the-site-are-made}

如果你想向 ethereum.org 添加术语表词汇且符合标准，请[在 GitHub 上创建议题](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=feature+%3Asparkles%3A%2Ccontent+%3Afountain_pen%3A&template=suggest_glossary_term.yaml)。
