我们希望确保列出尽可能最好的开发者资源，以便人们能够充满信心地进行构建并获得所需的支持。

如果我们遗漏了某个有用的开发者工具，请随时在合适的地方提出建议。

我们目前在整个[开发者门户](/developers/)中列出了开发者工具。

**欢迎随时向相关页面建议添加新内容。**

## 我们的决定方式 {#ways-to-contribute}

开发者工具的提交将根据以下标准进行评估：

**它与已列出的工具是否有显著区别？**

- 新的工具类别或类型
- 与现有同类工具相比具有新功能
- 针对现有同类工具未涵盖的特定用例

**该工具的文档是否完善？**

- 是否有文档？
- 文档是否足以指导使用该工具？
- 最近是否更新过？

**该工具是否被广泛使用？**

- 我们将考虑 GitHub 星数、下载统计数据以及是否被知名公司或项目使用等指标

**该工具的质量是否足够好？**

- 是否存在反复出现的错误？
- 该工具是否可靠？
- 该工具是否在积极维护中？

**该工具是否开源？**

以太坊领域的许多项目都是开源的。我们更倾向于列出允许社区开发者检查代码并为其做出贡献的开源项目。

---

## 产品排序 {#product-ordering}

除非产品有特定的排序方式（例如按字母顺序），否则产品将按添加到页面的时间先后顺序显示。换句话说，最新添加的产品会排在列表的底部。
