## 如何贡献故事 {#how-to-contribute}

社区故事展示在 [ethereum.org/stories/](/stories/)，重点介绍个人在开源和避风港（sanctuary）技术方面的体验、以太坊生态系统如何影响人们的生活，以及世界各地的社区如何使用以太坊。任何人都可以建议或提交故事。

贡献故事有三种方式：

1. **精选故事** - 作为网站上的独立页面发布的完整书面故事。
2. **短篇故事** - 在故事登陆页面上以卡片形式显示的简短感言。
3. **视频故事** - 通过[视频库](/videos/)提交并在故事页面上精选的视频。

### 上架政策 {#listing-policy}

Ethereum.org 是一个中立的教育资源。故事页面的策划旨在：

- 通过真实的个人和社区体验，使以太坊的目标和技术**更具人情味**
- **强调**开源和避风港技术如何真正改变人们的生活
- **赞颂**全球以太坊社区的多样性

本网站不会上架主要用于推广特定产品、代币或商业服务的故事。所有提交的内容均由 ethereum.org 团队审核。

### 收录标准 {#criteria-for-inclusion}

#### 必备条件 {#must-haves}

- **真实的人类故事** - 故事必须描述真实的个人或社区体验。这可以包括你在开源和避风港技术方面的体验、以太坊生态系统如何影响你的生活，或者你和你的社区如何使用以太坊。虚构、假设或纯技术内容属于其他部分（例如，针对技术构建者领域文章的 `/latest/`）。
- **与以太坊的联系** - 故事必须以有意义的方式涉及以太坊、基于以太坊的应用程序（包括 L2、以太坊上的稳定币、去中心化应用 (dapp) 等），或更广泛的开源和避风港技术生态系统。不接受没有特定以太坊焦点、关于加密货币和区块链的一般性故事。
- **非促销性质** - 故事不应主要推广特定产品、代币、公司或商业服务。提及所使用的工具是可以接受的；但将故事包装成产品感言则不被允许。
- **内容适当** - 故事必须遵守 ethereum.org 的[行为准则](/community/code-of-conduct/)。不接受充满仇恨、误导性或非法的内容。
- **英语优先** - 故事可以使用任何[支持的语言](/contributing/translation-program/)提交。设置 `lang` 字段以匹配故事所用的语言（例如，英语为 `en`，西班牙语为 `es`）。通过[翻译计划](/contributing/translation-program/)，英文提交的内容将被翻译成其他语言，非英文提交的内容也可以被翻译成英文。

#### 加分项 {#nice-to-haves}

- **地理和主题多样性** - 来自代表性不足地区以及涵盖代表性不足用例的故事将获得优先考虑。
- **多媒体** - 鼓励附带视频、音频或摄影的故事。
- **开源和避风港价值观** - 能够说明开源或避风港技术如何实现抗审查、开放访问、普惠金融、隐私或安全的故事尤其受到重视。

### 如何添加精选故事 {#how-to-add-featured-story}

#### 选项 1：通过接收表单提交 {#submit-via-intake-form}

如果你有故事要分享，可以使用接收表单提交。ethereum.org 团队将审核你的提交并跟进。

<ButtonLink href="https://ethereumstory.paperform.co/">
  提交你的故事
</ButtonLink>

#### 选项 2：在 GitHub 上开启 issue {#open-an-issue}

如果你想为 ethereum.org 建议一个社区故事并且它符合标准，请在 GitHub 上创建一个 issue。

<ButtonLink href="https://github.com/ethereum/ethereum-org-website/issues/new?template=content-suggestion.yml">
  创建 issue
</ButtonLink>

### 贡献故事的其他方式 {#other-ways-to-contribute}

你也可以建议**短篇故事**或**视频故事**：

- **短篇故事**在[故事登陆页面](/stories/)上以卡片形式显示。这些故事可以直接发布在 ethereum.org 上，也可以链接到在推特上分享的故事。
- **视频故事**是在[视频库](/videos/)中带有 `community-stories` 类别标签的视频。要添加视频故事，请遵循现有的[添加视频](/contributing/adding-videos/)贡献指南，并在主题标签中包含 `community-stories`。

如果你想建议这两种类型中的任何一种，请在 GitHub 上创建一个 issue。

<ButtonLink href="https://github.com/ethereum/ethereum-org-website/issues/new?template=BLANK_ISSUE">
  创建 issue
</ButtonLink>

### 维护 {#maintenance}

- 如果事实细节发生变化，精选故事可能会更新。
- 短篇故事会经过适当性和准确性审查。
- 故事页面会自动抓取带有 `community-stories` 标签的新视频条目。
