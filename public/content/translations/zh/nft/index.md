## 什么是 NFT？ {#what-are-nfts}

NFT 是**独一无二**的代币。每个 NFT 都有不同的属性（非同质化），并且其稀缺性是可证明的。这与 [ETH](/glossary/#ether) 等代币或其他基于[以太坊](/)的代币（如 USDC）不同，后者的每个代币都是相同的，并具有相同的属性（“同质化”）。你不会在意自己钱包里具体是哪一张美元钞票（或哪一个 ETH），因为它们都是相同的，价值也一样。但是，你_确实_会在意自己拥有的是哪一个特定的 NFT，因为它们都有各自的属性，使其与其他 NFT 区分开来（“非同质化”）。

每个 NFT 的独特性使得艺术品、收藏品甚至房地产等事物的代币化成为可能，其中一个特定的独特 NFT 代表某个特定的独特现实世界或数字物品。资产的所有权在以太坊[区块链](/glossary/#blockchain)上是可公开验证的。

## 资产互联网 {#internet-of-assets}

NFT 和以太坊解决了当今互联网上存在的一些问题。随着一切变得更加数字化，我们需要以一种不受中心化组织控制的方式，复制物理物品的属性，如稀缺性、独特性和所有权证明。例如，借助 NFT，你可以在所有基于以太坊的应用程序中拥有一个音乐 mp3 文件，而不受限于某家公司的特定音乐应用程序（如 Spotify 或 Apple Music）。你可以拥有一个社交媒体账号，你可以出售或兑换它，但平台提供商**不能随意将其从你手中夺走**。

以下是 NFT 互联网与我们大多数人今天使用的互联网的比较……

### 比较 {#nft-comparison}

| NFT 互联网                                                                                                         | 当今的互联网                                                                                                                                     |
| ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **你拥有自己的资产！** 只有你可以出售或兑换它们。                                                                    | **你从某个组织租用资产**，并且它可能会被夺走。                                                                            |
| NFT 是**数字上独一无二的**，没有两个 NFT 是相同的。                                                                    | **副本通常无法与原件区分开来**。                                                                                   |
| NFT 的所有权存储在区块链上，任何人都可以**公开验证**。                                                   | 数字物品所有权记录的访问权限**由机构控制**——你只能相信他们的话。                                 |
| NFT 是以太坊上的[智能合约](/glossary/#smart-contract)。这意味着它们**可以轻松地在以太坊上的其他智能合约**和应用程序中使用！ | 拥有数字物品的公司通常**需要他们自己的“围墙花园”基础设施**。                                                                 |
| 内容**创作者可以在任何地方出售他们的作品**，并可以进入全球市场。                                           | 创作者依赖于他们所使用平台的基础设施和分发渠道。这些通常受使用条款和**地理限制**的约束。 |
| NFT 创作者**可以保留对其作品的所有权**，并将版税直接编程到 NFT 合约中。     | 平台（如音乐**流媒体服务）保留了销售的大部分利润**。                                                                |

## NFT 有什么用途？ {#nft-use-cases}

NFT 可用于许多事物，包括：

- 证明你参加了某个活动
- 证明你完成了一门课程
- 游戏中的可拥有物品
- 数字艺术
- 将[现实世界资产](/real-world-assets/)代币化
- 证明你的在线身份
- 限制对内容的访问
- 票务
- 去中心化的互联网域名
- [去中心化金融 (DeFi)](/glossary/#defi)中的抵押品

也许你是一位艺术家，希望使用 NFT 分享自己的作品，而不想失去控制权并将利润牺牲给中间商。你可以创建一个新合约，并指定 NFT 的数量、它们的属性以及指向某个特定艺术品的链接。作为艺术家，**你可以将你应该获得的版税编程到智能合约中**（例如，每次转账 NFT 时，将销售价格的 5% 转账给合约所有者）。你还可以随时证明是你创建了这些 NFT，因为你拥有部署该合约的[钱包](/glossary/#wallet)。你的买家可以轻松证明他们拥有你收藏中的**正版 NFT**，因为他们的钱包[地址](/glossary/#address)与你智能合约中的代币相关联。他们可以在整个以太坊生态系统中使用它，并对其真实性充满信心。

## NFT 是如何工作的？ {#how-nfts-work}

与以太坊区块链上的任何数字物品一样，NFT 是通过一种称为“智能合约”的基于以太坊的特殊计算机程序创建的。这些合约遵循某些规则，例如 [ERC-721](/glossary/#erc-721) 或 [ERC-1155](/glossary/#erc-1155) 标准，这些标准决定了合约可以做什么。

NFT 智能合约可以做几件关键的事情：

- **创建 NFT：** 它可以制作新的 NFT。
- **分配所有权：** 它通过将 NFT 链接到特定的以太坊地址来跟踪谁拥有哪些 NFT。
- **为每个 NFT 分配一个 ID：** 每个 NFT 都有一个使其独一无二的编号。此外，通常还会附加一些信息（元数据），描述该 NFT 代表什么。

当有人“创建”或“铸造”一个 NFT 时，他们基本上是在告诉智能合约赋予他们特定 NFT 的所有权。这些信息安全且公开地存储在区块链中。

此外，合约的创建者可以添加额外的规则。他们可能会限制某种 NFT 的制作数量，或者决定每当 NFT 易手时，他们应该获得一小笔版税。

### NFT 安全性 {#nft-security}

以太坊的安全性来自[权益证明 (PoS)](/glossary/#pos)。该系统旨在从经济上抑制恶意行为，使以太坊具有防篡改性。这就是 NFT 成为可能的原因。一旦包含你的 NFT 交易的[区块](/glossary/#block)[已最终确定](/glossary/#finality)，攻击者将需要花费数百万 ETH 才能更改它。任何运行以太坊软件的人都能立即检测到对 NFT 的不诚实篡改，而作恶者将受到经济惩罚并被驱逐。

与 NFT 相关的安全问题通常与网络钓鱼诈骗、智能合约漏洞或用户错误（例如无意中暴露私钥）有关，因此良好的钱包安全性对 NFT 所有者至关重要。
