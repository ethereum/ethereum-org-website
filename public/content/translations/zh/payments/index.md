## 使用自托管加密货币卡支付 {#pay-with-self-custodial-crypto-cards}

自托管加密货币卡的工作原理就像使用你自己的背包，而不是把钱锁在别人的金库里。使用传统银行卡时，银行或托管人持有你的资金，并在你消费时释放它们。使用自托管卡，你可以始终控制自己的资产——没有中间人——同时仍然可以通过刷卡或感应支付咖啡、杂货甚至航班费用。

这些卡直接链接到非托管钱包或智能合约账户，允许用户在日常场景中消费 ETH 和稳定币，而无需放弃所有权。与要求用户将资金存入第三方的托管卡不同，自托管卡支持 Visa 和万事达卡 (Mastercard) 等现实世界支付，同时保留链上控制权。

### 示例 {#crypto-cards-examples}

- **梅塔马斯克 (MetaMask) 卡：** 这张万事达借记卡与梅塔马斯克钱包链接，允许用户消费 ETH、稳定币和其他受支持的代币。它支持 Apple Pay 和 Google Pay，包含加密货币现金返还奖励，并提供赚取收益的选项。

- **Tuyo 卡：** 一种基于智能合约的 Visa 卡，可自动将加密货币转换为 USDC，以便在任何接受 Visa 的地方消费。用户保留其资产的托管权，并可使用收益、交易和消费功能。

- **Gnosis Pay：** 第一张与 Gnosis Safe 智能账户绑定的自托管 Visa 卡。用户直接从钱包中消费加密货币，无需支付 Gas 费、外汇费或出金费用。还支持通过以太坊域名服务 (ENS) 进行卡片个性化。

- **Ether.Fi Cash 卡：** 该卡与 ether.fi 的质押协议集成，允许用户在 ETH 保持质押状态的同时进行消费。支付通过智能合约处理，即使在消费时也能保持自托管。

### 自托管加密货币卡比较 {#crypto-card-custody-comparison}

| **加密货币卡** | **自托管** | **非托管** | **关键说明**                                                |
| --------------- | :----------------: | :---------------: | ------------------------------------------------------------ |
| 梅塔马斯克卡   |         ✅         |        ✅         | 钱包保留在梅塔马斯克中；支付时自动出金           |
| Tuyo 卡       |         ✅         |        ✅         | 智能钱包转换为 USDC；用户保留控制权          |
| Gnosis Pay      |         ✅         |        ✅         | 链接到用户的 Gnosis Safe；使用期间托管权不转移    |
| Ether.Fi Cash   |         ✅         |        ✅         | ETH 保持质押状态；智能合约控制消费权限。 |

> **注意：**“自托管”是指用户控制的钱包，用户对其资金拥有完全的访问和控制权。
> “非托管”是指在没有第三方托管的情况下管理资金的钱包，通常通过智能合约实现。
> 虽然所有自托管卡都是非托管的，但并非所有非托管卡都是自托管的。

## 网站和代理的微支付 (x402) {#x402}

[x402](https://www.x402.org/) 是一种开放的支付标准，它将原生的按次付费模式引入网络。通过在低成本的[以太坊二层网络 (l2)](/layer-2/)上使用[稳定币](/stablecoins/)，x402 标准使得人类和机器直接为单一操作（例如阅读新闻文章或调用 API）付费变得经济实惠，而无需管理 API 密钥、订阅或通过关注广告来“付费”。

- **消除付费墙和登录：** 你的钱包可以支付解锁文章所需的几美分，而无需为了阅读一篇新闻文章而创建账户并分享个人信息。
- **为 AI 代理支付：** x402 允许自主软件（“AI 代理”）在无需人工干预的情况下，为它们运行所需的数据和 API 调用付费。

### x402 支付标准的工作原理 {#x402-how}

当客户端请求资源时，服务器会发送一个 `402 Payment Required` 错误代码以及支付指令（价格、账户以及支持哪些代币和链）。

- 你的[钱包](/wallets/)会检测到该请求并处理支付（通常只需单击即可授权，或使用预先批准的授权额度自动处理）
- 能够访问预先批准的钱包余额的[AI 代理](/ai-agents/)可以自动检测价格并即时支付以访问数据或服务
- 客户端需要在其钱包中拥有受支持的稳定币之一，但不需要拥有任何 ETH 来支付[Gas 费用](/gas/)

这开启了一种新的“机器对机器”经济，AI 代理可以自行购买资源，并且可以更高效地访问 API 服务。

然后，签名的消息会被传递到服务器。服务器通常使用[x402 促进者](https://x402.gitbook.io/x402/core-concepts/facilitator)来处理区块链的复杂性（发送交易、获取付款、促进 Gas 费等），这意味着开发者可以轻松接受加密货币微支付，而无需管理支付基础设施。

## 以太坊上的加密货币支付与法定货币支付 {#ethereum-vs-fiat}

为了真正体会以太坊支付的影响，值得将它们与传统的法定货币进行比较：

|                       | **以太坊**            | **传统银行**                                        |
| --------------------- | ----------------------- | ------------------------------------------------------------ |
| **速度**             | 几秒到几分钟      | 几小时到几天                                                |
| **全球覆盖**      | 无国界，全天候        | 受国际银行限制和工作时间影响 |
| **透明度**      | 完全透明       | 因机构而异                                        |
| **可编程性**   | 支持智能合约 | 仅限于基本交易                                |
| **通胀控制** | 可预测的发行    | 受中央银行政策影响                             |
| **可访问性**     | 任何有互联网的人    | 受国家和国际限制           |

在其核心，以太坊是一个去中心化平台，允许进行安全、快速和透明的交易。然而，许多组件使其有别于传统的支付方式。让我们深入探讨使以太坊支付成为游戏规则改变者的优势：

### 可编程性 {#programmability}

以太坊的独特功能之一是它支持智能合约的能力。智能合约是自动执行的协议，其条款直接写入代码中。这为基于条件的自动支付打开了无限可能，可以极大地改善以下交易：

- 托管服务
- 定期付款
- 基于绩效的薪酬

### 速度 {#speed}

你还记得上一次等待国际银行转账结清花了几天时间吗？排长队？还有你必须填写的各种表格？有了以太坊，那些日子一去不复返了。以太坊网络上的交易在几分钟内即可结算，无论发送方和接收方位于何处。由于以太坊是无需许可的，因此在汇款时没有监管上的繁文缛节。这种速度在时间紧迫的情况下尤为关键，例如紧急救援行动。

### 更低的费用 {#lower-fees}

传统的国际汇款费用有时会吞噬掉汇款金额的很大一部分，尤其是在处理数百美元的交易时。以太坊交易虽然不是免费的，但通常费用较低。这意味着你更多的钱会流向你想要的地方，而不是落入中间人的口袋。

### 透明度 {#transparency}

以太坊区块链上的每笔交易都记录在公共账本上。这意味着任何人都可以验证资金的流动，使其成为以下方面的绝佳工具：

- 慈善组织展示捐款的使用情况
- 企业证明向供应商或员工支付了款项
- 个人跟踪其财务活动

借助以太坊，每个人都可以看到资金是如何流动的以及成本是如何执行的，这与传统组织中大多数情况仍然未知不同。
