你是否厌倦了寻找列出你所有最爱代币的交易所？你可以使用[去中心化交易所](/glossary/#dex)兑换大多数代币。

代币兑换涉及交换存在于以太坊网络上的两种不同资产，例如将 ETH 兑换为 DAI（一种 [ERC-20](/glossary/#erc-20) 代币）。这个过程非常快速且便宜。你需要拥有一个加密货币钱包才能兑换代币。

**先决条件：**

- 拥有一个[加密货币钱包](/glossary/#wallet)；如果没有，你可以按照这篇关于[如何创建以太坊账户](/guides/how-to-create-an-ethereum-account/)的指南进行操作
- 向你的钱包充值资金

## 1. 将你的钱包连接到你选择的去中心化交易所 (DEX) {#1-connect-your-wallet-to-the-decentralized-exchange-dex-of-your-choice}

一些受欢迎的交易所包括：

- [尤尼斯瓦普](https://app.uniswap.org/#/swap)
- [Sushiswap](https://www.sushi.com/swap)
- [1Inch](https://app.1inch.io/#/1/unified/swap/ETH/DAI)
- [Curve](https://www.curve.finance/dex/ethereum/swap/)

感兴趣吗？了解更多关于什么是[去中心化金融 (DeFi)](/defi/)以及这些新型交易所是如何运作的。

## 2. 选择你希望兑换的代币对 {#2-select-the-pair-of-tokens-you-wish-to-swap}

例如，ETH 和 DAI。确保你拥有这两种代币中其中一种的资金。
![Common interface for swapping](./swap1.png)

## 3. 输入你想要交易的代币数量并点击兑换 {#3-enter-the-amount-of-tokens-you-want-to-trade-and-click-swap}

交易所将自动计算你将获得多少代币。

![Common interface for swapping](./swap2.png)

## 4. 确认交易 {#4-confirm-the-transaction}

检查交易详情。查看汇率和任何其他费用，以防出现意外情况。

![Common interface for reviewing the transaction](./swap3.png)

## 5. 等待交易被处理 {#5-wait-for-the-transaction-to-be-processed}

你可以在任何区块链浏览器上查看交易进度。这个过程通常不会超过 10 分钟。

一旦交易被处理，你将自动在你的钱包中收到兑换后的代币。

## 常见问题 {#frequently-asked-questions}

### 我可以从我的钱包中将 ETH 兑换为 BTC 吗？ {#can-i-swap-eth-for-btc-from-my-wallet}

不可以，你只能兑换以太坊网络原生的代币，例如 ETH、ERC-20 代币或 NFT。你只能兑换存在于以太坊上的“包装”形式的比特币。

### 什么是滑点？ {#what-is-slippage}

它是你预期的汇率与实际汇率之间的差额。
