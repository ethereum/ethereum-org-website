本文档旨在帮助你使用[以太坊](/)进行构建。它涵盖了以太坊的概念，解释了以太坊技术栈，并记录了针对更复杂应用和用例的高级主题。

这里的所有内容都是开源且由社区维护的，因此如果某个页面已过时或缺少有用的内容，请提交 issue 或 pull request。[编辑指南](https://github.com/ethereum/ethereum-org-website/blob/dev/docs/editing-markdown.md)详细说明了具体操作方法。

## 选择一个起点 {#pick-a-starting-point}

读者带着不同的目标来到这里，通读这些文档的最快路径取决于你想要构建什么。以下是一些常见的切入点：

- **构建与以太坊交互的去中心化应用 (dapp)。** 从[技术简介](/developers/docs/intro-to-ethereum/)开始，然后学习[账户](/developers/docs/accounts/)和[交易](/developers/docs/transactions/)。当你准备好编写代码时，选择一个[框架](/developers/docs/frameworks/)。
- **编写智能合约。** 如果 EVM 概念对你来说是全新的，请浏览[简介](/developers/docs/intro-to-ethereum/)，然后跳转到[智能合约](/developers/docs/smart-contracts/)和[编程语言](/developers/docs/programming-languages/)。
- **运行节点或质押。** 前往[节点和客户端](/developers/docs/nodes-and-clients/)，然后了解[网络](/developers/docs/networking-layer/)和[共识机制](/developers/docs/consensus-mechanisms/)。
- **自下而上地理解协议。** 下面的模块就是为此排序的。请按顺序阅读。
