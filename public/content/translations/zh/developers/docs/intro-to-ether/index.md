## 前提条件 {#prerequisites}

为了帮助你更好地理解本页面，我们建议你先阅读[以太坊简介](/developers/docs/intro-to-ethereum/)。

## 什么是加密货币？ {#what-is-a-cryptocurrency}

加密货币是一种由基于区块链的账本提供安全保障的交换媒介。

交换媒介是指被广泛接受作为商品和服务支付方式的任何事物，而账本则是记录交易的数据存储。区块链技术允许用户在账本上进行交易，而无需依赖受信任的第三方来维护账本。

第一种加密货币是中本聪 (Satoshi Nakamoto) 创造的比特币。自 2009 年比特币发布以来，人们已经在许多不同的区块链上创造了数以千计的加密货币。

## 什么是以太币？ {#what-is-ether}

**以太币 (ETH)** 是在以太坊网络上用于许多事物的加密货币。从根本上说，它是唯一可接受的交易费支付形式，并且在[合并](/roadmap/merge)之后，在主网上验证和提议区块都需要以太币。以太币还被用作[去中心化金融 (DeFi)](/defi) 借贷市场中的主要抵押品形式、NFT 市场中的记账单位、提供服务或销售现实世界商品赚取的报酬等等。

以太坊允许开发者创建[**去中心化应用 (dapp)**](/developers/docs/dapps)，这些应用共享一个计算能力池。这个共享池是有限的，因此以太坊需要一种机制来决定谁可以使用它。否则，某个 dapp 可能会意外或恶意地消耗所有网络资源，从而阻止其他人访问。

以太币这种加密货币支持以太坊计算能力的定价机制。当用户想要进行交易时，他们必须支付以太币才能让他们的交易在区块链上得到确认。这些使用成本被称为 [gas 费](/developers/docs/gas/)，gas 费取决于执行交易所需的计算能力以及当时全网对计算能力的需求。

因此，即使恶意的 dapp 提交了无限循环，该交易最终也会耗尽以太币并终止，从而使网络恢复正常。

人们[经常混淆](https://abcnews.go.com/Business/bitcoin-slumps-week-low-amid-renewed-worries-chinese/story?id=78399845)以太坊和以太币——当人们提到“以太坊的价格”时，他们实际上是在描述以太币的价格。

## 铸造以太币 {#minting-ether}

铸造是在以太坊账本上创建新以太币的过程。底层的以太坊协议创建新的以太币，用户是不可能创建以太币的。

以太币被铸造出来，作为对提议的每个区块的奖励，并在每个时段检查点作为对与达成共识相关的其他验证者活动的奖励。发行的总金额取决于验证者的数量以及他们质押了多少以太币。在所有验证者都诚实且在线的理想情况下，总发行量在验证者之间平均分配，但在现实中，它会根据验证者的表现而有所不同。总发行量的大约 1/8 归区块提议者所有；其余部分分配给其他验证者。区块提议者还会从交易费和与 MEV 相关的收入中获得小费，但这些来自回收的以太币，而不是新发行的以太币。

## 销毁以太币 {#burning-ether}

除了通过区块奖励创建以太币外，以太币还可以通过称为“销毁”的过程被销毁。当以太币被销毁时，它将永久退出流通。

以太坊上的每笔交易都会发生以太币销毁。当用户为他们的交易付款时，由网络根据交易需求设定的基础费用将被销毁。这与可变区块大小和最大 gas 费相结合，简化了以太坊上的交易费估算。当网络需求很高时，[区块](https://eth.blockscout.com/block/22580057)销毁的以太币可能多于铸造的以太币，从而有效地抵消了以太币的发行。

销毁基础费用阻碍了区块生产者操纵交易的能力。例如，如果区块生产者收到基础费用，他们就可以免费包含自己的交易，并提高其他所有人的基础费用。或者，他们可以在链下向某些用户退还基础费用，从而导致交易费市场变得更加不透明和复杂。

## 以太币的面额 {#denominations}

由于以太坊上许多交易的价值很小，以太币有几种面额，可以作为较小的记账单位被引用。在这些面额中，Wei 和 Gwei 尤为重要。

Wei 是以太币的最小可能数量，因此，许多技术实现（例如[以太坊黄皮书](https://ethereum.github.io/yellowpaper/paper.pdf)）将所有计算都基于 Wei。

Gwei 是 giga-wei 的缩写，通常用于描述以太坊上的 gas 成本。

| 面额 | 以太币价值 | 常见用途 |
| ------------ | ---------------- | ------------------------- |
| Wei          | 10<sup>-18</sup> | 技术实现 |
| Gwei         | 10<sup>-9</sup>  | 人类可读的 gas 费 |

## 转移以太币 {#transferring-ether}

以太坊上的每笔交易都包含一个 `value` 字段，该字段指定要从发送者地址发送到接收者地址的以太币转移金额（以 Wei 为单位）。

当接收者地址是[智能合约](/developers/docs/smart-contracts/)时，转移的以太币可用于在智能合约执行其代码时支付 gas 费。

[关于交易的更多信息](/developers/docs/transactions/)

## 查询以太币 {#querying-ether}

用户可以通过检查账户的 `balance` 字段来查询任何[账户](/developers/docs/accounts/)的以太币余额，该字段显示以 Wei 为单位的以太币持有量。

[Etherscan](https://etherscan.io) 和 [Blockscout](https://eth.blockscout.com) 是通过基于 Web 的应用程序检查地址余额的流行工具。例如，[这个 Blockscout 页面](https://eth.blockscout.com/address/0xde0B295669a9FD93d5F28D9Ec85E40f4cb697BAe)显示了以太坊基金会的余额。账户余额也可以使用钱包查询，或者直接通过向节点发出请求来查询。
