一个常见的误解是，开发者必须编写[智能合约](/developers/docs/smart-contracts/)才能在以太坊上进行开发。这是错误的。
以太坊网络和社区的魅力之一在于，你可以使用几乎任何编程语言来[参与](/community/)其中。

以太坊及其社区拥抱开源。你可以找到各种语言的社区项目——客户端实现、API、开发框架、测试工具。

### 如果不支持我的语言怎么办 {#other-lang}

如果你想链接到其他编程语言的资源或指向其虚拟社区，你可以通过[开启一个议题 (issue)](https://github.com/ethereum/ethereum-org-website/issues/new/choose) 来请求创建一个新页面。

如果你只是想使用目前不受支持的语言编写代码来与区块链进行交互，你可以使用 [JSON-RPC 接口](/developers/docs/apis/json-rpc/)连接到以太坊网络。任何可以使用 TCP/IP 的编程语言都可以使用此接口。
