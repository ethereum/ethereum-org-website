## 使用以太坊客户端 {#working-with-ethereum-clients}

你可以使用以太坊创建去中心化应用（或称“dapp”），从而利用加密货币和区块链技术的优势。
目前至少有两个维护中的 Dart 库，可用于调用以太坊的 [JSON-RPC API](/developers/docs/apis/json-rpc/)。

1. [来自 pwa.ir 的 Web3dart](https://pub.dev/packages/web3dart)
1. [来自 darticulate.com 的 Ethereum 5.0.0](https://pub.dev/packages/ethereum)

还有一些其他的库允许你操作特定的以太坊地址，或者让你获取各种加密货币的价格。
[你可以在这里查看完整列表](https://pub.dev/dart/packages?q=ethereum)。
