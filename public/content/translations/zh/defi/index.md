去中心化金融 (DeFi) 是为互联网时代构建的开放式全球金融系统，它是对不透明、受严格控制且由几十年前的基础设施和流程支撑的系统的替代方案。它让你对自己的资金拥有控制权和可见性。它让你能够接触全球市场，并提供当地货币或银行选项之外的替代方案。DeFi 产品向任何有互联网连接的人开放金融服务，并且它们主要由用户拥有和维护。到目前为止，已有价值数百亿美元的加密货币流经 DeFi 应用，并且这个数字每天都在增长。

## 什么是 DeFi？ {#what-is-defi}

DeFi 是金融产品和服务的统称，任何能够使用[以太坊](/)的人（即任何有互联网连接的人）都可以访问这些产品和服务。借助 DeFi，市场始终开放，没有中心化机构可以阻止付款或拒绝你访问任何内容。以前缓慢且存在人为错误风险的服务，现在由任何人都可以检查和审查的代码处理，变得自动化且更加安全。

这里有一个蓬勃发展的加密货币经济，你可以在其中进行借贷、借款、做多/做空、赚取利息等。精通加密货币的阿根廷人已经使用 DeFi 来逃避严重的通货膨胀。公司已经开始向员工实时流支付工资。有些人甚至在不需要任何个人身份证明的情况下，借出并偿还了价值数百万美元的贷款。

## DeFi 与传统金融 {#defi-vs-tradfi}

了解 DeFi 潜力的最佳方法之一是了解当今存在的问题。

- 有些人未获准开设银行账户或使用金融服务。
- 无法获得金融服务可能会阻碍人们就业。
- 金融服务可能会阻止你获得报酬。
- 金融服务的一项隐藏费用是你的个人数据。
- 政府和中心化机构可以随意关闭市场。
- 交易时间通常仅限于特定时区的营业时间。
- 由于内部人工流程，资金转账可能需要几天时间。
- 金融服务存在溢价，因为中介机构需要从中抽成。

### 比较 {#defi-comparison}

| DeFi                                                                                               | 传统金融                                                                                                            |
| -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| 你持有自己的资金。                                                                               | 你的资金由公司持有。                                                                                               |
| 你控制资金的去向和使用方式。                                              | 你必须相信公司不会对你的资金管理不善，例如借给高风险借款人。                                      |

| 市场始终开放。                                                                       | 市场会关闭，因为员工需要休息。                                                                                   |
| 它建立在透明度之上——任何人都可以查看产品的数据并检查系统的工作原理。 | 金融机构是不公开的：你不能要求查看他们的贷款历史、管理资产记录等。 |

## 一切始于比特币…… {#bitcoin}

在许多方面，比特币是第一个去中心化金融 (DeFi) 应用。比特币让你真正拥有和控制价值，并将其发送到世界各地。它通过提供一种方式，让大量互不信任的人在不需要受信任中介的情况下，就账户账本达成一致来实现这一点。比特币向所有人开放，没有人有权更改其规则。比特币的规则（如其稀缺性和开放性）被写入了技术之中。它不像传统金融那样，政府可以印钞使你的储蓄贬值，公司可以关闭市场。

以太坊以此为基础。像比特币一样，规则不能随意更改，每个人都可以访问。但它还使用[智能合约](/glossary/#smart-contract)使这种数字货币具有可编程性，因此你可以超越仅仅存储和发送价值的范畴。

## 可编程货币 {#programmable-money}

这听起来很奇怪……“我为什么要对我的钱进行编程”？然而，这不仅仅是以太坊上代币的默认功能。任何人都可以将逻辑编程到支付中。因此，你可以获得比特币的控制权和安全性，并将其与金融机构提供的服务结合起来。这让你能够用加密货币做一些用比特币做不到的事情，比如借贷和借款、安排付款、投资指数基金等。

### 获取稳定货币 {#stablecoins}

加密货币的波动性是许多金融产品和日常消费面临的一个问题。DeFi 社区通过稳定币解决了这个问题。它们的价值与另一种资产挂钩，通常是像美元这样的流行货币。

像 DAI 或 USDC 这样的代币，其价值与美元的偏差保持在几美分以内。这使得它们非常适合用于赚取收益或零售。在政府发行的货币存在极大不确定性的时期，拉丁美洲的许多人使用稳定币作为保护其储蓄的一种方式。

### 借款 {#lending}

从去中心化提供商处借款主要有两种形式。

- 点对点，意味着借款人将直接从特定的出借人那里借款。
- 基于资金池，出借人向资金池提供资金（流动性），借款人可以从中借款。

<ButtonLink href="/apps/categories/defi">

使用去中心化借贷平台有很多优势……

#### 保护隐私的借款 {#borrowing-privacy}

如今，借贷资金都围绕着相关的个人展开。银行在放贷前需要知道你是否有可能偿还贷款。

去中心化借贷的运作不需要任何一方表明身份。相反，借款人必须提供抵押品，如果贷款未偿还，出借人将自动获得该抵押品。一些借贷平台甚至接受 [NFT](/glossary/#nft) 作为抵押品。NFT 是独特资产（如画作）的契约。[更多关于 NFT 的信息](/nft/)

这让你无需信用检查或交出私人信息即可借款。

#### 获得全球资金 {#access-global-funds}

当你使用去中心化借贷平台时，你可以获得来自全球各地的存款，而不仅仅是你选择的银行或机构托管的资金。这使得贷款更容易获得，并改善了利率。

#### 税务效率 {#tax-efficiencies}

借款可以让你获得所需的资金，而无需出售你的 ETH（这是一种应税事件）。相反，你可以使用 ETH 作为稳定币贷款的抵押品。这为你提供了所需的现金流，并让你保留了 ETH。稳定币是当你需要现金时更好的代币，因为它们的价值不会像 ETH 那样波动。[更多关于稳定币的信息](#stablecoins)

#### 闪电贷 {#flash-loans}

闪电贷是去中心化借贷中一种更具实验性的形式，它让你无需抵押品或提供任何个人信息即可借款。

它们目前尚未被非技术人员广泛使用，但它们暗示了未来每个人都可能实现的事情。

它的运作基础是贷款在同一笔交易中借出并偿还。如果无法偿还，交易将回滚，就像什么都没发生过一样。

通常使用的资金存放在流动性池（用于借款的大型资金池）中。如果它们在特定时刻未被使用，这就为某人创造了机会，可以借用这些资金，用它们开展业务，并在借入的同时全额偿还。

这意味着必须在非常定制的交易中包含大量逻辑。一个简单的例子可能是，有人使用闪电贷以一个价格借入尽可能多的某种资产，以便他们可以在价格更高的另一家交易所出售。

因此，在单笔交易中，会发生以下情况：

- 你从交易所 A 以 1.00 美元的价格借入 X 数量的 $asset
- 你在交易所 B 以 1.10 美元的价格出售 X 数量的 $asset
- 你向交易所 A 偿还贷款
- 你保留扣除交易费后的利润

如果交易所 B 的供应量突然下降，并且用户无法购买足够的资产来偿还原始贷款，交易将直接失败。

要在传统金融世界中实现上述例子，你需要大量的资金。这些赚钱策略只有那些拥有现有财富的人才能使用。闪电贷展示了这样一个未来：拥有资金不一定是赚钱的先决条件。

#### 二次方融资 {#quadratic-funding}

以太坊是开源软件，到目前为止，许多工作都是由社区资助的。这导致了一种有趣的新融资模式的发展：二次方融资。这有可能改善我们未来资助所有类型公共物品的方式。

二次方融资确保获得最多资金的项目是那些具有最独特需求的项目。换句话说，就是那些有望改善最多人生活的项目。它的工作原理如下：

1. 有一个捐赠的匹配资金池。
2. 一轮公共融资开始。
3. 人们可以通过捐赠一些钱来表达他们对某个项目的需求。
4. 一旦该轮融资结束，匹配资金池将分配给各个项目。具有最独特需求的项目将从匹配资金池中获得最高金额。

这意味着获得 100 笔 1 美元捐款的项目 A 最终获得的资金可能比获得单笔 10,000 美元捐款的项目 B 更多（取决于匹配资金池的规模）。

## DeFi 是如何运作的？ {#how-defi-works}

DeFi 使用加密货币和智能合约来提供不需要中介的服务。在当今的金融世界中，金融机构充当交易的担保人。这赋予了这些机构巨大的权力，因为你的资金流经它们。此外，世界上有数十亿人甚至无法拥有银行账户。

在 DeFi 中，智能合约取代了交易中的金融机构。智能合约是一种以太坊账户，它可以持有资金，并可以根据特定条件发送/退还资金。当智能合约上线时，没有人可以更改它——它将始终按编程运行。

一个旨在发放津贴或零花钱的合约可以被编程为每周五将资金从账户 A 发送到账户 B。只要账户 A 有所需的资金，它就只会这样做。没有人可以更改合约并将账户 C 添加为收款人来窃取资金。

合约也是公开的，任何人都可以检查和审计。这意味着糟糕的合约通常会很快受到社区的审查。

这确实意味着目前需要信任以太坊社区中能够阅读代码的更具技术性的成员。基于开源的社区有助于约束开发人员，但随着智能合约变得更容易阅读以及开发出其他证明代码可信度的方法，这种需求将随着时间的推移而减少。

## 以太坊与 DeFi {#ethereum-and-defi}

以太坊是 DeFi 的完美基础，原因如下：

- 没有人拥有以太坊或存在于其上的智能合约——这让每个人都有机会使用 DeFi。这也意味着没有人可以随意更改规则。
- DeFi 产品在幕后都使用同一种语言：以太坊。这意味着许多产品可以无缝协作。你可以在一个平台上借出代币，并在完全不同的应用上的不同市场中兑换生息代币。这就像能够在你的银行兑现忠诚度积分一样。
- 代币和加密货币内置于以太坊这个共享账本中——跟踪交易和所有权正是以太坊的强项。
- 以太坊允许完全的财务自由——大多数产品永远不会托管你的资金，让你保持控制权。

你可以分层来理解 DeFi：

1. 区块链——以太坊包含交易历史和账户状态。
2. 资产——[ETH](/what-is-ether/) 和其他代币（货币）。
3. 协议——提供功能的[智能合约](/glossary/#smart-contract)，例如，允许去中心化借贷资产的服务。
4. [应用](/apps/)——我们用来管理和访问协议的产品。

注意：大部分 DeFi 使用 [ERC-20 标准](/glossary/#erc-20)。DeFi 中的应用使用一种名为封装以太币 (WETH) 的 ETH 封装器。[了解有关封装以太币的更多信息](/wrapped-eth)。

## 构建 DeFi {#build-defi}

DeFi 是一场开源运动。DeFi 协议和应用都是开放的，供你检查、分叉和创新。由于这种分层堆栈（它们都共享相同的基础区块链和资产），协议可以混合和匹配，以解锁独特的组合机会。

## 超越传统 DeFi {#beyond-traditional-defi}

DeFi 生态系统继续扩展到新领域：

- **[预测市场](/prediction-markets/)**——去中心化平台，你可以在没有中介的情况下对未来事件（从选举到体育赛事）的结果进行下注。
- **[现实世界资产 (RWA)](/real-world-assets/)**——在以太坊上将房地产、商品和债券等实物资产代币化，将数万亿美元的价值带到链上。
- **[支付](/payments/)**——使用以太坊和稳定币进行快速、低成本的全球支付，无需传统的银行基础设施。
- **[AI 代理](/ai-agents/)**——可以在以太坊上进行交易的自主软件代理，实现新形式的自动交易、投资组合管理和链上交互。
