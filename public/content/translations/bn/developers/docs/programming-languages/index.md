একটি সাধারণ ভুল ধারণা হলো যে ইথেরিয়ামে কিছু তৈরি করতে হলে ডেভেলপারদের অবশ্যই [স্মার্ট কন্ট্রাক্ট](/developers/docs/smart-contracts/) লিখতে হবে। এটি ভুল।
ইথেরিয়াম নেটওয়ার্ক এবং কমিউনিটির অন্যতম সৌন্দর্য হলো আপনি প্রায় যেকোনো প্রোগ্রামিং ভাষায় [অংশগ্রহণ](/community/) করতে পারবেন।

ইথেরিয়াম এবং এর কমিউনিটি ওপেন সোর্সকে গ্রহণ করে। আপনি বিভিন্ন ভাষায় কমিউনিটি প্রজেক্টগুলো - ক্লায়েন্ট ইমপ্লিমেন্টেশন, API, ডেভেলপমেন্ট ফ্রেমওয়ার্ক, টেস্টিং টুল - খুঁজে পেতে পারেন।

### যদি আমার ভাষা সমর্থিত না হয় {#other-lang}

আপনি যদি কোনো অতিরিক্ত প্রোগ্রামিং ভাষার জন্য রিসোর্স লিঙ্ক করতে চান বা কোনো ভার্চুয়াল কমিউনিটিকে নির্দেশ করতে চান, তবে আপনি [একটি ইস্যু ওপেন করে](https://github.com/ethereum/ethereum-org-website/issues/new/choose) নতুন পেজের জন্য অনুরোধ করতে পারেন।

আপনি যদি বর্তমানে অসমর্থিত কোনো ভাষা ব্যবহার করে ব্লকচেইন এর সাথে ইন্টারফেস করার জন্য কোড লিখতে চান, তবে আপনি ইথেরিয়াম নেটওয়ার্ক এর সাথে কানেক্ট করতে [জেসন-আরপিসি ইন্টারফেস](/developers/docs/apis/json-rpc/) ব্যবহার করতে পারেন। TCP/IP ব্যবহার করতে পারে এমন যেকোনো প্রোগ্রামিং ভাষা এই ইন্টারফেসটি ব্যবহার করতে পারে।
