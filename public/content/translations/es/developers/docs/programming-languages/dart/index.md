## Comenzar con contratos inteligentes y con el lenguaje Solidity {#getting-started-with-smart-contracts-and-solidity}

## Tutoriales {#tutorials}

- [Flutter y cadena de bloques: dapp de «Hola, mundo»](https://www.geeksforgeeks.org/flutter-and-blockchain-hello-world-dapp/) te guía a través de todos los pasos para empezar:
  1. Escribir un contrato inteligente en [Solidity](https://soliditylang.org/)
  2. Cómo escribir una interfaz de usuario en Dart
- [Crear una dapp móvil con Flutter](https://medium.com/dash-community/building-a-mobile-dapp-with-flutter-be945c80315a) es mucho más corto, lo que podría ser mejor
  si ya conoces los conceptos básicos
- Si prefieres aprender viendo un vídeo, puedes ver [Crea tu primera aplicación de cadena de bloques con Flutter](https://www.youtube.com/watch?v=3Eeh3pJ6PeA), que dura aproximadamente una hora
- Si eres impaciente, quizás prefieras [Crear una aplicación descentralizada de cadena de bloques con Flutter y Dart en Ethereum](https://www.youtube.com/watch?v=jaMFEOCq_1s), que dura solo unos veinte minutos
- [Integrar MetaMask en una aplicación Flutter con Web3Modal de WalletConnect](https://www.youtube.com/watch?v=v_M2buHCpc4): este breve vídeo te guía a través de los pasos para integrar MetaMask en tus aplicaciones Flutter con la biblioteca [Web3Modal](https://pub.dev/packages/web3modal_flutter) de WalletConnect
- [Curso intensivo de desarrollador de cadena de bloques móvil con Solidity y Flutter](https://youtube.com/playlist?list=PL4V4Unlk5luhQ26ERO6hWEbcUwHDSSmVH): lista de reproducción del curso para desarrolladores de pila completa de cadena de bloques móvil

## Trabajar con clientes de Ethereum {#working-with-ethereum-clients}

Puede usar Ethereum para crear aplicaciones decentralizadas (o "dapps") que utilicen los beneficios de las criptomonedas y la tecnología de cadena de bloques.
Actualmente, existen al menos dos bibliotecas para Dart con mantenimiento para utilizar la
[API JSON-RPC](/developers/docs/apis/json-rpc/) de Ethereum.

1. [Web3dart de pwa.ir](https://pub.dev/packages/web3dart)
2. [Ethereum 5.0.0 de darticulate.com](https://pub.dev/packages/ethereum)

También hay bibliotecas adicionales que le permiten manipular direcciones de Ethereum específicas, o que le permiten recuperar precios de diferentes criptomonedas.
[Puedes ver la lista completa aquí](https://pub.dev/dart/packages?q=ethereum).
