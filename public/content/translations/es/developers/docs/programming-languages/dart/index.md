## Primeros pasos con los contratos inteligentes y el lenguaje Solidity {#getting-started-with-smart-contracts-and-solidity}

## Tutoriales {#tutorials}

- [Flutter y cadena de bloques: dapp de Hola Mundo](https://www.geeksforgeeks.org/flutter-and-blockchain-hello-world-dapp/) le guía por todos los pasos para empezar:
  1.  Escribir un contrato inteligente en [Solidity](https://soliditylang.org/)
  2.  Escribir una interfaz de usuario en Dart
- [Crear una dapp móvil con Flutter](https://medium.com/dash-community/building-a-mobile-dapp-with-flutter-be945c80315a) es mucho más corto, lo cual podría ser mejor
  si ya conoce los conceptos básicos
- Si prefiere aprender viendo un video, puede ver [Cree su primera aplicación de cadena de bloques con Flutter](https://www.youtube.com/watch?v=3Eeh3pJ6PeA), que dura aproximadamente una hora
- Si es impaciente, puede que prefiera [Crear una aplicación descentralizada (dapp) de cadena de bloques con Flutter y Dart en Ethereum](https://www.youtube.com/watch?v=jaMFEOCq_1s), que dura solo unos veinte minutos
- [Integrar MetaMask en una aplicación de Flutter con Web3Modal de WalletConnect](https://www.youtube.com/watch?v=v_M2buHCpc4): este breve video le guía por los pasos para integrar MetaMask en sus aplicaciones de Flutter con la biblioteca [Web3Modal](https://pub.dev/packages/web3modal_flutter) de WalletConnect
- [Curso intensivo para desarrolladores de cadenas de bloques móviles con Solidity y Flutter](https://youtube.com/playlist?list=PL4V4Unlk5luhQ26ERO6hWEbcUwHDSSmVH): lista de reproducción del curso para desarrolladores full stack de cadenas de bloques móviles

## Trabajar con clientes de Ethereum {#working-with-ethereum-clients}

Puede usar Ethereum para crear aplicaciones descentralizadas (dapps) que utilicen los beneficios de la criptomoneda y la tecnología de cadena de bloques.
Hay al menos dos bibliotecas mantenidas actualmente para que Dart use la
[API JSON-RPC](/developers/docs/apis/json-rpc/) para Ethereum.

1. [Web3dart de pwa.ir](https://pub.dev/packages/web3dart)
1. [Ethereum 5.0.0 de darticulate.com](https://pub.dev/packages/ethereum)

También hay bibliotecas adicionales que le permiten manipular direcciones específicas de Ethereum,
o que le permiten recuperar precios de varias criptomonedas.
[Puede ver la lista completa aquí](https://pub.dev/dart/packages?q=ethereum).
