<FeaturedText>Aprenda a desarrollar para Ethereum utilizando proyectos y herramientas basados en Elixir.</FeaturedText>

Utilice Ethereum para crear aplicaciones descentralizadas (dapps) que aprovechen los beneficios de las criptomonedas y la tecnología de cadena de bloques. Estas dapps pueden funcionar sin necesidad de confianza, lo que significa que una vez que se despliegan en Ethereum, siempre se ejecutarán según lo programado. Pueden controlar activos digitales para crear nuevos tipos de aplicaciones financieras. Pueden ser descentralizadas, lo que significa que ninguna entidad o persona las controla y son casi imposibles de censurar.

## Proyectos y herramientas de Elixir {#elixir-projects-and-tools}

### Activos {#active}

- [block_keys](https://github.com/ExWeb3/block_keys) - _Implementación de BIP32 y BIP44 en Elixir (jerarquía multicuentas para billeteras deterministas)_
- [ethereumex](https://github.com/mana-ethereum/ethereumex) - _Cliente JSON-RPC de Elixir para la cadena de bloques de Ethereum_
- [ethers](https://github.com/ExWeb3/elixir_ethers) - _Una biblioteca Web3 completa para interactuar con contratos inteligentes en Ethereum usando Elixir_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) - _Una biblioteca de firma KMS para Ethers (firmar transacciones con AWS KMS)_
- [ex_abi](https://github.com/poanetwork/ex_abi) - _Implementación de analizador/decodificador/codificador de ABI de Ethereum en Elixir_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) - _Biblioteca de Elixir para calcular hashes Keccak SHA3-256 utilizando un crate de Rust tiny-keccak construido con NIF_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) - _Implementación en Elixir de la codificación RLP (Recursive Length Prefix) de Ethereum_

### Archivados / Sin mantenimiento {#archived--no-longer-maintained}

- [eth](https://hex.pm/packages/eth) - _Utilidades de Ethereum para Elixir_
- [exw3](https://github.com/hswick/exw3) - _Cliente RPC de Ethereum de alto nivel para Elixir_
- [mana](https://github.com/mana-ethereum/mana) - _Implementación de nodo completo de Ethereum escrita en Elixir_
