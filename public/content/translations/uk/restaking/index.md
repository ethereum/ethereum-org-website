Мережа Етеріум захищає мільярди доларів цінності 24/7, 365 днів на рік. Як саме?

Люди по всьому світу блокують (або «стейкають») [етер (ETH)](/what-is-ether/) у смарт-контрактах для запуску програмного забезпечення, яке обробляє транзакції Етеріуму та захищає мережу Етеріум. Натомість вони отримують винагороду у вигляді додаткового ETH.

Рестейкінг — це технологія, створена для [стейкерів](/staking/), щоб поширити цю безпеку на інші сервіси, застосунки або мережі. Натомість вони отримують додаткові винагороди за рестейкінг. Однак вони також піддають свій застейканий ETH більшому ризику.

## Що таке рестейкінг? {#what-is-restaking}

Рестейкінг — це коли стейкери використовують свій вже застейканий ETH для захисту інших децентралізованих сервісів. Натомість рестейкери можуть отримувати додаткові винагороди від цих інших сервісів на додаток до своїх звичайних винагород за стейкінг ETH.

Децентралізовані сервіси, захищені за допомогою рестейкінгу, відомі як «Активно валідовані сервіси» (Actively Validated Services, AVS).
Так само, як багато стейкерів ETH запускають програмне забезпечення для валідації Етеріуму, багато рестейкерів запускають спеціалізоване програмне забезпечення AVS.

## Стейкінг чи рестейкінг {#staking-vs-restaking}

| Стейкінг | Рестейкінг |
| ------------------------------ | ------------------------------------------------- |
| Отримання винагород в ETH | Отримання винагород в ETH + винагород AVS |
| Захищає мережу Етеріум | Захищає мережу Етеріум + AVS |
| Немає мінімуму ETH | Немає мінімуму ETH |
| Низький рівень ризику | Рівень ризику від низького до високого |
| Час виведення залежить від черги | Час виведення залежить від черги + періоду відв'язування (unbonding period) |

## Як працює рестейкінг? {#how-does-restaking-work}

У рестейкінгу беруть участь кілька суб'єктів — кожен з них відіграє важливу роль.

| **Термін** | **Опис** |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Платформи для рестейкінгу** | Платформа для рестейкінгу — це сервіс, який з'єднує AVS, стейкерів ETH та операторів. Вони створюють децентралізовані застосунки для стейкерів, щоб ті могли робити рестейкінг свого ETH, а також маркетплейси, де стейкери, AVS та оператори можуть знаходити одне одного. |
| **Нативні рестейкери** | Люди, які стейкають свій ETH, запускаючи власні валідатори Етеріуму, можуть підключити свій застейканий ETH до платформи для рестейкінгу, зокрема EigenLayer та інших, щоб отримувати винагороди за рестейкінг на додаток до винагород валідатора ETH. |
| **Ліквідні рестейкери** | Люди, які стейкають свій ETH через стороннього провайдера ліквідного стейкінгу, як-от Lido або Rocket Pool, отримують токени ліквідного стейкінгу (LST), які представляють їхній застейканий ETH. Вони можуть робити рестейкінг цих LST, щоб отримувати винагороди за рестейкінг, зберігаючи при цьому свій початковий ETH у стейкінгу. |
| **Оператори** | Оператори запускають програмне забезпечення для рестейкінгу AVS, виконуючи завдання з валідації, які вимагає кожен AVS. Оператори — це зазвичай професійні постачальники послуг, які гарантують такі речі, як час безперебійної роботи та продуктивність. Як і рестейкери, що не є операторами, оператори використовують застейканий ETH для захисту AVS, але оператори також отримують додаткові винагороди в обмін на свою роботу. |
| **AVS** | Це децентралізовані сервіси — такі як цінові оракули, мости токенів та системи даних — які отримують безпеку від рестейкерів і пропонують винагороди в токенах натомість. |

## Скільки я можу заробити на рестейкінгу? {#how-much-can-i-make-from-restaking}

Хоча AVS пропонують різні ставки, токени ліквідного рестейкінгу (LRT), такі як eETH, дають уявлення про те, скільки ви можете заробити. Так само, як ви отримуєте LST, наприклад stETH, за стейкінг вашого ETH, ви можете отримати LRT, наприклад eETH, за рестейкінг stETH. Ці токени приносять винагороди за стейкінг ETH та рестейкінг.

**Важливо усвідомлювати ризики, пов'язані з рестейкінгом. Потенційні винагороди можуть бути привабливими, але вони не позбавлені ризиків.**

## Які ризики рестейкінгу? {#what-are-the-risks-of-restaking}

| **Ризики** | **Опис** |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Штрафи (або «слешинг»)** | Як і у випадку зі стейкінгом ETH, якщо рестейкери/оператори переходять в офлайн, цензурують повідомлення або намагаються пошкодити мережу, їхній стейк може бути підданий слешингу (спалений) частково або повністю. |
| **Централізація** | Якщо кілька операторів домінують у більшій частині рестейкінгу, вони можуть мати великий вплив на рестейкерів, AVS і навіть на платформи для рестейкінгу. |
| **Ланцюгові реакції** | Якщо рестейкер піддається слешингу під час захисту кількох AVS, це може знизити безпеку для інших AVS, роблячи їх вразливими. |
| **Негайний доступ до коштів** | Існує час очікування (або «період відв'язування») для виведення застейканого ETH, тому ви не завжди можете отримати доступ до нього негайно. |
