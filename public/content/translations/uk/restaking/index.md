Мережа Ethereum забезпечує безпеку активів вартістю в мільярди доларів 24/7, 365 днів на рік. Як?

Люди по всьому світу блокують (або «стейкають») [ether (ETH)](/what-is-ether/) у смарт-контрактах, щоб запустити програмне забезпечення, яке обробляє транзакції Ethereum і захищає мережу Ethereum. Натомість вони отримують винагороду в ETH.

Рестейкінг — це технологія, створена для [стейкерів](/staking/), щоб поширити цю безпеку на інші сервіси, застосунки або мережі. Натомість вони заробляють додаткові винагороди за рестейкінг. Однак вони також піддають свій застейканий ETH більшому ризику.

## Що таке рестейкінг? {#what-is-restaking}

Рестейкінг — це коли стейкери використовують свій вже застейканий ETH для захисту інших децентралізованих сервісів. Натомість рестейкери можуть отримувати додаткові винагороди від цих інших сервісів на додаток до своїх звичайних винагород за стейкінг ETH.

Децентралізовані сервіси, захищені рестейкінгом, відомі як «активно валідовані сервіси» (AVS).
Так само як багато стейкерів ETH запускають програмне забезпечення для валідації Ethereum, багато рестейкерів запускають спеціалізоване програмне забезпечення AVS.

## Стейкінг проти рестейкінгу {#staking-vs-restaking}

| Стейкінг                         | Рестейкінг                                          |
| -------------------------------- | --------------------------------------------------- |
| Отримання винагород в ETH        | Отримання винагород в ETH + винагороди AVS          |
| Захищає мережу Ethereum          | Захищає мережу Ethereum + AVS                       |
| Немає мінімальної суми ETH       | Немає мінімальної суми ETH                          |
| Низький рівень ризику            | Рівень ризику від низького до високого              |
| Час виведення залежить від черги | Час виведення залежить від черги + період відв’язки |

## Як працює рестейкінг? Чи впливає це оновлення на всі вузли та валідаторів Ethereum? {#how-does-restaking-work}

У рестейкінгу задіяно кілька суб’єктів — кожен із них відіграє важливу роль.

| **Термін**                | **Опис**                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Платформи рестейкінгу** | Платформа для рестейкінгу — це сервіс, який з'єднує AVS, стейкерів ETH та операторів. Вони створюють децентралізовані застосунки для стейкерів, щоб ті могли рестейкати свої ETH, і торгові майданчики, де стейкери, AVS та оператори можуть знайти один одного.                                                                                                                                                                                      |
| **Нативні рестейкери**    | Люди, які стейкають свої ETH, запускаючи власні валідатори Ethereum, можуть підключити свої застейкані ETH до платформи рестейкінгу, включно з EigenLayer та іншими, щоб заробляти винагороди за рестейкінг на додаток до винагород валідатора ETH.                                                                                                                                                                                                                   |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Ліквідні рестейкери**   | Люди, які стейкають свої ETH через стороннього постачальника послуг ліквідного стейкінгу, як-от Lido або Rocket Pool, отримують ліквідні токени стейкінгу (LST), які представляють їхні застейкані ETH. Вони можуть рестейкати ці LST, щоб отримувати винагороди за рестейкінг, зберігаючи при цьому свої початкові ETH застейканими.                                                                                              |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Оператори**             | Оператори запускають програмне забезпечення для рестейкінгу AVS, виконуючи завдання валідації, яких вимагає кожен AVS. Оператори — це, як правило, професійні постачальники послуг, які гарантують такі речі, як час безвідмовної роботи та продуктивність. Як і рестейкери, що не є операторами, оператори використовують застейкані ETH для захисту AVS, але оператори також отримують додаткові винагороди в обмін на свою роботу. |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **AVS**                   | Це децентралізовані сервіси, як-от цінові оракули, токенові мости та системи даних, які отримують безпеку від рестейкерів і пропонують натомість винагороди в токенах.                                                                                                                                                                                                                                                                                                |

## Скільки я можу заробити на рестейкінгу? Як можна конвертувати Eth після хардфорку? {#how-much-can-i-make-from-restaking}

Хоча AVS пропонують різні ставки, ліквідні токени рестейкінгу (LRT), як-от eETH, дають уявлення про те, скільки ви можете заробити. Так само, як ви отримуєте LST, наприклад stETH, за стейкінг свого ETH, ви можете отримати LRT, наприклад eETH, за рестейкінг stETH. Ці токени приносять винагороди за стейкінг ETH та за рестейкінг.

**Важливо усвідомлювати ризики, пов'язані з рестейкінгом. Потенційні винагороди можуть бути привабливими, але вони не позбавлені ризику.**

## Які ризики пов’язані з рестейкінгом? {#what-are-the-risks-of-restaking}

| **Ризики**                                    | **Опис**                                                                                                                                                                                                                           |
| --------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Штрафи (або «слешинг»)** | Як і при стейкінгу ETH, якщо рестейкери/оператори переходять в офлайн, цензурують повідомлення або намагаються зашкодити мережі, їхня частка може бути частково або повністю слешена (спалена). |
| **Централізація**                             | Якщо декілька операторів домінуватимуть у більшій частині рестейкінгу, вони можуть мати великий вплив на рестейкерів, AVS і навіть на платформи рестейкінгу.                                                       |
| **Ланцюгові реакції**                         | Якщо рестейкер отримує слешинг під час захисту кількох AVS, це може знизити рівень безпеки для інших AVS, роблячи їх вразливими.                                                                                   |
| **Миттєвий доступ до коштів**                 | Існує час очікування (або «період відв’язки») для виведення рестейкнутих ETH, тому ви не завжди можете мати до них миттєвий доступ.                                                             |
