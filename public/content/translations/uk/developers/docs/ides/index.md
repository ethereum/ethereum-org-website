Коли йдеться про налаштування [інтегрованого середовища розробки (IDE)](https://wikipedia.org/wiki/Integrated_development_environment), програмування застосунків на Ethereum схоже на програмування будь-якого іншого програмного проєкту. Можна вибирати з великої кількості варіантів, тому врешті-решт виберіть той, який найкраще відповідає вашим уподобанням. Швидше за все, найкращим IDE вибором для розробки Ethereum є IDE ви вже використовуєте для традиційної розробки програмного забезпечення.

## Веб-орієнтовані IDE {#web-based-ides}

Якщо ви хочете поекспериментувати з кодом перед тим, як [налаштувати локальне середовище розробки](/developers/local-environment/), ці веб-застосунки спеціально розроблені для розробки смарт-контрактів Ethereum.

**[Remix](https://remix.ethereum.org/)** - **_Веб-орієнтована IDE із вбудованим статичним аналізом і тестовою віртуальною машиною блокчейну_**

- [Документація](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[ChainIDE](https://chainide.com/)** - **_Хмарна багатоланцюгова IDE_**

- [Документація](https://chainide.gitbook.io/chainide-english-1/)
- [Форум підтримки](https://forum.chainide.com/)

**[Replit (Solidity Starter - Beta)](https://replit.com/@replit/Solidity-starter-beta)** - **_Настроюване середовище розробки для Ethereum з гарячим перезавантаженням, перевіркою помилок і першокласною підтримкою тестових мереж_**

- [Документація](https://docs.replit.com/)

**[Tenderly Sandbox](https://sandbox.tenderly.co/)** - **_Швидке середовище для створення прототипів, у якому ви можете писати, виконувати та налагоджувати смарт-контракти в браузері, використовуючи Solidity та JavaScript_**

**[EthFiddle](https://ethfiddle.com/)** - **_Веб-орієнтована IDE, яка дає змогу писати, компілювати та налагоджувати ваші смарт-контракти_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## Настільні IDE {#desktop-ides}

Більшість прийнятих IDE мають вбудовані плагіни для покращення розробки Ethereum. Як мінімум, вони забезпечують підсвічування синтаксису для [мов смарт-контрактів](/developers/docs/smart-contracts/languages/).

**Visual Studio Code -** **_Професійна багатоплатформна IDE з офіційною підтримкою Ethereum_**

- [Visual Studio Code](https://code.visualstudio.com/)
- [Зразки коду](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**IDE від JetBrains (IntelliJ IDEA тощо)** -\*\* **_Основні інструменти для розробників програмного забезпечення та команд_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ Solidity](https://github.com/intellij-solidity/intellij-solidity/)

**Remix Desktop -** **_Спробуйте Remix IDE на вашому локальному комп'ютері_**

- [Завантажити](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)

## Плагіни та розширення {#plugins-extensions}

- [solidity](https://marketplace.visualstudio.com/items?itemName=JuanBlanco.solidity) - мова Ethereum Solidity для Visual Studio Code
- [Solidity + Hardhat for VS Code](https://marketplace.visualstudio.com/items?itemName=NomicFoundation.hardhat-solidity) - підтримка Solidity та Hardhat від команди Hardhat
- [Prettier Solidity](https://github.com/prettier-solidity/prettier-plugin-solidity) - форматувальник коду, що використовує prettier
