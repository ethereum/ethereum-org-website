## Вступ до фреймворків {#introduction-to-frameworks}

Створення повноцінного dapp вимагає
різних технологій. Фреймворк програми містить багато необхідних
функцій та надає легкі плагінні системи для вибору тих інструментів, які ви
бажаєте.

Ці фреймворки мають багато незвичайних функцій:

- Функції для створення локальної системи блокчейнів.
- Утиліти для компіляції і тестування cмарт-контрактів.
- Доповнення для розробки клієнтської частини для створення програми, орієнтованої на користувача,
  в межах того самого проєкту/репозиторію.
- Конфігурація для підключення до мереж Ethereum і розгортання
  контрактів, як на локальному екземплярі, так і в одній
  із загальнодоступних мереж Ethereum.
- Розповсюдження децентралізованих програм - інтеграції з такими
  сховищами, як IPFS.

## Передумови {#prerequisites}

Перш ніж занурюватися у фреймворки, ми рекомендуємо вам спочатку ознайомитися з нашим вступом до [dapps](/developers/docs/dapps/) та [стеку Ethereum](/developers/docs/ethereum-stack/).

## Доступні фреймворки {#available-frameworks}

**Foundry** - **_Foundry — це надзвичайно швидкий, портативний і модульний інструментарій для розробки програм на Ethereum_**

- [Установити Foundry](https://book.getfoundry.sh/)
- [Книга про Foundry](https://book.getfoundry.sh/)
- [Чат спільноти Foundry в Telegram](https://t.me/foundry_support)
- [Awesome Foundry](https://github.com/crisgarner/awesome-foundry)

**Hardhat -** **_Середовище розробки Ethereum для професіоналів._**

- [hardhat.org](https://hardhat.org)
- [GitHub](https://github.com/nomiclabs/hardhat)

**Ape -** **_Інструмент для розробки смартконтрактів для пітоністів, фахівців з даних та фахівців з безпеки._**

- [Документація](https://docs.apeworx.io/ape/stable/)
- [GitHub](https://github.com/ApeWorX/ape)

**Web3j -** **_Платформа для розробки блокчейн-додатків на JVM._**

- [Домашня сторінка](https://www.web3labs.com/web3j-sdk)
- [Документація](https://docs.web3j.io)
- [GitHub](https://github.com/web3j/web3j)

**ethers-kt -** **_Асинхронна, високопродуктивна бібліотека Kotlin/Java/Android для блокчейнів на основі EVM._**

- [GitHub](https://github.com/Kr1ptal/ethers-kt)
- [Приклади](https://github.com/Kr1ptal/ethers-kt/tree/master/examples)
- [Discord](https://discord.gg/rx35NzQGSb)

**Create Eth App -** **_Створюйте програми на базі Ethereum за допомогою однієї команди._** Відкривається з великим вибором frameworks інтерфейсу та шаблонів DeFi для вибору._\*\*

- [GitHub](https://github.com/paulrberg/create-eth-app)

**Scaffold-Eth -** **_Ethers.js + Hardhat + компоненти та хуки React для web3: усе, що вам потрібно, щоб почати створювати децентралізовані програми на основі смартконтрактів._**

**Tenderly -** **_Платформа розробки Web3, яка дозволяє розробникам блокчейнів створювати, тестувати, налагоджувати, відстежувати та використовувати смартконтракти, а також покращувати UX децентралізованих програм._**

**Catapulta -** **_Інструмент для розгортання смартконтрактів у кількох ланцюжках, що автоматизує перевірки в оглядачах блоків, відстежує розгорнуті смартконтракти та надає спільний доступ до звітів про розгортання; готове рішення для проєктів на Foundry та Hardhat._**

- [Вебсайт](https://goldrush.dev/)
