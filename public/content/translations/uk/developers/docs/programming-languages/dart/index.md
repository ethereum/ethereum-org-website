## Робота з клієнтами Ethereum {#working-with-ethereum-clients}

Ви можете використовувати Ethereum для створення децентралізованих додатків (або "dapps"), які використовують переваги криптовалюти та технології блокчейну.
Існує принаймні дві бібліотеки для Dart, що наразі підтримуються, для використання
[JSON-RPC API](/developers/docs/apis/json-rpc/) для Ethereum.

1. [Web3dart від pwa.ir](https://pub.dev/packages/web3dart)
2. [Ethereum 5.0.0 від darticulate.com](https://pub.dev/packages/ethereum)

Також існують додаткові бібліотеки, які дають змогу маніпулювати конкретними адресами Ethereum або отримувати ціни на різні криптовалюти.
[Повний список можна переглянути тут](https://pub.dev/dart/packages?q=ethereum).
