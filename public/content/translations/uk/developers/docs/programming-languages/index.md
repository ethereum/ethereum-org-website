Поширена помилка полягає в тому, що розробники повинні писати [смарт-контракти](/developers/docs/smart-contracts/), щоб створювати продукти на Етеріумі. Це не так.
Одна з переваг мережі Етеріум та її спільноти полягає в тому, що ви можете [брати участь](/community/), використовуючи майже будь-яку мову програмування.

Етеріум та його спільнота підтримують відкритий вихідний код. Ви можете знайти проєкти спільноти — реалізації клієнтів, API, фреймворки для розробки, інструменти для тестування — найрізноманітнішими мовами.

### Що робити, якщо моя мова не підтримується {#other-lang}

Якщо ви хочете додати посилання на ресурси або вказати віртуальну спільноту для додаткової мови програмування, ви можете надіслати запит на створення нової сторінки, [відкривши issue](https://github.com/ethereum/ethereum-org-website/issues/new/choose).

Якщо ви просто хочете написати код для взаємодії з блокчейном, використовуючи мову, яка наразі не підтримується, ви можете скористатися [інтерфейсом JSON-RPC](/developers/docs/apis/json-rpc/) для підключення до мережі Етеріум. Будь-яка мова програмування, яка може використовувати TCP/IP, може використовувати цей інтерфейс.
