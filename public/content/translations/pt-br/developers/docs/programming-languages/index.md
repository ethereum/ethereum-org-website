Um equívoco comum é que os desenvolvedores devem escrever [contratos inteligentes](/developers/docs/smart-contracts/) para construir no Ethereum. Isso é falso.
Uma das belezas da rede Ethereum e de sua comunidade é que você pode [participar](/community/) usando praticamente qualquer linguagem de programação.

O Ethereum e sua comunidade adotam o código aberto. Você pode encontrar projetos da comunidade - implementações de clientes, APIs, frameworks de desenvolvimento, ferramentas de teste - em uma ampla variedade de linguagens.

### E se minha linguagem não for suportada {#other-lang}

Se você quiser criar um link para recursos ou apontar para uma comunidade virtual de uma linguagem de programação adicional, você pode solicitar uma nova página [abrindo uma issue](https://github.com/ethereum/ethereum-org-website/issues/new/choose).

Se você apenas deseja escrever código para interagir com a blockchain usando uma linguagem atualmente não suportada, você pode usar a [interface JSON-RPC](/developers/docs/apis/json-rpc/) para se conectar à rede Ethereum. Qualquer linguagem de programação que possa usar TCP/IP pode usar essa interface.
