## Trabalhando com clientes Ethereum {#working-with-ethereum-clients}

Você pode usar o Ethereum para criar aplicativos descentralizados (ou "dapps") que utilizam os benefícios da criptomoeda e da tecnologia blockchain.
Existem pelo menos duas bibliotecas atualmente mantidas para Dart para usar a [API JSON-RPC](/developers/docs/apis/json-rpc/) para o Ethereum.

1. [Web3dart da pwa.ir](https://pub.dev/packages/web3dart)
1. [Ethereum 5.0.0 da darticulate.com](https://pub.dev/packages/ethereum)

Também existem bibliotecas adicionais que permitem manipular endereços específicos do Ethereum, ou que permitem recuperar preços de várias criptomoedas.
[Você pode ver a lista completa aqui](https://pub.dev/dart/packages?q=ethereum).
