## Overview {#overview}

Ini adalah versi terbaru dari halaman referensi EVM di [wolflo/evm-opcodes](https://github.com/wolflo/evm-opcodes).
Juga diambil dari implementasi [Yellow Paper](https://ethereum.github.io/yellowpaper/paper.pdf), [Jello Paper](https://jellopaper.org/evm/), dan [geth](https://github.com/ethereum/go-ethereum).
Ini dimaksudkan untuk menjadi referensi yang dapat diakses, tetapi tidak terlalu ketat.
Jika Anda ingin memastikan kebenaran dan memahami setiap kemungkinan kasus tepi, disarankan untuk menggunakan Jello Paper atau implementasi klien.

Mencari referensi interaktif? Kunjungi [evm.codes](https://www.evm.codes/).

Untuk operasi dengan biaya gas dinamis, lihat [gas.md](https://github.com/wolflo/evm-opcodes/blob/main/gas.md).
