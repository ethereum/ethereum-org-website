Dalam hal menyiapkan [lingkungan pengembangan terpadu (IDE)](https://wikipedia.org/wiki/Integrated_development_environment), memprogram aplikasi di Ethereum mirip dengan memprogram proyek perangkat lunak lainnya. Ada banyak pilihan yang dapat dipilih, jadi pada akhirnya, pilihlah IDE atau editor kode yang paling sesuai dengan preferensi Anda. Kemungkinan besar pilihan IDE terbaik untuk pengembangan Ethereum Anda adalah IDE yang sudah Anda gunakan untuk pengembangan perangkat lunak tradisional.

## IDE berbasis web {#web-based-ides}

Jika Anda ingin mengutak-atik kode sebelum Anda [menyiapkan lingkungan pengembangan lokal](/developers/local-environment/), aplikasi web ini dibuat khusus untuk pengembangan kontrak pintar Ethereum.

**[Remix](https://remix.ethereum.org/)** - **_IDE berbasis web dengan analisis statis bawaan, dan mesin virtual blockchain pengujian_**

- [Dok](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[ChainIDE](https://chainide.com/)** - **_IDE multi-chain berbasis cloud_**

- [Dok](https://chainide.gitbook.io/chainide-english-1/)
- [Forum bantuan](https://forum.chainide.com/)

**[Replit (Solidity Starter - Beta)](https://replit.com/@replit/Solidity-starter-beta)** - **_Lingkungan pengembangan yang dapat disesuaikan untuk Ethereum dengan pemuatan ulang cepat (hot reloading), pemeriksaan kesalahan, dan dukungan testnet kelas satu_**

- [Dok](https://docs.replit.com/)

**[Tenderly Sandbox](https://sandbox.tenderly.co/)** - **_Lingkungan pembuatan prototipe cepat di mana Anda dapat menulis, mengeksekusi, dan men-debug kontrak pintar di peramban menggunakan Solidity dan JavaScript_**

**[EthFiddle](https://ethfiddle.com/)** - **_IDE berbasis web yang memungkinkan Anda menulis, mengompilasi, dan men-debug kontrak pintar Anda_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## IDE Desktop {#desktop-ides}

Sebagian besar IDE yang sudah mapan telah membangun plugin untuk meningkatkan pengalaman pengembangan Ethereum. Setidaknya, mereka menyediakan penyorotan sintaksis untuk [bahasa kontrak pintar](/developers/docs/smart-contracts/languages/).

**Visual Studio Code -** **_IDE lintas platform profesional dengan dukungan resmi Ethereum_**

- [Visual Studio Code](https://code.visualstudio.com/)
- [Sampel kode](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**IDE JetBrains (IntelliJ IDEA, dll.) -** **_Alat penting untuk pengembang dan tim perangkat lunak_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ Solidity](https://github.com/intellij-solidity/intellij-solidity/)

**Remix Desktop -** **_Rasakan pengalaman Remix IDE di mesin lokal Anda_**

- [Unduh](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)

## Plugin dan ekstensi {#plugins-extensions}

- [solidity](https://marketplace.visualstudio.com/items?itemName=JuanBlanco.solidity) - Bahasa Solidity Ethereum untuk Visual Studio Code
- [Solidity + Hardhat for VS Code](https://marketplace.visualstudio.com/items?itemName=NomicFoundation.hardhat-solidity) - Dukungan Solidity dan Hardhat oleh tim Hardhat
- [Prettier Solidity](https://github.com/prettier-solidity/prettier-plugin-solidity) - Pemformat kode menggunakan prettier
