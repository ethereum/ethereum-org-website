## Pengenalan {#introduction}

Banyak standar pengembangan Ethereum yang berfokus pada antarmuka token. Standar-standar ini membantu memastikan kontrak pintar tetap dapat disusun, sehingga ketika sebuah proyek baru menerbitkan token, token tersebut tetap kompatibel dengan bursa terdesentralisasi dan aplikasi yang sudah ada.

Standar token mendefinisikan bagaimana token berperilaku dan berinteraksi di seluruh ekosistem Ethereum. Standar ini memudahkan pengembang untuk membangun tanpa harus memulai dari awal, memastikan bahwa token bekerja dengan lancar dengan dompet, bursa, dan platform DeFi. Baik dalam permainan, tata kelola, atau kasus penggunaan lainnya, standar-standar ini memberikan konsistensi dan membuat Ethereum lebih saling terhubung.

## Standar token {#token-standards}

Berikut adalah beberapa standar token paling populer di Ethereum:

- [ERC-20](/developers/docs/standards/tokens/erc-20/) - Antarmuka standar untuk token fungible (dapat dipertukarkan), seperti token pemungutan suara, token staking, atau mata uang virtual.

### Standar NFT {#nft-standards}

- [ERC-721](/developers/docs/standards/tokens/erc-721/) - Antarmuka standar untuk token non-fungible, seperti akta untuk karya seni atau lagu.
- [ERC-1155](/developers/docs/standards/tokens/erc-1155/) - ERC-1155 memungkinkan perdagangan dan pengelompokan transaksi yang lebih efisien – sehingga menghemat biaya. Standar token ini memungkinkan untuk membuat kedua token utilitas (seperti $BNB atau $BAT) dan Token yang Tak Dapat Dipertukarkan seperti CryptoPunk.

Daftar lengkap proposal [ERC](https://eips.ethereum.org/erc).
