## Pengantar {#introduction}

Banyak standar pengembangan [Ethereum](/) berfokus pada antarmuka token. Standar-standar ini membantu memastikan kontrak pintar tetap dapat disusun, sehingga ketika sebuah proyek baru menerbitkan token, token tersebut tetap kompatibel dengan bursa dan aplikasi terdesentralisasi yang ada.

Standar token mendefinisikan bagaimana token berperilaku dan berinteraksi di seluruh ekosistem Ethereum. Standar ini memudahkan pengembang untuk membangun tanpa harus membuat ulang dari awal, memastikan bahwa token berfungsi dengan lancar bersama dompet, bursa, dan platform DeFi. Baik dalam permainan, tata kelola, atau kasus penggunaan lainnya, standar-standar ini memberikan konsistensi dan membuat Ethereum lebih saling terhubung.

## Standar token {#token-standards}

Berikut adalah beberapa standar token paling populer di Ethereum:

- [ERC-20](/developers/docs/standards/tokens/erc-20/) - Antarmuka standar untuk token yang sepadan (dapat dipertukarkan), seperti token suara, token staking, atau mata uang virtual.

### Standar NFT {#nft-standards}

- [ERC-721](/developers/docs/standards/tokens/erc-721/) - Antarmuka standar untuk token yang tidak sepadan (non-fungible), seperti akta untuk karya seni atau lagu.
- [ERC-1155](/developers/docs/standards/tokens/erc-1155/) - ERC-1155 memungkinkan perdagangan dan penggabungan transaksi yang lebih efisien – sehingga menghemat biaya. Standar token ini memungkinkan pembuatan token utilitas (seperti $BNB atau $BAT) dan Token Non-Fungible (NFT) seperti CryptoPunks.

Daftar lengkap proposal [ERC](https://eips.ethereum.org/erc).
