<FeaturedText>Pelajari cara mengembangkan untuk Ethereum menggunakan proyek dan perkakas berbasis Elixir.</FeaturedText>

Gunakan Ethereum untuk membuat aplikasi terdesentralisasi (atau "dapps") yang memanfaatkan keuntungan dari mata uang kripto dan teknologi blockchain. Dapps ini bisa bersifat trustless, yang berarti setelah disebarkan ke Ethereum, mereka akan selalu berjalan sesuai dengan yang diprogramkan. Mereka dapat mengontrol aset digital untuk membuat jenis aplikasi keuangan baru. Mereka dapat didesentralisasi, yang berarti tidak ada entitas atau orang tunggal yang mengendalikannya dan hampir tidak mungkin untuk disensor.

## Proyek dan perkakas Elixir {#elixir-projects-and-tools}

### Aktif {#active}

- [block_keys](https://github.com/ExWeb3/block_keys) - _Implementasi BIP32 & BIP44 di Elixir (Hierarki Multi-Akun untuk Dompet Deterministik)_
- [ethereumex](https://github.com/mana-ethereum/ethereumex) - _Klien JSON-RPC Elixir untuk blockchain Ethereum_
- [ethers](https://github.com/ExWeb3/elixir_ethers) - _Pustaka Web3 komprehensif untuk berinteraksi dengan kontrak pintar di Ethereum menggunakan Elixir_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) - _Pustaka penandatangan KMS untuk Ethers (menandatangani transaksi dengan AWS KMS)_
- [ex_abi](https://github.com/poanetwork/ex_abi) - _Implementasi parser/dekoder/enkoder ABI Ethereum di Elixir_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) - _Pustaka Elixir untuk menghitung hash Keccak SHA3-256 menggunakan NIF yang dibangun dari crate Rust tiny-keccak_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) - _Implementasi Elixir dari pengodean RLP (Recursive Length Prefix) Ethereum_

### Diarsipkan / Tidak lagi dipelihara {#archived--no-longer-maintained}

- [eth](https://hex.pm/packages/eth) - _Utilitas Ethereum untuk Elixir_
- [exw3](https://github.com/hswick/exw3) - _Klien RPC Ethereum tingkat tinggi untuk Elixir_
- [mana](https://github.com/mana-ethereum/mana) - _Implementasi node penuh Ethereum yang ditulis dalam Elixir_
