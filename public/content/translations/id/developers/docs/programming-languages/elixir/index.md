<FeaturedText>Pelajari cara mengembangkan di Ethereum dengan menggunakan proyek dan tooling berbasis Elixir.</FeaturedText>

Gunakan Ethereum untuk membuat aplikasi terdesentralisasi (atau "dapp") yang menggunakan keuntungan teknologi mata uang kripto dan blockchain. Dapps ini bisa tanpa kepercayaan, artinya setelah mereka diterapkan di Ethereum, mereka akan selalu berjalan sesuai yang diprogram. Mereka dapat mengontrol aset digital untuk membuat jenis aplikasi keuangan baru. Dapp juga terdesentralisasi, yang berarti tidak ada entitas atau orang yang mengontrolnya dan hampir mustahil untuk disensor.

## Proyek dan alat Elixir {#elixir-projects-and-tools}

### Aktif {#active}

- [block_keys](https://github.com/ExWeb3/block_keys)-Implementasi BIP32 & BIP44 di Elixir (Hierarki Multi-Akun untuk Dompet Deterministik)_
- ethereumex](https://github.com/mana-ethereum/ethereumex) - _Elixir JSON-RPC client for the Ethereum blockchain_
- [ethers](https://github.com/ExWeb3/elixir_ethers) - Library Web3 yang komprehensif untuk berinteraksi dengan smart contract di Ethereum menggunakan Elixir_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) - Library penandatangan KMS untuk Ethers (menandatangani transaksi dengan AWS KMS)_
- [ex_abi](https://github.com/poanetwork/ex_abi) - _Ethereum ABI parser/decoder/encoder implementation in Elixir_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) - Library Elixir untuk menghitung hash Keccak SHA3-256 menggunakan NIF yang dibangun dari crate Rust tiny-keccak_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) - _Elixir implementation of Ethereum's RLP (Recursive Length Prefix) encoding_

### Diarsipkan / Tidak lagi dipelihara {#archived--no-longer-maintained}

- [eth](https://hex.pm/packages/eth) - _Ethereum utilities for Elixir_
- [exw3](https://github.com/hswick/exw3) - _level Tinggi Ethereum RPC Client untuk Elixir_
- [mana](https://github.com/mana-ethereum/mana) - Implementasi full node Ethereum yang ditulis dalam Elixir_
