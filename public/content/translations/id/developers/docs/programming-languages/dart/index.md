## Bekerja dengan klien Ethereum {#working-with-ethereum-clients}

Anda dapat menggunakan Ethereum untuk membuat aplikasi terdesentralisasi (atau "dapps") yang menggunakan manfaat dari teknologi blockchain dan mata uang kripto.
Saat ini ada setidaknya dua pustaka yang masih dikelola untuk Dart guna menggunakan
[JSON-RPC API](/developers/docs/apis/json-rpc/) untuk Ethereum.

1. [Web3dart dari pwa.ir](https://pub.dev/packages/web3dart)
2. [Ethereum 5.0.0 dari darticulate.com](https://pub.dev/packages/ethereum)

Ada juga pustaka tambahan yang memungkinkan Anda memanipulasi alamat khusus Ethereum,
atau yang memungkinkan Anda mengambil harga dari berbagai mata uang kripto.
[Anda dapat melihat daftar lengkapnya di sini](https://pub.dev/dart/packages?q=ethereum).
