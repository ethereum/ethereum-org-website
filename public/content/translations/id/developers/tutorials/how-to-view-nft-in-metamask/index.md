Tutorial ini adalah Bagian 3/3 dalam seri Tutorial NFT, di mana kita melihat NFT kita yang baru di-mint. Namun, Anda dapat menggunakan tutorial umum ini untuk token ERC-721 apa pun menggunakan MetaMask, termasuk di mainnet atau testnet apa pun. Jika Anda ingin mempelajari cara melakukan mint NFT Anda sendiri di Ethereum, Anda harus memeriksa [Bagian 1 tentang Cara Menulis & Menerapkan kontrak pintar NFT](/developers/tutorials/how-to-write-and-deploy-an-nft)!

Selamat! Anda telah berhasil mencapai bagian terpendek dan paling sederhana dari seri tutorial NFT kami — cara melihat NFT Anda yang baru di-mint di dompet virtual. Kita akan menggunakan MetaMask untuk contoh ini karena itulah yang kita gunakan di dua bagian sebelumnya.

Sebagai prasyarat, Anda harus sudah menginstal MetaMask di ponsel, dan itu harus menyertakan akun tempat Anda melakukan mint NFT Anda — Anda bisa mendapatkan aplikasinya secara gratis di [iOS](https://apps.apple.com/us/app/metamask-blockchain-wallet/id1438144202) atau [Android](https://play.google.com/store/apps/details?id=io.metamask&hl=en_US&gl=US).

## Langkah 1: Atur jaringan Anda ke Sepolia {#set-network-to-sepolia}

Di bagian atas aplikasi, tekan tombol "Wallet" (Dompet), setelah itu Anda akan diminta untuk memilih jaringan. Karena NFT kita di-mint di jaringan Sepolia, Anda harus memilih Sepolia sebagai jaringan Anda.

![Cara mengatur Sepolia sebagai jaringan Anda di MetaMask Mobile](./goerliMetamask.gif)

## Langkah 2: Tambahkan barang koleksi Anda ke MetaMask {#add-nft-to-metamask}

Setelah Anda berada di jaringan Sepolia, pilih tab "Collectibles" (Barang Koleksi) di sebelah kanan dan tambahkan alamat kontrak pintar NFT dan ID token ERC-721 dari NFT Anda — yang seharusnya dapat Anda temukan di Etherscan berdasarkan hash transaksi dari NFT Anda yang diterapkan di Bagian II dari tutorial kami.

![Cara menemukan hash transaksi dan ID token ERC-721 Anda](./findNFTEtherscan.png)

Anda mungkin perlu menyegarkan beberapa kali untuk melihat NFT Anda — tetapi itu akan ada di sana <Emoji text="😄" size={1} />!

![Cara mengunggah NFT Anda ke MetaMask](./findNFTMetamask.gif)

Selamat! Anda telah berhasil melakukan mint NFT, dan sekarang Anda dapat melihatnya! Kami tidak sabar untuk melihat bagaimana Anda akan menggemparkan dunia NFT!
