これは遠い夢ではありません。今日、イーサリアム上で実際に起きていることです。伝統的な金融機関は何十年もかけて堅牢な決済システムを構築してきましたが、国境、営業時間、レガシーなインフラによる制約を依然として受けています。イーサリアムは新しいパラダイムを提供します。それは、インターネットにアクセスできる人なら誰でも、ほぼ瞬時にプログラム可能なトランザクションを実行できる、グローバルで24時間365日稼働する金融プラットフォームです。

## グローバル通貨へのアクセス {#access-to-global-currencies}

多くの国において、インフレは差し迫った懸念事項であり、しばしば外貨へのアクセス制限を伴います。このような状況にある人々は、急速に価値が下がる貯蓄を保持せざるを得ず、資産を維持するのに苦労しています。

イーサリアムコミュニティは、いかなる国の金融政策や管理からも独立した、**堅牢な代替金融システム**を構築しました。

イーサリアムユーザーは、**米ドルのような強力な通貨に通常連動するトークンであるステーブルコイン**を使用できます。暗号資産で稼ぎ、貯蓄することで、人々は自国の高いインフレから身を守り、購買力を維持、あるいは向上させることができます。これにより、国内および世界中の両方で、商品やサービスに対する支払いがより簡単になります。

- **Travala.com:** この旅行予約プラットフォームでは、ユーザーはイーサリアムを使用してホテルやフライトの支払いを行うことができます。

## セルフカストディ型の暗号資産カードでの支払い {#pay-with-self-custodial-crypto-cards}

セルフカストディ型の暗号資産カードは、お金を他人のヴォールトに閉じ込めるのではなく、自分のバックパックを使うように機能します。従来のカードでは、銀行やカストディアンが資金を保持し、使用時にそれを解放します。セルフカストディ型カードでは、仲介者なしで常に資産を管理し続けながら、タップやスワイプでコーヒー、食料品、さらにはフライトの支払いを行うことができます。

これらのカードは、ノン・カストディアルなウォレットやスマート・コントラクトのアカウントに直接リンクしており、ユーザーは所有権を手放すことなく、日常の場面でETHやステーブルコインを使用できます。ユーザーが第三者に資金を預ける必要があるカストディアル型カードとは異なり、セルフカストディ型カードは、オンチェーンでのコントロールを維持しながら、VisaやMastercardなどの現実世界での支払いを可能にします。

### 例 {#crypto-cards-examples}

- **メタマスクカード:** メタマスクのウォレットにリンクされたこのMastercardデビットカードを使用すると、ユーザーはETH、ステーブルコイン、およびその他のサポートされているトークンを使用できます。Apple PayとGoogle Payをサポートし、暗号資産のキャッシュバック報酬が含まれており、利回り獲得のオプションを提供します。

- **Tuyoカード:** Visaが利用できる場所ならどこでも使用できるように、暗号資産を自動的にUSDCに変換するスマート・コントラクトベースのVisaカードです。ユーザーは資産のセルフカストディを維持し、利回り、取引、および支払い機能にアクセスできます。

- **Gnosis Pay:** Gnosis Safeのスマート・アカウントに結び付けられた初のセルフカストディ型Visaカードです。ユーザーは、ガス代、為替手数料、またはオフ・ランプ手数料なしで、ウォレットから直接暗号資産を使用できます。Ethereum Name Service (ENS) を介したカードのパーソナライズもサポートされています。

- **Ether.Fi Cashカード:** ether.fiのステーキングプロトコルと統合されたこのカードを使用すると、ユーザーはETHをステークしたまま支払いに使用できます。支払いはスマート・コントラクトを介して処理され、支払い中もセルフカストディが維持されます。

### セルフカストディ型暗号資産カードの比較 {#crypto-card-custody-comparison}

| **暗号資産カード** | **セルフカストディ** | **ノン・カストディアル** | **主な特徴**                                                |
| --------------- | :----------------: | :---------------: | ------------------------------------------------------------ |
| メタマスクカード   |         ✅         |        ✅         | ウォレットはメタマスク内に留まる。支払い時に自動でオフ・ランプ           |
| Tuyoカード       |         ✅         |        ✅         | スマートウォレットがUSDCに変換。ユーザーがコントロールを維持          |
| Gnosis Pay      |         ✅         |        ✅         | ユーザーのGnosis Safeにリンク。使用中のカストディ移行なし    |
| Ether.Fi Cash   |         ✅         |        ✅         | ETHはステークされたまま。スマート・コントラクトが支払いアクセスを制御 |

> **注:** 「セルフカストディ」とは、ユーザーが資金への完全なアクセスとコントロールを持つ、ユーザー管理のウォレットを指します。
> 「ノン・カストディアル」とは、多くの場合スマート・コントラクトを通じて、第三者のカストディなしで資金が管理されるウォレットを指します。
> すべてのセルフカストディ型カードはノン・カストディアルですが、すべてのノン・カストディアル型カードがセルフカストディ型であるとは限りません。

## ウェブサイトとエージェント向けのマイクロペイメント (x402) {#x402}

[x402](https://www.x402.org/)は、ウェブにネイティブな従量課金制の支払いをもたらすオープンな支払い標準です。低コストの[イーサリアムのレイヤー2 (L2) ネットワーク](/layer-2/)上で[ステーブルコイン](/stablecoins/)を使用することで、x402標準は、APIキーやサブスクリプションを管理したり、広告に注意を向けることで「支払う」のではなく、ニュース記事を読んだりAPIを呼び出したりするような単一のアクションに対して、人間や機械が直接支払うことを経済的に可能にします。

- **ペイウォールとログインの排除:** 1つのニュース記事を読むためにアカウントを作成して個人情報を共有する代わりに、ウォレットからロック解除に必要な数セントを支払うことができます。
- **AIエージェントの支払い:** x402により、自律型ソフトウェア (「AIエージェント」) は、人間の介入なしに、機能するために必要なデータやAPI呼び出しの支払いを行うことができます。

### x402支払い標準の仕組み {#x402-how}

クライアントがリソースをリクエストすると、サーバーは`402 Payment Required`エラーコードとともに支払い指示 (価格、アカウント、サポートされているトークンとチェーン) を送信します。

- [ウォレット](/wallets/)がリクエストを検出し、支払いを処理します (多くの場合、ワンクリックで承認するか、事前に承認されたアローワンスを使用して自動的に行われます)。
- 事前に承認されたウォレット残高にアクセスできる[AIエージェント](/ai-agents/)は、価格を自動的に検出し、即座に支払いを行ってデータやサービスにアクセスできます。
- クライアントは、サポートされているステーブルコインのいずれかをウォレットに持っている必要がありますが、[ガス代](/gas/)のためにETHを持っている必要はありません。

これにより、AIエージェントが自らリソースを購入し、APIサービスにより効率的にアクセスできる、新しい「マシン・ツー・マシン」経済が解き放たれます。

署名されたメッセージはサーバーに配信されます。サーバーは通常、ブロックチェーンの複雑さ (トランザクションの送信、支払いの取得、ガス代の促進など) を処理するために[x402ファシリテーター](https://x402.gitbook.io/x402/core-concepts/facilitator)を使用します。つまり、開発者は支払いインフラストラクチャを管理することなく、暗号資産のマイクロペイメントを簡単に受け入れることができます。

## 給与の支払い {#salary-payments}

多くの先進的な企業が現在、従業員に対し、給与の全額または一部をイーサ (ETH) などの暗号資産で受け取るオプションを提供しています。

- **Gipsybee:** 電子機器、ロボット工学、ゲーム制作、その他のサービスを扱う組織です。従業員にイーサリアムで支払いを受けるオプションを提供しています。
- **SC5:** このフィンランドの企業は、ビットコインでの給与支払いをいち早く提供した企業の1つであり、イーサリアムでの同様の取り決めの道を開きました。
- **ブロックチェーンのスタートアップ:** ブロックチェーン分野の多くの企業は、当然のことながら従業員に暗号資産での給与オプションを提供しています。
- **DAO:** DAOへの貢献者の特異性と多様性のため、ほとんどの貢献と給与は暗号資産で報われます。

この傾向は、国境のない支払いや、有利になる可能性のある為替レートの恩恵を受けることができるリモートワーカーやデジタルノマドにとって特に魅力的です。

## グローバルな救援活動 {#global-relief-efforts}

2023年2月、トルコとシリアを壊滅的な地震が襲った際、世界の暗号資産コミュニティはすぐに行動を起こしました。救援活動のための資金を集めるさまざまなキャンペーンが立ち上げられ、危機の時代におけるイーサリアムの力を示しました。トルコでは暗号資産が[認められた支払い形態ではない](https://www.reuters.com/technology/no-more-kebabs-bitcoins-turkeys-crypto-payment-ban-looms-2021-04-28/)にもかかわらず、当局は一部の組織が寄付を集めるための[例外](https://x.com/haluklevent/status/1622913175409623041)を設けました。いくつかの例を挙げます。

- [Refik Anadol](https://x.com/refikanadol/status/1622623521104089090): 資金調達キャンペーンを開始した著名なデジタルアーティストです。
- DAOの力: [Anka Relief DAO](https://ankarelief.org/)と[Bankless DAO](https://x.com/banklessDAO)が[Giveth](https://x.com/Giveth/status/1623493672149843969)と協力して資金を調達しました。
- 著名なNFTアーティストである[Pak](https://cause.quest/)もこの活動に貢献しました。
- イーサリアムの共同創設者である[ヴィタリック・ブテリン](https://cointelegraph.com/news/vitalik-buterin-donates-227k-to-help-earthquake-victims-in-turkey-syria)でさえ、複数のキャンペーンに個人的な寄付を行いました。

この結果はどうなったでしょうか？ [Dune](https://dune.com/davy42/turkiye-earthquake-donations) Analyticsのダッシュボードで追跡されたように、数日のうちに600万ドル以上が調達されました。

インドやウクライナで起きた悲劇に対しても、同様の迅速な対応が見られました。この迅速な対応は、通貨の変換、時間のかかる銀行振込、または法外な手数料といったハードルなしに、グローバルな支援を迅速に動員できるという、イーサリアムでの支払いの重要な利点を浮き彫りにしています。

## イーサリアムでの暗号資産決済と法定通貨決済の比較 {#ethereum-vs-fiat}

イーサリアムでの支払いの影響を真に理解するには、従来の法定通貨と比較する価値があります。

|                       | **イーサリアム**            | **従来の銀行**                                        |
| --------------------- | ----------------------- | ------------------------------------------------------------ |
| **スピード**             | 数秒から数分      | 数時間から数日                                                |
| **グローバルな到達範囲**      | 国境なし、24時間365日        | 国際的な銀行の制限や営業時間の対象となる |
| **透明性**      | 完全に透明       | 機関によって異なる                                        |
| **プログラマビリティ**   | スマート・コントラクトが有効 | 基本的なトランザクションに限定                                |
| **インフレ制御** | 予測可能な発行    | 中央銀行の政策の対象となる                             |
| **アクセシビリティ**     | インターネットがあれば誰でも    | 国内および国際的な制限の対象となる           |

根本的に、イーサリアムは安全で高速、かつ透明性の高いトランザクションを可能にする分散型プラットフォームです。しかし、多くの要素が従来の支払い方法とは一線を画しています。イーサリアムでの支払いをゲームチェンジャーにする利点について詳しく見ていきましょう。

### プログラマビリティ {#programmability}

イーサリアムのユニークな機能の1つは、スマート・コントラクトをサポートする機能です。スマート・コントラクトは、条件がコードに直接書き込まれた自己実行型の契約です。これにより、以下のようなトランザクションを大幅に改善できる、自動化された条件ベースの支払いの可能性が広がります。

- エスクローサービス
- 定期支払い
- 成果報酬

### スピード {#speed}

国際銀行振込が完了するまで何日も待った最後の時のことを覚えていますか？ 長い行列？ そして記入しなければならなかった複数のフォーム？ イーサリアムがあれば、そんな日々は遠い昔のことです。イーサリアムネットワーク上のトランザクションは、送信者と受信者の場所に関係なく、数分で決済されます。イーサリアムはパーミッションレスであるため、送金時に規制上の官僚主義はありません。このスピードは、緊急の救援活動など、時間に敏感な状況において特に重要です。

### より低い手数料 {#lower-fees}

従来の国際送金手数料は、特に数百ドルのトランザクションを扱う場合、送金額のかなりの部分を食いつぶすことがあります。イーサリアムのトランザクションは無料ではありませんが、多くの場合、より低い手数料で済みます。これは、仲介業者の懐を潤すのではなく、より多くのお金が意図した場所に届くことを意味します。

### 透明性 {#transparency}

イーサリアムのブロックチェーン上のすべてのトランザクションは、公開台帳に記録されます。これは、誰でも資金の移動を検証できることを意味し、以下のための優れたツールになります。

- 慈善団体が寄付金の使途を証明するため
- 企業がサプライヤーや従業員への支払いを証明するため
- 個人が自身の財務活動を追跡するため

イーサリアムを使用すると、これらのほとんどが不明なままである従来の組織とは異なり、お金がどのように移動し、コストがどのように適用されるかを誰もが確認できます。

法定通貨には広く受け入れられ、安定しているという利点がありますが、イーサリアムは特定の種類のトランザクションにとって魅力的な選択肢となる独自の利点を提供します。

迅速な災害救援の促進からグローバルな労働者のエンパワーメントまで、イーサリアムでの支払いは、お金の長い歴史に新しい章を書き加えています。課題は残っていますが、このテクノロジーが提供する独自の利点により、幅広いユースケースにとって魅力的な選択肢となっています。
