ウォレットのすべての基本機能を操作する方法を学びましょう。まだウォレットを持っていない場合は、[イーサリアムアカウントの作成方法](/guides/how-to-create-an-ethereum-account/)をご覧ください。

## ウォレットを開く {#open-your-wallet}

残高が表示され、トークンを送受信するためのボタンが含まれたダッシュボードが表示されるはずです。

## 暗号資産を受け取る {#receive-cryptocurrency}

ウォレットに暗号資産を受け取りたいですか？

各イーサリアムアカウントには、数字と文字の固有のシーケンスである独自の受取アドレスがあります。アドレスは銀行の口座番号のように機能します。イーサリアムのアドレスは常に「0x」で始まります。このアドレスは誰とでも共有でき、共有しても安全です。

アドレスは自宅の住所のようなものです。あなたを見つけてもらうためには、それが何であるかを人々に伝える必要があります。住んでいる場所を知られても、自分だけが管理する別の鍵で玄関のドアを施錠できるため、誰も中に入ることはできず、安全です。

あなたに送金したい人には、あなたのパブリックアドレスを教える必要があります。多くのウォレットアプリでは、アドレスをコピーしたり、スキャン用のQRコードを表示したりして、簡単に使用できるようになっています。イーサリアムのアドレスを手動で入力することは避けてください。入力ミスにより、資金を失う原因になりやすいからです。

アプリによって異なったり、違う用語が使われたりする場合がありますが、資金を送金しようとする場合は、同様のプロセスを経るはずです。

1. ウォレットアプリを開きます。
2. 「受信（Receive）」（または同様の表現のオプション）をクリックします。
3. イーサリアムのアドレスをクリップボードにコピーします。
4. 送信者に受取用のイーサリアムアドレスを伝えます。

## 暗号資産を送る {#send-cryptocurrency}

別のウォレットにETHを送りたいですか？

1. ウォレットアプリを開きます。
2. 受取アドレスを取得し、受取人と同じネットワークに接続していることを確認します。
3. アドレスを手動で入力しなくて済むように、受取アドレスを入力するか、カメラでQRコードをスキャンします。
4. ウォレットの「送信（Send）」ボタン（または同様の表現のボタン）をクリックします。

![Send field for crypto address](./send.png)
<br/>

5. DAIやUSDCのような多くの資産は、複数のネットワーク上に存在します。暗号資産トークンを送金する際は、これらは互換性がないため、受取人があなたと同じネットワークを使用していることを確認してください。
6. ネットワークの状況によって変動するトランザクション手数料をカバーするのに十分なETHがウォレットにあることを確認してください。ほとんどのウォレットは、推奨される手数料をトランザクションに自動的に追加し、それを確認できるようになっています。
7. トランザクションが処理されると、対応する暗号資産の金額が受取人のアカウントに表示されます。ネットワークの現在の使用状況に応じて、これには数秒から数分かかる場合があります。

## プロジェクトへの接続 {#connecting-to-projects}

あなたのアドレスは、すべてのイーサリアムプロジェクトで同じになります。プロジェクトごとに個別に登録する必要はありません。ウォレットを持っていれば、追加情報なしで任意のイーサリアムプロジェクトに接続できます。メールアドレスやその他の個人情報は必要ありません。

1. 任意のプロジェクトのウェブサイトにアクセスします。
2. プロジェクトのランディングページがプロジェクトの静的な説明のみである場合、メニューの「アプリを開く（Open the App）」ボタンをクリックすると、実際のウェブアプリに移動できるはずです。
3. アプリに入ったら、「接続（Connect）」をクリックします。

![Button allowing user to connect to the website with a wallet](./connect1.png)

4. 提供されたオプションリストからウォレットを選択します。ウォレットが見当たらない場合は、「WalletConnect」オプションの下に隠れている可能性があります。

![Selecting from a list of wallets to connect with](./connect2.png)

5. 接続を確立するために、ウォレットで署名リクエストを確認します。**このメッセージへの署名にETHを消費する必要はありません**。
6. これで完了です！アプリを使い始めましょう。興味深いプロジェクトは、私たちの[分散型アプリケーション (dapp) ページ](/apps/#explore)で見つけることができます。

## よくある質問 {#frequently-asked-questions}

### ETHのアドレスを持っていれば、他のブロックチェーンでも同じアドレスを持てるのですか？ {#if-i-own-an-eth-address-do-i-own-the-same-address-on-other-blockchains}

（リカバリーフレーズを持つタイプのウォレットであれば）すべてのEVM互換ブロックチェーンで同じアドレスを使用できます。この[リスト](https://chainlist.org/)で、同じアドレスを使用できるブロックチェーンを確認できます。ビットコインのようないくつかのブロックチェーンは、完全に異なるネットワークルールのセットを実装しているため、異なるフォーマットの異なるアドレスが必要になります。スマート・コントラクトウォレットをお持ちの場合は、サポートされているブロックチェーンの詳細について、その製品のウェブサイトを確認してください。

### 複数のデバイスで同じアドレスを使用できますか？ {#can-i-use-the-same-address-on-multiple-devices}

はい、複数のデバイスで同じアドレスを使用できます。ウォレットは技術的には残高を表示し、トランザクションを行うためのインターフェースにすぎず、アカウントはウォレットの中ではなく、ブロックチェーン上に保存されています。

### 暗号資産を受け取っていません。トランザクションのステータスはどこで確認できますか？ {#i-have-not-received-the-crypto-where-can-i-check-the-status-of-a-transaction}

[ブロックエクスプローラー](/developers/docs/data-and-analytics/block-explorers/)を使用すると、あらゆるトランザクションのステータスをリアルタイムで確認できます。必要なのは、ウォレットアドレスまたはトランザクションのIDを検索することだけです。

### トランザクションをキャンセルしたり、元に戻したりすることはできますか？ {#can-i-cancel-or-return-transactions}

いいえ、トランザクションが一度承認されると、そのトランザクションをキャンセルすることはできません。
