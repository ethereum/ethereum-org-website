開発者が自信を持って構築し、必要なサポートを受けられるように、可能な限り最高レベルの開発者リソースを掲載したいと考えています。

掲載漏れとなっている有用な開発者ツールがある場合は、適切な場所で遠慮なく提案してください。

現在、開発者ツールは[開発者ポータル](/developers/)の各所に掲載されています。

**適切なページへの新規追加をぜひご提案ください。**

## 決定方法 {#ways-to-contribute}

開発者ツールの提案は、以下の基準によって評価されます。

**すでに掲載されているツールと意味のある差別化がされているか？**

- 新しいカテゴリや種類のツールであるか
- 既存の類似ツールと比較して新しい機能があるか
- 既存の類似ツールではカバーされていない明確なユースケースを対象としているか

**ツールのドキュメントは充実しているか？**

- ドキュメントが存在するか
- ツールを使用するのに十分な内容か
- 最近更新されているか

**ツールは広く使用されているか？**

- GitHub のスター数、ダウンロード統計、既知の企業やプロジェクトで使用されているかなどの指標を考慮します。

**ツールは十分な品質を備えているか？**

- 繰り返し発生するバグはないか
- ツールは信頼できるか
- ツールは積極的にメンテナンスされているか

**ツールはオープンソースか？**

イーサリアム分野の多くのプロジェクトはオープンソースです。コミュニティの開発者がコードを検査し、貢献できるオープンソースプロジェクトの方が、掲載される可能性が高くなります。

---

## プロダクトの順序 {#product-ordering}

アルファベット順など、プロダクトが特定の順序で並べられていない限り、ページに追加された順（古いものから新しいものへ）で表示されます。つまり、最新のプロダクトはリストの一番下に追加されます。
