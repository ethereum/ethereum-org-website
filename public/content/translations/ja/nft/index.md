## NFTとは？ {#what-are-nfts}

NFTは、**それぞれが固有**のトークンです。各NFTは異なる特性を持ち（非代替性）、希少性が証明可能です。これは、[ETH](/glossary/#ether)やUSDCのような他の[イーサリアム](/)ベースのトークンとは異なります。これらのトークンはすべて同一であり、同じ特性を持っています（「代替可能」）。ウォレットにどの1ドル紙幣（またはETH）が入っているかは気にしないでしょう。なぜなら、それらはすべて同一であり、同じ価値があるからです。しかし、どの特定のNFTを所有しているかは_重要_です。なぜなら、それらはすべて他のものと区別される個別の特性を持っているからです（「非代替性」）。

各NFTの独自性により、アート、収集品、さらには不動産などのトークン化が可能になります。ここでは、1つの特定の固有のNFTが、特定の固有の現実世界またはデジタルのアイテムを表します。資産の所有権は、イーサリアムの[ブロックチェーン](/glossary/#blockchain)上で公的に検証可能です。

## 資産のインターネット {#internet-of-assets}

NFTとイーサリアムは、今日のインターネットに存在するいくつかの問題を解決します。すべてがデジタル化するにつれて、希少性、独自性、所有権の証明といった物理的なアイテムの特性を、中央組織に管理されない方法で再現する必要性が生じています。たとえば、NFTを使用すると、すべてのイーサリアムベースのアプリで音楽のmp3ファイルを所有でき、SpotifyやApple Musicのような特定の企業の音楽アプリに縛られることはありません。販売やスワップが可能なソーシャルメディアのハンドル名を所有できますが、プラットフォームプロバイダーによって**恣意的に奪われることはありません**。

NFTのインターネットと、私たちが今日使用しているインターネットの比較は以下の通りです。

### 比較 {#nft-comparison}

| NFTのインターネット                                                                                                         | 今日のインターネット                                                                                                                                     |
| ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **資産を所有できます！** あなただけが販売やスワップを行えます。                                                                    | 組織から**資産を借りており**、奪われる可能性があります。                                                                            |
| NFTは**デジタル的に固有**であり、同じNFTは2つとありません。                                                                    | 多くの場合、**コピーとオリジナルを区別できません**。                                                                                   |
| NFTの所有権はブロックチェーンに保存され、誰でも**公的に検証**できます。                                                   | デジタルアイテムの所有権記録へのアクセスは**機関によって管理**されており、彼らの言葉を信じるしかありません。                                 |
| NFTはイーサリアム上の[スマート・コントラクト](/glossary/#smart-contract)です。つまり、イーサリアム上の**他のスマート・コントラクトやアプリで簡単に使用できます**！ | デジタルアイテムを扱う企業は通常、**独自の「ウォールドガーデン（閉鎖的な環境）」インフラを必要とします**。                                                                 |
| コンテンツ**クリエイターはどこでも作品を販売でき**、グローバル市場にアクセスできます。                                           | クリエイターは、使用するプラットフォームのインフラと流通に依存します。これらは多くの場合、利用規約や**地理的な制限**の対象となります。 |
| NFTクリエイターは自身の作品に対する**所有権を保持**し、NFTコントラクトに直接ロイヤリティをプログラムできます。     | 音楽**ストリーミングサービスなどのプラットフォームは、売上から利益の大部分を保持します**。                                                                |

## NFTの用途 {#nft-use-cases}

NFTは、以下を含む多くの用途で使用されています。

- イベントに参加したことの証明
- コースを修了したことの証明
- ゲーム内の所有可能なアイテム
- デジタルアート
- [現実世界の資産](/real-world-assets/)のトークン化
- オンラインアイデンティティの証明
- コンテンツへのアクセスの制限
- チケット発行
- 分散型インターネットドメイン名
- [分散型金融 (DeFi)](/glossary/#defi)における担保

あなたは、コントロールを失ったり仲介者に利益を犠牲にしたりすることなく、NFTを使用して作品を共有したいアーティストかもしれません。新しいコントラクトを作成し、NFTの数、その特性、および特定のアートワークへのリンクを指定できます。アーティストとして、支払われるべき**ロイヤリティをスマート・コントラクトにプログラムできます**（例：NFTが送金されるたびに、販売価格の5%をコントラクトの所有者に送金する）。また、コントラクトをデプロイした[ウォレット](/glossary/#wallet)を所有しているため、自分がNFTを作成したことを常に証明できます。購入者は、自分のウォレット[アドレス](/glossary/#address)があなたのスマート・コントラクト内のトークンに関連付けられているため、あなたのコレクションの**本物のNFT**を所有していることを簡単に証明できます。彼らはその真正性に自信を持って、イーサリアムエコシステム全体でそれを使用できます。

あるいは、スポーツイベントのチケットを考えてみましょう。**イベントの主催者が販売するチケットの数を選択できる**のと同じように、NFTのクリエイターはレプリカの数を決定できます。5000枚の一般入場券のように、これらが完全に同じレプリカである場合もあります。指定席のチケットのように、非常に似ていてもそれぞれがわずかに異なるものがいくつかミント（発行）される場合もあります。これらはチケット取扱業者に手数料を支払うことなくピア・ツー・ピアで売買でき、購入者はコントラクトアドレスを確認することで常にチケットの真正性を保証されます。

ethereum.orgでは、**NFTは、人々が私たちのGitHubリポジトリに有意義な貢献をしたこと**（ウェブサイトのプログラミング、記事の執筆や修正など）、コンテンツを翻訳したこと、またはコミュニティコールに参加したことを証明するために使用されており、独自のNFTドメイン名も持っています。ethereum.orgに貢献すると、[POAP](/glossary/#poap) NFTを請求できます。一部の暗号資産ミートアップでは、POAPをチケットとして使用しています。[貢献についての詳細](/contributing/#poap)。また、[ethereum.orgの収集品](/collectibles/)ページで、コミュニティの達成を示すNFTを見ることもできます。

![ethereum.org POAP](./poap.png)

このウェブサイトには、NFTを利用した代替ドメイン名である**ethereum.eth**もあります。私たちの`.org`アドレスはドメインネームシステム（DNS）プロバイダーによって中央集権的に管理されていますが、ethereum`.eth`はEthereum Name Service（ENS）を通じてイーサリアム上に登録されています。そして、それは私たちが所有し、管理しています。[私たちのENSレコードを確認する](https://app.ens.domains/name/ethereum.eth)

## NFTの仕組み {#how-nfts-work}

NFTは、イーサリアムブロックチェーン上の他のデジタルアイテムと同様に、「スマート・コントラクト」と呼ばれるイーサリアムベースの特別なコンピュータプログラムを通じて作成されます。これらのコントラクトは、[ERC-721](/glossary/#erc-721)や[ERC-1155](/glossary/#erc-1155)標準などの特定のルールに従い、コントラクトができることを決定します。

NFTのスマート・コントラクトは、いくつかの重要なことを実行できます。

- **NFTの作成:** 新しいNFTを作成できます。
- **所有権の割り当て:** NFTを特定のイーサリアムアドレスにリンクすることで、誰がどのNFTを所有しているかを追跡します。
- **各NFTへのIDの付与:** 各NFTには、それを固有にする番号があります。さらに、通常はNFTが何を表しているかを説明する情報（メタデータ）が添付されています。

誰かがNFTを「作成」または「ミント」するとき、彼らは基本的にスマート・コントラクトに対して、特定のNFTの所有権を自分に与えるように指示しています。この情報は、安全かつ公的にブロックチェーンに保存されます。

さらに、コントラクトの作成者は追加のルールを加えることができます。特定のNFTを作成できる数を制限したり、NFTの所有者が変わるたびに少額のロイヤリティを受け取るように決定したりできます。

### NFTのセキュリティ {#nft-security}

イーサリアムのセキュリティは、[プルーフ・オブ・ステーク (PoS)](/glossary/#pos)によってもたらされます。このシステムは、悪意のある行動を経済的に抑制するように設計されており、イーサリアムを改ざん不可能にしています。これがNFTを可能にしている理由です。NFTのトランザクションを含む[ブロック](/glossary/#block)が[ファイナライズ済み](/glossary/#finality)になると、攻撃者がそれを変更するには数百万ETHのコストがかかります。イーサリアムソフトウェアを実行している人なら誰でも、NFTの不正な改ざんを即座に検出でき、悪意のあるアクターは経済的なペナルティを受けて排除されます。

NFTに関連するセキュリティ問題は、フィッシング詐欺、スマート・コントラクトの脆弱性、またはユーザーのミス（秘密鍵をうっかり公開してしまうなど）に関連していることが最も多く、NFT所有者にとって優れたウォレットセキュリティが不可欠です。
