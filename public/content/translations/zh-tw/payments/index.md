每天，數以百萬計的人面臨著同樣的挑戰：跨境匯款速度慢、費用高，而且往往令人沮喪。峇里島的自由工作者要等上好幾天，才能收到紐約客戶的付款。這對銀行基礎設施有限地區的人們影響尤為嚴重，使他們難以參與全球經濟。

這不是遙不可及的夢想——它今天正在以太坊上發生。雖然傳統金融機構在過去幾十年裡建立了強大的支付系統，但它們往往仍然受到國界、工作時間和傳統基礎設施的限制。以太坊提供了一個新典範：一個全球化、全天候的金融平台，讓任何有網路連線的人都能進行近乎即時、可程式化的交易。

## 獲取全球貨幣 {#access-to-global-currencies}

在許多國家，通貨膨脹是一個迫切的問題，通常伴隨著難以獲取外幣的困境。處於這些情況下的人們難以保存他們的財富，因為他們被迫持有迅速貶值的存款。

以太坊社群創建了**一個強大的替代金融系統**，該系統獨立於任何國家的貨幣政策或控制。

以太坊使用者可以使用**穩定幣——通常與美元等強勢貨幣掛鉤的代幣**。透過賺取和儲蓄加密貨幣，人們可以保護自己免受國內高通膨的影響，幫助保存甚至增加他們的購買力。這也使得在本地和全球範圍內支付商品和服務變得更加容易。

## 使用自行託管加密貨幣卡支付 {#pay-with-self-custodial-crypto-cards}

自行託管加密貨幣卡的運作方式就像使用自己的背包，而不是將錢鎖在別人的金庫裡。使用傳統卡片時，銀行或託管人持有您的資金，並在您消費時釋放資金。使用自行託管卡，您始終掌控自己的資產——沒有中間人——同時仍然可以透過感應或刷卡來支付咖啡、雜貨甚至航班的費用。

這些卡片直接連結到非託管錢包或智能合約帳戶，讓使用者可以在日常環境中花費 ETH 和穩定幣，而無需放棄所有權。與要求使用者將資金存入第三方的託管卡不同，自行託管卡支援 Visa 和 Mastercard 等現實世界支付，同時保留鏈上控制權。

### 範例 {#crypto-cards-examples}

- **梅塔馬斯克 (MetaMask) 卡：** 這張 Mastercard 簽帳金融卡與梅塔馬斯克錢包連結，讓使用者可以花費 ETH、穩定幣和其他支援的代幣。它支援 Apple Pay 和 Google Pay，包含加密貨幣現金回饋獎勵，並提供賺取收益的選項。

- **Tuyo 卡：** 一張基於智能合約的 Visa 卡，可自動將加密貨幣轉換為 USDC，以便在任何接受 Visa 的地方消費。使用者保留其資產的託管權，並可使用收益、交易和消費功能。

- **Gnosis Pay：** 第一張與 Gnosis Safe 智能帳戶綁定的自行託管 Visa 卡。使用者直接從他們的錢包花費加密貨幣，無需支付燃料費、外匯手續費或出金費用。也支援透過以太坊域名服務 (ENS) 進行卡片個人化。

- **Ether.Fi Cash 卡：** 這張卡與 ether.fi 的質押協定整合，讓使用者在消費的同時，他們的 ETH 仍保持質押狀態。支付透過智能合約處理，即使在消費時也能維持自行託管。

### 自行託管加密貨幣卡比較 {#crypto-card-custody-comparison}

| **加密貨幣卡** | **自行託管** | **非託管** | **主要特點**                                                 |
| --------------- | :----------------: | :---------------: | ------------------------------------------------------------ |
| 梅塔馬斯克卡   |         ✅         |        ✅         | 錢包保留在梅塔馬斯克中；付款時自動出金           |
| Tuyo 卡       |         ✅         |        ✅         | 智能錢包轉換為 USDC；使用者保留控制權          |
| Gnosis Pay      |         ✅         |        ✅         | 連結至使用者的 Gnosis Safe；使用期間託管權不轉移    |
| Ether.Fi Cash   |         ✅         |        ✅         | ETH 保持質押狀態；智能合約控制消費權限。 |

> **注意：**「自行託管」是指由使用者控制的錢包，使用者對其資金擁有完全的存取權和控制權。
> 「非託管」是指在沒有第三方託管的情況下管理資金的錢包，通常透過智能合約進行。
> 雖然所有自行託管卡都是非託管的，但並非所有非託管卡都是自行託管的。

## 網站與代理程式的微型支付 (x402) {#x402}

[x402](https://www.x402.org/) 是一種開放的支付標準，為網路帶來了原生的按次使用支付功能。透過在低成本的[以太坊第二層 (L2) 網路](/layer-2/)上使用[穩定幣](/stablecoins/)，x402 標準讓人類和機器能夠以符合經濟效益的方式直接為單一操作付費（例如閱讀新聞文章或呼叫 API），而無需管理 API 金鑰、訂閱，或透過觀看廣告來「付費」。

- **移除付費牆和登入：** 您無需為了閱讀一篇新聞文章而建立帳戶並分享個人資訊，您的錢包可以支付解鎖所需的幾美分。
- **為 AI 代理程式付款：** x402 讓自主軟體（「AI 代理程式」）能夠為其運作所需的資料和 API 呼叫付費，而無需人工干預。

### x402 支付標準的運作方式 {#x402-how}

當客戶端請求資源時，伺服器會發送 `402 Payment Required` 錯誤代碼以及付款指示（價格、帳戶，以及支援哪些代幣和區塊鏈）。

- 您的[錢包](/wallets/)會偵測到該請求並處理付款（通常只需點擊一次即可授權，或使用預先核准的授權額度自動付款）
- 能夠存取預先核准錢包餘額的 [AI 代理程式](/ai-agents/)可以自動偵測價格並立即付款，以存取資料或服務
- 客戶端需要在其錢包中擁有其中一種受支援的穩定幣，但不需要擁有任何 ETH 來支付[燃料費用](/gas/)

這開啟了全新的「機器對機器」經濟，AI 代理程式可以自行購買資源，並且可以更有效率地存取 API 服務。

然後，已簽署的訊息會傳遞到伺服器。伺服器通常使用 [x402 促進器](https://x402.gitbook.io/x402/core-concepts/facilitator)來處理區塊鏈的複雜性（發送交易、獲取付款、協助處理燃料費等），這意味著開發人員可以輕鬆接受加密貨幣微型支付，而無需管理支付基礎設施。

## 薪資支付 {#salary-payments}

許多具前瞻性思維的公司現在為員工提供選項，讓他們以以太幣 (ETH) 等加密貨幣領取全部或部分薪資：

- **Gipsybee：** 是一家從事電子、機器人、遊戲創作和其他服務的組織。他們讓員工可以選擇以以太坊領取薪資。
- **SC5：** 這家芬蘭公司是最早提供比特幣薪資的公司之一，為以太坊的類似安排鋪平了道路。
- **區塊鏈新創公司：** 許多區塊鏈領域的公司自然會為其員工提供加密貨幣薪資選項。
- **去中心化自治組織 (DAO)：** 由於 DAO 貢獻者的特殊性和多樣性，大多數的貢獻和薪資都是以加密貨幣作為報酬。

這種趨勢對遠距工作者和數位遊牧民族特別有吸引力，他們可以從無國界支付和潛在的有利匯率中受益。

## 全球救援行動 {#global-relief-efforts}

2023 年 2 月，當土耳其和敘利亞發生毀滅性地震時，全球加密貨幣社群迅速採取行動。發起了各種活動來為救援行動籌集資金，展現了以太坊在危機時刻的力量。儘管加密貨幣在土耳其[不是公認的](https://www.reuters.com/technology/no-more-kebabs-bitcoins-turkeys-crypto-payment-ban-looms-2021-04-28/)支付形式，但當局為一些組織籌集捐款[破例](https://x.com/haluklevent/status/1622913175409623041)。一些例子包括：

- [Refik Anadol](https://x.com/refikanadol/status/1622623521104089090)：是一位著名的數位藝術家，他發起了一項募款活動。
- DAO 力量：[Anka Relief DAO](https://ankarelief.org/) 和 [Bankless DAO](https://x.com/banklessDAO) 與 [Giveth](https://x.com/Giveth/status/1623493672149843969) 聯手籌集資金。
- 著名的 NFT 藝術家 [Pak](https://cause.quest/) 也為這項事業做出了貢獻。
- 甚至以太坊共同創辦人[維塔利克·布特林](https://cointelegraph.com/news/vitalik-buterin-donates-227k-to-help-earthquake-victims-in-turkey-syria)也向多個活動進行了個人捐款。

結果如何？正如 [Dune](https://dune.com/davy42/turkiye-earthquake-donations) Analytics 儀表板所追蹤的，在短短幾天內就籌集了超過 600 萬美元。

對於發生在印度和烏克蘭的悲劇，也有類似的反應時間。這種快速反應凸顯了以太坊支付的一個關鍵優勢，即能夠迅速動員全球支援，而沒有貨幣轉換、漫長的銀行轉帳或過高費用的障礙。

## 以太坊上的加密貨幣支付與法定貨幣支付 {#ethereum-vs-fiat}

為了真正體會以太坊支付的影響，值得將其與傳統法定貨幣進行比較：

|                       | **以太坊**            | **傳統銀行**                                        |
| --------------------- | ----------------------- | ------------------------------------------------------------ |
| **速度**             | 幾秒到幾分鐘      | 幾小時到幾天                                                |
| **全球覆蓋範圍**      | 無國界，全天候        | 受限於國際銀行限制和工作時間 |
| **透明度**      | 完全透明       | 依機構而異                                        |
| **可程式化**   | 支援智能合約 | 僅限於基本交易                                |
| **通膨控制** | 可預測的發行    | 受限於中央銀行政策                             |
| **可及性**     | 任何有網路的人    | 受限於國家和國際限制           |

以太坊的核心是一個去中心化的平台，允許進行安全、快速和透明的交易。然而，許多組成部分使其有別於傳統支付方式。讓我們深入探討使以太坊支付成為遊戲規則改變者的優勢：

### 可程式化 {#programmability}

以太坊的獨特功能之一是其支援智能合約的能力。智能合約是自動執行的協議，其條款直接寫入程式碼中。這為自動化、基於條件的支付開啟了無限可能，可以大幅改善以下交易：

- 託管服務
- 定期付款
- 基於績效的報酬

### 速度 {#speed}

您還記得上次等待國際銀行轉帳結算等了好幾天嗎？大排長龍？還有您必須填寫的多份表格？有了以太坊，那些日子已經一去不復返了。以太坊網路上的交易在幾分鐘內即可結算，無論發送者和接收者位於何處。由於以太坊是無需許可的，因此在匯款時沒有監管的官僚作風。這種速度在時間緊迫的情況下（例如緊急救援行動）尤為關鍵。

### 較低的費用 {#lower-fees}

傳統的國際匯款費用有時會吃掉匯款金額的很大一部分，尤其是在處理數百美元的交易時。以太坊交易雖然不是免費的，但通常費用較低。這意味著您更多的錢會流向您預期的地方，而不是落入中間人的口袋。

### 透明度 {#transparency}

以太坊區塊鏈上的每筆交易都記錄在公共帳本上。這意味著任何人都可以驗證資金的流動，使其成為以下用途的絕佳工具：

- 慈善機構展示捐款的使用方式
- 企業證明對供應商或員工的付款
- 個人追蹤其財務活動

有了以太坊，每個人都可以看到資金如何流動以及成本如何執行，這與傳統組織中大部分資訊仍然未知的情況不同。
