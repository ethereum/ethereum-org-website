<FeaturedText>了解如何使用基於 Java 的專案與工具來開發以太坊</FeaturedText>

使用以太坊建立去中心化應用程式 (dapp)，利用加密貨幣與區塊鏈技術的優勢。這些 dapp 是值得信賴的，這意味著一旦它們被部署到以太坊，就會始終按照程式設計的方式運行。它們可以控制數位資產，以建立新型態的金融應用程式。它們可以是去中心化的，這意味著沒有單一實體或個人控制它們，且幾乎不可能被審查。
