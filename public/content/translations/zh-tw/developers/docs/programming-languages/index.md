一個常見的誤解是，開發人員必須撰寫[智能合約](/developers/docs/smart-contracts/)才能在以太坊上進行開發。這是不正確的。
以太坊網路與社群的美妙之處在於，您幾乎可以使用任何程式語言來[參與](/community/)。

以太坊及其社群擁抱開源。您可以找到各種語言的社群專案，包括用戶端實作、API、開發框架和測試工具。

### 如果不支援我的語言怎麼辦？ {#other-lang}

如果您想連結到其他程式語言的資源或虛擬社群，您可以透過[建立 issue](https://github.com/ethereum/ethereum-org-website/issues/new/choose) 來請求新增頁面。

如果您只是想使用目前不支援的語言撰寫程式碼來與區塊鏈互動，您可以使用 [JSON-RPC 介面](/developers/docs/apis/json-rpc/)連接到以太坊網路。任何可以使用 TCP/IP 的程式語言都可以使用此介面。
