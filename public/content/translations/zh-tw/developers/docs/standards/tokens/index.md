## 簡介 {#introduction}

許多 [以太坊](/) 開發標準都專注於代幣介面。這些標準有助於確保智能合約保持可組合的特性，因此當新專案發行代幣時，它能與現有的去中心化交易所和應用程式保持相容。

代幣標準定義了代幣在整個以太坊生態系中的行為與互動方式。它們讓開發者更容易進行建置而無需重複造輪子，確保代幣能與錢包、交易所和去中心化金融 (DeFi) 平台無縫運作。無論是在遊戲、治理或其他使用案例中，這些標準都提供了一致性，並讓以太坊的互連性更強。

## 代幣標準 {#token-standards}

以下是以太坊上最受歡迎的一些代幣標準：

- [ERC-20](/developers/docs/standards/tokens/erc-20/) - 同質化（可互換）代幣的標準介面，例如投票代幣、質押代幣或虛擬貨幣。

### NFT 標準 {#nft-standards}

- [ERC-721](/developers/docs/standards/tokens/erc-721/) - 非同質化代幣的標準介面，例如藝術品或歌曲的契據。
- [ERC-1155](/developers/docs/standards/tokens/erc-1155/) - ERC-1155 允許更有效率的交易與交易打包，從而節省成本。此代幣標準允許建立實用型代幣（例如 $BNB 或 $BAT）以及非同質化代幣（如 CryptoPunks）。
