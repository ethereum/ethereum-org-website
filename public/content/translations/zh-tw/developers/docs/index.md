本文件旨在協助您使用 [以太坊](/) 進行開發。內容涵蓋以太坊的概念、解釋以太坊技術堆疊，並記錄適用於更複雜應用程式與使用案例的進階主題。

這裡的所有內容皆為開源且由社群維護，因此如果頁面內容過時或缺少有用的資訊，請開啟 issue 或提交 pull request。[編輯指南](https://github.com/ethereum/ethereum-org-website/blob/dev/docs/editing-markdown.md)會逐步說明如何操作。

## 選擇起點 {#pick-a-starting-point}

讀者帶著不同的目標來到這裡，而瀏覽這些文件的最快路徑取決於您想要開發什麼。以下是幾個常見的切入點：

- **開發與以太坊互動的去中心化應用程式 (dapp)。** 從[技術簡介](/developers/docs/intro-to-ethereum/)開始，接著了解[帳戶](/developers/docs/accounts/)與[交易](/developers/docs/transactions/)。準備好撰寫程式碼時，請選擇一個[框架](/developers/docs/frameworks/)。
- **撰寫智能合約。** 如果對 EVM 概念還很陌生，請先瀏覽[簡介](/developers/docs/intro-to-ethereum/)，然後跳至[智能合約](/developers/docs/smart-contracts/)與[程式語言](/developers/docs/programming-languages/)。
- **執行節點或質押。** 前往[節點與客戶端](/developers/docs/nodes-and-clients/)，接著了解[網路](/developers/docs/networking-layer/)與[共識機制](/developers/docs/consensus-mechanisms/)。
- **由下而上了解協定。** 下方的模組正是為此排序。請依序閱讀。
