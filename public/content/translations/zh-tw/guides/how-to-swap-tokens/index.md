厭倦了尋找列出所有您最愛代幣的交易所嗎？您可以使用[去中心化交易所](/glossary/#dex)兌換大部分的代幣。

代幣兌換涉及交換存在於以太坊網路上的兩種不同資產，例如將 ETH 兌換為 DAI（一種 [ERC-20](/glossary/#erc-20) 代幣）。這個過程非常快速且便宜。您需要擁有一個加密貨幣錢包才能兌換代幣。

**先決條件：**

- 擁有一個[加密貨幣錢包](/glossary/#wallet)；如果您沒有，可以遵循這篇關於[如何建立以太坊帳戶](/guides/how-to-create-an-ethereum-account/)的指南
- 將資金存入您的錢包

## 1. 將您的錢包連接到您選擇的去中心化交易所 (DEX) {#1-connect-your-wallet-to-the-decentralized-exchange-dex-of-your-choice}

一些受歡迎的交易所包括：

- [尤尼斯瓦普](https://app.uniswap.org/#/swap)
- [Sushiswap](https://www.sushi.com/swap)
- [1Inch](https://app.1inch.io/#/1/unified/swap/ETH/DAI)
- [Curve](https://www.curve.finance/dex/ethereum/swap/)

覺得有趣嗎？了解更多關於什麼是[去中心化金融 (DeFi)](/defi/)，以及這些新型交易所是如何運作的。

## 2. 選擇您想要兌換的代幣對 {#2-select-the-pair-of-tokens-you-wish-to-swap}

例如，ETH 和 DAI。請確保您擁有這兩種代幣中其中一種的資金。
![Common interface for swapping](./swap1.png)

## 3. 輸入您想要交易的代幣數量並點擊兌換 {#3-enter-the-amount-of-tokens-you-want-to-trade-and-click-swap}

交易所將自動計算您會獲得多少代幣。

![Common interface for swapping](./swap2.png)

## 4. 確認交易 {#4-confirm-the-transaction}

檢查交易的詳細資訊。確認匯率和任何其他費用，以避免出現不愉快的意外。

![Common interface for reviewing the transaction](./swap3.png)

## 5. 等待交易處理完成 {#5-wait-for-the-transaction-to-be-processed}

您可以在任何區塊鏈瀏覽器上查看交易進度。這個過程通常不會超過 10 分鐘。

一旦交易處理完成，您將自動在錢包中收到兌換的代幣。

## 常見問題 {#frequently-asked-questions}

### 我可以從我的錢包將 ETH 兌換為 BTC 嗎？ {#can-i-swap-eth-for-btc-from-my-wallet}

不行，您只能兌換以太坊網路的原生代幣，例如 ETH、ERC-20 代幣或 NFT。您只能兌換存在於以太坊上的「包裝 (wrapped)」形式的比特幣。

### 什麼是滑價？ {#what-is-slippage}

它是您預期的匯率與實際匯率之間的差額。
