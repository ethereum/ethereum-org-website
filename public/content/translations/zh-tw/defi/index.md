去中心化金融 (DeFi) 是為網際網路時代打造的開放式全球金融體系——它是對不透明、受嚴格控制且由數十年前的基礎設施和流程所維繫的系統的一種替代方案。它讓你對自己的資金擁有控制權和可見度。它讓你接觸全球市場，並提供當地貨幣或銀行選項的替代方案。DeFi 產品向任何有網際網路連線的人開放金融服務，且主要由其使用者擁有和維護。到目前為止，已有價值數百億美元的加密貨幣流經 DeFi 應用程式，而且每天都在增長。

## 什麼是 DeFi？ {#what-is-defi}

DeFi 是金融產品和服務的總稱，任何可以使用 [以太坊](/) 的人（任何有網際網路連線的人）都可以存取。透過 DeFi，市場永遠開放，沒有中心化機構可以阻止付款或拒絕你存取任何內容。以前緩慢且有發生人為錯誤風險的服務，現在因為由任何人都可以檢查和審查的程式碼處理，變得自動化且更安全。

現在有一個蓬勃發展的加密貨幣經濟，你可以在其中借貸、借款、做多/做空、賺取利息等等。精通加密貨幣的阿根廷人已經使用 DeFi 來逃避嚴重的通貨膨脹。公司已經開始即時向員工支付工資。有些人甚至在不需要任何個人身分證明的情況下，借出並償還了價值數百萬美元的貸款。

## DeFi 與傳統金融 {#defi-vs-tradfi}

了解 DeFi 潛力的最佳方法之一，就是了解當今存在的問題。

- 有些人無法開設銀行帳戶或使用金融服務。
- 無法獲得金融服務可能會阻礙人們就業。
- 金融服務可能會阻止你獲得報酬。
- 金融服務的一項隱藏費用是你的個人資料。
- 政府和中心化機構可以隨意關閉市場。
- 交易時間通常僅限於特定時區的營業時間。
- 由於內部人工作業流程，資金轉帳可能需要數天時間。
- 金融服務存在溢價，因為中介機構需要從中抽成。

### 比較 {#defi-comparison}

| DeFi                                                                                               | 傳統金融                                                                                                            |
| -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| 你持有自己的資金。                                                                               | 你的資金由公司持有。                                                                                               |
| 你控制資金的去向和使用方式。                                              | 你必須信任公司不會管理不善你的資金，例如貸款給高風險的借款人。                                      |
| 資金轉帳在幾分鐘內完成。                                                              | 由於人工作業流程，付款可能需要數天時間。                                                                                |
| 交易活動是假名的。                                                              | 金融活動與你的身分緊密相關。                                                                      |
| DeFi 對任何人開放。                                                                            | 你必須申請才能使用金融服務。                                                                                      |
| 市場永遠開放。                                                                       | 市場會關閉，因為員工需要休息。                                                                                   |
| 它建立在透明度之上——任何人都可以查看產品的資料並檢查系統的運作方式。 | 金融機構是不公開的：你不能要求查看他們的貸款歷史、管理資產的記錄等等。 |

## 一切始於比特幣…… {#bitcoin}

在許多方面，比特幣是第一個 DeFi 應用程式。比特幣讓你真正擁有和控制價值，並將其發送到世界各地。它透過提供一種方式，讓大量互不信任的人在不需要受信任中介的情況下，就帳戶分類帳達成共識來實現這一點。比特幣對任何人開放，沒有人有權更改其規則。比特幣的規則（如其稀缺性和開放性）都寫入了技術中。它不像傳統金融，政府可以印鈔票使你的儲蓄貶值，公司可以關閉市場。

以太坊以此為基礎。與比特幣一樣，規則不能隨意更改，每個人都可以存取。但它還使用[智能合約](/glossary/#smart-contract)使這種數位貨幣具有可程式化特性，因此你不僅可以儲存和發送價值。

## 可程式化貨幣 {#programmable-money}

這聽起來很奇怪……「為什麼我要對我的錢進行程式設計」？然而，這不僅僅是以太坊上代幣的預設功能。任何人都可以將邏輯編寫到付款中。因此，你可以獲得比特幣的控制權和安全性，並結合金融機構提供的服務。這讓你可以使用加密貨幣做一些比特幣無法做到的事情，例如借貸和借款、安排付款、投資指數基金等等。

### 快速向全球匯款 {#send-money}

#### 向全球串流支付資金…… {#stream-money}

### 存取穩定貨幣 {#stablecoins}

加密貨幣的波動性是許多金融產品和日常消費面臨的問題。DeFi 社群已經用穩定幣解決了這個問題。它們的價值與另一種資產掛鉤，通常是像美元這樣受歡迎的貨幣。

像 DAI 或 USDC 這樣的代幣，其價值與美元的差距保持在幾美分以內。這使得它們非常適合用於賺取收益或零售。在政府發行的貨幣存在極大不確定性的時期，許多拉丁美洲人使用穩定幣作為保護儲蓄的一種方式。

### 借款 {#lending}

從去中心化提供商借款主要有兩種形式。

- 點對點，這意味著借款人將直接向特定的貸款人借款。
- 基於資金池，貸款人向資金池提供資金（流動性），借款人可以從中借款。

<ButtonLink href="/apps/categories/defi">

</ButtonLink>

使用去中心化貸款機構有許多優勢……

#### 具備隱私的借款 {#borrowing-privacy}

如今，借貸和借款都圍繞著相關的個人展開。銀行在貸款之前需要知道你是否有可能償還貸款。

去中心化借貸的運作不需要任何一方表明身分。相反，借款人必須提供抵押品，如果貸款未償還，貸款人將自動獲得該抵押品。一些貸款機構甚至接受 [NFT](/glossary/#nft) 作為抵押品。NFT 是獨特資產（如畫作）的契約。[更多關於 NFT 的資訊](/nft/)

這讓你可以借款而無需進行信用檢查或交出私人資訊。

#### 存取全球資金 {#access-global-funds}

當你使用去中心化貸款機構時，你可以存取來自全球各地的存款資金，而不僅僅是你選擇的銀行或機構託管的資金。這使得貸款更容易獲得，並改善了利率。

#### 稅務效率 {#tax-efficiencies}

借款可以讓你獲得所需的資金，而無需出售你的 ETH（應稅事件）。相反，你可以使用 ETH 作為穩定幣貸款的抵押品。這為你提供了所需的現金流，並讓你保留你的 ETH。穩定幣是當你需要現金時更好的代幣，因為它們的價值不會像 ETH 那樣波動。[更多關於穩定幣的資訊](#stablecoins)

#### 閃電貸 {#flash-loans}

閃電貸是一種更具實驗性的去中心化借貸形式，讓你在沒有抵押品或提供任何個人資訊的情況下借款。

目前非技術人員還無法廣泛使用它們，但它們暗示了未來每個人都可能實現的目標。

它的運作基礎是貸款在同一筆交易中借出並償還。如果無法償還，交易就會還原，就像什麼都沒發生過一樣。

通常使用的資金保存在流動性池（用於借款的大型資金池）中。如果它們在特定時刻沒有被使用，這就為某人創造了機會來借用這些資金，用它們進行業務，並在借入的同時全額償還。

這意味著必須在非常客製化的交易中包含大量邏輯。一個簡單的例子可能是，有人使用閃電貸以一個價格借入盡可能多的資產，然後在價格較高的另一個交易所出售。

因此，在單筆交易中，會發生以下情況：

- 你從交易所 A 以 1.00 美元的價格借入 X 數量的 $asset
- 你在交易所 B 以 1.10 美元的價格出售 X 數量的 $asset
- 你向交易所 A 償還貸款
- 你保留扣除交易手續費後的利潤

如果交易所 B 的供應量突然下降，且使用者無法購買足夠的資產來償還原始貸款，交易就會直接失敗。

要在傳統金融世界中實現上述例子，你需要大量的資金。這些賺錢策略只有擁有現有財富的人才能使用。閃電貸是未來的一個例子，在未來，擁有資金不一定是賺錢的先決條件。

### 開始使用加密貨幣儲蓄 {#saving}

#### 借貸 {#lending-2}

你可以透過借出加密貨幣來賺取利息，並即時看到你的資金增長。目前，利率遠高於你在當地銀行可能獲得的利率（如果你夠幸運能使用銀行的話）。這是一個例子：

- 你將 100 DAI（一種[穩定幣](/stablecoins/)）借給像 Aave 這樣的產品。
- 你會收到 100 Aave Dai (aDai)，這是一種代表你借出 DAI 的代幣。
- 你的 aDai 將根據利率增加，你可以看到錢包中的餘額不斷增長。根據 [APR](/glossary/#apr)，幾天甚至幾小時後，你的錢包餘額可能會顯示為 100.1234！
- 你可以隨時提取與你的 aDai 餘額相等的常規 DAI。

### 兌換代幣 {#swaps}

以太坊上有成千上萬種代幣。去中心化交易所 (DEX) 讓你可以隨時交易不同的代幣。你永遠不會放棄對資產的控制權。這就像在訪問不同國家時使用貨幣兌換一樣。但 DeFi 版本永遠不會關閉。市場全年無休，每週 7 天，每天 24 小時開放，技術保證總會有人接受交易。

例如，如果你想使用無損彩券 PoolTogether（如上所述），你需要像 DAI 或 USDC 這樣的代幣。這些 DEX 允許你將 ETH 兌換成這些代幣，並在完成後再兌換回來。

### 進階交易 {#trading}

對於喜歡更多控制權的交易者，有更進階的選項。限價單、永續合約、保證金交易等都是可能的。透過去中心化交易，你可以獲得全球流動性，市場永遠不會關閉，而且你始終控制著自己的資產。

當你使用中心化交易所時，你必須在交易前存入資產，並信任他們會妥善保管。當你的資產被存入時，它們處於風險之中，因為中心化交易所是駭客極具吸引力的目標。

#### 二次方融資 {#quadratic-funding}

以太坊是開源軟體，到目前為止，許多工作都是由社群資助的。這導致了一種有趣的新募資模式的發展：二次方融資。這有潛力改善我們未來資助所有類型公共財的方式。

二次方融資確保獲得最多資金的專案是那些具有最獨特需求的專案。換句話說，就是那些有望改善最多人生活的專案。它的運作方式如下：

1. 有一個捐贈的配對資金池。
2. 開始一輪公共融資。
3. 人們可以透過捐贈一些資金來表達他們對專案的需求。
4. 一旦該輪融資結束，配對資金池將分配給各個專案。具有最獨特需求的專案將從配對資金池中獲得最高金額。

這意味著獲得 100 筆 1 美元捐款的專案 A，最終獲得的資金可能比獲得單筆 10,000 美元捐款的專案 B 還多（取決於配對資金池的規模）。

### 保險 {#insurance}

去中心化保險旨在使保險更便宜、理賠更快且更透明。隨著自動化程度的提高，保險範圍變得更實惠，理賠也快得多。用於決定你申領的資料是完全透明的。

與任何軟體一樣，以太坊產品也可能遭受錯誤和漏洞利用。因此，目前該領域的許多保險產品都專注於保護其使用者免受資金損失。然而，有些專案開始為生活中可能發生的各種情況建立保險範圍。一個很好的例子是 Etherisc 的農作物保險，旨在[保護肯亞的小農免受乾旱和洪水的影響](https://blog.etherisc.com/etherisc-teams-up-with-chainlink-to-deliver-crop-insurance-in-kenya-137e433c29dc)。去中心化保險可以為經常因價格過高而無法購買傳統保險的農民提供更便宜的保險。

## DeFi 如何運作？ {#how-defi-works}

DeFi 使用加密貨幣和智能合約來提供不需要中介機構的服務。在當今的金融世界中，金融機構充當交易的擔保人。這賦予了這些機構巨大的權力，因為你的資金流經它們。此外，世界上有數十億人甚至無法開設銀行帳戶。

在 DeFi 中，智能合約取代了交易中的金融機構。智能合約是一種以太坊帳戶，可以持有資金，並可以根據特定條件發送/退還資金。當智能合約上線時，沒有人可以更改它——它將始終按照程式設計執行。

一個設計用來發放授權額度或零用錢的合約，可以被編寫成每週五從帳戶 A 向帳戶 B 發送資金。而且只要帳戶 A 有所需的資金，它就只會這樣做。沒有人可以更改合約並將帳戶 C 新增為收款人來竊取資金。

合約也是公開的，任何人都可以檢查和稽核。這意味著不良合約通常會很快受到社群的審查。

這確實意味著目前需要信任以太坊社群中能夠閱讀程式碼的技術人員。基於開源的社群有助於監督開發人員，但隨著智能合約變得更容易閱讀，以及開發出其他證明程式碼可信度的方法，這種需求將隨著時間的推移而減少。

## 以太坊與 DeFi {#ethereum-and-defi}

以太坊是 DeFi 的完美基礎，原因如下：

- 沒有人擁有以太坊或存在於其上的智能合約——這讓每個人都有機會使用 DeFi。這也意味著沒有人可以隨意更改規則。
- DeFi 產品在幕後都使用相同的語言：以太坊。這意味著許多產品可以無縫協作。你可以在一個平台上借出代幣，並在完全不同的應用程式上的不同市場中兌換生息代幣。這就像能夠在你的銀行兌現忠誠度積分一樣。
- 代幣和加密貨幣內建於以太坊這個共享分類帳中——追蹤交易和所有權正是以太坊的專長。
- 以太坊允許完全的財務自由——大多數產品永遠不會託管你的資金，讓你保持控制權。

你可以將 DeFi 想像成幾個層次：

1. 區塊鏈——以太坊包含交易歷史和帳戶狀態。
2. 資產——[ETH](/what-is-ether/) 和其他代幣（貨幣）。
3. 協定——提供功能的[智能合約](/glossary/#smart-contract)，例如，允許去中心化借貸資產的服務。
4. [應用程式](/apps/)——我們用來管理和存取協定的產品。

注意：大部分 DeFi 使用 [ERC-20 標準](/glossary/#erc-20)。DeFi 中的應用程式使用一種稱為包裝以太幣 (wETH) 的 ETH 包裝器。[了解更多關於包裝以太幣的資訊](/wrapped-eth)。

## 開發 DeFi {#build-defi}

DeFi 是一場開源運動。DeFi 協定和應用程式都是開放的，供你檢查、分叉和創新。由於這種分層堆疊（它們都共享相同的基礎區塊鏈和資產），協定可以混合搭配，以解鎖獨特的組合機會。

## 超越傳統 DeFi {#beyond-traditional-defi}

DeFi 生態系統繼續擴展到新領域：

- **[預測市場](/prediction-markets/)**——去中心化平台，你可以在沒有中介機構的情況下，對從選舉到體育賽事等未來事件的結果進行下注。
- **[現實世界資產 (RWA)](/real-world-assets/)**——在以太坊上將房地產、商品和債券等實體資產代幣化，將數兆美元的價值帶到鏈上。
- **[付款](/payments/)**——使用以太坊和穩定幣進行快速、低成本的全球付款，而無需傳統的銀行基礎設施。
- **[AI 代理](/ai-agents/)**——可以在以太坊上進行交易的自主軟體代理，實現新形式的自動化交易、投資組合管理和鏈上互動。
