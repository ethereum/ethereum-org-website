#### 同質化與非同質化 (0:00) {#fungible-vs-non-fungible-000}

讓我們先從「同質化 (fungible)」這個詞開始。同質化意味著資產的各個單位是可以互換且無法區分的。同質化資產的一個好例子是貨幣。一張五元鈔票的價值總是等於任何其他五元鈔票。你並不會真正在意收到的是哪一張特定的五元鈔票，因為它們的價值都一樣。

然而，當談到非同質化資產時，每個單位都是獨一無二的，不能直接被另一個取代。一個好例子是機票。儘管機票看起來可能很相似，但每張機票都載有不同的乘客姓名、目的地、出發時間和座位號碼。試圖將一張機票換成另一張機票可能會導致嚴重的問題。

另一個例子是集換式卡牌。儘管它們看起來可能很相似，但每張卡牌都有不同的屬性。生產年份或卡牌的保存狀況等因素都會產生影響。非同質化物品的一個極端例子是藝術品——例如，一幅畫通常只會創作出一份原件。

#### NFT 的屬性 (2:13) {#properties-of-nfts-213}

既然我們知道「非同質化」是什麼意思了，讓我們來看看 NFT 最常見的屬性。

- **獨一無二** — 每個 NFT 都有不同的屬性，這些屬性通常儲存在代幣的中繼資料中
- **可證明的稀缺性** — NFT 的數量通常是有限的，極端的例子是只有一份副本；代幣的數量可以在區塊鏈上進行驗證
- **不可分割** — 大多數 NFT 不能分割成更小的面額，因此你不能購買或轉帳 NFT 的一小部分

與標準代幣類似，NFT 也保證了資產的所有權，易於轉帳，並且具有防偽功能。

#### 代幣標準：ERC-20、ERC-721 與 ERC-1155 (3:17) {#token-standards-erc-20-erc-721-and-erc-1155-317}

雖然 NFT 可以在任何支援智能合約程式設計的區塊鏈上實作，但最著名的標準是以太坊上的 ERC-721 和 ERC-1155。在我們深入探討 NFT 標準之前，讓我們先快速回顧一下 ERC-20，因為這將有助於比較。

**ERC-20** 是在以太坊區塊鏈上建立代幣的著名標準。例子包括 USDT 或 DAI 等穩定幣，以及 LEND、YFI、SNX 和 UNI 等 DeFi 代幣。ERC-20 允許建立同質化代幣——在該標準下建立的所有代幣都是完全無法區分的。無論你是從朋友那裡還是從交易所收到 USDT 都沒關係；每個代幣的價值都是一樣的。

**ERC-721** 是建立非同質化代幣的標準。它允許建立能產生具有不同屬性且可區分代幣的合約。一個常見的例子是著名的迷戀貓 (CryptoKitties)——這是一款允許收集和繁殖虛擬小貓的遊戲。

**ERC-1155** 是非同質化代幣建立的下一步。這個標準允許建立同時支援同質化和非同質化代幣的合約。它是由專注於區塊鏈遊戲的專案 Enjin 所建立。在許多遊戲（如《魔獸世界》）中，玩家可以同時持有非同質化物品（劍、盾牌、盔甲）和同質化物品（如黃金或箭矢）。ERC-1155 允許開發人員定義同質化和非同質化代幣，並決定每種代幣應該存在多少數量。

#### NFT 使用案例 (5:28) {#nft-use-cases-528}

除了迷戀貓 (CryptoKitties) 之外，還有其他幾款利用 NFT 的熱門遊戲，例如 Gods Unchained 和 Decentraland。Decentraland 是一個有趣的例子，因為玩家可以購買數位土地，這些土地稍後可以轉售，甚至在遊戲中用作廣告空間。

其他例子包括數位藝術品市場，如 Rarible 和 SuperRare，甚至是像 OpenSea 這樣的市場聚合器。另一個可以表示為 NFT 的稀缺物品例子是網域名稱——例如，帶有 .eth 擴展名的以太坊名稱服務 (Ethereum Name Service) 和帶有 .crypto 擴展名的 Unstoppable Domains。

有些 NFT 可能非常昂貴。最昂貴的迷戀貓 Dragon 在 2017 年底以 600 ETH 的價格售出——當時價值約十七萬美元。稀缺的網域名稱（如 exchange.eth）價值可能高達五十萬美元以上。

#### NFT 作為 DeFi 中的抵押品 (6:48) {#nfts-as-collateral-in-defi-648}

談到 DeFi，NFT 可以為去中心化金融釋放更多潛力。目前，絕大多數的 DeFi 借貸協定都是有抵押的。其中一個最有趣的點子是將 NFT 用作抵押品。這意味著你將能夠提供代表藝術品、數位土地甚至代幣化房地產的 NFT 作為抵押品，並以此借款。

這聽起來很有希望，但存在一個問題。在 Compound 或 Aave 等標準的 DeFi 借貸平台上，所提供抵押品的價值可以透過整合價格預言機輕鬆衡量。這些預言機會匯總來自多個流動性來源（如中心化和去中心化交易所）的價格。然而，當談到 NFT 時，特定代幣的市場通常缺乏流動性，這使得價格發現過程變得棘手。

為了更了解這個問題，想像一下有人以 10 ETH 買了一隻稀有的迷戀貓。這個 NFT 隨後被用作抵押品，借款人提取了 1,700 DAI——假設 10 ETH 價值 3,500 美元，且這個特定的 NFT 具有 50% 的貸款價值比 (LTV)。在此之後，如果沒有其他人願意購買這隻特定的迷戀貓，那麼這個 NFT 的市場就缺乏流動性，甚至根本不存在。唯一的假設是該 NFT 的價值仍然與上次售出時相同——這並不是一個安全的假設，因為 NFT 的價值可能會發生相當劇烈的變化。

這就是為什麼一些提供 NFT 抵押貸款的專案使用了一種稍微不同的模型：點對點貸款。在這個市場模型中，借款人可以提供他們的 NFT 作為抵押品，而貸款人可以在啟動貸款之前選擇他們願意接受哪個 NFT。用作抵押品的 NFT 保存在託管合約中，如果借款人違約，沒有按時償還借款金額加上利息，該 NFT 就會轉帳給貸款人。這個領域還很新，但使用這種模型的公司之一是 NFTfi。

#### NFT 作為金融產品 (9:32) {#nfts-as-financial-products-932}

除了用作抵押品之外，NFT 還可以代表更複雜的金融產品，如保險、債券或選擇權。Yearn Finance 的 Yinsure 是 NFT 在保險領域使用的一個好例子。在 Yinsure 中，每份保險合約都表示為一個 NFT，也可以在 Rarible 等二級市場上進行交易。

我們最近也開始看到 NFT 專案使用流動性挖礦等 DeFi 原生概念。例如，Rarible 開始用 RARI 治理代幣獎勵在其平台上建立、購買和銷售 NFT 的使用者。

#### 不斷成長的 NFT 市場 (10:30) {#the-growing-nft-market-1030}

隨著超過 1 億美元價值的 NFT 被交易，且僅在最近一個月就達到了 600 萬美元，NFT 領域是加密貨幣中成長最快的利基市場之一。從數位小貓到複雜的金融產品，它具有巨大的潛力。
