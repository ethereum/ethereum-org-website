#### 科學與進步簡介 (0:10) {#introduction-to-science-and-progress-010}

好的，大家好。我是 Juan。我今天來這裡是為了談談 DeSci。我想探討我們如何利用 DeSci 來資助、組織和開放科學。首先，這是我們將要討論的內容：我會先簡短談談一般科學，接著討論什麼是去中心化科學 (DeSci) 運動，然後探討我們如何資助科學公共財。接著是 DeSci 如何組織圍繞科學的人員、專案和工作。然後我想稍微談談開放取用和可重現的科學，最後以行動呼籲作為總結。所以這會非常緊湊。我們有很多內容要涵蓋，所以我會講得很快。

首先，我想說的是，在過去幾個世紀裡，我們取得了巨大的進步。幾乎所有的人類指標都在改善。從我們能想到的幾乎任何標準來看，人類的生存條件都得到了顯著的改善，而實現這種進步的很大一部分歸功於科學事業。透過擴展我們的知識，透過將我們所知轉化為技術和解決各種問題的方案，我們已經能夠讓世界上很大一部分人脫離貧困。我們能夠養活無數的人，為每個人提供住所，治癒各種疾病等等。多虧了科學，我們取得了巨大的進步。

科學是一項龐大的事業，包含許多不同的子領域和許多不同的知識領域。你可以想到任何特定的領域和任何研究領域，科學都是其中的重要組成部分。歸根究底，科學的本質就是發現事物的過程。創造新知識並結合新概念的過程。想想科學方法。費曼有一句名言：「如果它與實驗不符，它就是錯的。」這就是科學的關鍵。

你可以把科學看作是一項涉及全球人類的大規模事業。有各種各樣的努力和系統。你可以想到地球上各種大學、各種研究小組、不同領域和期刊。圍繞著綜合我們所知、提出新想法、將這些想法轉化為研究專案、將其轉化為實際的假設檢驗，以及收集資料以測試假設是否正確的過程，有許多不同的活動。一直到將這些結果寫成某種論文，然後由科學社群進行同儕審查，將其添加到知識樹中，進而擴展我們的知識。

也許故事就到此為止，或者也許後來發現，實際上那無法重現，我們必須撤回它。或者實際上那是正確的，但它為大量其他新知識打開了大門。所以這是一個高度動態的領域，有著許多不同的活動。

然而，科學也面臨著許多問題。科學事業存在各種各樣的議題。儘管它一直是推動進步的巨大引擎，但其中也出現了各種問題。特別是，許多領域都缺乏資金。同時，儘管缺乏資金，但整體而言仍有大量資金投入科學。人們有一種感覺，資金的運用不如以往有效，科學的投資報酬率不再那麼高。許多領域在爭取補助金方面競爭過於激烈。

一旦研究完成並發表，只有一小部分能夠被重現。因此，有各種各樣的科學研究被發表、被接受並被認為是正確的，後來卻發現其中很大一部分實際上無法重現。所以存在著巨大的可重現性危機。甚至有些科學發現的產物正在遺失。想想與某些結果相關的實際論文、程式碼或資料從我們的知識庫中消失。因此，圍繞科學有各種各樣的問題需要解決，而這正是 DeSci 的部分目標。這是在解決一系列此類問題，雖然不是全面、徹底地解決，但 DeSci 社群正試圖解決其中的許多問題。

#### DeSci 運動 (5:11) {#the-desci-movement-511}

那麼什麼是 DeSci？DeSci 是一項使用 Web3 技術和工具來改善科學的運動。想像一下，能夠利用雜湊連結、區塊鏈和智慧合約的所有魔力，來創建能夠改善全球各領域科學研究方式的系統和結構。

這裡有許多不同的重點領域。想像一下，能夠擁有開放取用的論文和資料公共財，擁有更好的可重現實驗，並且能夠更好地組織實驗室和團隊。想像創建像 DAO 這樣的結構，使研究小組能夠組成和組織、籌集資金，並向參與者分配獎勵。還有全新的資金結構，例如 IPNFT。還有帶有獎勵的同儕審查協定。從歷史上看，同儕審查一直處於一種掠奪性的情況，學者們投入大量的時間和精力來對所有工作進行同儕審查，而期刊實際上並沒有為這些勞動支付任何人報酬。現在有各種各樣的新激勵結構正在被實驗。

這是一個相當新的運動。它已經存在一段時間了。當我創立 IPFS 時，在 DeSci 這個詞出現之前，它就有點像是 DeSci 運動。我創立 IPFS 的目標是讓人們能夠更好地散佈資料，以用於科學研究。所以這些想法中有很多是該專案的核心部分。然而，這項運動在過去一兩年裡發展迅速，許多新組織不斷湧現。這張地圖在過去一年裡擴大了一到兩倍，這真是太棒了。

現在有幾個團體正在進行去中心化的生物科技資金籌集，例如 VitaDAO、Molecule 等團體。有許多組織試圖提出資助科學的新結構。有幾個 DAO 本身就是試圖進行研發的科學組織。有幾個基金會和機構正在支持許多 DeSci 工作，或者以某種方式將自己與 DeSci 聯繫起來。有許多團體正在探索不同的出版方式、許多科學 NFT 等等。這個社群在過去一兩年裡發展迅速。

現在也有許多不同的聚會和會議將這些社群聚集在一起。例如 DeSci Day、DeSci 柏林、Gitcoin 社群的 Schelling Point 以及 Funding the Commons。這些會議匯集了許多關於 DeSci 的討論。

#### 資助公共財 (10:40) {#funding-the-commons-1040}

讓我們來談談資助公共財。也許你們有些人看過我過去用過的這張關於創新鴻溝的圖表。在科學到技術的轉化過程中，DeSci 部分主要集中在左半部——也就是純科學部分——試圖構思更好的激勵結構和更好的團隊協調方式，以產生更好的科學成果。值得注意的是，全球研發資金總額從某個角度來看是巨大的，但從另一個角度來看，其實並沒有那麼大，而且在過去幾十年裡也沒有太大的變化，儘管我們正在建立的技術的吞吐量和成果已經有了巨大的增長。

這些資金規模並非區塊鏈無法企及。想想美國非國防研發，每年大約是 700 億美元。這確實很多，但並非天文數字。單看 NSF（美國國家科學基金會），每年大約 100 億美元，這完全可以透過區塊鏈來實現。想想加密貨幣領域的規模，取決於你觀察的時間點，大約在 1 到 3 兆美元之間。

想像一下，如果區塊鏈每年將其供應量的一部分投入研發。想像每年提取 1% 的 Filecoin、以太坊或比特幣，並將其投入研發。你就會開始達到國家級別資助科學的數字範圍。如果加密貨幣再增長一兩個數量級，加密貨幣將能夠以國家規模資助研發和科學，這想想就覺得很瘋狂。因此，在我們達到那個階段之前，弄清楚結構並找出良好的資金途徑將會非常棒。

當你開始分析這些機構的資金時，你會遇到各種問題。某些領域受到的關注太少，或者計畫本身存在反常的激勵機制，或者競爭過於激烈，導致科學家花費大量時間只是在寫補助金申請。在 COVID 期間有一項名為 Fast Grants 的計畫，同樣的效果也在 Impetus Grants 中被複製，這些計畫構建了一個非常快速的補助金計畫。他們能夠以科學家投入時間的一小部分，發放 2 萬到 20 萬美元不等的補助金。

在一項針對申請這些補助金的科學家的調查中，他們指出了他們通常花多少時間申請補助金。想像一下，科學家 25% 到 50% 的時間都花在說明他們在做什麼以及申請各種補助金上。這有點瘋狂。理想情況下，你會希望科學家將絕大部分時間花在思考他們的工作、提出新想法和分析工作上。還有一種效應是，補助金計畫限制了人們最終探索的內容。許多科學家有更具野心的研究想要追求，但他們最終卻陷入了追求其他影響力遠不及此的工作，因為他們必須符合補助金計畫的限制。

Web3 公共財來救援了！有許多不同的團體。當然，這仍然很微小；與全球科學研發資金相比，Web3 運動非常小，但如果我們能把結構弄對，把激勵機制調整好，並證明它是有效的，那麼我們就可以隨著加密貨幣將其擴展幾個數量級。我們應該探索許多不同種類的科學過程資金：不同的補助金計畫、影響力憑證、影響力市場等等。Funding the Commons 社群一直在嘗試許多不同的機制。

例如，像 VitaDAO 這樣的團體正在創建一種資料結構，向團體發放補助金以換取資料、知識和智慧財產權 (IP)。然後，他們將這些 IP 捆綁成具有法律效力的 IPNFT，將 IP 權利授予生物科技公司，並資助這些生物科技公司，目的是透過它們的成功來獲得投資回報。我傾向於稱之為基礎發展基金，透過本身不是公司的實驗室進行重要工作，產生 IP 以資助公司。像 Molecule 這樣的團體正在為這些工作的發生創建市場。

影響力憑證是另一種代表追溯性資金的迷人結構。它們使參與者在取得某些影響力後，能夠圍繞該影響力鑄造憑證，並在市場上將其出售給任何想要申領該影響力的人。這使得投機市場得以出現，跨越時間形成一個閉環，以追溯性地資助極其重要的工作。這至關重要，因為很多時候，你只有在工作完成很久之後，才會意識到某件事有多麼有價值。

#### 組織人員與資料 DAO (15:28) {#organizing-people-and-data-daos-1528}

現在來快速談談關於組織人員的想法。過去，GitHub 在幫助組織科學發現方面取得了巨大的成功。整本教科書和整個領域都是透過 GitHub 發展起來的。許多團體利用 GitHub 圍繞議題、程式碼協作和版本控制的基本原語，來組織實踐和科學社群。但那裡不完整的是，你沒有辦法創建進行研究、處理資金或向貢獻者支付報酬的組織。

有一些有趣的實驗，例如 LabDAO，創建實驗室團隊，讓團體可以組成、籌集資金並進行分配。你能夠對參與者的不同貢獻程度進行編碼，以公平地獎勵他們。還有一些更具野心的專案，圍繞著在更大網路中的參與者之間分配信用，在不同的合作團隊之間傳播獎勵。

有些團體正在實驗同儕審查協定，觀察同儕審查系統的經濟學和動態，以激勵這項工作並適當地獎勵它的發生。一個名為 Ants Review 的協定已經在做這件事，你可以與梅塔馬斯克一起使用它。Gitcoin Grants 已經開創了大量可以在這裡使用的工作，並且已經在為想要以這些方式組織的參與者提供工具支援。

這裡一個非常關鍵的組成部分是透過雜湊連結內容。你可以凍結一組資訊，獲得一個內容定址的雜湊連結，並引用事物。這是你在文獻中想要的核心原語。當你有一篇論文對另一篇論文的引用，或者從一篇論文對其資料或程式碼的引用時，CID 正是你想要的東西。想像一下，能夠透過版本控制凍結整個文獻，並凍結再次運行這些實驗所需的所有重要資料集和程式碼。許多團體正在探索這一點，提出透過 IPFS 進行同儕審查和科學發展的不同方式。

你可以考慮將這種活動和資料生成與稱為資料 DAO 的東西捆綁在一起。與我之前提到已經開始運作的 DAO 不同，資料 DAO 非常新。想像一個能夠收集、策展、轉換和計算資料的團體，並管理這些資料隨著時間的推移如何被使用、如何貨幣化以及如何被分享。

最後關於開放取用和可重現科學的一些注意事項。IPFS 已經被大量用於許多種類的開放科學工作。它已經實現了開放許多科學存取權限的夢想，支援分散式的維基百科副本、大量的論文檔案和資料集。

#### 開放取用、可重現科學與行動呼籲 (20:40) {#open-access-reproducible-science-and-call-to-action-2040}

我們在完全可重現性方面還沒有完全達到目標。這是一個需要更多工作的領域，但許多人已經進行了思考。關於使用 IPFS 的標準可重現性來凍結所有資產並建立完全可重現的管道，有非常好的規格和想法。你可以召回過去的特定實驗，帶回完全凍結的虛擬機 (VM) 或容器，重新運行所有的資料管道，並驗證實驗是否正確。

還有一個完全不同的角度，就是以 DeSci 導向的方式進行資料科學本身，其中筆記本、資料分析和產物都使用由 Web3 驅動的應用程式。像 Jupyter 筆記本、IPython 筆記本和 Wolfram 筆記本等已經與 CID 結合。我認為隨著 Filecoin 網路的巨大增長，這在未來將會得到極大的增強。Filecoin 網路擁有大量與計算結合的儲存空間——儲存提供者在資料旁邊擁有大量的 GPU。這些將在明年連接起來，具備圍繞該資料發布計算管道的能力。想像一下，為科學家生成一個大規模進行資料科學的平台，利用 Web3 計算平台來進行資訊的定址和儲存，以及計算，創建一個完整的端到端資料科學管道。

最後，一個簡短的行動呼籲。科學是進步的引擎。透過擴展我們的知識，我們能夠生產更多技術並改善我們的生活。如果我們能改善科學家的生活，讓他們的工作更輕鬆，加速他們的發展，降低他們的成本，並讓他們花更多時間解決問題而不是寫補助金申請，那麼我們就能以獨特的方式更快地推動社會進步。

DeSci 運動需要你。考慮嘗試新的資金機制、建立開放取用和開放科學工具，或者研究公開資料集。考慮加入 DeSci 團隊或 DAO。探索這些社群，我希望能在那裡見到你。非常感謝，後會有期。
