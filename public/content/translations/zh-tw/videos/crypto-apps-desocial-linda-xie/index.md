由 **Linda Xie** 在 Devconnect 發表關於去中心化社群媒體的演講。Linda 探討了為何使用者與創作者能從開放社交圖譜中受益、內建錢包與迷你應用程式如何在 Farcaster 上創造全新體驗、採用過程中所面臨的真實挑戰，以及與觀眾的問答環節。

#### 中心化社群的問題 (0:00) {#the-problem-with-centralized-social-000}

我已經在加密貨幣領域工作了 11 年，目前在 Farcaster 團隊。在此之前，我曾在 Farcaster 上進行建構。因此，我對加密貨幣與去中心化社群，以及它能為社會帶來什麼充滿熱情。

我認為首先釐清目前傳統社群媒體的問題會很有幫助。我想我們都經歷過類似的問題：你在一個平台上起步，然後最終不得不在另一個平台上建立帳戶，並完全從頭開始。在這種情況下，你的受眾實際上屬於平台本身。當你被平台踢出，或者平台最終被某個國家封禁時，你將無法接觸到任何受眾分發管道。對於使用者和內容創作者來說，能夠接觸到自己的受眾是非常重要的。

最後，如果帳戶和資料歸平台所有，這意味著一旦公司本身倒閉，你將無法存取這些資訊。我們已經屢次看到這種情況發生。在美國，TikTok 面臨潛在的封禁危機，許多創作者對此感到擔憂。在許多國家，這些平台甚至會自我封禁。因此，這是人們在日常生活中實際經歷的問題。

#### 開放社交圖譜 (1:30) {#open-social-graphs-130}

去中心化社群的好處之一，就是你可以隨時存取社交圖譜。你無需許可即可使用這些資料。最棒的是，你可以在這裡的底部看到一個開放社交圖譜。所有這些資料都可以被任何客戶端、任何開發者、任何平台提取，你可以使用這些資料在此平台上進行建構。

一個很好的例子——你有 Farcaster 應用程式，然後你也有 Coinbase 建構的 Base 應用程式。你的追隨者數量會很相似，雖然會有些微差異，因為每個客戶端都有自己的垃圾訊息過濾機制。但我能夠在 Farcaster 上建立起追隨者群體，然後立即開始使用 Base 應用程式，而不需要從頭開始。當你體驗到這一點時，那真的是一個神奇的時刻。對於所有其他 Farcaster 客戶端（如 Zapper、Uno 等）來說，情況也是一樣的。

當你決定使用這個平台時，你不再需要從頭開始。你能夠保留自己的受眾。你在網路上發布一次貼文，就能在多個客戶端上獲得分發。你可以是 Farcaster 的使用者，同時透過 Base 應用程式獲得更多分發，因為他們正在進行更多使用者與創作者的入門引導。這終究是最佳的使用者體驗。現在客戶端正在競爭這些使用者，因為轉換成本非常低——你不需要從頭開始，你保留了你的受眾，因此客戶端必須競爭以建構更好的使用者體驗。

#### 建構者的好處 (3:13) {#benefits-for-builders-313}

接著是建構者方面，建構者不斷被平台踢出。我們在 Twitter 上看到了這種情況——它踢出了許多加密貨幣應用程式和建構者，這對他們的公司來說是一個單點故障。Bankr 最近在同一天被 Twitter 和 Telegram 踢出。幸運的是，他們在幾天後得以恢復，但這確實表明了這些單點故障的存在。如果你是一個投入所有時間和資源試圖建構應用程式的建構者，分發管道突然被切斷絕對是一次極度痛苦的經歷。

我們也從歷史中看到，像 Reddit 這樣的應用程式改變了他們的 API 定價，突然之間營運業務變得極其昂貴。過去有許多應用程式因為無法負擔 Reddit 新的 API 定價而關閉。這在社群中引起了軒然大波。這種情況在這些平台上屢見不鮮。他們可以出於任意原因將你踢出。他們不需要說明踢出你的確切原因。如果他們只是將你視為競爭對手，他們就可以將你從平台上移除。我親自與幾個直接經歷過這種情況的團隊交談過。

去中心化社群真正令人驚嘆的一點是，就像你作為使用者可以在網路上發布一次貼文一樣，你也可以無需許可地在網路上進行建構。開發者能夠無需許可地進行建構，而不必去每個平台詢問：「嘿，我可以在你們的平台上建構嗎？」我今天早些時候剛好在 Farcaster 的攤位，人們問如何將自己加入白名單以便在 Farcaster 上建構，我可以直接告訴他們，這是無需許可的——你可以使用任何這些資料，直接將其整合到你的應用程式中，而且你不需要與團隊溝通。這就是未來事物建構應有的方式。

你可以在這裡看到——我發布了一個名為 Miniword 的填字遊戲。每天人們都在這上面競爭，就像紐約時報的填字遊戲一樣。我在 Farcaster 上發布一次，它也會出現在 Base 應用程式和所有其他客戶端上。作為建構者，你能夠獲得分發量的提升。如果我點擊打開這個迷你應用程式，我就可以開始與它互動，而且因為社交資料是無需許可即可利用的，你可以提取網路上不同人的所有時間記錄，讓人們競爭最高分。這變成了一種更具社交性的體驗，而不僅僅是自己一個人玩拼圖，你可以透過利用社交圖譜獲得更多的病毒式傳播。

#### 加密貨幣與內建錢包 (6:30) {#crypto-and-embedded-wallets-630}

去中心化社群已經存在——我們在 Bluesky 上看到了這一點，隨著平台在不同國家被封禁，他們在吸引使用者方面做得非常出色。然而，真正特別的是當你將加密貨幣與去中心化社群結合時，因為你現在正在創造在傳統平台上永遠無法獲得的體驗。這正是我們團隊非常投入的方向。

Farcaster 上的每個使用者預設都有一個加密貨幣地址。在 Farcaster 應用程式中，你有一個內建錢包，使用者可以輕鬆地向網路上的其他人發送和接收資金。顯然，這是加密貨幣，所以你只需貼上地址，就可以將資金發送給世界各地的任何人。這就是加密貨幣如此特別和獨特的原因。透過將其與開放社交圖譜結合，我們認為這將引領去中心化社群領域的下一波成長。

使用者也可以在鏈上與迷你應用程式互動，因為他們擁有內建錢包。你可以在這裡看到一個在 ETH Global 建構的 eSIM 迷你應用程式——在阿根廷或任何其他國家，你可以購買 eSIM，使用你的內建錢包以 USDC 支付，這是一個非常無縫的體驗。你現在可以建構以前從未存在過的應用程式。

你也可以讓代理程式在動態消息中互動。如果你真的相信這個充滿代理程式的未來，其中有許多代理程式相互互動，那麼加密貨幣就是它們用來互相發送資金的自然來源。你不需要協調地址或某人如何獲得報酬——你自動知道使用者的地址，你擁有所有資訊，無論他們住在世界上的哪個地方都無所謂。你能夠直接與他們互動。我堅信加密貨幣、代理程式和開放社交圖譜將發揮更大的作用。

#### 微型小費 (8:36) {#micro-tipping-836}

關於加密貨幣中開放社交圖譜的獨特之處，我最喜歡的例子之一是微型小費。在加密貨幣的極早期，人們總是談論如果能給微型小費會有多棒——如果你是一個發布優質內容的創作者，能夠發送極少量的資金給創作者。我曾使用過這些早期的比特幣小費應用程式。但早期的問題是，發送極少量的資金實際上相當緩慢且昂貴。

現在，由於 L2 的出現及其低廉快速的特性，你有能力發送一美分甚至更低金額的交易。有像 Tipin 和 Noise 這樣的應用程式，讓你可以為每次互動（追蹤、轉發、按讚貼文、回覆）自動給予小費。你能夠設定你想給予金額的授權額度。以我為例，我設定了每次按讚別人的貼文就給一美分，這在背景會自動發生。每次我在網路上按讚內容時，我就會自動發送一美分給該使用者。

你可以看到我的錢包活動歷史記錄——我正在向網路上的其他使用者發送資金，我也從設定了這些授權額度的其他使用者那裡接收資金。甚至還有低於一美分的交易，這是一個相當瘋狂的概念。這些人住在世界各地。如果你是加密貨幣的新手並加入了 Farcaster，你預設會有一個錢包。一旦你開始發布高品質的內容，人們就可以開始自動給你小費，你可以在不曾購買加密貨幣的情況下累積餘額。我與許多使用者交談過，他們說這實際上是他們因為創作內容或與迷你應用程式互動而賺取的第一筆加密貨幣。

#### 迷你應用程式與創意工具 (11:01) {#mini-apps-and-creative-tools-1101}

我最喜歡的迷你應用程式之一，它很好地展示了內建錢包、加密貨幣交易和開放社交圖譜的使用，那就是由 Atown 及其團隊建構的 Emerge。它讓創作者能夠組合一個提示詞，將你的個人資料圖片轉換為該提示詞的某種版本。這裡有一個例子——網路上非常受歡迎的創作者 Peachy 寫了一個提示詞，將個人資料圖片轉換為塔羅牌。我能夠使用我的個人資料圖片生成一張圖片，並分享這個迷你應用程式，讓其他感興趣的人也能生成他們自己的圖片。這裡有表現優異的提示詞排行榜。如果你是一個有創意的人，你可以創建自己的提示詞並開始賺錢——我為這筆提示詞交易支付了 25 美分，還有 418 位其他使用者也生成了它，所以 Peachy 因為創建這個內容而獲得了收入，而且它可以在動態消息中像病毒一樣傳播。

#### Warplet 現象 (12:41) {#the-warplet-phenomenon-1241}

實際上，由於最近發生的病毒式傳播，我們在 Farcaster 上迎來了日活躍使用者數最高的一天。首先，我們有一個名為 Warplet 的 Farcaster 非官方吉祥物。Farcaster 的共同創辦人 Dan 創造了它。團隊——包括我在內——可能覺得它作為官方吉祥物不是最可愛的，所以他把它發布到社群中，並說人們可以隨心所欲地使用它。

一位名叫 Angel 的開發者採用了它，並決定建構一個迷你應用程式，你可以在其中鑄造一個混合了你自己個人資料圖片 DNA 的 Warplet 生物版本。NishProf 分享了她從個人資料圖片客製化的 Warplet——結果有超過 49,000 個帳戶生成了他們的 Warplet。它完全像病毒一樣傳播開來。Angel 僅在幾天內就透過創建這個迷你應用程式賺了將近 100,000 美元。這確實表明，當你有一個任何人都可以輕鬆進行交易的內建錢包時，作為建構者，你可以透過建構在社交動態中病毒式傳播的應用程式來賺錢。

同樣非常酷的是看到了不同的混音文化。建構者們說：「好吧，你們現在都有自己的 Warplet 了，所以讓我們在上面建構遊戲和有趣的體驗吧。」出現了「最醜的 Warplet 是哪個？」以及「如果你把這兩個結合起來，Warplet 寶寶會長什麼樣？」的排名。所有這些有趣的遊戲都是在網路上創建的，這表明當你利用社交圖譜時，作為應用程式開發者，你可以創造出更具吸引力的體驗。

#### 為了社會公益的迷你應用程式 (15:02) {#mini-apps-for-social-good-1502}

這是一個迷你應用程式的例子——實際上是團隊中的 Horsefax 建構的。我認為這非常重要——你可以支持 Roman Storm 的法律辯護基金。我們使用了 Daimo Pay，你可以用任何代幣支付並為法律辯護基金捐款。因為它是一個內建錢包，你永遠不會被踢出去使用另一個應用程式。你可以立即在你的動態消息中分享，告訴你的追隨者你捐款了。然後他們可以發現這個應用程式並繼續捐款。這是一種在每個人自己建立的追隨者和受眾中進一步分享應用程式的機制。

這是另一個類似的例子——由 Seed Club 的 Nicholas 建構的群眾募資，就像 GoFundMe 或 Kickstarter 平台一樣。你可以看到不同使用者的捐款。你可以在你的內建錢包中點擊，直接透過 Farcaster 應用程式捐款。你永遠不需要被踢出去。創作者能夠籌集到遠超過他們目標的資金，你可以在你的動態消息中分享，這樣你所有的追隨者也能發現這個迷你應用程式。

#### 去中心化社群的挑戰 (16:41) {#challenges-of-decentralized-social-1641}

所有這些都非常棒，去中心化社群聽起來很棒，但我想非常誠實地談談挑戰。需要注意的一點是，它很小——相對於所有這些擁有數十億和數億使用者的主要傳統社群平台來說，它非常小。

然而，我真的想強調——我已經在加密貨幣領域工作了 11 年，從比特幣的早期就開始了。你可以比較像黃金這樣的東西——看看比特幣已經走了多遠。你也可以看看去中心化金融 (DeFi) 以及所有的總鎖倉量 (TVL) 和兌換交易量，看看它成長了多少。這一切都是社群、建構者和使用者非常有可能實現的。我堅信社群將在加密貨幣領域經歷另一個成長階段，因為這非常合理。這對使用者來說是更好的體驗，對建構者來說也是更好的體驗，而且你現在可以創造以前根本不存在的東西。隨著加密貨幣變得更加主流，我們希望能與之共同成長。

特別是對於 Farcaster，我們發現人們對內建錢包產生了強烈的共鳴，所以我們非常投入這些功能。對非加密貨幣參與者進行入門引導是任何加密貨幣應用程式都在經歷的事情——如果你是一個完全的新手，必須了解什麼是錢包以及如何保持安全。自早期以來，我們已經改進了許多，但與傳統消費者體驗相比，仍有許多需要改進的地方。

垃圾訊息也是一個問題。這在中心化社群平台上也存在——你可能在 Twitter 上體驗過，要分辨什麼是真實帳戶有多難。在 Farcaster 層面，你可以擁有在協定層面幫助客戶端進行過濾的垃圾訊息服務，或者客戶端可以進行自己的垃圾訊息過濾。

然後是內容審核——這也是傳統社群平台面臨的問題和挑戰，以確保使用者有良好的體驗並發現他們真正感興趣的內容。這些是我們一直持續努力解決的問題。

#### 為什麼是現在？ (19:48) {#why-now-1948}

去中心化社群已經存在一段時間了，但為什麼是現在？我認為這是所有這些基礎元件結合在一起的結果。首先，你有任何人都可以使用和利用的開放社交圖譜。我經常與以前甚至不認識的團隊交談——他們只是在建構 Farcaster 迷你應用程式，因為他們能夠在沒有我們的情況下利用這些資料。

L2 是其中關鍵的一部分，使交易變得非常便宜和快速——你可以看到我們之前分享的微型小費等體驗。內建錢包也是關鍵——我真的想強調，在使用消費者應用程式時，它們的體驗有多好。我們很多人都經歷過被踢出到錢包，然後必須被帶回來，有時還會失敗。有了內建錢包，你留在應用程式體驗中，流失率會大大降低。

像迷你應用程式和代理程式這樣的新建構區塊，是你能夠使用的全新事物，我們以前未必見過如此規模的應用。而在加密貨幣之外發生的一個大趨勢是 AI 變得多麼強大——你現在可以使用提示詞來創建應用程式。這確實降低了人們實際建構的成本和門檻。我們有像 Neynar 這樣的服務，讓你在不是開發者的情況下也能創建 Farcaster 迷你應用程式。

這一切將如何發展——我認為電子郵件是一個很好的比喻。當你發送電子郵件時，你不需要建立一個 AOL 帳戶來與擁有 AOL 電子郵件的人交談。你可以使用 Gmail、Yahoo、Outlook 或任何你想要的，你們都能夠在協定層面相互溝通。我認為這就是去中心化社群應該發展的方式——所有這些網路資料、所有這些社交圖譜都在協定層面，每個客戶端都可以在其上建構針對其使用者群體的特定功能。他們可以根據使用者實際喜歡的內容，擁有不同的演算法和不同的功能。

Farcaster 越來越多地將時間花在交易功能上；如果你對此不感興趣，Base 應用程式和 Uno 在展示社交功能方面非常出色。從長遠來看，這才是社群應有的樣子。我們還有很長的路要走，但我真的認為這就是社群未來應該發展的方向。

#### 問答 (23:47) {#qa-2347}
