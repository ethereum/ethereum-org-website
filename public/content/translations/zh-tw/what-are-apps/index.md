## 擁有超能力的應用程式 {#apps-with-superpowers}

以太坊應用程式用起來可能感覺像一般的應用程式。但在幕後，它們具有一些特殊的性質。

一旦應用程式發佈在以太坊區塊鏈上，它就變得無法阻擋。這是因為以太坊網路是去中心化的，分佈在世界各地成千上萬台電腦上。沒有人能關閉運行在以太坊上的應用程式，因為沒有單一的伺服器可以作為攻擊目標。以太坊也是中立的，因此世界上任何地方的任何人都可以使用它，甚至連接到它並在上面建立自己的修改版本。

## 什麼是 dapp？ {#what-is-a-dapp}

以太坊應用程式的邏輯運行在以太坊區塊鏈上，而不是在中心化伺服器中。這就是為什麼它們通常被稱為去中心化應用程式 (dapp)。

## 為什麼這很重要 {#why-does-this-matter}

以太坊應用程式可以做到傳統應用程式根本無法做到的事情。例如，將錢借給一個完全陌生的人，並保證您能拿回本金加上利息。而且無需支付像律師這樣「值得信賴」的中介機構來處理交易。

這裡有適用於各種用途的應用程式：遊戲、金融、工作、通訊、儲存等等。對於大多數應用程式，您不會受到廣告的干擾，也不會受到存取權限的限制。

您只需要一個以太坊錢包和一點 ETH 就可以開始使用任何以太坊應用程式。

## 它是如何運作的 {#how-does-it-work}

應用程式由智能合約驅動——這些是存在於以太坊區塊鏈上的程式碼片段。與傳統應用程式不同，它們不需要公司來運行。

| 功能 | 傳統應用程式 | 以太坊應用程式 |
| ------- | ---------------- | ------------- |
| **誰控制它？** | 一家公司 | 沒有人 |
| **運行於** | 私人公司伺服器 | 公共以太坊區塊鏈 |
| **可以被審查嗎？** | 是 | 否 |
| **誰擁有您的資料？** | 通常不是您 | 您擁有自己的資料 |

## 以太坊應用程式就像樂高積木 {#ethereum-apps-are-like-legos}

當應用程式都建立在以太坊上時，它們都是相容的。一個應用程式的代幣，可以在一個完全不同的應用程式上使用。這就像能夠將推文發佈到您的臉書塗鴉牆上一樣。事實上，您通常可以在許多不同的以太坊應用程式中重複使用同一個個人檔案，而無需在每個地方分別註冊。
