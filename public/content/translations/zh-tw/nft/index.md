## 什麼是 NFT？ {#what-are-nfts}

NFT 是**獨一無二**的代幣。每個 NFT 都有不同的屬性（非同質化），並且具有可證明的稀缺性。這與 [ETH](/glossary/#ether) 或其他基於[以太坊](/)的代幣（如 USDC）不同，後者的每個代幣都是相同的，並具有相同的屬性（「同質化」）。你不會在意你的錢包裡具體是哪一張鈔票（或哪一個 ETH），因為它們都是相同的，價值也一樣。然而，你_確實_會在意你擁有的是哪一個特定的 NFT，因為它們都有各自的屬性，使其與眾不同（「非同質化」）。

每個 NFT 的獨特性使得藝術品、收藏品甚至房地產等事物的代幣化成為可能，其中一個特定的獨特 NFT 代表某個特定的獨特現實世界或數位物品。資產的所有權可以在以太坊[區塊鏈](/glossary/#blockchain)上公開驗證。

## 資產網際網路 {#internet-of-assets}

NFT 和以太坊解決了當今網際網路上存在的一些問題。隨著一切變得更加數位化，我們需要以一種不受中心化組織控制的方式，複製實體物品的屬性，如稀缺性、獨特性和所有權證明。例如，透過 NFT，你可以在所有基於以太坊的應用程式中擁有一個音樂 mp3 檔案，而不受限於某家公司的特定音樂應用程式（如 Spotify 或 Apple Music）。你可以擁有一個社群媒體帳號，你可以出售或兌換它，但平台供應商**不能隨意將其從你手中奪走**。

以下是 NFT 網際網路與我們大多數人今天使用的網際網路的比較...

### 比較 {#nft-comparison}

| NFT 網際網路                                                                                                         | 當今的網際網路                                                                                                                                     |
| ----------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **你擁有你的資產！** 只有你可以出售或兌換它們。                                                                    | **你向某個組織租用資產**，而且它可能會被奪走。                                                                            |
| NFT 是**數位上獨一無二的**，沒有兩個 NFT 是相同的。                                                                    | **複製品通常無法與原件區分**。                                                                                   |
| NFT 的所有權儲存在區塊鏈上，任何人都可以**公開驗證**。                                                   | 數位物品所有權記錄的存取權限由**機構控制**——你只能相信他們的一面之詞。                                 |
| NFT 是以太坊上的[智能合約](/glossary/#smart-contract)。這意味著它們**可以輕鬆地在以太坊上的其他智能合約**和應用程式中使用！ | 擁有數位物品的公司通常**需要他們自己的「封閉花園」基礎設施**。                                                                 |
| 內容**創作者可以在任何地方出售他們的作品**，並能進入全球市場。                                           | 創作者依賴他們所使用平台的基礎設施和發行管道。這些通常受限於使用條款和**地理限制**。 |
| NFT 創作者**可以保留對自己作品的所有權**，並將版稅直接編寫進 NFT 合約中。     | 平台（如音樂**串流服務）保留了銷售的大部分利潤**。                                                                |

## NFT 有什麼用途？ {#nft-use-cases}

NFT 有許多用途，包括：

- 證明你參加過某個活動
- 證明你完成了一門課程
- 遊戲中可擁有的物品
- 數位藝術
- 將[現實世界資產](/real-world-assets/)代幣化
- 證明你的線上身分
- 限制內容的存取權限
- 票務
- 去中心化的網際網路網域名稱
- [去中心化金融 (DeFi)](/glossary/#defi)中的抵押品

也許你是一位藝術家，希望使用 NFT 分享自己的作品，而不想失去控制權並將利潤犧牲給仲介機構。你可以建立一個新的合約，並指定 NFT 的數量、屬性以及指向某個特定藝術品的連結。身為藝術家，**你可以將你應得的版稅編寫進智能合約中**（例如，每次轉帳 NFT 時，將銷售價格的 5% 轉帳給合約擁有者）。你也可以隨時證明是你創造了這些 NFT，因為你擁有部署該合約的[錢包](/glossary/#wallet)。你的買家可以輕鬆證明他們擁有來自你收藏的**正版 NFT**，因為他們的錢包[地址](/glossary/#address)與你智能合約中的代幣相關聯。他們可以在整個以太坊生態系統中使用它，並對其真實性充滿信心。

或者考慮一張體育賽事的門票。就像**活動主辦單位可以選擇出售多少張門票**一樣，NFT 的創作者也可以決定存在多少個複製品。有時這些是完全相同的複製品，例如 5000 張普通門票。有時會鑄造幾個非常相似但每個都略有不同的 NFT，例如帶有指定座位的門票。這些門票可以點對點買賣，無需支付票務代理費用，而且買家始終可以透過檢查合約地址來確保門票的真實性。

在 ethereum.org 上，**NFT 被用來證明人們對我們的 GitHub 儲存庫做出了有意義的貢獻**（編寫網站程式碼、撰寫或修改文章...）、翻譯了我們的內容，或參加了我們的社群電話會議，我們甚至擁有自己的 NFT 網域名稱。如果你對 ethereum.org 做出貢獻，你可以申領一個 [POAP](/glossary/#poap) NFT。一些加密貨幣聚會已經使用 POAP 作為門票。[更多關於貢獻的資訊](/contributing/#poap)。你也可以探索我們的 [ethereum.org 收藏品](/collectibles/)頁面，查看社群成就 NFT。

## NFT 是如何運作的？ {#how-nfts-work}

與以太坊區塊鏈上的任何數位物品一樣，NFT 是透過一種稱為「智能合約」的特殊以太坊電腦程式建立的。這些合約遵循某些規則，例如 [ERC-721](/glossary/#erc-721) 或 [ERC-1155](/glossary/#erc-1155) 標準，這些標準決定了合約可以做什麼。

NFT 智能合約可以執行幾個關鍵操作：

- **建立 NFT：** 它可以製作新的 NFT。
- **分配所有權：** 它透過將 NFT 連結到特定的以太坊地址來追蹤誰擁有哪個 NFT。
- **為每個 NFT 提供一個 ID：** 每個 NFT 都有一個使其獨一無二的編號。此外，通常還會附加一些資訊（中繼資料），描述該 NFT 代表什麼。

當有人「建立」或「鑄造」一個 NFT 時，他們基本上是在告訴智能合約賦予他們特定 NFT 的所有權。這些資訊安全且公開地儲存在區塊鏈中。

此外，合約的建立者可以新增額外的規則。他們可能會限制某種 NFT 的製作數量，或者決定每當 NFT 易手時，他們應該獲得一小筆版稅。

### NFT 安全性 {#nft-security}

以太坊的安全性來自於[權益證明 (PoS)](/glossary/#pos)。該系統的設計旨在從經濟上抑制惡意行為，使以太坊具有防篡改性。這正是 NFT 得以實現的原因。一旦包含你 NFT 交易的[區塊](/glossary/#block)變成[已定案](/glossary/#finality)，攻擊者將需要花費數百萬 ETH 才能更改它。任何執行以太坊軟體的人都能立即偵測到對 NFT 的不誠實篡改，而惡意行為者將受到經濟懲罰並被驅逐。

與 NFT 相關的安全問題通常與網路釣魚詐騙、智能合約漏洞或使用者錯誤（例如不慎暴露私鑰）有關，因此良好的錢包安全性對 NFT 擁有者來說至關重要。
