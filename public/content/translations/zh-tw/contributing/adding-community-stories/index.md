## 如何貢獻故事 {#how-to-contribute}

社群故事會顯示在 [ethereum.org/stories/](/stories/)，並突顯個人在開源與庇護技術方面的經驗、以太坊生態系統如何影響人們的生活，以及世界各地的社群如何使用以太坊。任何人都可以建議或提交故事。

貢獻故事有三種方式：

1. **精選故事** - 完整長度的文字故事，在網站上作為獨立頁面發布。
2. **短篇故事** - 簡短的見證，在故事登陸頁面上以卡片形式顯示。
3. **影片故事** - 透過[影片庫](/videos/)提交並在故事頁面上精選的影片。

### 上架政策 {#listing-policy}

Ethereum.org 是一個中立的教育資源。故事頁面的策展目的是：

- 透過真實的個人與社群經驗，將以太坊的目的與技術**人性化**
- <strong>突顯</strong>開源與庇護技術如何為人們的生活帶來真正的改變
- <strong>慶祝</strong>全球以太坊社群的多樣性

本網站不會列出主要用於推廣特定產品、代幣或商業服務的故事。所有提交的內容都會由 ethereum.org 團隊進行審核。

### 收錄標準 {#criteria-for-inclusion}

#### 必備條件 {#must-haves}

- **真實的人類故事** - 故事必須描述真實的個人或社群經驗。這可以包含您在開源與庇護技術方面的經驗、以太坊生態系統如何影響您的生活，或是您與您的社群如何使用以太坊。虛構、假設或純技術內容屬於其他區塊（例如，技術建構者領域文章請見 `/latest/`）。
- **與以太坊的關聯** - 故事必須以有意義的方式涉及以太坊、基於以太坊的應用程式（包含 L2、以太坊上的穩定幣、去中心化應用程式 (dapp) 等），或是更廣泛的開源與庇護技術生態系統。不接受沒有特定以太坊焦點、關於加密貨幣與區塊鏈的一般性故事。
- **非宣傳性質** - 故事不應主要用於推廣特定產品、代幣、公司或商業服務。提及所使用的工具是可以接受的；但將故事包裝成產品見證則不被允許。
- **適當的內容** - 故事必須遵守 ethereum.org 的[行為準則](/community/code-of-conduct/)。不接受充滿仇恨、誤導性或非法的內容。
- **英文優先** - 故事可以使用任何[支援的語言](/contributing/translation-program/)提交。設定 `lang` 欄位以符合故事撰寫的語言（例如，英文為 `en`，西班牙文為 `es`）。英文提交的內容將被翻譯成其他語言，而非英文提交的內容則可透過[翻譯計畫](/contributing/translation-program/)翻譯成英文。

#### 加分條件 {#nice-to-haves}

- **地理與主題多樣性** - 來自代表性不足地區以及涵蓋代表性不足使用案例的故事將獲得優先考慮。
- **多媒體** - 鼓勵附帶影片、音訊或攝影照片的故事。
- **開源與庇護價值觀** - 特別重視能說明開源或庇護技術如何實現抗審查性、開放存取、普惠金融、隱私或安全性的故事。

### 如何新增精選故事 {#how-to-add-featured-story}

#### 選項 1：透過接收表單提交 {#submit-via-intake-form}

如果您有故事想分享，可以使用接收表單提交。ethereum.org 團隊將會審核您提交的內容並進行後續聯繫。

<ButtonLink href="https://ethereumstory.paperform.co/">
  提交您的故事
</ButtonLink>

#### 選項 2：在 GitHub 上開啟 issue {#open-an-issue}

如果您想為 ethereum.org 建議社群故事，且該故事符合標準，請在 GitHub 上建立一個 issue。

<ButtonLink href="https://github.com/ethereum/ethereum-org-website/issues/new?template=content-suggestion.yml">
  建立 issue
</ButtonLink>

### 其他貢獻故事的方式 {#other-ways-to-contribute}

您也可以建議**短篇故事**或**影片故事**：

- <strong>短篇故事</strong>會以卡片形式顯示在[故事登陸頁面](/stories/)上。這些故事可以直接發布在 ethereum.org 上，或是連結到在推特上分享的故事。
- <strong>影片故事</strong>是在[影片庫](/videos/)中標記為 `community-stories` 類別的影片。若要新增影片故事，請遵循現有的[新增影片](/contributing/adding-videos/)貢獻指南，並在主題標籤中包含 `community-stories`。

如果您想建議這兩種故事類型中的任何一種，請在 GitHub 上建立一個 issue。

<ButtonLink href="https://github.com/ethereum/ethereum-org-website/issues/new?template=BLANK_ISSUE">
  建立 issue
</ButtonLink>

### 維護 {#maintenance}

- 如果事實細節發生變化，精選故事可能會進行更新。
- 短篇故事會經過適當性與準確性的審核。
- 故事頁面會自動擷取標記為 `community-stories` 的新影片項目。
