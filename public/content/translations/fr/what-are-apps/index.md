## Des applications avec des super-pouvoirs {#apps-with-superpowers}

Les applications Ethereum peuvent ressembler à des applications ordinaires. Mais en coulisses, elles possèdent des qualités particulières.

Une fois qu'une application est publiée sur la chaîne de blocs Ethereum, elle devient inarrêtable. C'est parce que le réseau Ethereum est décentralisé sur des milliers d'ordinateurs à travers le monde. Personne ne peut mettre hors ligne les applications fonctionnant sur Ethereum, car il n'y a pas de serveur unique à cibler. Ethereum est également neutre, de sorte que n'importe qui, n'importe où dans le monde, peut l'utiliser ou même s'y connecter et construire ses propres modifications par-dessus.

## Qu'est-ce qu'une dapp ? {#what-is-a-dapp}

Les applications Ethereum ont leur logique qui s'exécute sur la chaîne de blocs Ethereum au lieu de serveurs centralisés. C'est pourquoi elles ont souvent été appelées applications décentralisées (dapp) en abrégé.

## Pourquoi est-ce important ? {#why-does-this-matter}

Les applications Ethereum peuvent faire des choses qui sont tout simplement impossibles avec les applications traditionnelles. Comme prêter de l'argent à un parfait inconnu avec la garantie que vous récupérerez votre argent, plus les intérêts. Sans payer un intermédiaire « de confiance », comme un avocat, pour gérer la transaction.

Il existe des applications pour tout : les jeux, la finance, le travail, la messagerie, le stockage et plus encore. Avec la plupart des applications, vous n'êtes pas soumis à des publicités ni limité par un accès restreint.

Tout ce dont vous avez besoin, c'est d'un portefeuille Ethereum et d'un peu d'ETH pour commencer à utiliser n'importe quelle application Ethereum.

## Comment ça marche ? {#how-does-it-work}

Les applications sont propulsées par des contrats intelligents — des morceaux de code qui vivent sur la chaîne de blocs Ethereum. Contrairement aux applications traditionnelles, elles n'ont pas besoin d'une entreprise pour les faire fonctionner.

| Fonctionnalité | Applications traditionnelles | Applications Ethereum |
| ------- | ---------------- | ------------- |
| **Qui la contrôle ?** | Une entreprise | Personne |
| **S'exécute sur** | Des serveurs d'entreprises privées | La chaîne de blocs publique Ethereum |
| **Peut-elle être censurée ?** | Oui | Non |
| **À qui appartiennent vos données ?** | Généralement pas à vous | Vous possédez vos données |

## Les applications Ethereum sont comme des Legos {#ethereum-apps-are-like-legos}

Lorsque les applications sont toutes construites sur Ethereum, elles sont toutes compatibles. Un jeton pour une application fonctionnera sur une application complètement différente. C'est comme pouvoir publier des tweets sur votre mur Facebook. En fait, vous pouvez souvent réutiliser le même profil dans de nombreuses applications Ethereum différentes sans avoir besoin de vous inscrire partout séparément.
