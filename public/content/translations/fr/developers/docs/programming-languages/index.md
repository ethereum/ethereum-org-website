Une idée fausse très répandue est que les développeurs doivent écrire des [contrats intelligents](/developers/docs/smart-contracts/) pour construire sur Ethereum. C'est faux.
L'une des beautés du réseau Ethereum et de sa communauté est que vous pouvez y [participer](/community/) dans presque n'importe quel langage de programmation.

Ethereum et sa communauté adoptent l'open source. Vous pouvez trouver des projets communautaires - implémentations de clients, API, frameworks de développement, outils de test - dans une grande variété de langages.

### Que faire si mon langage n'est pas pris en charge ? {#other-lang}

Si vous souhaitez créer un lien vers des ressources ou indiquer une communauté virtuelle pour un langage de programmation supplémentaire, vous pouvez demander une nouvelle page en [ouvrant une issue](https://github.com/ethereum/ethereum-org-website/issues/new/choose).

Si vous souhaitez simplement écrire du code pour interagir avec la chaîne de blocs en utilisant un langage actuellement non pris en charge, vous pouvez utiliser l'[interface JSON-RPC](/developers/docs/apis/json-rpc/) pour vous connecter au réseau Ethereum. Tout langage de programmation capable d'utiliser TCP/IP peut utiliser cette interface.
