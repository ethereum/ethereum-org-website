<FeaturedText>تعلم كيفية التطوير على إيثريوم باستخدام المشاريع والأدوات القائمة على Ruby.</FeaturedText>

استخدم إثيريوم لإنشاء تطبيقات لامركزية (أو "dapp") تستخدم فوائد العملات المشفرة وتكنولوجيا سلسلة الكتل. يمكن أن تكون هذه التطبيقات اللامركزية غير موثوقة، مما يعني أنه بمجرد نشرها على Ethereum، فستعمل دائمًا كما هو مبرمج. يمكن أن تكون هذه التطبيقات اللامركزية غير موثوقة، مما يعني أنه بمجرد نشرها على Ethereum، فستعمل دائمًا كما هو مبرمج. كما أنها قد تكون لامركزية، بمعنى عدم وجود كيان أو شخص واحد يتحكم فيها ويكاد يكون من المستحيل مراقبتها.

## مشاريع وأدوات Ruby {#ruby-projects-and-tools}

### نشيطٌ {#active}

- [eth.rb](https://github.com/q9f/eth.rb) - _مكتبة Ruby وعميل RPC للتعامل مع حسابات ورسائل ومعاملات إيثريوم_
- [keccak.rb](https://github.com/q9f/keccak.rb) - _تجزئة (هاش) Keccak (SHA3) المُستخدمة بواسطة إيثريوم_
- [siwe-ruby](https://github.com/signinwithethereum/siwe-ruby) - _تطبيق Ruby لتسجيل الدخول باستخدام إيثريوم_
- [siwe-rails](https://github.com/signinwithethereum/siwe-rails) - _إضافة Rails التي تضيف مسارات تسجيل الدخول المحلية لـ SIWE_
- [siwe-rails-examples](https://github.com/signinwithethereum/siwe-rails-examples) - _مثال SIWE باستخدام Ruby on Rails مع وحدة تحكم مخصصة_
- [omniauth-siwe](https://github.com/signinwithethereum/omniauth-siwe) - _استراتيجية OmniAuth لتسجيل الدخول باستخدام إيثريوم (SIWE)_
- [omniauth-nft](https://github.com/valthon/omniauth-nft) - _استراتيجية OmniAuth للمصادقة عبر ملكية NFT_
- [ethereum-on-rails](https://github.com/q9f/ethereum-on-rails) - _قالب Ethereum on Rails الذي يسمح بربط MetaMask بـ Ruby on Rails_

### أرشيفي / لم يعد مدعومًا بعد الآن{#archived--no-longer-maintained}

- [web3-eth](https://github.com/spikewilliams/vtada-ethereum) - _استدعاء أساليب RPC لعقدة إيثريوم باستخدام Ruby_
- [ethereum_tree](https://github.com/longhoangwkm/ethereum_tree) - _مكتبة Ruby لتوليد عناوين ETH من محفظة هرمية حتمية وفقًا لمعيار BIP32_
- [etherlite](https://github.com/budacom/etherlite) - _تكامل إيثريوم لـ Ruby on Rails_
- [ethereum.rb](https://github.com/EthWorks/ethereum.rb) - _عميل إيثريوم لـ Ruby يستخدم واجهة JSON-RPC لإرسال المعاملات، وإنشاء العقود والتفاعل معها، بالإضافة إلى مجموعة أدوات مفيدة للعمل مع عقدة إيثريوم_
- [omniauth-ethereum.rb](https://github.com/q9f/omniauth-ethereum.rb) - _يطبق استراتيجية مزود إيثريوم لـ OmniAuth_
