इथेरियम नेटवर्क 24/7, 365 दिवस अब्जावधी डॉलर्सचे मूल्य सुरक्षित ठेवते. कसे?

जगभरातील लोक इथेरियम व्यवहार प्रक्रिया करणारे आणि इथेरियम नेटवर्क सुरक्षित करणारे सॉफ्टवेअर चालवण्यासाठी स्मार्ट कॉन्ट्रॅक्ट्समध्ये [इथर (ETH)](/what-is-ether/) लॉक करतात (किंवा "स्टेक" करतात). याच्या मोबदल्यात, त्यांना अधिक ETH बक्षीस म्हणून मिळतात.

रिस्टेकिंग हे [स्टेकर्ससाठी](/staking/) तयार केलेले एक तंत्रज्ञान आहे, जे ही सुरक्षा इतर सेवा, ॲप्लिकेशन्स किंवा नेटवर्क्सपर्यंत वाढवते. याच्या मोबदल्यात, ते अतिरिक्त रिस्टेकिंग बक्षिसे मिळवतात. तथापि, ते त्यांचे स्टेक केलेले ETH अधिक धोक्यात देखील टाकतात.

## रिस्टेकिंग म्हणजे काय? {#what-is-restaking}

जेव्हा स्टेकर्स इतर विकेंद्रित सेवा सुरक्षित करण्यासाठी त्यांच्या आधीच स्टेक केलेल्या ETH चा वापर करतात तेव्हा त्याला रिस्टेकिंग म्हणतात. याच्या मोबदल्यात, रिस्टेकर्सना त्यांच्या नियमित ETH स्टेकिंग बक्षिसांव्यतिरिक्त त्या इतर सेवांकडून अतिरिक्त बक्षिसे मिळू शकतात.

रिस्टेकिंगद्वारे सुरक्षित केलेल्या विकेंद्रित सेवांना "ॲक्टिव्हली व्हॅलिडेटेड सर्व्हिसेस" (AVSs) म्हणून ओळखले जाते.
ज्याप्रमाणे अनेक ETH स्टेकर्स इथेरियम व्हॅलिडेशन सॉफ्टवेअर चालवतात, त्याचप्रमाणे अनेक रिस्टेकर्स विशेष AVS सॉफ्टवेअर चालवतात.

## स्टेकिंग विरुद्ध रिस्टेकिंग {#staking-vs-restaking}

| स्टेकिंग                        | रिस्टेकिंग                                         |
| ------------------------------ | ------------------------------------------------- |
| ETH बक्षिसे मिळवा               | ETH बक्षिसे + AVS बक्षिसे मिळवा                    |
| इथेरियम नेटवर्क सुरक्षित करते   | इथेरियम नेटवर्क + AVSs सुरक्षित करते               |
| किमान ETH ची अट नाही                 | किमान ETH ची अट नाही                                    |
| कमी धोक्याची पातळी                 | कमी ते उच्च धोक्याची पातळी                            |
| पैसे काढण्याची वेळ रांगेवर अवलंबून असते | पैसे काढण्याची वेळ रांग + अनबॉन्डिंग कालावधीवर अवलंबून असते |

## रिस्टेकिंग कसे काम करते? {#how-does-restaking-work}

रिस्टेकिंगमध्ये अनेक घटक सामील असतात — त्यापैकी प्रत्येकजण महत्त्वाची भूमिका बजावतो.

| **संज्ञा**                | **वर्णन**                                                                                                                                                                                                                                                                                                                                   |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **रिस्टेकिंग प्लॅटफॉर्म्स** | रिस्टेकिंग प्लॅटफॉर्म ही एक सेवा आहे जी AVSs, ETH स्टेकर्स आणि ऑपरेटर्सना जोडते. ते स्टेकर्सना त्यांचे ETH रिस्टेक करण्यासाठी विकेंद्रित ॲप्लिकेशन्स तयार करतात आणि अशी मार्केटप्लेसेस तयार करतात जिथे स्टेकर्स, AVSs आणि ऑपरेटर्स एकमेकांना शोधू शकतात.                                                                                                                |
| **नेटिव्ह रिस्टेकर्स**    | जे लोक स्वतःचे इथेरियम प्रमाणक चालवून त्यांचे ETH स्टेक करतात, ते ETH प्रमाणक बक्षिसांव्यतिरिक्त रिस्टेकिंग बक्षिसे मिळवण्यासाठी त्यांचे स्टेक केलेले ETH EigenLayer आणि इतरांसह रिस्टेकिंग प्लॅटफॉर्मशी जोडू शकतात.                                                                                                                             |
| **लिक्विड रिस्टेकर्स**    | जे लोक लिडो (Lido) किंवा Rocket Pool सारख्या तृतीय-पक्ष लिक्विड स्टेकिंग प्रदात्याद्वारे त्यांचे ETH स्टेक करतात, त्यांना तरल स्टेकिंग टोकन (LST) मिळतात जे त्यांच्या स्टेक केलेल्या ETH चे प्रतिनिधित्व करतात. ते त्यांचे मूळ ETH स्टेक केलेले ठेवून रिस्टेकिंग बक्षिसे मिळवण्यासाठी या LSTs ला रिस्टेक करू शकतात.                                                                                  |
| **ऑपरेटर्स**           | ऑपरेटर्स AVSs चे रिस्टेकिंग सॉफ्टवेअर चालवतात, प्रत्येक AVS ला आवश्यक असलेली प्रमाणीकरण कार्ये करतात. ऑपरेटर्स सहसा व्यावसायिक सेवा प्रदाते असतात जे अपटाइम आणि कार्यप्रदर्शनासारख्या गोष्टींची हमी देतात. नॉन-ऑपरेटर रिस्टेकर्सप्रमाणेच, ऑपरेटर्स AVSs सुरक्षित करण्यासाठी स्टेक केलेल्या ETH चा वापर करतात, परंतु ऑपरेटर्सना त्यांच्या कामाच्या बदल्यात अतिरिक्त बक्षिसे देखील मिळतात. |
| **AVSs**                | या विकेंद्रित सेवा आहेत — जसे की प्राइस ओरॅकल्स, टोकन ब्रिजेस आणि डेटा सिस्टीम्स — ज्यांना रिस्टेकर्सकडून सुरक्षा मिळते आणि त्या बदल्यात टोकन बक्षिसे देतात.                                                                                                                                                                              |

## मी रिस्टेकिंगमधून किती कमाई करू शकतो? {#how-much-can-i-make-from-restaking}

जरी AVSs वेगवेगळे दर देतात, तरी eETH सारखे लिक्विड रिस्टेकिंग टोकन्स (LRTs) तुम्हाला किती कमाई करू शकता याची कल्पना देतात. ज्याप्रमाणे तुम्हाला तुमचे ETH स्टेक करण्यासाठी stETH सारखे LSTs मिळतात, त्याचप्रमाणे तुम्ही stETH रिस्टेक करण्यासाठी eETH सारखे LRTs मिळवू शकता. ही टोकन्स ETH स्टेकिंग आणि रिस्टेकिंग बक्षिसे मिळवतात.

**रिस्टेकिंगमधील धोके ओळखणे महत्त्वाचे आहे. संभाव्य बक्षिसे आकर्षक असू शकतात, परंतु ती धोक्यामुक्त नाहीत.**

## रिस्टेकिंगचे धोके काय आहेत? {#what-are-the-risks-of-restaking}

| **धोके**                     | **वर्णन**                                                                                                                                                |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **दंड (किंवा "स्लॅशिंग")** | ETH स्टेकिंगप्रमाणेच, जर रिस्टेकर्स/ऑपरेटर्स ऑफलाइन गेले, संदेश सेन्सॉर केले किंवा नेटवर्क भ्रष्ट करण्याचा प्रयत्न केला, तर त्यांचा स्टेक अंशतः किंवा पूर्णपणे स्लॅशिंग (नष्ट) केला जाऊ शकतो. |
| **केंद्रीकरण**            | जर काही मोजकेच ऑपरेटर्स बहुतांश रिस्टेकिंगवर वर्चस्व गाजवत असतील, तर त्यांचा रिस्टेकर्स, AVSs आणि अगदी रिस्टेकिंग प्लॅटफॉर्म्सवर मोठा प्रभाव असू शकतो.                             |
| **साखळी प्रतिक्रिया**           | जर एकाधिक AVSs सुरक्षित करताना एखाद्या रिस्टेकरचे स्लॅशिंग झाले, तर यामुळे इतर AVSs ची सुरक्षा कमी होऊ शकते, ज्यामुळे ते असुरक्षित बनू शकतात.                             |
| **निधीचा त्वरित ॲक्सेस** | रिस्टेक केलेले ETH काढण्यासाठी प्रतीक्षा वेळ (किंवा "अनबॉन्डिंग कालावधी") असतो, त्यामुळे तुम्हाला नेहमीच त्वरित ॲक्सेस मिळणार नाही.                                       |
