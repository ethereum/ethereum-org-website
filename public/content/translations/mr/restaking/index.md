Ethereum नेटवर्क 24/7, 365 अब्जावधी डॉलर्सच्या मूल्याला सुरक्षित करते. कसे?

जगभरातील लोक Ethereum व्यवहार प्रक्रिया करणारे आणि Ethereum नेटवर्क सुरक्षित करणारे सॉफ्टवेअर चालवण्यासाठी स्मार्ट कॉन्ट्रॅक्टमध्ये [ईथर (ETH)](/what-is-ether/) लॉक करतात (किंवा 'स्टेक' करतात). त्या बदल्यात, त्यांना अधिक ETH चे रिवॉर्ड्स मिळतात.

रिस्टेकिंग हे [स्टेकर्स](/staking/) साठी तयार केलेले एक तंत्रज्ञान आहे जे ही सुरक्षा इतर सेवा, ॲप्लिकेशन्स किंवा नेटवर्कपर्यंत विस्तारित करते. त्या बदल्यात, ते अतिरिक्त रिस्टेकिंग रिवॉर्ड्स मिळवतात. तथापि, ते त्यांच्या स्टेक केलेल्या ETH ला अधिक धोक्यात टाकतात.

## रिस्टेकिंग म्हणजे काय? {#what-is-restaking}

रिस्टेकिंग म्हणजे जेव्हा स्टेकर्स त्यांच्या आधीच स्टेक केलेल्या ETH चा वापर इतर विकेंद्रित सेवा सुरक्षित करण्यासाठी करतात. त्या बदल्यात, रिस्टेकर्सना त्यांच्या नियमित ETH स्टॅकिंग रिवॉर्ड्स व्यतिरिक्त त्या इतर सेवांमधून अतिरिक्त रिवॉर्ड्स मिळू शकतात.

रिस्टेकिंगद्वारे सुरक्षित केलेल्या विकेंद्रित सेवांना "ॲक्टिव्हली व्हॅलिडेटेड सर्व्हिसेस" (AVSs) म्हणून ओळखले जाते.
ज्याप्रमाणे अनेक ETH स्टेकर्स Ethereum व्हॅलिडेशन सॉफ्टवेअर चालवतात, त्याचप्रमाणे अनेक रिस्टेकर्स विशेष AVS सॉफ्टवेअर चालवतात.

## स्टेकिंग विरुद्ध रिस्टेकिंग {#staking-vs-restaking}

| स्टॅकिंग                                | रिस्टेकिंग                                                     |
| --------------------------------------- | -------------------------------------------------------------- |
| ETH रिवॉर्ड्स मिळवा                     | ETH रिवॉर्ड्स + AVS रिवॉर्ड्स मिळवा                            |
| Ethereum नेटवर्क सुरक्षित करते          | Ethereum नेटवर्क + AVSs सुरक्षित करते                          |
| किमान ETH नाही                          | किमान ETH नाही                                                 |
| कमी जोखमीची पातळी                       | कमी ते उच्च जोखमीची पातळी                                      |
| पैसे काढण्याची वेळ रांगेवर अवलंबून असते | पैसे काढण्याची वेळ रांगेवर + अनबॉन्डिंग कालावधीवर अवलंबून असते |

## रिस्टेकिंग कसे काम करते? {#how-does-restaking-work}

रिस्टेकिंगमध्ये अनेक संस्था सामील आहेत — त्यापैकी प्रत्येक एक महत्त्वाची भूमिका बजावते.

| **संज्ञा**                | **वर्णन**                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **रिस्टेकिंग प्लॅटफॉर्म** | रिस्टेकिंग प्लॅटफॉर्म ही एक सेवा आहे जी AVSs, ETH स्टेकर्स आणि ऑपरेटर्सना जोडते. ते स्टेकर्सना त्यांचे ETH रिस्टेक करण्यासाठी विकेंद्रित ॲप्लिकेशन्स तयार करतात आणि असे मार्केटप्लेस तयार करतात जिथे स्टेकर्स, AVSs, आणि ऑपरेटर्स एकमेकांना शोधू शकतात.                                                                                                                                                |
| **नेटिव्ह रिस्टेकर्स**    | जे लोक स्वतःचे Ethereum व्हॅलिडेटर्स चालवून त्यांचे ETH स्टेक करतात, ते त्यांचे स्टेक केलेले ETH EigenLayer आणि इतरांसह एका रिस्टेकिंग प्लॅटफॉर्मशी कनेक्ट करू शकतात, आणि ETH व्हॅलिडेटर रिवॉर्ड्सच्या वर रिस्टेकिंग रिवॉर्ड्स मिळवू शकतात.                                                                                                                                                                            |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **लिक्विड रिस्टेकर्स**    | जे लोक Lido किंवा Rocket Pool सारख्या थर्ड-पार्टी लिक्विड स्टेकिंग प्रोव्हायडरद्वारे त्यांचे ETH स्टेक करतात, त्यांना लिक्विड स्टेकिंग टोकन्स (LSTs) मिळतात जे त्यांच्या स्टेक केलेल्या ETH चे प्रतिनिधित्व करतात. ते त्यांचे मूळ ETH स्टेक केलेले ठेवून रिस्टेकिंग रिवॉर्ड्स मिळवण्यासाठी हे LSTs रिस्टेक करू शकतात.                                                               |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **ऑपरेटर्स**              | ऑपरेटर्स AVSs चे रिस्टेकिंग सॉफ्टवेअर चालवतात, आणि प्रत्येक AVS ला आवश्यक असलेले व्हॅलिडेशन कार्ये करतात. ऑपरेटर्स सामान्यतः व्यावसायिक सेवा प्रदाता असतात जे अपटाइम आणि परफॉर्मन्ससारख्या गोष्टींची हमी देतात. नॉन-ऑपरेटर रिस्टेकर्सप्रमाणे, ऑपरेटर्स AVSs सुरक्षित करण्यासाठी स्टेक केलेले ETH वापरतात, परंतु ऑपरेटर्सना त्यांच्या कामाच्या बदल्यात अतिरिक्त रिवॉर्ड्स देखील मिळतात. |
|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **AVSs**                  | या विकेंद्रित सेवा आहेत — जसे की प्राइस ओरॅकल्स, टोकन ब्रिजेस, आणि डेटा सिस्टीम्स — ज्या रिस्टेकर्सकडून सुरक्षा प्राप्त करतात आणि बदल्यात टोकन रिवॉर्ड्स देतात.                                                                                                                                                                                                                                                        |

## मी रिस्टेकिंगमधून किती कमावू शकेन? {#how-much-can-i-make-from-restaking}

AVSs वेगवेगळे दर देत असले तरी, eETH सारखे लिक्विड रिस्टेकिंग टोकन्स (LRTs) तुम्हाला कल्पना देतात की तुम्ही किती कमावू शकता. ज्याप्रमाणे तुम्हाला तुमचे ETH स्टेक करण्यासाठी stETH सारखे LSTs मिळतात, त्याचप्रमाणे तुम्ही stETH रिस्टेक करण्यासाठी eETH सारखे LRTs मिळवू शकता. हे टोकन्स ETH स्टेकिंग आणि रिस्टेकिंग रिवॉर्ड्स मिळवतात.

**रिस्टेकिंगमधील धोके स्वीकारणे महत्त्वाचे आहे. संभाव्य रिवॉर्ड्स आकर्षक असू शकतात, पण ते धोकामुक्त नाहीत.**

## रिस्टेकिंगचे धोके काय आहेत? {#what-are-the-risks-of-restaking}

| **धोके**                                      | **वर्णन**                                                                                                                                                                                                                         |
| --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **दंड (किंवा 'स्लॅशिंग')** | ETH स्टेकिंगप्रमाणे, जर रिस्टेकर्स/ऑपरेटर्स ऑफलाइन गेले, मेसेज सेन्सॉर केले किंवा नेटवर्कमध्ये फेरफार करण्याचा प्रयत्न केला, तर त्यांचा स्टेक अंशतः किंवा पूर्णपणे स्लॅश (बर्न) केला जाऊ शकतो. |
| **केंद्रीकरण**                                | जर काही ऑपरेटर्स बहुतेक रिस्टेकिंगवर वर्चस्व गाजवत असतील, तर त्यांचा रिस्टेकर्स, AVSs आणि अगदी रिस्टेकिंग प्लॅटफॉर्मवर मोठा प्रभाव पडू शकतो.                                                                      |
| **साखळी प्रतिक्रिया**                         | एकापेक्षा जास्त AVSs सुरक्षित करताना एखाद्या रिस्टेकरला स्लॅश केले गेल्यास, यामुळे इतर AVSs ची सुरक्षा कमी होऊ शकते, ज्यामुळे ते असुरक्षित बनू शकतात.                                                             |
| **निधीमध्ये त्वरित प्रवेश**                   | रिस्टेक केलेला ETH काढण्यासाठी प्रतीक्षा कालावधी (किंवा 'अनबॉन्डिंग कालावधी') असतो, त्यामुळे तुम्हाला नेहमीच त्वरित प्रवेश मिळणार नाही.                                                        |
