जेव्हा [इंटिग्रेटेड डेव्हलपमेंट एन्व्हायर्नमेंट (IDE)](https://wikipedia.org/wiki/Integrated_development_environment) सेट करण्याचा विचार येतो, तेव्हा इथेरियमवर ॲप्लिकेशन्स प्रोग्राम करणे हे इतर कोणत्याही सॉफ्टवेअर प्रोजेक्टच्या प्रोग्रामिंगसारखेच असते. निवडण्यासाठी अनेक पर्याय आहेत, त्यामुळे शेवटी, तुमच्या पसंतीनुसार सर्वोत्तम वाटणारा IDE किंवा कोड एडिटर निवडा. बहुधा तुमच्या इथेरियम डेव्हलपमेंटसाठी सर्वोत्तम IDE पर्याय तोच असेल जो तुम्ही आधीपासून पारंपारिक सॉफ्टवेअर डेव्हलपमेंटसाठी वापरत आहात.

## वेब-आधारित IDEs {#web-based-ides}

जर तुम्ही [लोकल डेव्हलपमेंट एन्व्हायर्नमेंट सेट](/developers/local-environment/) करण्यापूर्वी कोडसोबत प्रयोग करू इच्छित असाल, तर हे वेब ॲप्स इथेरियम स्मार्ट कॉन्ट्रॅक्ट डेव्हलपमेंटसाठी खास बनवलेले आहेत.

**[Remix](https://remix.ethereum.org/)** - **_बिल्ट-इन स्टॅटिक ॲनालिसिस आणि टेस्ट ब्लॉकचेन व्हर्च्युअल मशीनसह वेब-आधारित IDE_**

- [डॉक्युमेंटेशन](https://remix-ide.readthedocs.io/en/latest/#)
- [Gitter](https://gitter.im/ethereum/remix)

**[ChainIDE](https://chainide.com/)** - **_क्लाउड-आधारित मल्टी-चेन IDE_**

- [डॉक्युमेंटेशन](https://chainide.gitbook.io/chainide-english-1/)
- [मदत फोरम](https://forum.chainide.com/)

**[Replit (Solidity Starter - Beta)](https://replit.com/@replit/Solidity-starter-beta)** - **_हॉट रीलोडिंग, एरर चेकिंग आणि फर्स्ट-क्लास टेस्टनेट सपोर्टसह इथेरियमसाठी कस्टमाइझ करण्यायोग्य डेव्हलपमेंट एन्व्हायर्नमेंट_**

- [डॉक्युमेंटेशन](https://docs.replit.com/)

**[Tenderly Sandbox](https://sandbox.tenderly.co/)** - **_एक जलद प्रोटोटायपिंग एन्व्हायर्नमेंट जिथे तुम्ही Solidity आणि JavaScript वापरून ब्राउझरमध्ये स्मार्ट कॉन्ट्रॅक्ट लिहू शकता, कार्यान्वित करू शकता आणि डीबग करू शकता_**

**[EthFiddle](https://ethfiddle.com/)** - **_वेब-आधारित IDE जे तुम्हाला तुमचे स्मार्ट कॉन्ट्रॅक्ट लिहिण्याची, कंपाईल करण्याची आणि डीबग करण्याची सुविधा देते_**

- [Gitter](https://gitter.im/loomnetwork/ethfiddle)

## डेस्कटॉप IDEs {#desktop-ides}

बहुतेक प्रस्थापित IDEs ने इथेरियम डेव्हलपमेंटचा अनुभव वाढवण्यासाठी प्लगइन्स तयार केले आहेत. किमानपक्षी, ते [स्मार्ट कॉन्ट्रॅक्ट भाषांसाठी](/developers/docs/smart-contracts/languages/) सिंटॅक्स हायलाइटिंग प्रदान करतात.

**Visual Studio Code -** **_अधिकृत इथेरियम सपोर्टसह व्यावसायिक क्रॉस-प्लॅटफॉर्म IDE_**

- [Visual Studio Code](https://code.visualstudio.com/)
- [कोड सॅम्पल्स](https://github.com/Azure-Samples/blockchain/blob/master/blockchain-workbench/application-and-smart-contract-samples/readme.md)
- [GitHub](https://github.com/microsoft/vscode)

**JetBrains IDEs (IntelliJ IDEA, इ.) -** **_सॉफ्टवेअर डेव्हलपर्स आणि टीम्ससाठी आवश्यक टूल्स_**

- [JetBrains](https://www.jetbrains.com/)
- [GitHub](https://github.com/JetBrains)
- [IntelliJ Solidity](https://github.com/intellij-solidity/intellij-solidity/)

**Remix Desktop -** **_तुमच्या लोकल मशीनवर Remix IDE चा अनुभव घ्या_**

- [डाउनलोड](https://github.com/ethereum/remix-desktop/releases)
- [GitHub](https://github.com/ethereum/remix-desktop)

## प्लगइन्स आणि एक्स्टेंशन्स {#plugins-extensions}
