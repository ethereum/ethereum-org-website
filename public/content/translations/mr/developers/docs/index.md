हे दस्तऐवजीकरण तुम्हाला Ethereum सह बिल्ड करण्यास मदत करण्यासाठी डिझाइन केलेले आहे. हे एक संकल्पना म्हणून Ethereum समाविष्ट करते, Ethereum टेक स्टॅक स्पष्ट करते, आणि अधिक जटिल ऍप्लिकेशन्स आणि वापर प्रकरणांसाठी प्रगत विषयांचे दस्तऐवजीकरण करते.

हा एक मुक्त-स्रोत समुदाय प्रयत्न आहे, त्यामुळे नवीन विषय सुचवण्यास, नवीन सामग्री जोडण्यास आणि जिथे तुम्हाला वाटेल तिथे उदाहरणे देण्यास मोकळ्या मनाने. सर्व दस्तऐवजीकरण GitHub द्वारे संपादित केले जाऊ शकते – तुम्हाला कसे करायचे याची खात्री नसल्यास, [या सूचनांचे पालन करा](https://github.com/ethereum/ethereum-org-website/blob/dev/docs/editing-markdown.md).
