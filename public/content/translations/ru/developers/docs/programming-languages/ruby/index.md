<FeaturedText>Узнайте, как разрабатывать для Эфириума, используя проекты и инструменты на базе Ruby.</FeaturedText>

Используйте Эфириум для создания децентрализованных приложений (dapp), которые используют преимущества криптовалюты и технологии блокчейн. Эти dapp могут быть не требующими доверия, что означает, что после того, как они будут развернуты в Эфириуме, они всегда будут работать так, как запрограммировано. Они могут управлять цифровыми активами для создания новых видов финансовых приложений. Они могут быть децентрализованными, что означает, что ни одна организация или человек не контролирует их, и их практически невозможно подвергнуть цензуре.

## Проекты и инструменты на Ruby {#ruby-projects-and-tools}

### Активные {#active}

- [eth.rb](https://github.com/q9f/eth.rb) — _библиотека Ruby и RPC-клиент для работы с аккаунтами, сообщениями и транзакциями Эфириума_
- [keccak.rb](https://github.com/q9f/keccak.rb) — _хеш Keccak (SHA3), используемый Эфириумом_
- [siwe-ruby](https://github.com/signinwithethereum/siwe-ruby) — _реализация входа с помощью Эфириума (Sign-In with Ethereum) на Ruby_
- [siwe-rails](https://github.com/signinwithethereum/siwe-rails) — _gem для Rails, добавляющий локальные маршруты входа SIWE_
- [siwe-rails-examples](https://github.com/signinwithethereum/siwe-rails-examples) — _пример SIWE с использованием Ruby on Rails с пользовательским контроллером_
- [omniauth-siwe](https://github.com/signinwithethereum/omniauth-siwe) — _стратегия OmniAuth для входа с помощью Эфириума (SIWE)_
- [omniauth-nft](https://github.com/valthon/omniauth-nft) — _стратегия OmniAuth для аутентификации через владение NFT_
- [ethereum-on-rails](https://github.com/q9f/ethereum-on-rails) — _шаблон Ethereum on Rails, который позволяет подключить МетаМаск к Ruby on Rails_

### В архиве / Больше не поддерживаются {#archived--no-longer-maintained}

- [web3-eth](https://github.com/spikewilliams/vtada-ethereum) — _вызов RPC-методов узла Эфириума с помощью Ruby_
- [ethereum_tree](https://github.com/longhoangwkm/ethereum_tree) — _библиотека Ruby для генерации адресов ETH из иерархически детерминированного кошелька в соответствии со стандартом BIP32_
- [etherlite](https://github.com/budacom/etherlite) — _интеграция Эфириума для Ruby on Rails_
- [ethereum.rb](https://github.com/EthWorks/ethereum.rb) — _клиент Эфириума на Ruby, использующий интерфейс JSON-RPC для отправки транзакций, создания и взаимодействия с контрактами, а также полезный набор инструментов для работы с узлом Эфириума_
- [omniauth-ethereum.rb](https://github.com/q9f/omniauth-ethereum.rb) — _реализует стратегию провайдера Эфириума для OmniAuth_

Ищете больше ресурсов? Посетите [нашу главную страницу для разработчиков](/developers/).
