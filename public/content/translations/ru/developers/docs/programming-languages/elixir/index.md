<FeaturedText>Узнайте, как разрабатывать для Эфириума, используя проекты и инструменты на базе Elixir.</FeaturedText>

Используйте Эфириум для создания децентрализованных приложений (dapp), которые используют преимущества криптовалюты и технологии блокчейн. Эти децентрализованные приложения (dapp) могут быть не требующими доверия, что означает, что после того, как они будут развернуты в Эфириуме, они всегда будут работать так, как запрограммировано. Они могут управлять цифровыми активами для создания новых видов финансовых приложений. Они могут быть децентрализованными, что означает, что ни одна организация или человек не контролирует их, и их практически невозможно подвергнуть цензуре.

## Проекты и инструменты Elixir {#elixir-projects-and-tools}

### Активные {#active}

- [block_keys](https://github.com/ExWeb3/block_keys) — _Реализация BIP32 и BIP44 на Elixir (иерархия нескольких аккаунтов для детерминированных кошельков)_
- [ethereumex](https://github.com/mana-ethereum/ethereumex) — _Клиент JSON-RPC на Elixir для блокчейна Эфириума_
- [ethers](https://github.com/ExWeb3/elixir_ethers) — _Комплексная библиотека Web3 для взаимодействия со смарт-контрактами в Эфириуме с использованием Elixir_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) — _Библиотека подписи KMS для Ethers (подписание транзакций с помощью AWS KMS)_
- [ex_abi](https://github.com/poanetwork/ex_abi) — _Реализация парсера/декодера/кодировщика ABI Эфириума на Elixir_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) — _Библиотека Elixir для вычисления хешей Keccak SHA3-256 с использованием крейта Rust tiny-keccak, встроенного в NIF_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) — _Реализация кодирования RLP (Recursive Length Prefix) Эфириума на Elixir_

### В архиве / Больше не поддерживаются {#archived--no-longer-maintained}

- [eth](https://hex.pm/packages/eth) — _Утилиты Эфириума для Elixir_
- [exw3](https://github.com/hswick/exw3) — _Высокоуровневый RPC-клиент Эфириума для Elixir_
- [mana](https://github.com/mana-ethereum/mana) — _Реализация полного узла Эфириума, написанная на Elixir_
