## Работа с клиентами Эфириума {#working-with-ethereum-clients}

Вы можете использовать Эфириум для создания децентрализованных приложений (или «dapp»), которые используют преимущества криптовалюты и технологии блокчейн.
В настоящее время поддерживаются как минимум две библиотеки для Dart, позволяющие использовать [JSON-RPC API](/developers/docs/apis/json-rpc/) для Эфириума.

1. [Web3dart от pwa.ir](https://pub.dev/packages/web3dart)
1. [Ethereum 5.0.0 от darticulate.com](https://pub.dev/packages/ethereum)

Существуют также дополнительные библиотеки, которые позволяют вам манипулировать определенными адресами Эфириума или получать цены на различные криптовалюты.
[Вы можете посмотреть полный список здесь](https://pub.dev/dart/packages?q=ethereum).
