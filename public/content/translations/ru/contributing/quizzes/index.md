# Викторины {#quizzes}

Тесты — это возможность для пользователей проверить себя, чтобы понять, усвоили ли они содержание страницы, которую только что прочитали. Вопросы должны основываться только на содержимом, представленном на странице, и не должны затрагивать информацию, которая на странице не была упомянута.

Вопросы структурированы следующим образом. Содержание вопроса, 1 правильный ответ с объяснением, почему он правильный, 3 неправильных ответа с объяснением, почему они неправильные.

Некоторые примеры текущих тестов можно найти здесь:

- [Уровень 2](/layer-2)
- [NFT](/nft/)
- [Что такое Ethereum?](/what-is-ethereum/)
- [Что такое ETH?](/what-is-ether/)

## Добавление обучающего теста

Если для какой-либо страницы еще не создана обучающая викторина, [отправьте запрос](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=&template=suggest_quiz.yaml) на ее создание.

Пожалуйста, предоставьте следующую информацию:

- Страница, на которую вы хотите добавить тест
- 5 вопросов со следующей информацией:
  - Раздел страницы, на основе которого составлен вопрос
  - Содержание вопроса
  - 1 правильный ответ с объяснением, почему он правильный
  - 3 неправильных ответа, с объяснением, почему каждый из них неверен

## Добавление вопроса в тест

Если вы хотите добавить вопрос в банк вопросов для викторины, пожалуйста, [отправьте запрос](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=&template=suggest_quiz.yaml) и предоставьте следующую информацию:

- Страница, на которой вы хотите добавить вопрос для теста
- Для каждого вопроса предоставьте следующую информацию:
  - Раздел страницы, на основе которого составлен вопрос
  - Содержание вопроса
  - 1 правильный ответ с объяснением, почему он правильный
  - 3 неправильных ответа, с объяснением, почему каждый из них неверен

## Обновление вопроса в тесте

Если вы хотите обновить вопрос в банке вопросов для викторины, пожалуйста, [отправьте запрос](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=&template=suggest_quiz.yaml) и предоставьте следующую информацию:

- Страница, на которой вы хотите обновить вопрос в тесте
- Для каждого вопроса, который обновляется, предоставьте следующую информацию:
  - Раздел страницы, на основе которого составлен вопрос
  - Содержание вопроса, который вы хотите обновить
  - Обновленное содержание вопроса
  - 1 правильный ответ с объяснением, почему он правильный
  - 3 неправильных ответа, с объяснением, почему каждый из них неверен

## Удаление вопроса из теста

Если содержимое вопроса больше не существует на странице и его необходимо удалить, пожалуйста, [отправьте запрос](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=&template=suggest_quiz.yaml) на удаление вопроса и предоставьте следующую информацию:

- Страница, на которой вы хотите удалить вопрос из теста
- Вопрос, который вы хотите удалить
- Обоснование, если необходимо, почему вопрос должен быть удалён
