# Ресурсы {#resources}

Вы можете найти несколько полезных руководств и инструментов для переводчиков ethereum.org, а также сообщества переводов и новости ниже.

## Руководства {#guides}

- [Руководство по стилю перевода](/contributing/translation-program/translators-guide/) _– инструкции и советы для переводчиков ethereum.org_
- [FAQ по переводам](/contributing/translation-program/faq/) _– часто задаваемые вопросы и ответы о программе переводов ethereum.org_
- [Руководство по онлайн-редактору Crowdin](https://support.crowdin.com/online-editor/) _– подробное руководство по использованию онлайн-редактора Crowdin и некоторых его расширенных функций_

## Инструменты {#tools}

- [Linguee](https://www.linguee.com/)
  _– поисковая система для переводов и словарь, позволяющий осуществлять поиск по слову или фразе_
- [Поиск терминов Proz](https://www.proz.com/search/)
  _– база данных словарей для перевода и глоссариев по специализированным терминам_
- [Eurotermbank](https://www.eurotermbank.com/)
  _– сборники европейской терминологии на 42 языках_

## Сообщества {#communities}

- [Группы Discord для переводчиков на конкретный язык](https://discord.gg/ethereum-org)
  _– инициатива по объединению переводчиков ethereum.org в группы переводчиков_
- [Группа китайских переводчиков](https://www.notion.so/Ethereum-org-05375fe0a94c4214acaf90f42ba40171)
  _– страница Notion для облегчения координации между китайскими переводчиками_

## Часы приема переводчиков {#office-hours}

У нас есть часы приема для переводчиков во вторую среду каждого месяца. Они проводятся в голосовом канале #office-hours на [Discord-сервере ethereum.org](https://discord.gg/ethereum-org), где вы также можете уточнить время проведения и дополнительную информацию.

Часы приема позволяют нашим переводчикам задавать вопросы о процессе перевода, оставлять отзывы о программе перевода, делиться своими идеями или просто пообщаться с командой ethereum.org.
Наконец, мы хотим использовать эти разговоры, чтобы сообщать о последних изменениях в программе перевода и делиться ключевыми советами и инструкциями с нашими участниками.

Если вы являетесь переводчиком ethereum.org или хотели бы им стать, не стесняйтесь присоединиться к нам во время одной из таких сессий.
