Сеть Ethereum обеспечивает безопасность активов на миллиарды долларов круглосуточно, 365 дней в году. Как?

Люди по всему миру блокируют (или "стейкают") [эфир (ETH)](/eth/) в смарт-контрактах для запуска программного обеспечения, которое обрабатывает транзакции Ethereum и обеспечивает безопасность сети Ethereum. Взамен они получают вознаграждение в виде большего количества ETH.

Рестейкинг — это технология, созданная для [стейкеров](/staking/), чтобы расширить эту безопасность на другие сервисы, приложения или сети. Взамен они получают дополнительные вознаграждения за рестейкинг. Однако они также подвергают свои ETH в стейкинге большему риску.

## Что такое рестейкинг? {#what-is-restaking}

Рестейкинг — это когда стейкеры используют свои уже застейканные ETH для обеспечения безопасности других децентрализованных сервисов. Взамен рестейкеры могут получать дополнительные вознаграждения от этих других сервисов сверх своих обычных вознаграждений за стейкинг ETH.

Децентрализованные сервисы, защищенные рестейкингом, известны как "Активно валидируемые сервисы" (AVS).
Точно так же, как многие стейкеры ETH запускают программное обеспечение для валидации Ethereum, многие рестейкеры запускают специализированное программное обеспечение AVS.

## Стейкинг и рестейкинг {#staking-vs-restaking}

| Стейкинг                                | Рестейкинг                                            |
| --------------------------------------- | ----------------------------------------------------- |
| Получайте вознаграждения в ETH          | Получайте вознаграждения в ETH + вознаграждения AVS   |
| Обеспечивает безопасность сети Ethereum | Обеспечивает безопасность сети Ethereum + AVS         |
| Нет минимальной суммы ETH               | Нет минимальной суммы ETH                             |
| Низкий уровень риска                    | От низкого до высокого уровня риска                   |
| Время вывода зависит от очереди         | Время вывода зависит от очереди + периода открепления |

## Как работает рестейкинг? {#how-does-restaking-work}

В рестейкинге участвуют несколько субъектов, и каждый из них играет важную роль.

| **Термин**                    | **Описание**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Платформы для рестейкинга** | Платформа для рестейкинга — это сервис, который соединяет AVS, стейкеров ETH и операторов. Они создают децентрализованные приложения для стейкеров, чтобы они могли осуществлять рестейкинг своих ETH, и рынки, где стейкеры, AVS и операторы могут найти друг друга.                                                                                                                                                                                                           |
| **Нативные рестейкеры**       | Люди, которые стейкают свои ETH, запуская собственные валидаторы Ethereum, могут подключить свои застейканные ETH к платформе для рестейкинга, включая EigenLayer и другие, чтобы получать вознаграждения за рестейкинг поверх вознаграждений валидатора ETH.                                                                                                                                                                                                                                   |
|                               |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Ликвидные рестейкеры**      | Люди, которые стейкают свои ETH через стороннего провайдера ликвидного стейкинга, такого как Lido или Rocket Pool, получают токены ликвидного стейкинга (LST), которые представляют их застейканные ETH. Они могут осуществлять рестейкинг этих LST, чтобы получать вознаграждения за рестейкинг, сохраняя при этом свои первоначальные ETH в стейкинге.                                                                                                     |
|                               |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Операторы**                 | Операторы запускают программное обеспечение для рестейкинга AVS, выполняя задачи валидации, которые требует каждый AVS. Операторы обычно являются профессиональными поставщиками услуг, которые гарантируют такие вещи, как время безотказной работы и производительность. Как и рестейкеры, не являющиеся операторами, операторы используют застейканные ETH для защиты AVS, но операторы также получают дополнительные вознаграждения в обмен на свою работу. |
|                               |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **AVS**                       | Это децентрализованные сервисы, такие как ценовые оракулы, мосты токенов и системы данных, которые получают безопасность от рестейкеров и предлагают взамен вознаграждения в токенах.                                                                                                                                                                                                                                                                                                           |

## Сколько я могу заработать на рестейкинге? {#how-much-can-i-make-from-restaking}

Хотя AVS предлагают разные ставки, токены ликвидного рестейкинга (LRT), такие как eETH, дают вам представление о том, сколько вы можете заработать. Точно так же, как вы получаете LST, такие как stETH, за стейкинг своих ETH, вы можете получить LRT, такие как eETH, за рестейкинг stETH. Эти токены приносят вознаграждения за стейкинг ETH и рестейкинг.

**Важно осознавать риски, связанные с рестейкингом. Потенциальные вознаграждения могут быть привлекательными, но они не безрисковые.**

## Каковы риски рестейкинга? {#what-are-the-risks-of-restaking}

| **Риски**                                     | **Описание**                                                                                                                                                                                                                      |
| --------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Штрафы (или "слешинг")** | Как и в случае со стейкингом ETH, если рестейкеры/операторы отключаются, цензурируют сообщения или пытаются повредить сеть, их стейк может быть частично или полностью урезан (слэшен/сожжен). |
| **Централизация**                             | Если немногие операторы доминируют в большей части рестейкинга, они могут оказывать большое влияние на рестейкеров, AVS и даже на платформы для рестейкинга.                                                      |
| **Цепные реакции**                            | Если рестейкер подвергается слешингу при защите нескольких AVS, это может снизить безопасность других AVS, делая их уязвимыми.                                                                                    |
| **Немедленный доступ к средствам**            | Существует время ожидания (или "период открепления") для вывода застейканных ETH, поэтому у вас не всегда может быть немедленный доступ.                                                       |

## Как начать рестейкинг? {#how-to-get-started-with-restaking}

| 🫡 Начинающим                                                                                                              | 🤓 Опытные пользователи                                                                                                            |
| -------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| 1. Стейкайте ETH на таких платформах, как Lido или Rocket Pool, чтобы получить LST. | 1. Стейкайте свои ETH в качестве валидатора на Ethereum.                                    |
| 2. Используйте эти LST, чтобы начать рестейкинг на сервисе для рестейкинга.         | 2. Сравните сервисы для рестейкинга, такие как EigenLayer, Symbiotic и другие.              |
