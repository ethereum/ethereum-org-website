ایتھیریم نیٹ ورک <span dir="ltr">24/7, 365</span> اربوں ڈالر کی مالیت کو محفوظ بناتا ہے۔ کیسے؟

دنیا بھر میں لوگ ایتھیریم کی ٹرانزیکشنز پر کارروائی کرنے اور ایتھیریم نیٹ ورک کو محفوظ بنانے والے سافٹ ویئر کو چلانے کے لیے سمارٹ کنٹریکٹ میں [ایتھر (<span dir="ltr">ETH</span>)](/what-is-ether/) کو مقفل (یا "اسٹیک") کرتے ہیں۔ اس کے بدلے میں، انہیں مزید <span dir="ltr">ETH</span> سے انعام دیا جاتا ہے۔

ری اسٹیکنگ ایک ایسی ٹیکنالوجی ہے جو [اسٹیکرز](/staking/) کے لیے بنائی گئی ہے تاکہ اس سیکیورٹی کو دیگر خدمات، ایپلی کیشنز، یا نیٹ ورکس تک بڑھایا جا سکے۔ اس کے بدلے میں، وہ اضافی ری اسٹیکنگ انعامات کماتے ہیں۔ تاہم، وہ اپنے اسٹیک شدہ <span dir="ltr">ETH</span> کو مزید خطرے میں بھی ڈالتے ہیں۔

## ری اسٹیکنگ کیا ہے؟ {#what-is-restaking}

ری اسٹیکنگ اس وقت ہوتی ہے جب اسٹیکرز دیگر لامركزی خدمات کو محفوظ بنانے کے لیے اپنے پہلے سے اسٹیک شدہ <span dir="ltr">ETH</span> کا استعمال کرتے ہیں۔ اس کے بدلے میں، ری اسٹیکرز اپنے باقاعدہ <span dir="ltr">ETH</span> اسٹیکنگ انعامات کے علاوہ ان دیگر خدمات سے اضافی انعامات حاصل کر سکتے ہیں۔

ری اسٹیکنگ کے ذریعے محفوظ کی جانے والی لامركزی خدمات کو "ایکٹیولی ویلیڈیٹڈ سروسز" (<span dir="ltr">AVSs</span>) کے نام سے جانا جاتا ہے۔
جس طرح بہت سے <span dir="ltr">ETH</span> اسٹیکرز ایتھیریم کی توثیق کا سافٹ ویئر چلاتے ہیں، اسی طرح بہت سے ری اسٹیکرز مخصوص <span dir="ltr">AVS</span> سافٹ ویئر چلاتے ہیں۔

## اسٹیکنگ بمقابلہ ری اسٹیکنگ {#staking-vs-restaking}

| اسٹیکنگ | ری اسٹیکنگ |
| ------------------------------ | ------------------------------------------------- |
| <span dir="ltr">ETH</span> انعامات کمائیں | <span dir="ltr">ETH</span> انعامات + <span dir="ltr">AVS</span> انعامات کمائیں |
| ایتھیریم نیٹ ورک کو محفوظ بناتا ہے | ایتھیریم نیٹ ورک + <span dir="ltr">AVSs</span> کو محفوظ بناتا ہے |
| کوئی کم از کم <span dir="ltr">ETH</span> نہیں | کوئی کم از کم <span dir="ltr">ETH</span> نہیں |
| کم خطرے کی سطح | کم سے زیادہ خطرے کی سطح |
| نکالنے کا وقت قطار پر منحصر ہے | نکالنے کا وقت قطار + ان بانڈنگ کی مدت پر منحصر ہے |

## ری اسٹیکنگ کیسے کام کرتی ہے؟ {#how-does-restaking-work}

ری اسٹیکنگ میں کئی ادارے شامل ہیں — ان میں سے ہر ایک اہم کردار ادا کرتا ہے۔

| **اصطلاح** | **تفصیل** |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ری اسٹیکنگ پلیٹ فارمز** | ری اسٹیکنگ پلیٹ فارم ایک ایسی سروس ہے جو <span dir="ltr">AVSs</span>، <span dir="ltr">ETH</span> اسٹیکرز، اور آپریٹرز کو جوڑتی ہے۔ وہ اسٹیکرز کے لیے اپنے <span dir="ltr">ETH</span> کو ری اسٹیک کرنے کے لیے لامركزی ایپلی کیشنز بناتے ہیں، اور ایسے بازار بناتے ہیں جہاں اسٹیکرز، <span dir="ltr">AVSs</span>، اور آپریٹرز ایک دوسرے کو تلاش کر سکیں۔ |
| **نیٹو ری اسٹیکرز** | وہ لوگ جو اپنے ایتھیریم توثیق کاروں کو چلا کر اپنا <span dir="ltr">ETH</span> اسٹیک کرتے ہیں، وہ اپنے اسٹیک شدہ <span dir="ltr">ETH</span> کو ری اسٹیکنگ پلیٹ فارم سے جوڑ سکتے ہیں، بشمول EigenLayer اور دیگر، تاکہ <span dir="ltr">ETH</span> توثیق کار کے انعامات کے علاوہ ری اسٹیکنگ انعامات حاصل کر سکیں۔ |
| **لیکویڈ ری اسٹیکرز** | وہ لوگ جو فریق ثالث لیکویڈ اسٹیکنگ فراہم کنندہ، جیسے لیڈو یا Rocket Pool کے ذریعے اپنا <span dir="ltr">ETH</span> اسٹیک کرتے ہیں، انہیں سیال اسٹیکنگ ٹوکنز (<span dir="ltr">LSTs</span>) ملتے ہیں جو ان کے اسٹیک شدہ <span dir="ltr">ETH</span> کی نمائندگی کرتے ہیں۔ وہ اپنے اصل <span dir="ltr">ETH</span> کو اسٹیک رکھتے ہوئے ری اسٹیکنگ انعامات حاصل کرنے کے لیے ان <span dir="ltr">LSTs</span> کو ری اسٹیک کر سکتے ہیں۔ |
| **آپریٹرز** | آپریٹرز <span dir="ltr">AVSs</span> کا ری اسٹیکنگ سافٹ ویئر چلاتے ہیں، اور توثیق کے وہ کام انجام دیتے ہیں جن کی ہر <span dir="ltr">AVS</span> کو ضرورت ہوتی ہے۔ آپریٹرز عام طور پر پیشہ ورانہ سروس فراہم کرنے والے ہوتے ہیں جو اپ ٹائم اور کارکردگی جیسی چیزوں کی ضمانت دیتے ہیں۔ نان آپریٹر ری اسٹیکرز کی طرح، آپریٹرز <span dir="ltr">AVSs</span> کو محفوظ بنانے کے لیے اسٹیک شدہ <span dir="ltr">ETH</span> کا استعمال کرتے ہیں، لیکن آپریٹرز کو ان کے کام کے بدلے اضافی انعامات بھی ملتے ہیں۔ |
| **<span dir="ltr">AVSs</span>** | یہ لامركزی خدمات ہیں — جیسے پرائس اوریکلز، ٹوکن برجز، اور ڈیٹا سسٹمز — جو ری اسٹیکرز سے سیکیورٹی حاصل کرتی ہیں اور بدلے میں ٹوکن انعامات پیش کرتی ہیں۔ |

## میں ری اسٹیکنگ سے کتنا کما سکتا ہوں؟ {#how-much-can-i-make-from-restaking}

اگرچہ <span dir="ltr">AVSs</span> مختلف شرحیں پیش کرتے ہیں، لیکویڈ ری اسٹیکنگ ٹوکنز (<span dir="ltr">LRTs</span>) جیسے <span dir="ltr">eETH</span> آپ کو اندازہ دیتے ہیں کہ آپ کتنا کما سکتے ہیں۔ جس طرح آپ کو اپنا <span dir="ltr">ETH</span> اسٹیک کرنے کے لیے <span dir="ltr">stETH</span> جیسے <span dir="ltr">LSTs</span> ملتے ہیں، اسی طرح آپ <span dir="ltr">stETH</span> کو ری اسٹیک کرنے کے لیے <span dir="ltr">eETH</span> جیسے <span dir="ltr">LRTs</span> حاصل کر سکتے ہیں۔ یہ ٹوکنز <span dir="ltr">ETH</span> اسٹیکنگ اور ری اسٹیکنگ کے انعامات کماتے ہیں۔

**ری اسٹیکنگ کے خطرات کو تسلیم کرنا ضروری ہے۔ ممکنہ انعامات پرکشش ہو سکتے ہیں، لیکن وہ خطرے سے پاک نہیں ہیں۔**

## ری اسٹیکنگ کے خطرات کیا ہیں؟ {#what-are-the-risks-of-restaking}

| **خطرات** | **تفصیل** |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **جرمانے (یا "کٹوتی")** | <span dir="ltr">ETH</span> اسٹیکنگ کی طرح، اگر ری اسٹیکرز/آپریٹرز آف لائن ہو جاتے ہیں، پیغامات کو سنسر کرتے ہیں یا نیٹ ورک کو خراب کرنے کی کوشش کرتے ہیں، تو ان کے اسٹیک کی جزوی یا مکمل طور پر کٹوتی (جلا دیا) کی جا سکتی ہے۔ |
| **مرکزیت** | اگر چند آپریٹرز زیادہ تر ری اسٹیکنگ پر غلبہ حاصل کر لیتے ہیں تو وہ ری اسٹیکرز، <span dir="ltr">AVSs</span> اور یہاں تک کہ ری اسٹیکنگ پلیٹ فارمز پر بھی بہت زیادہ اثر و رسوخ رکھ سکتے ہیں۔ |
| **سلسلہ وار ردعمل** | اگر ایک سے زیادہ <span dir="ltr">AVSs</span> کو محفوظ بناتے ہوئے کسی ری اسٹیکر کی کٹوتی ہو جاتی ہے، تو اس سے دیگر <span dir="ltr">AVSs</span> کی سیکیورٹی کم ہو سکتی ہے، جس سے وہ غیر محفوظ ہو جاتے ہیں۔ |
| **فنڈز تک فوری رسائی** | ری اسٹیک شدہ <span dir="ltr">ETH</span> نکالنے کے لیے انتظار کا وقت (یا "ان بانڈنگ کی مدت") ہوتا ہے لہذا ہو سکتا ہے کہ آپ کو ہمیشہ فوری رسائی حاصل نہ ہو۔ |
