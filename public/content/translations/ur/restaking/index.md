ایتھیریم نیٹ ورک 24/7، 365 اربوں ڈالر کی مالیت کو محفوظ بناتا ہے۔ کیسے۔؟

دنیا بھر کے لوگ ایتھیریم ٹرانزیکشنز پر کارروائی کرنے اور ایتھیریم نیٹ ورک کو محفوظ بنانے والے سافٹ ویئر کو چلانے کے لیے اسمارٹ کانٹریکٹس میں [ایتھر (ETH)](/what-is-ether/) کو لاک (یا "اسٹیک") کرتے ہیں۔ اس کے بدلے میں، انہیں مزید ETH سے نوازا جاتا ہے۔

ری اسٹیکنگ ایک ایسی ٹیکنالوجی ہے جو [اسٹیکرز](/staking/) کے لیے بنائی گئی ہے تاکہ اس سیکیورٹی کو دیگر سروسز، ایپلی کیشنز، یا نیٹ ورکس تک بڑھایا جا سکے۔ اس کے بدلے میں، وہ اضافی ری اسٹیکنگ انعامات حاصل کرتے ہیں۔ تاہم، وہ اپنے اسٹیک شدہ ETH کو مزید خطرے میں بھی ڈالتے ہیں۔

## ری اسٹیکنگ کیا ہے؟ {#what-is-restaking}

ری اسٹیکنگ وہ عمل ہے جب اسٹیکرز اپنے پہلے سے اسٹیک شدہ ETH کو دیگر ڈی سینٹرلائزڈ سروسز کو محفوظ بنانے کے لیے استعمال کرتے ہیں۔ اس کے بدلے میں، ری اسٹیکرز اپنے باقاعدہ ETH اسٹیکنگ انعامات کے علاوہ ان دیگر سروسز سے اضافی انعامات حاصل کر سکتے ہیں۔

ری اسٹیکنگ کے ذریعے محفوظ کی جانے والی ڈی سینٹرلائزڈ سروسز کو "Actively Validated Services" (AVSs) کہا جاتا ہے۔
جس طرح بہت سے ETH اسٹیکرز ایتھیریم کی توثیق کا سافٹ ویئر چلاتے ہیں، اسی طرح بہت سے ری اسٹیکرز مخصوص AVS سافٹ ویئر چلاتے ہیں۔

## اسٹیکنگ بمقابلہ ری اسٹیکنگ {#staking-vs-restaking}

| اسٹیکنگ | ری اسٹیکنگ |
| ------------------------------ | ------------------------------------------------- |
| ETH انعامات حاصل کریں | ETH انعامات + AVS انعامات حاصل کریں |
| ایتھیریم نیٹ ورک کو محفوظ بناتا ہے | ایتھیریم نیٹ ورک + AVSs کو محفوظ بناتا ہے |
| کوئی کم از کم ETH نہیں | کوئی کم از کم ETH نہیں |
| کم خطرے کی سطح | کم سے زیادہ خطرے کی سطح |
| نکالنے کا وقت قطار پر منحصر ہے | نکالنے کا وقت قطار + ان بانڈنگ (unbonding) کی مدت پر منحصر ہے |

## ری اسٹیکنگ کیسے کام کرتی ہے؟ {#how-does-restaking-work}

ری اسٹیکنگ میں کئی ادارے شامل ہیں — ان میں سے ہر ایک اہم کردار ادا کرتا ہے۔

| **اصطلاح** | **تفصیل** |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ری اسٹیکنگ پلیٹ فارمز** | ری اسٹیکنگ پلیٹ فارم ایک ایسی سروس ہے جو AVSs، ETH اسٹیکرز، اور آپریٹرز کو جوڑتی ہے۔ وہ اسٹیکرز کے لیے اپنے ETH کو ری اسٹیک کرنے کے لیے ڈی سینٹرلائزڈ ایپلی کیشنز بناتے ہیں، اور ایسی مارکیٹ پلیسز بناتے ہیں جہاں اسٹیکرز، AVSs، اور آپریٹرز ایک دوسرے کو تلاش کر سکیں۔ |
| **نیٹو ری اسٹیکرز** | وہ لوگ جو اپنے ایتھیریم ویلیڈیٹرز چلا کر اپنا ETH اسٹیک کرتے ہیں، وہ اپنے اسٹیک شدہ ETH کو EigenLayer اور دیگر سمیت کسی ری اسٹیکنگ پلیٹ فارم سے جوڑ سکتے ہیں، تاکہ ETH ویلیڈیٹر انعامات کے علاوہ ری اسٹیکنگ انعامات حاصل کر سکیں۔ |
| **لیکویڈ ری اسٹیکرز** | وہ لوگ جو اپنا ETH کسی تھرڈ پارٹی لیکویڈ اسٹیکنگ پرووائیڈر، جیسے Lido یا Rocket Pool کے ذریعے اسٹیک کرتے ہیں، انہیں Liquid Staking Tokens (LSTs) ملتے ہیں جو ان کے اسٹیک شدہ ETH کی نمائندگی کرتے ہیں۔ وہ اپنے اصل ETH کو اسٹیک رکھتے ہوئے ری اسٹیکنگ انعامات حاصل کرنے کے لیے ان LSTs کو ری اسٹیک کر سکتے ہیں۔ |
| **آپریٹرز** | آپریٹرز AVSs کا ری اسٹیکنگ سافٹ ویئر چلاتے ہیں، اور وہ توثیقی کام انجام دیتے ہیں جن کی ہر AVS کو ضرورت ہوتی ہے۔ آپریٹرز عام طور پر پیشہ ور سروس پرووائیڈرز ہوتے ہیں جو اپ ٹائم اور کارکردگی جیسی چیزوں کی ضمانت دیتے ہیں۔ نان آپریٹر ری اسٹیکرز کی طرح، آپریٹرز AVSs کو محفوظ بنانے کے لیے اسٹیک شدہ ETH کا استعمال کرتے ہیں، لیکن آپریٹرز کو ان کے کام کے بدلے اضافی انعامات بھی ملتے ہیں۔ |
| **AVSs** | یہ ڈی سینٹرلائزڈ سروسز ہیں — جیسے پرائس اوریکلز، ٹوکن برجز، اور ڈیٹا سسٹمز — جو ری اسٹیکرز سے سیکیورٹی حاصل کرتی ہیں اور بدلے میں ٹوکن انعامات پیش کرتی ہیں۔ |

## میں ری اسٹیکنگ سے کتنا کما سکتا ہوں؟ {#how-much-can-i-make-from-restaking}

اگرچہ AVSs مختلف شرحیں پیش کرتے ہیں، Liquid Restaking Tokens (LRTs) جیسے eETH آپ کو اندازہ دیتے ہیں کہ آپ کتنا کما سکتے ہیں۔ جس طرح آپ کو اپنا ETH اسٹیک کرنے پر stETH جیسے LSTs ملتے ہیں، اسی طرح آپ stETH کو ری اسٹیک کرنے پر eETH جیسے LRTs حاصل کر سکتے ہیں۔ یہ ٹوکنز ETH اسٹیکنگ اور ری اسٹیکنگ انعامات کماتے ہیں۔

**ری اسٹیکنگ سے وابستہ خطرات کو تسلیم کرنا ضروری ہے۔ ممکنہ انعامات پرکشش ہو سکتے ہیں، لیکن وہ خطرے سے خالی نہیں ہیں۔**

## ری اسٹیکنگ کے خطرات کیا ہیں؟ {#what-are-the-risks-of-restaking}

| **خطرات** | **تفصیل** |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **جرمانے (یا "سلیشنگ")** | ETH اسٹیکنگ کی طرح، اگر ری اسٹیکرز/آپریٹرز آف لائن ہو جاتے ہیں، پیغامات کو سنسر کرتے ہیں یا نیٹ ورک کو خراب کرنے کی کوشش کرتے ہیں، تو ان کا اسٹیک جزوی یا مکمل طور پر سلیش (جلا دیا) جا سکتا ہے۔ |
| **سینٹرلائزیشن** | اگر چند آپریٹرز زیادہ تر ری اسٹیکنگ پر غلبہ حاصل کر لیتے ہیں تو وہ ری اسٹیکرز، AVSs اور یہاں تک کہ ری اسٹیکنگ پلیٹ فارمز پر بھی بہت زیادہ اثر و رسوخ رکھ سکتے ہیں۔ |
| **سلسلہ وار ردعمل (Chain reactions)** | اگر کوئی ری اسٹیکر متعدد AVSs کو محفوظ بناتے ہوئے سلیش ہو جاتا ہے، تو اس سے دیگر AVSs کی سیکیورٹی کم ہو سکتی ہے، جس سے وہ غیر محفوظ ہو سکتے ہیں۔ |
| **فنڈز تک فوری رسائی** | ری اسٹیک شدہ ETH نکالنے کے لیے انتظار کا وقت (یا "ان بانڈنگ کی مدت") ہوتا ہے اس لیے ہو سکتا ہے کہ آپ کو ہمیشہ فوری رسائی حاصل نہ ہو۔ |
