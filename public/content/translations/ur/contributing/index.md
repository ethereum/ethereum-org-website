# ethereum.org میں تعاون کریں 🦄 {#contributing-to-ethereumorg}

Ethereum.org ایک اوپن سورس پروجیکٹ ہے جس میں **12,000+** سے زیادہ معاونین شامل ہیں جو ویب سائٹ کا ترجمہ کرنے، لکھنے، ڈیزائن کرنے اور اسے برقرار رکھنے میں مدد کرتے ہیں۔

ہم ایک خوش آئند کمیونٹی ہیں جو آپ کو [Ethereum](/) ایکو سسٹم میں ترقی کرنے اور سیکھنے میں مدد دے گی، جبکہ آپ بامعنی تعاون بھی کر سکیں گے اور متعلقہ عملی تجربہ حاصل کر سکیں گے!

## تعاون کے طریقے {#ways-to-contribute}

**تراجم**
- [ترجمہ پروگرام میں شامل ہوں](/contributing/translation-program/) – ethereum.org کو نئی زبانوں میں لانے میں ہماری مدد کریں

**ڈیولپمنٹ**
- [کسی اوپن ایشو پر کام کریں](https://github.com/ethereum/ethereum-org-website/issues) – وہ کام جس کی ہم نے نشاندہی کی ہے اور جسے کرنے کی ضرورت ہے

**ڈیزائن**
- [ویب سائٹ ڈیزائن کرنے میں مدد کریں](/contributing/design/) – ہر سطح کے ڈیزائنرز ویب سائٹ کو بہتر بنانے میں اپنا حصہ ڈال سکتے ہیں

**مواد**
- [مواد بنائیں/ترمیم کریں](/contributing/#how-to-update-content) – نئے صفحات تجویز کریں یا موجودہ مواد میں تبدیلیاں کریں
- [کمیونٹی وسائل شامل کریں](/contributing/content-resources/) – کسی متعلقہ صفحے پر کوئی مفید مضمون یا وسیلہ شامل کریں
- [ڈیزائن کے وسائل تجویز کریں](/contributing/design/adding-design-resources/) – مفید ڈیزائن وسائل شامل کریں، اپ ڈیٹ کریں اور حذف کریں
- [کوئزز](/contributing/quizzes/) – کسی متعلقہ صفحے کے لیے کوئز سوالات کے بینک شامل کریں، اپ ڈیٹ کریں اور حذف کریں

**فیچر کے آئیڈیاز**
- [کسی فیچر کی درخواست کریں](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=Type%3A+Feature&template=feature_request.yaml&title=) – ہمیں کسی نئے فیچر یا ڈیزائن کے حوالے سے اپنے آئیڈیاز کے بارے میں بتائیں

**پروڈکٹ کی فہرستیں**
- [ایکسچینج شامل کریں](/contributing/adding-exchanges/) – ہمارے [ایکسچینج فائنڈر](/get-eth/#country-picker) میں ایک ایکسچینج شامل کریں
- [پروڈکٹ شامل کریں](/contributing/adding-products/) – کسی متعلقہ صفحے پر dapp یا والیٹ شامل کریں
- [ڈیولپر ٹولز شامل کریں](/contributing/adding-developer-tools/) – کسی متعلقہ صفحے پر ڈیولپر ٹول شامل کریں
- [لیئر 2 شامل کریں](/contributing/adding-layer-2s/) – کسی متعلقہ صفحے پر لیئر 2 شامل کریں
- [اسٹیکنگ پروڈکٹ یا سروس شامل کریں](/contributing/adding-staking-products/) – ایک ایسا پروجیکٹ شامل کریں جو سولو اسٹیکنگ، پولڈ اسٹیکنگ، یا بطور سروس اسٹیکنگ میں سہولت فراہم کرے
- [والیٹ شامل کریں](/contributing/adding-wallets/) – [والیٹس تلاش کریں صفحہ](/wallets/find-wallet/) کے لیے ایک والیٹ شامل کریں
- [ہمارے DeSci صفحے کے لیے پروجیکٹ تجویز کریں](/contributing/adding-desci-projects/) – Ethereum پر بنایا گیا ایک پروجیکٹ شامل کریں جو ڈی سینٹرلائزڈ سائنس میں حصہ ڈالتا ہو
- [وسیلہ شامل کریں](/contributing/adding-resources/) – کسی بھی متعلقہ صفحے پر ایک مفید وسیلہ شامل کریں

کوئی سوال ہے؟ 🤔 ہمارے [Discord سرور](https://discord.gg/ethereum-org) میں شامل ہوں

## تعاون شروع کرنے کے لیے ابتدائی بہترین ٹاسکس

یہ کچھ موجودہ ٹاسکس ہیں جنہیں حل کرنے اور ان کی ذمہ داری لینے میں آپ ہماری مدد کر سکتے ہیں۔ زیادہ تر کے لیے آپ کو GitHub اکاؤنٹ کی ضرورت ہوگی کیونکہ ویب سائٹ میں زیادہ تر تبدیلیاں GitHub کے ذریعے کی جاتی ہیں۔

## ethereum.org پر کام کیسے کریں {#how-to-update-content}

اگر آپ [ترجمہ پروگرام](/contributing/translation-program/) میں حصہ لینا چاہتے ہیں، تو ہم آپ سے [Crowdin](https://crowdin.com/project/ethereum-org) پر اکاؤنٹ بنانے کی درخواست کرتے ہیں۔ باقی تمام چیزوں کے لیے – ویب سائٹ پر مواد یا بصریات شامل کرنا یا ان میں ترمیم کرنا، بگز ٹھیک کرنا، اوپن ٹاسکس پر کام کرنا – آپ کو ایک [GitHub](https://github.com/) اکاؤنٹ کی ضرورت ہوگی۔

تمام اپ ڈیٹس GitHub PR کے عمل کے ذریعے کی جاتی ہیں۔ اس کا مطلب ہے کہ آپ ویب سائٹ کی ایک مقامی کاپی بناتے ہیں، اپنی تبدیلیاں کرتے ہیں اور اپنی تبدیلیوں کو ضم (merge) کرنے کی درخواست کرتے ہیں۔ اگر آپ نے پہلے کبھی ایسا نہیں کیا ہے، تو ہماری [GitHub ریپوزٹری](https://github.com/ethereum/ethereum-org-website) کے آخر میں دی گئی ہدایات پر عمل کریں۔

آپ کو کسی بھی چیز پر کام کرنے کے لیے اجازت کی ضرورت نہیں ہے، لیکن یہ ہمیشہ بہتر ہوتا ہے کہ آپ ہمیں بتائیں کہ آپ کیا کرنے کا ارادہ رکھتے ہیں۔ آپ یہ درج ذیل طریقوں سے کر سکتے ہیں:

- [GitHub](https://github.com/ethereum/ethereum-org-website) میں کسی ایشو یا PR پر تبصرہ کر کے
- ہمارے [Discord سرور](https://discord.gg/ethereum-org) پر پیغام بھیج کر

تعاون کرنے سے پہلے، یقینی بنائیں کہ آپ درج ذیل سے واقف ہیں:

- ethereum.org کا ابھرتا ہوا [وژن](/about/)
- ہمارے [ڈیزائن کے اصول](/contributing/design-principles/)
- ہماری [اسٹائل گائیڈ](/contributing/style-guide/)
- ہمارا [ضابطہ اخلاق](/community/code-of-conduct)

## سائٹ کے بارے میں فیصلے کیسے کیے جاتے ہیں {#how-decisions-about-the-site-are-made}

انفرادی PRs، ڈیزائن کے ارتقاء اور بڑے اپ گریڈز کے بارے میں فیصلے Ethereum ایکو سسٹم کی ایک ٹیم کرتی ہے۔ اس ٹیم میں پروجیکٹ مینیجرز، ڈیولپرز، ڈیزائنرز، مارکیٹنگ اور کمیونیکیشنز، اور متعلقہ مضامین کے ماہرین شامل ہیں۔ کمیونٹی کی رائے ہر فیصلے میں رہنمائی کرتی ہے: لہذا براہ کرم ایشوز میں سوالات اٹھائیں، PRs جمع کرائیں، یا ٹیم سے رابطہ کریں:

- [website@ethereum.org](mailto:website@ethereum.org)
- [@ethdotorg](https://twitter.com/ethdotorg)
- [Discord سرور](https://discord.gg/ethereum-org)

### سرقہ (Plagiarism) پر ایک نوٹ {#plagiarism}

ethereum.org میں کوئی بھی مواد یا آرٹفیکٹ شامل کرتے وقت صرف اپنا اصل کام یا وہ مواد استعمال کریں جسے استعمال کرنے کی آپ کو اجازت ہو۔ Ethereum ایکو سسٹم کے اندر بہت سے پروجیکٹس اوپن سورس لائسنسنگ کا استعمال کرتے ہیں جو معلومات کے مفت اشتراک کی اجازت دیتا ہے۔ تاہم، اگر آپ کو یہ معلومات نہیں مل سکتیں، تو اسے ethereum.org میں شامل کرنے کی کوشش نہ کریں۔ سرقہ سمجھی جانے والی کوئی بھی پل ریکوئسٹ (pull request) مسترد کر دی جائے گی۔

## اوپن سورس میں نئے ہیں؟ {#new-to-open-source}

ہماری GitHub ریپوزٹری پر انٹری کے آسان ایشوز موجود ہیں جو خاص طور پر ان ڈیولپرز کے لیے بنائے گئے ہیں جو اوپن سورس میں نئے ہیں، جن پر [good first issue](https://github.com/ethereum/ethereum-org-website/issues?q=is%3Aopen+is%3Aissue+label%3A%22good+first+issue%22) کا لیبل لگا ہوا ہے۔

## اپنا Onchain Achievement Token (OAT) کلیم کریں {#oat}

اگر آپ کا تعاون ethereum.org میں ضم (merge) ہو جاتا ہے، تو آپ کو [Galxe](https://app.galxe.com/quest/ethereumorg) پر ایک خاص بیج کلیم کرنے کا موقع ملے گا۔ Onchain Achievement Token (OAT) اس بات کا ثبوت ہے کہ آپ نے ایکو سسٹم کو قدرے مزید شاندار بنانے میں مدد کی۔

[OATs کے بارے میں مزید](https://help.galxe.com/en/articles/9645630-create-quest-rewards#h_1c5d63ba03)

### کلیم کرنے کا طریقہ

1. ہمارے [Discord سرور](https://discord.gg/ethereum-org) میں شامل ہوں۔
2. اپنے تعاون کا لنک `#🥇 | proof-of-contribution` چینل میں پیسٹ کریں۔
3. ہماری ٹیم کے کسی رکن کی جانب سے آپ کو آپ کے OAT کا لنک بھیجنے کا انتظار کریں۔
4. اپنا OAT کلیم کریں!

OATs کلیم کرنے کے لیے آپ کو صرف سیلف کسٹڈی والیٹس کا استعمال کرنا چاہیے۔ ایکسچینج اکاؤنٹس یا ایسے دیگر اکاؤنٹس استعمال نہ کریں جن کی پرائیویٹ کیز آپ کے پاس نہیں ہیں، کیونکہ یہ آپ کو اپنے OATs تک رسائی اور ان کا نظم کرنے کی اجازت نہیں دیں گے۔

## اپنا GitPOAP کلیم کریں {#claim-gitpoap}

GitPOAP بھی خود بخود آپ کے ضم شدہ تعاون کو پہچان لے گا اور آپ کو ان کے پلیٹ فارم پر ہی ایک الگ منفرد معاونین کا POAP منٹ (mint) کرنے دے گا!


### کلیم کرنے کا طریقہ {#how-to-claim}

1. [GitPOAP](https://www.gitpoap.io) پر جائیں۔
2. سائن ان آپشن کے ذریعے اپنے والیٹ یا اپنے ای میل سے کنیکٹ کریں۔
3. یہ چیک کرنے کے لیے کہ آیا آپ اہل ہیں، اپنا GitHub یوزر نیم، ETH ایڈریس، ENS نام یا کوئی بھی GitPOAP تلاش کریں۔
4. اگر آپ کا GitHub اکاؤنٹ اہل ہے، تو آپ GitPOAP منٹ کر سکیں گے!
