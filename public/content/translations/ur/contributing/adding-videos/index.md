# ویڈیوز شامل کرنا {#adding-videos}

[<span dir="ltr">ethereum.org</span> کی ویڈیو گیلری](/videos/) میں کمیونٹی کے تخلیق کاروں اور قابل اعتماد ذرائع کی جانب سے ایتھیریم اور ایتھیریم ایکو سسٹم کے بارے میں ویڈیوز شامل ہیں۔ کوئی بھی شخص شامل کرنے کے لیے ویڈیو تجویز کر سکتا ہے۔

## فہرست سازی کی پالیسی {#listing-policy}

<span dir="ltr">Ethereum.org</span> ایک غیر جانبدار، تعلیمی وسیلہ ہے۔ ویڈیو گیلری کو درج ذیل مقاصد کے لیے ترتیب دیا گیا ہے:

- صارفین کو ایتھیریم ٹیکنالوجی، ایکو سسٹم، اور کمیونٹی کے بارے میں **تعلیم دینا**
- اپنے تکنیکی مواد میں **درست رہنا**
- ایتھیریم کمیونٹی کے لیے **متعلقہ رہنا**

یہ سائٹ ایسی ویڈیوز کو فہرست میں شامل نہیں کرتی جو بنیادی طور پر کسی مخصوص پروڈکٹ، ٹوکن، یا تجارتی سروس کی تشہیر کرتی ہوں۔


## شمولیت کا معیار {#criteria-for-inclusion}

### لازمی شرائط {#must-haves}

- **ایتھیریم پر مرکوز** – ویڈیو بنیادی طور پر ایتھیریم، اس کی ٹیکنالوجی، ایکو سسٹم، یا کمیونٹی کے بارے میں ہونی چاہیے۔ عام بلاک چین موضوعات پر ویڈیوز صرف اسی صورت میں قابل قبول ہیں جب وہ سائٹ پر موجود کسی تعلیمی صفحے کی نمایاں طور پر حمایت کرتی ہوں یا اس سے متعلق ہوں، یا ایتھیریم کا حوالہ دیتی ہوں۔
- **تعلیمی اہمیت** – ویڈیو کو ناظرین کو ایتھیریم کے بارے میں کچھ سکھانا چاہیے، یا عالمی ایتھیریم کمیونٹی کو سراہنا چاہیے۔ تشہیری یا مارکیٹنگ کے مواد کو قبول نہیں کیا جائے گا۔
- **درست معلومات** – تکنیکی مواد حقائق پر مبنی اور اپ ٹو ڈیٹ ہونا چاہیے۔ متروک خصوصیات کے بارے میں پرانی ویڈیوز کو ہٹایا جا سکتا ہے۔
- **معیاری پروڈکشن** – ویڈیو کی آڈیو اور ویڈیو کوالٹی معقول حد تک واضح ہونی چاہیے۔
- **عوامی طور پر دستیاب** – ویڈیو کو کسی کھلے وسیلے یا قابل رسائی پلیٹ فارم جیسے YouTube پر ہوسٹ کیا جانا چاہیے، اور بغیر کسی پے وال یا سائن اپ کی ضرورت کے آزادانہ طور پر قابل رسائی ہونا چاہیے۔

### اضافی خوبیاں {#nice-to-haves}

- **ٹرانسکرپٹ کا ہونا** – ٹرانسکرپٹ والی ویڈیوز رسائی اور SEO کو بہتر بناتی ہیں۔ اگر آپ کے پاس یہ نہیں ہے، تو <span dir="ltr">ethereum.org</span> کی ٹیم اسے تیار کرنے میں مدد کر سکتی ہے۔
- **قابل اعتماد ذریعہ سے** – مستند ماہرین تعلیم، محققین، اور ذرائع کے مواد کو ترجیح دی جاتی ہے۔
- **بروقت اور سدا بہار** – ایسا مواد جو وقت گزرنے کے ساتھ متعلقہ رہے، اسے وقت کے لحاظ سے حساس مواد پر ترجیح دی جاتی ہے۔


## ویڈیو کیسے شامل کریں {#how-to-add-a-video}

### آپشن 1: ایک ایشو کھولیں {#open-an-issue}

اگر آپ کوئی ویڈیو تجویز کرنا چاہتے ہیں لیکن خود فائلیں نہیں بنانا چاہتے، تو ویڈیو کی تفصیلات کے ساتھ ایک GitHub ایشو کھولیں اور کوئی معاون اسے آپ کے لیے شامل کرنے میں مدد کر سکتا ہے۔

<ButtonLink href="https://github.com/ethereum/ethereum-org-website/issues/new?template=suggest_video.yaml">
  ویڈیو تجویز کریں
</ButtonLink>

### آپشن 2: پل ریکوئسٹ کھولیں {#open-a-pull-request}

اگر آپ خود ویڈیو شامل کرنا چاہتے ہیں، تو ان اقدامات پر عمل کریں:

#### مرحلہ 1: ویڈیو فائل بنائیں {#step-1}

ایک نئی ڈائرکٹری اور `index.md` فائل یہاں بنائیں:

```
public/content/videos/{your-video-slug}/index.md
```

سلگ URL کے لیے محفوظ، چھوٹے حروف (lowercase) میں، اور ہائفن کے ساتھ ہونا چاہیے (جیسے، `blockchain-101-visual-demo`)۔

#### مرحلہ 2: فرنٹ میٹر شامل کریں {#step-2}

اپنی `index.md` میں درج ذیل YAML فرنٹ میٹر شامل کریں:

```yaml
---
title: "Your Video Title"
description: "A brief 1–3 sentence summary of the video."
lang: en
youtubeId: "dQw4w9WgXcQ"
uploadDate: 2025-01-15
duration: "12:30"
educationLevel: beginner
topic:
  - "your-topic"
  - "another-topic"
format: explainer
author: Channel Name
---
```

**فیلڈ کا حوالہ:**

| فیلڈ | درکار | تفصیل |
|---|---|---|
| `title` | ہاں | ویڈیو کا عنوان |
| `description` | ہاں | 1–3 جملوں کا خلاصہ |
| `lang` | ہاں | فی الحال ہمیشہ `en` |
| `youtubeId` | ہاں | YouTube ویڈیو کی ID (`v=` کے بعد URL سے) |
| `uploadDate` | ہاں | `YYYY-MM-DD` فارمیٹ میں اصل اپ لوڈ کی تاریخ |
| `duration` | ہاں | ویڈیو کی طوالت `H:MM:SS` یا `M:SS` کے طور پر |
| `educationLevel` | ہاں | `beginner`، `intermediate`، یا `advanced` |
| `topic` | ہاں | گیلری فلٹرنگ کے لیے موضوع کے ٹیگز کی ارے (Array) |
| `format` | ہاں | `explainer`، `presentation`، `interview`، `tutorial`، یا `panel` |
| `author` | ہاں | تخلیق کار یا چینل کا نام |
| `breadcrumb` | نہیں | بریڈ کرمب نیویگیشن کے لیے حسب ضرورت مختصر لیبل |
| `customThumbnailUrl` | نہیں | حسب ضرورت تھمب نیل URL (پہلے سے طے شدہ YouTube تھمب نیل ہوتا ہے) |

#### مرحلہ 3: ٹرانسکرپٹ شامل کریں (تجویز کردہ) {#step-3}

فرنٹ میٹر `---` کے نیچے، مارک ڈاؤن فارمیٹ میں ویڈیو کا ٹرانسکرپٹ شامل کریں:

```markdown
---
title: "..."
# ... باقی فرنٹ میٹر
---

ویڈیو کے مواد کا مختصر تعارف۔

### سیکشن کا عنوان (0:00)

اس سیکشن کے لیے ٹرانسکرپٹ کا متن...

### اگلا سیکشن (5:30)

مزید ٹرانسکرپٹ کا متن...
```

بڑے سیکشنز کو نشان زد کرنے کے لیے ٹائم اسٹامپس کے ساتھ `###` ہیڈنگز کا استعمال کریں۔ اس سے ٹرانسکرپٹ کو اسکین کرنا آسان ہو جاتا ہے اور SEO بہتر ہوتا ہے۔

اگر آپ کے پاس ٹرانسکرپٹ نہیں ہے، تو آپ باڈی کو خالی چھوڑ سکتے ہیں اور ٹیم اسے تیار کر لے گی۔

#### مرحلہ 5: اپنی PR جمع کرائیں {#step-5}

`dev` برانچ میں اپنی تبدیلیوں کے ساتھ ایک پل ریکوئسٹ کھولیں۔ ٹیم آپ کی جمع کرائی گئی درخواست کا جائزہ لے گی اور فیڈ بیک فراہم کرے گی۔


## دیکھ بھال {#maintenance}

فہرست میں شامل ویڈیوز کا باقاعدگی سے جائزہ لیا جاتا ہے تاکہ یہ یقینی بنایا جا سکے کہ وہ:

- اب بھی فہرست سازی کے معیار پر پورا اترتی ہیں
- درست، اپ ٹو ڈیٹ معلومات پر مشتمل ہیں
- ان کے ہوسٹنگ/YouTube لنکس کام کر رہے ہیں

اگر آپ کو فہرست میں شامل کسی ویڈیو میں کوئی مسئلہ نظر آتا ہے، تو [ایک ایشو بنائیں](https://github.com/ethereum/ethereum-org-website/issues/new?assignees=&labels=feature+✨,content+🖋️&template=suggest_video.yaml) یا [<span dir="ltr">website@ethereum.org</span>](mailto:website@ethereum.org) پر ای میل بھیجیں۔
