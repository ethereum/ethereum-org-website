یہ ٹیوٹوریل NFT ٹیوٹوریل سیریز کا حصہ 3/3 ہے، جہاں ہم اپنا نیا منٹ کیا گیا NFT دیکھتے ہیں۔ تاہم، آپ MetaMask کا استعمال کرتے ہوئے کسی بھی ERC-721 ٹوکن کے لیے یہ عمومی ٹیوٹوریل استعمال کر سکتے ہیں، بشمول Mainnet یا کسی بھی ٹیسٹ نیٹ پر۔ اگر آپ یہ سیکھنا چاہتے ہیں کہ Ethereum پر اپنا NFT کیسے منٹ کیا جائے، تو آپ کو [NFT سمارٹ کنٹریکٹ لکھنے اور ڈیپلائے کرنے کے طریقے پر حصہ 1](/developers/tutorials/how-to-write-and-deploy-an-nft) دیکھنا چاہیے!

مبارک ہو! آپ ہماری NFT ٹیوٹوریل سیریز کے سب سے چھوٹے اور آسان ترین حصے تک پہنچ گئے ہیں — ورچوئل والیٹ پر اپنا تازہ ترین منٹ کیا گیا NFT کیسے دیکھیں۔ ہم اس مثال کے لیے MetaMask کا استعمال کریں گے کیونکہ ہم نے پچھلے دو حصوں میں یہی استعمال کیا تھا۔

ایک شرط کے طور پر، آپ کے موبائل پر پہلے سے ہی MetaMask انسٹال ہونا چاہیے، اور اس میں وہ اکاؤنٹ شامل ہونا چاہیے جس پر آپ نے اپنا NFT منٹ کیا تھا — آپ یہ ایپ [iOS](https://apps.apple.com/us/app/metamask-blockchain-wallet/id1438144202) یا [Android](https://play.google.com/store/apps/details?id=io.metamask&hl=en_US&gl=US) پر مفت حاصل کر سکتے ہیں۔

## مرحلہ 1: اپنا نیٹ ورک Sepolia پر سیٹ کریں {#set-network-to-sepolia}

ایپ کے اوپری حصے میں، "Wallet" بٹن دبائیں، جس کے بعد آپ کو نیٹ ورک منتخب کرنے کا کہا جائے گا۔ چونکہ ہمارا NFT Sepolia نیٹ ورک پر منٹ کیا گیا تھا، اس لیے آپ کو اپنے نیٹ ورک کے طور پر Sepolia کو منتخب کرنا ہوگا۔

![MetaMask موبائل پر Sepolia کو اپنے نیٹ ورک کے طور پر کیسے سیٹ کریں](./goerliMetamask.gif)

## مرحلہ 2: اپنا کلیکٹ ایبل MetaMask میں شامل کریں {#add-nft-to-metamask}

ایک بار جب آپ Sepolia نیٹ ورک پر آ جائیں، تو دائیں جانب "Collectibles" ٹیب کو منتخب کریں اور اپنے NFT کا سمارٹ کنٹریکٹ ایڈریس اور ERC-721 ٹوکن ID شامل کریں — جسے آپ ہمارے ٹیوٹوریل کے حصہ II میں ڈیپلائے کیے گئے اپنے NFT کے ٹرانزیکشن ہیش کی بنیاد پر Etherscan پر تلاش کر سکتے ہیں۔

![اپنا ٹرانزیکشن ہیش اور ERC-721 ٹوکن ID کیسے تلاش کریں](./findNFTEtherscan.png)

اپنا NFT دیکھنے کے لیے آپ کو چند بار ریفریش کرنے کی ضرورت پڑ سکتی ہے — لیکن یہ وہاں موجود ہوگا <Emoji text="😄" size={1} />!

![اپنا NFT MetaMask پر کیسے اپ لوڈ کریں](./findNFTMetamask.gif)

مبارک ہو! آپ نے کامیابی کے ساتھ ایک NFT منٹ کر لیا ہے، اور اب آپ اسے دیکھ سکتے ہیں! ہم یہ دیکھنے کے لیے بے تاب ہیں کہ آپ NFT کی دنیا میں کیسے تہلکہ مچاتے ہیں!
