<FeaturedText>سیکھیں کہ <span dir="ltr">Elixir</span> پر مبنی پروجیکٹس اور ٹولنگ کا استعمال کرتے ہوئے ایتھیریم کے لیے کیسے ڈیولپ کیا جائے۔</FeaturedText>

ایتھیریم کا استعمال کرتے ہوئے غیر مرکزی ایپلی کیشنز (یا "<span dir="ltr">dapps</span>") بنائیں جو کرپٹو کرنسی اور بلاک چین ٹیکنالوجی کے فوائد کو استعمال کرتی ہیں۔ یہ <span dir="ltr">dapps</span> بلا اعتماد ہو سکتی ہیں، جس کا مطلب ہے کہ ایک بار جب انہیں ایتھیریم پر تعینات کر دیا جاتا ہے، تو وہ ہمیشہ پروگرام کے مطابق چلیں گی۔ وہ نئی قسم کی مالیاتی ایپلی کیشنز بنانے کے لیے ڈیجیٹل اثاثوں کو کنٹرول کر سکتی ہیں۔ وہ لامركزی ہو سکتی ہیں، جس کا مطلب ہے کہ کوئی ایک ادارہ یا شخص انہیں کنٹرول نہیں کرتا اور انہیں سنسر کرنا تقریباً ناممکن ہے۔

## <span dir="ltr">Elixir</span> پروجیکٹس اور ٹولز {#elixir-projects-and-tools}

### فعال {#active}

- [block_keys](https://github.com/ExWeb3/block_keys) - _<span dir="ltr">Elixir</span> میں <span dir="ltr">BIP32</span> اور <span dir="ltr">BIP44</span> کا نفاذ (ڈیٹرمینسٹک والیٹس کے لیے ملٹی اکاؤنٹ ہائیرارکی)_
- [ethereumex](https://github.com/mana-ethereum/ethereumex) - _ایتھیریم بلاک چین کے لیے <span dir="ltr">Elixir</span> جے سن آر پی سی کلائنٹ_
- [ethers](https://github.com/ExWeb3/elixir_ethers) - _<span dir="ltr">Elixir</span> کا استعمال کرتے ہوئے ایتھیریم پر سمارٹ کنٹریکٹس کے ساتھ تعامل کے لیے ایک جامع <span dir="ltr">Web3</span> لائبریری_
- [ethers_kms](https://github.com/ExWeb3/elixir_ethers_kms) - _<span dir="ltr">Ethers</span> کے لیے ایک <span dir="ltr">KMS</span> سائنر لائبریری (<span dir="ltr">AWS KMS</span> کے ساتھ ٹرانزیکشنز پر دستخط کریں)_
- [ex_abi](https://github.com/poanetwork/ex_abi) - _<span dir="ltr">Elixir</span> میں ایتھیریم <span dir="ltr">ABI</span> پارسر/ڈیکوڈر/انکوڈر کا نفاذ_
- [ex_keccak](https://github.com/ExWeb3/ex_keccak) - _<span dir="ltr">NIF</span> سے بنے <span dir="ltr">tiny-keccak</span> <span dir="ltr">Rust</span> کریٹ کا استعمال کرتے ہوئے <span dir="ltr">Keccak SHA3-256</span> ہیشز کا حساب لگانے کے لیے <span dir="ltr">Elixir</span> لائبریری_
- [ex_rlp](https://github.com/mana-ethereum/ex_rlp) - _ایتھیریم کی <span dir="ltr">RLP</span> (ریکرسیو لینتھ پریفکس) انکوڈنگ کا <span dir="ltr">Elixir</span> نفاذ_

### آرکائیو شدہ / اب برقرار نہیں رکھا جاتا {#archived--no-longer-maintained}

- [exw3](https://github.com/hswick/exw3) - _<span dir="ltr">Elixir</span> کے لیے ہائی لیول ایتھیریم <span dir="ltr">RPC</span> کلائنٹ_
- [mana](https://github.com/mana-ethereum/mana) - _<span dir="ltr">Elixir</span> میں لکھا گیا ایتھیریم مکمل نوڈ کا نفاذ_
