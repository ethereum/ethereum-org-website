ایک عام غلط فہمی یہ ہے کہ ڈیولپرز کو ایتھیریم پر تعمیر کرنے کے لیے [اسمارٹ کانٹریکٹس](/developers/docs/smart-contracts/) لکھنا لازمی ہے۔ یہ غلط ہے۔
ایتھیریم نیٹ ورک اور کمیونٹی کی ایک خوبصورتی یہ ہے کہ آپ تقریباً کسی بھی پروگرامنگ زبان میں [حصہ لے](/community/) سکتے ہیں۔

ایتھیریم اور اس کی کمیونٹی اوپن سورس کو اپناتی ہے۔ آپ مختلف زبانوں میں کمیونٹی پروجیکٹس - کلائنٹ امپلیمینٹیشنز، <span dir="ltr">APIs</span>، ڈیولپمنٹ فریم ورکس، ٹیسٹنگ ٹولز - تلاش کر سکتے ہیں۔

### اگر میری زبان تعاون یافتہ نہیں ہے تو کیا ہوگا {#other-lang}

اگر آپ کسی اضافی پروگرامنگ زبان کے لیے وسائل سے لنک کرنا چاہتے ہیں یا کسی ورچوئل کمیونٹی کی طرف اشارہ کرنا چاہتے ہیں تو آپ [ایک ایشو کھول کر](https://github.com/ethereum/ethereum-org-website/issues/new/choose) نئے صفحے کی درخواست کر سکتے ہیں۔

اگر آپ صرف ایک ایسی زبان کا استعمال کرتے ہوئے بلاک چین کے ساتھ انٹرفیس کرنے کے لیے کوڈ لکھنا چاہتے ہیں جو فی الحال تعاون یافتہ نہیں ہے، تو آپ ایتھیریم نیٹ ورک سے جڑنے کے لیے [جے سن آر پی سی انٹرفیس](/developers/docs/apis/json-rpc/) استعمال کر سکتے ہیں۔ کوئی بھی پروگرامنگ زبان جو <span dir="ltr">TCP/IP</span> استعمال کر سکتی ہے وہ اس انٹرفیس کو استعمال کر سکتی ہے۔
