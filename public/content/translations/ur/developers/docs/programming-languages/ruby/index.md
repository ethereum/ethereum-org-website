<FeaturedText><span dir="ltr">Ruby</span> پر مبنی پروجیکٹس اور ٹولز کا استعمال کرتے ہوئے ایتھیریم کے لیے ڈیولپ کرنا سیکھیں۔</FeaturedText>

ایتھیریم کا استعمال کرتے ہوئے غیر مرکزی ایپلی کیشنز (dapps) بنائیں جو کرپٹو کرنسی اور بلاک چین ٹیکنالوجی کے فوائد کو استعمال کرتی ہیں۔ یہ dapps بلا اعتماد ہو سکتی ہیں، جس کا مطلب ہے کہ ایک بار جب انہیں ایتھیریم پر تعینات کر دیا جاتا ہے، تو وہ ہمیشہ پروگرام کے مطابق چلیں گی۔ وہ نئی قسم کی مالیاتی ایپلی کیشنز بنانے کے لیے ڈیجیٹل اثاثوں کو کنٹرول کر سکتی ہیں۔ وہ لامركزی ہو سکتی ہیں، جس کا مطلب ہے کہ کوئی ایک ادارہ یا شخص انہیں کنٹرول نہیں کرتا اور انہیں سنسر کرنا تقریباً ناممکن ہے۔

## <span dir="ltr">Ruby</span> پروجیکٹس اور ٹولز {#ruby-projects-and-tools}

### فعال {#active}

- [<span dir="ltr">eth.rb</span>](https://github.com/q9f/eth.rb) - _ایتھیریم اکاؤنٹس، پیغامات، اور ٹرانزیکشنز کو سنبھالنے کے لیے <span dir="ltr">Ruby</span> لائبریری اور <span dir="ltr">RPC</span> کلائنٹ_
- [<span dir="ltr">keccak.rb</span>](https://github.com/q9f/keccak.rb) - _ایتھیریم کے ذریعے استعمال ہونے والا <span dir="ltr">Keccak (SHA3)</span> ہیش_
- [<span dir="ltr">siwe-ruby</span>](https://github.com/signinwithethereum/siwe-ruby) - _ایتھیریم کے ساتھ سائن ان (<span dir="ltr">Sign-In with Ethereum</span>) کا <span dir="ltr">Ruby</span> نفاذ_
- [<span dir="ltr">siwe-rails</span>](https://github.com/signinwithethereum/siwe-rails) - _<span dir="ltr">Rails gem</span> جو <span dir="ltr">SIWE</span> کے مقامی سائن ان روٹس کا اضافہ کرتا ہے_
- [<span dir="ltr">siwe-rails-examples</span>](https://github.com/signinwithethereum/siwe-rails-examples) - _کسٹم کنٹرولر کے ساتھ <span dir="ltr">Ruby on Rails</span> کا استعمال کرتے ہوئے <span dir="ltr">SIWE</span> کی مثال_
- [<span dir="ltr">omniauth-siwe</span>](https://github.com/signinwithethereum/omniauth-siwe) - _ایتھیریم کے ساتھ سائن ان (<span dir="ltr">SIWE</span>) کے لیے <span dir="ltr">OmniAuth</span> حکمت عملی_
- [<span dir="ltr">omniauth-nft</span>](https://github.com/valthon/omniauth-nft) - _<span dir="ltr">NFT</span> کی ملکیت کے ذریعے تصدیق کے لیے <span dir="ltr">OmniAuth</span> حکمت عملی_
- [<span dir="ltr">ethereum-on-rails</span>](https://github.com/q9f/ethereum-on-rails) - _<span dir="ltr">Ethereum on Rails</span> ٹیمپلیٹ جو میٹاماسک کو <span dir="ltr">Ruby on Rails</span> سے جوڑنے کی اجازت دیتا ہے_
