# ایتھیریم گائیڈز

کیا آپ اپنا ایتھیریم کا سفر شروع کرنا چاہتے ہیں؟ ہماری عملی گائیڈز آپ کو شروع کرنے کے لیے قدم بہ قدم رہنمائی کرتی ہیں، اور اس نئی ٹیکنالوجی کو سمجھنا آسان بناتی ہیں۔

## شروعات کریں

2. [والیٹ کا استعمال کیسے کریں](/guides/how-to-use-a-wallet/) - اپنے والیٹ میں ٹوکن بھیجنے اور وصول کرنے کا طریقہ سیکھیں اور والیٹ کو پروجیکٹس سے کیسے جوڑیں۔
