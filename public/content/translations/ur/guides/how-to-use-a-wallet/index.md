# والیٹ کا استعمال کیسے کریں

والیٹ کے تمام بنیادی فنکشنز کو چلانے کا طریقہ سیکھیں۔ اگر آپ کے پاس ابھی تک کوئی والیٹ نہیں ہے، تو ہمارا [ایتھریم اکاؤنٹ کیسے بنائیں](/guides/how-to-create-an-ethereum-account/) دیکھیں۔

## اپنا والیٹ کھولیں

آپ کو ایک ڈیش بورڈ نظر آنا چاہیے جو ممکنہ طور پر آپ کا بیلنس دکھائے گا اور اس میں ٹوکن بھیجنے اور وصول کرنے کے بٹن ہوں گے۔

## کرپٹو کرنسی وصول کریں

کیا آپ اپنے والیٹ میں کرپٹو وصول کرنا چاہتے ہیں؟

ہر ایتھریم اکاؤنٹ کا اپنا وصولی کا پتہ (receiving address) ہوتا ہے جو نمبروں اور حروف کی ایک منفرد ترتیب ہے۔ یہ پتہ بینک اکاؤنٹ نمبر کی طرح کام کرتا ہے۔ ایتھریم پتے ہمیشہ "0x" سے شروع ہوں گے۔ آپ اس پتے کو کسی کے ساتھ بھی شیئر کر سکتے ہیں: ایسا کرنا محفوظ ہے۔

آپ کا پتہ آپ کے گھر کے پتے کی طرح ہے: آپ کو لوگوں کو بتانا ہوگا کہ یہ کیا ہے تاکہ وہ آپ کو تلاش کر سکیں۔ ایسا کرنا محفوظ ہے، کیونکہ آپ اب بھی اپنے سامنے والے دروازے کو ایک اور چابی سے مقفل کر سکتے ہیں جسے صرف آپ کنٹرول کرتے ہیں تاکہ کوئی اندر نہ آ سکے، چاہے وہ جانتے ہوں کہ آپ کہاں رہتے ہیں۔

جو بھی آپ کو رقم بھیجنا چاہتا ہے، آپ کو اسے اپنا عوامی پتہ (public address) فراہم کرنا ہوگا۔ بہت سی والیٹ ایپس آپ کو اپنا پتہ کاپی کرنے یا آسانی کے لیے اسکین کرنے کے لیے QR کوڈ دکھانے کی سہولت دیتی ہیں۔ کسی بھی ایتھریم پتے کو دستی طور پر ٹائپ کرنے سے گریز کریں۔ اس سے آسانی سے ٹائپنگ کی غلطیاں ہو سکتی ہیں اور فنڈز ضائع ہو سکتے ہیں۔

مختلف ایپس مختلف ہو سکتی ہیں یا مختلف زبان استعمال کر سکتی ہیں، لیکن اگر آپ فنڈز منتقل کرنے کی کوشش کر رہے ہیں تو انہیں آپ کو اسی طرح کے عمل سے گزارنا چاہیے۔

1. اپنی والیٹ ایپ کھولیں۔
2. "Receive" (یا اسی طرح کے الفاظ والے آپشن) پر کلک کریں۔
3. اپنا ایتھریم پتہ کلپ بورڈ پر کاپی کریں۔
4. بھیجنے والے کو اپنا وصولی کا ایتھریم پتہ فراہم کریں۔

## کرپٹو کرنسی بھیجیں

کیا آپ کسی دوسرے والیٹ میں ETH بھیجنا چاہیں گے؟

1. اپنی والیٹ ایپ کھولیں۔
2. وصول کرنے کا پتہ حاصل کریں اور یقینی بنائیں کہ آپ اسی نیٹ ورک سے منسلک ہیں جس سے وصول کنندہ ہے۔
3. وصول کرنے کا پتہ درج کریں یا اپنے کیمرے سے QR کوڈ اسکین کریں تاکہ آپ کو پتہ دستی طور پر نہ لکھنا پڑے۔
4. اپنے والیٹ میں "Send" بٹن (یا اسی طرح کے الفاظ والے متبادل) پر کلک کریں۔

![Send field for crypto address](./send.png)
<br/>

5. بہت سے اثاثے، جیسے DAI یا USDC، متعدد نیٹ ورکس پر موجود ہیں۔ کرپٹو ٹوکن منتقل کرتے وقت، یقینی بنائیں کہ وصول کنندہ وہی نیٹ ورک استعمال کر رہا ہے جو آپ کر رہے ہیں، کیونکہ یہ قابل تبادلہ (interchangeable) نہیں ہیں۔
6. یقینی بنائیں کہ آپ کے والیٹ میں ٹرانزیکشن فیس ادا کرنے کے لیے کافی ETH موجود ہے، جو نیٹ ورک کے حالات کے لحاظ سے مختلف ہوتی ہے۔ زیادہ تر والیٹس خود بخود تجویز کردہ فیس کو ٹرانزیکشن میں شامل کر دیں گے جس کی آپ پھر تصدیق کر سکتے ہیں۔
7. ایک بار جب آپ کی ٹرانزیکشن پروسیس ہو جاتی ہے، تو متعلقہ کرپٹو رقم وصول کنندہ کے اکاؤنٹ میں ظاہر ہو جائے گی۔ نیٹ ورک کے موجودہ استعمال کے لحاظ سے اس میں چند سیکنڈ سے لے کر چند منٹ تک لگ سکتے ہیں۔

## پروجیکٹس سے منسلک ہونا

تمام ایتھریم پروجیکٹس میں آپ کا پتہ ایک ہی ہوگا۔ آپ کو کسی بھی پروجیکٹ پر انفرادی طور پر رجسٹر کرنے کی ضرورت نہیں ہے۔ ایک بار جب آپ کے پاس والیٹ ہو جائے، تو آپ بغیر کسی اضافی معلومات کے کسی بھی ایتھریم پروجیکٹ سے منسلک ہو سکتے ہیں۔ کسی ای میل یا کسی اور ذاتی معلومات کی ضرورت نہیں ہے۔

1. کسی بھی پروجیکٹ کی ویب سائٹ پر جائیں۔
2. اگر پروجیکٹ کا لینڈنگ پیج صرف پروجیکٹ کی ایک جامد (static) تفصیل ہے، تو آپ کو مینو میں "Open the App" بٹن پر کلک کرنے کے قابل ہونا چاہیے جو آپ کو اصل ویب ایپ پر لے جائے گا۔
3. ایک بار جب آپ ایپ میں ہوں تو "Connect" پر کلک کریں۔

![Button allowing user to connect to the website with a wallet](./connect1.png)

4. فراہم کردہ آپشنز کی فہرست سے اپنا والیٹ منتخب کریں۔ اگر آپ اپنا والیٹ نہیں دیکھ سکتے ہیں، تو یہ "WalletConnect" آپشن کے تحت چھپا ہو سکتا ہے۔

![Selecting from a list of wallets to connect with](./connect2.png)

5. کنکشن قائم کرنے کے لیے اپنے والیٹ میں دستخط کی درخواست (signature request) کی تصدیق کریں۔ **اس پیغام پر دستخط کرنے کے لیے کوئی ETH خرچ کرنے کی ضرورت نہیں ہونی چاہیے**۔
6. بس اتنا ہی! ایپ کا استعمال شروع کریں۔ آپ ہمارے [dApps صفحہ](/apps/#explore) پر کچھ دلچسپ پروجیکٹس تلاش کر سکتے ہیں۔

## اکثر پوچھے گئے سوالات

### اگر میرے پاس ایک ETH پتہ ہے، تو کیا میرے پاس دیگر بلاک چینز پر بھی وہی پتہ ہے؟

آپ تمام EVM ہم آہنگ (compatible) بلاک چینز پر ایک ہی پتہ استعمال کر سکتے ہیں (اگر آپ کے پاس ریکوری فریز والا والیٹ ہے)۔ یہ [فہرست](https://chainlist.org/) آپ کو دکھائے گی کہ آپ کن بلاک چینز کو اسی پتے کے ساتھ استعمال کر سکتے ہیں۔ کچھ بلاک چینز، جیسے Bitcoin، نیٹ ورک کے قواعد کا ایک بالکل الگ سیٹ نافذ کرتی ہیں اور آپ کو ایک مختلف فارمیٹ کے ساتھ ایک مختلف پتے کی ضرورت ہوگی۔ اگر آپ کے پاس اسمارٹ کانٹریکٹ والیٹ ہے تو آپ کو اس کی پروڈکٹ ویب سائٹ چیک کرنی چاہیے تاکہ مزید معلومات مل سکیں کہ کون سی بلاک چینز تعاون یافتہ ہیں۔

### کیا میں ایک سے زیادہ ڈیوائسز پر ایک ہی پتہ استعمال کر سکتا ہوں؟

جی ہاں، آپ ایک سے زیادہ ڈیوائسز پر ایک ہی پتہ استعمال کر سکتے ہیں۔ والیٹس تکنیکی طور پر صرف ایک انٹرفیس ہیں جو آپ کو آپ کا بیلنس دکھاتے ہیں اور ٹرانزیکشنز کرنے میں مدد دیتے ہیں، آپ کا اکاؤنٹ والیٹ کے اندر نہیں، بلکہ بلاک چین پر محفوظ ہوتا ہے۔

### مجھے کرپٹو موصول نہیں ہوئی، میں ٹرانزیکشن کا اسٹیٹس کہاں چیک کر سکتا ہوں؟

آپ کسی بھی ٹرانزیکشن کا اسٹیٹس ریئل ٹائم میں دیکھنے کے لیے [بلاک ایکسپلوررز](/developers/docs/data-and-analytics/block-explorers/) کا استعمال کر سکتے ہیں۔ آپ کو بس اپنا والیٹ کا پتہ یا ٹرانزیکشن کی ID تلاش کرنے کی ضرورت ہے۔

### کیا میں ٹرانزیکشنز منسوخ یا واپس کر سکتا ہوں؟

نہیں، ایک بار جب ٹرانزیکشن کی تصدیق ہو جاتی ہے، تو آپ ٹرانزیکشن کو منسوخ نہیں کر سکتے۔
