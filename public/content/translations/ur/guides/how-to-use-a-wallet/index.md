# والیٹ کا استعمال کیسے کریں

والیٹ کے تمام بنیادی فنکشنز کو آپریٹ کرنے کا طریقہ کار جانیں. اگر آپ کے پاس ابھی تک کوئی نہیں ہے، تو ہمارا [ایتھیریم اکاؤنٹ کیسے بنائیں](/guides/how-to-create-an-ethereum-account/) دیکھیں۔

## اپنا والیٹ کھولیں

آپ کو ایک ڈیش بورڈ نظر آئے گا جو ممکنہ طور پر آپ کا بیلنس دکھائے گا اور اس میں ٹوکن بھیجنے اور وصول کرنے کے بٹن ہوں گے۔

## کرپٹو کرنسی وصول کریں

کیا آپ اپنے والیٹ میں کرپٹو وصول کرنا چاہتے ہیں؟

ہر ایتھیریم اکاؤنٹ کا اپنا وصول کرنے والا ایڈریس ہوتا ہے جو نمبروں اور حروف کا ایک منفرد سلسلہ ہوتا ہے۔ یہ ایڈریس بینک اکاؤنٹ نمبر کی طرح کام کرتا ہے۔ ایتھیریم ایڈریس ہمیشہ "0x" سے شروع ہوں گے۔ آپ یہ ایڈریس کسی کے ساتھ بھی شیئر کر سکتے ہیں: ایسا کرنا محفوظ ہے۔

آپ کا ایڈریس آپ کے گھر کے ایڈریس کی طرح ہے: آپ کو لوگوں کو بتانا ہوگا کہ یہ کیا ہے تاکہ وہ آپ کو تلاش کر سکیں۔ ایسا کرنا محفوظ ہے، کیونکہ آپ اب بھی اپنے سامنے کے دروازے کو دوسری چابی سے لاک کر سکتے ہیں جسے صرف آپ کنٹرول کرتے ہیں تاکہ کوئی اندر نہ آ سکے، چاہے وہ جانتے ہوں کہ آپ کہاں رہتے ہیں۔

جو کوئی بھی آپ کو رقم بھیجنا چاہتا ہے، آپ کو اسے اپنا پبلک ایڈریس فراہم کرنے کی ضرورت ہے۔ بہت سی والیٹ ایپس آسان استعمال کے لیے آپ کو اپنا ایڈریس کاپی کرنے یا اسکین کرنے کے لیے QR کوڈ دکھانے کی اجازت دیتی ہیں۔ کسی بھی ایتھیریم ایڈریس کو دستی طور پر ٹائپ کرنے سے گریز کریں۔ اس سے آسانی سے کلریکل غلطیاں اور فنڈز کا نقصان ہو سکتا ہے۔

مختلف ایپس مختلف ہو سکتی ہیں یا مختلف زبان استعمال کر سکتی ہیں، لیکن اگر آپ فنڈز کی منتقلی کی کوشش کر رہے ہیں تو انہیں آپ کو اسی طرح کے عمل سے گزرنا چاہیے۔

1. اپنی والیٹ ایپ کھولیں۔
2. "وصول کریں" (یا اسی طرح کے الفاظ والے آپشن) پر کلک کریں۔
3. اپنا ایتھیریم ایڈریس کلپ بورڈ پر کاپی کریں۔
4. بھیجنے والے کو اپنا وصول کرنے والا ایتھیریم ایڈریس فراہم کریں۔

## کرپٹو کرنسی بھیجیں

کیا آپ کسی دوسرے والیٹ میں ETH بھیجنا چاہیں گے؟

1. اپنی والیٹ ایپ کھولیں۔
2. وصول کرنے والا ایڈریس حاصل کریں اور یقینی بنائیں کہ آپ وصول کنندہ کے نیٹ ورک سے ہی جڑے ہوئے ہیں۔
3. وصول کرنے والا ایڈریس درج کریں یا اپنے کیمرے سے QR کوڈ اسکین کریں تاکہ آپ کو ایڈریس دستی طور پر نہ لکھنا پڑے۔
4. اپنے والیٹ میں "بھیجیں" کے بٹن پر کلک کریں (یا اسی طرح کے الفاظ والے متبادل پر)۔

![کرپٹو ایڈریس کے لیے بھیجنے کا فیلڈ](./send.png) <br/>

5. بہت سے اثاثے، جیسے DAI یا USDC، متعدد نیٹ ورکس پر موجود ہیں۔ کرپٹو ٹوکن منتقل کرتے وقت، اس بات کو یقینی بنائیں کہ وصول کنندہ وہی نیٹ ورک استعمال کر رہا ہے جو آپ کر رہے ہیں، کیونکہ یہ قابل تبادلہ نہیں ہیں۔
6. یقینی بنائیں کہ آپ کے والیٹ میں ٹرانزیکشن فیس کو پورا کرنے کے لیے کافی ETH ہے، جو نیٹ ورک کے حالات کے لحاظ سے مختلف ہوتی ہے۔ زیادہ تر والیٹس خود بخود تجویز کردہ فیس کو ٹرانزیکشن میں شامل کر دیں گے جس کی آپ پھر تصدیق کر سکتے ہیں۔
7. ایک بار جب آپ کے ٹرانزیکشن پر کارروائی ہو جاتی ہے، تو متعلقہ کرپٹو رقم وصول کنندہ کے اکاؤنٹ میں ظاہر ہو جائے گی۔ اس میں چند سیکنڈ سے لے کر چند منٹ تک کا وقت لگ سکتا ہے، اس بات پر منحصر ہے کہ فی الحال نیٹ ورک کتنا استعمال ہو رہا ہے۔

## پروجیکٹس سے جڑنا

تمام ایتھیریم پروجیکٹس میں آپ کا ایڈریس ایک ہی ہوگا۔ آپ کو کسی بھی پروجیکٹ پر انفرادی طور پر رجسٹر کرنے کی ضرورت نہیں ہے۔ ایک بار جب آپ کے پاس والیٹ ہو جاتا ہے، تو آپ بغیر کسی اضافی معلومات کے کسی بھی ایتھیریم پروجیکٹ سے جڑ سکتے ہیں۔ کسی ای میل یا کسی دوسری ذاتی معلومات کی ضرورت نہیں ہے۔

1. کسی بھی پروجیکٹ کی ویب سائٹ پر جائیں۔
2. اگر پروجیکٹ کا لینڈنگ پیج صرف پروجیکٹ کی ایک جامد تفصیل ہے، تو آپ کو مینو میں "ایپ کھولیں" بٹن پر کلک کرنے کے قابل ہونا چاہئے جو آپ کو اصل ویب ایپ پر لے جائے گا۔
3. ایپ میں داخل ہونے کے بعد "کنیکٹ کریں" پر کلک کریں۔

![بٹن جو صارف کو والیٹ کے ساتھ ویب سائٹ سے جڑنے کی اجازت دیتا ہے](./connect1.png)

4. فراہم کردہ آپشنز کی فہرست سے اپنا والیٹ منتخب کریں۔ اگر آپ اپنا والیٹ نہیں دیکھ پا رہے ہیں، تو یہ "WalletConnect" آپشن کے تحت پوشیدہ ہو سکتا ہے۔

![جڑنے کے لیے والیٹس کی فہرست سے انتخاب کرنا](./connect2.png)

5. کنکشن قائم کرنے کے لیے اپنے والیٹ میں دستخط کی درخواست کی تصدیق کریں۔ **اس پیغام پر دستخط کرنے کے لیے کوئی ETH خرچ کرنے کی ضرورت نہیں ہونی چاہیے**۔
6. بس ہو گیا! ایپ کا استعمال شروع کریں۔ آپ کو ہمارے [dApps صفحہ](/apps/#explore) پر کچھ دلچسپ پروجیکٹس مل سکتے ہیں۔ <br />

## اکثر پوچھے جانے والے سوالات

### اگر میرا ETH ایڈریس ہے، کیا میرا وہی ایڈریس دیگر بلاک چینز پر بھی ہوگا؟

آپ تمام EVM ہم آہنگ بلاک چینز پر ایک ہی ایڈریس استعمال کر سکتے ہیں (اگر آپ کے پاس ریکوری فریز والا والیٹ ہے)۔ یہ [فہرست](https://chainlist.org/) آپ کو دکھائے گی کہ آپ کون سی بلاک چینز پر وہی ایڈریس استعمال کر سکتے ہیں۔ کچھ بلاک چینز، جیسے بٹ کوائن، مکمل طور پر الگ نیٹ ورک قوانین استعمال کرتے ہیں اور آپ کو مختلف فارمیٹ کے ساتھ ایک مختلف ایڈریس کی ضرورت ہوگی۔ اگر آپ کے پاس اسمارٹ کانٹریکٹ والیٹ ہے تو آپ کو اس کی پروڈکٹ ویب سائٹ پر مزید معلومات کے لیے چیک کرنا چاہیے کہ کون سی بلاک چینز سپورٹڈ ہیں۔

### کیا میں ایک ہی ایڈریس کو متعدد ڈیوائسز پر استعمال کر سکتا ہوں؟

جی ہاں، آپ ایک ہی ایڈریس کو متعدد ڈیوائسز پر استعمال کر سکتے ہیں۔ والیٹس تکنیکی طور پر صرف ایک انٹرفیس ہیں جو آپ کو آپ کا بیلنس دکھاتے ہیں اور ٹرانزیکشنز کرتے ہیں، آپ کا اکاؤنٹ والیٹ کے اندر نہیں بلکہ بلاک چین پر اسٹور ہوتا ہے۔

### مجھے کرپٹو موصول نہیں ہوا ہے، میں ٹرانزیکشن کا اسٹیٹس کہاں چیک کر سکتا ہوں؟

آپ کسی بھی ٹرانزیکشن کا اسٹیٹس حقیقی وقت میں دیکھنے کے لیے [بلاک ایکسپلوررز](/developers/docs/data-and-analytics/block-explorers/) کا استعمال کر سکتے ہیں۔ آپ کو صرف اپنے والیٹ ایڈریس یا ٹرانزیکشن کی ID کو تلاش کرنا ہے۔

### کیا میں ٹرانزیکشنز کو منسوخ یا واپس کر سکتا ہوں؟

نہیں، ایک بار ٹرانزیکشن کی تصدیق ہو جانے کے بعد، آپ ٹرانزیکشن کو منسوخ نہیں کر سکتے۔
