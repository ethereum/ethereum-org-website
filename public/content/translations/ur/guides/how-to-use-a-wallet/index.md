والیٹ کے تمام بنیادی فنکشنز کو چلانے کا طریقہ سیکھیں۔ اگر آپ کے پاس ابھی تک کوئی والیٹ نہیں ہے، تو ہمارا [ایتھیریم اکاؤنٹ بنانے کا طریقہ](/guides/how-to-create-an-ethereum-account/) دیکھیں۔

## اپنا والیٹ کھولیں {#open-your-wallet}

آپ کو ایک ڈیش بورڈ نظر آنا چاہیے جو ممکنہ طور پر آپ کا بیلنس دکھائے گا اور اس میں ٹوکن بھیجنے اور وصول کرنے کے بٹن ہوں گے۔

## کرپٹو کرنسی وصول کریں {#receive-cryptocurrency}

کیا آپ اپنے والیٹ میں کرپٹو وصول کرنا چاہتے ہیں؟

ہر ایتھیریم اکاؤنٹ کا اپنا وصولی کا پتہ ہوتا ہے جو اعداد اور حروف کی ایک منفرد ترتیب ہے۔ یہ پتہ بینک اکاؤنٹ نمبر کی طرح کام کرتا ہے۔ ایتھیریم پتے ہمیشہ <span dir="ltr">"0x"</span> سے شروع ہوں گے۔ آپ اس پتے کو کسی کے ساتھ بھی شیئر کر سکتے ہیں: ایسا کرنا محفوظ ہے۔

آپ کا پتہ آپ کے گھر کے پتے کی طرح ہے: آپ کو لوگوں کو بتانا ہوگا کہ یہ کیا ہے تاکہ وہ آپ کو تلاش کر سکیں۔ ایسا کرنا محفوظ ہے، کیونکہ آپ اب بھی اپنے سامنے والے دروازے کو ایک اور کلید سے مقفل کر سکتے ہیں جسے صرف آپ کنٹرول کرتے ہیں تاکہ کوئی اندر نہ آ سکے، چاہے وہ جانتے ہوں کہ آپ کہاں رہتے ہیں۔

جو بھی آپ کو رقم بھیجنا چاہتا ہے، آپ کو اسے اپنا عوامی پتہ فراہم کرنا ہوگا۔ بہت سی والیٹ ایپس آپ کو اپنا پتہ کاپی کرنے یا آسانی کے لیے اسکین کرنے کے لیے <span dir="ltr">QR</span> کوڈ دکھانے کی سہولت دیتی ہیں۔ کسی بھی ایتھیریم پتے کو دستی طور پر ٹائپ کرنے سے گریز کریں۔ اس سے آسانی سے کلیریکل غلطیاں ہو سکتی ہیں اور فنڈز ضائع ہو سکتے ہیں۔

مختلف ایپس مختلف ہو سکتی ہیں یا مختلف زبان استعمال کر سکتی ہیں، لیکن اگر آپ فنڈز کی منتقلی کی کوشش کر رہے ہیں تو انہیں آپ کو اسی طرح کے عمل سے گزارنا چاہیے۔

1. اپنی والیٹ ایپ کھولیں۔
2. "وصول کریں" (یا اسی طرح کے الفاظ والے آپشن) پر کلک کریں۔
3. اپنا ایتھیریم پتہ کلپ بورڈ پر کاپی کریں۔
4. بھیجنے والے کو اپنا وصولی کا ایتھیریم پتہ فراہم کریں۔

## کرپٹو کرنسی بھیجیں {#send-cryptocurrency}

کیا آپ کسی دوسرے والیٹ میں <span dir="ltr">ETH</span> بھیجنا چاہیں گے؟

1. اپنی والیٹ ایپ کھولیں۔
2. وصولی کا پتہ حاصل کریں اور یقینی بنائیں کہ آپ اسی نیٹ ورک سے منسلک ہیں جس سے وصول کنندہ ہے۔
3. وصولی کا پتہ درج کریں یا اپنے کیمرے سے <span dir="ltr">QR</span> کوڈ اسکین کریں تاکہ آپ کو پتہ دستی طور پر نہ لکھنا پڑے۔
4. اپنے والیٹ میں "بھیجیں" بٹن (یا اسی طرح کے الفاظ والے متبادل) پر کلک کریں۔

![Send field for crypto address](./send.png)
<br/>

5. بہت سے اثاثے، جیسے <span dir="ltr">DAI</span> یا <span dir="ltr">USDC</span>، متعدد نیٹ ورکس پر موجود ہیں۔ کرپٹو ٹوکنز کی منتقلی کرتے وقت، یقینی بنائیں کہ وصول کنندہ وہی نیٹ ورک استعمال کر رہا ہے جو آپ کر رہے ہیں، کیونکہ یہ قابل تبادلہ نہیں ہیں۔
6. یقینی بنائیں کہ آپ کے والیٹ میں لین دین کی فیس ادا کرنے کے لیے کافی <span dir="ltr">ETH</span> موجود ہے، جو نیٹ ورک کے حالات کے لحاظ سے مختلف ہوتی ہے۔ زیادہ تر والیٹس خود بخود تجویز کردہ فیس کو ٹرانزیکشن میں شامل کر دیں گے جس کی آپ پھر تصدیق کر سکتے ہیں۔
7. ایک بار جب آپ کی ٹرانزیکشن پر کارروائی ہو جاتی ہے، تو متعلقہ کرپٹو رقم وصول کنندہ کے اکاؤنٹ میں ظاہر ہو جائے گی۔ نیٹ ورک کے موجودہ استعمال کے لحاظ سے اس میں چند سیکنڈ سے لے کر چند منٹ تک لگ سکتے ہیں۔

## پروجیکٹس سے منسلک ہونا {#connecting-to-projects}

تمام ایتھیریم پروجیکٹس میں آپ کا پتہ ایک ہی ہوگا۔ آپ کو کسی بھی پروجیکٹ پر انفرادی طور پر رجسٹر ہونے کی ضرورت نہیں ہے۔ ایک بار جب آپ کے پاس والیٹ ہو جائے، تو آپ بغیر کسی اضافی معلومات کے کسی بھی ایتھیریم پروجیکٹ سے منسلک ہو سکتے ہیں۔ کسی ای میل یا کسی دوسری ذاتی معلومات کی ضرورت نہیں ہے۔

1. کسی بھی پروجیکٹ کی ویب سائٹ پر جائیں۔
2. اگر پروجیکٹ کا لینڈنگ پیج صرف پروجیکٹ کی ایک جامد تفصیل ہے، تو آپ کو مینو میں "ایپ کھولیں" بٹن پر کلک کرنے کے قابل ہونا چاہیے جو آپ کو اصل ویب ایپ پر لے جائے گا۔
3. ایک بار جب آپ ایپ میں ہوں تو "منسلک کریں" پر کلک کریں۔

![Button allowing user to connect to the website with a wallet](./connect1.png)

4. فراہم کردہ آپشنز کی فہرست سے اپنا والیٹ منتخب کریں۔ اگر آپ اپنا والیٹ نہیں دیکھ سکتے ہیں، تو یہ <span dir="ltr">WalletConnect</span> آپشن کے تحت چھپا ہو سکتا ہے۔

![Selecting from a list of wallets to connect with](./connect2.png)

5. کنکشن قائم کرنے کے لیے اپنے والیٹ میں دستخط کی درخواست کی تصدیق کریں۔ **اس پیغام پر دستخط کرنا کسی <span dir="ltr">ETH</span> کو خرچ کرنے کا تقاضا نہیں کرتا**۔
6. بس اتنا ہی! ایپ کا استعمال شروع کریں۔ آپ ہمارے [غیر مرکزی ایپلی کیشنز (dapps) کے صفحے](/apps/#explore) پر کچھ دلچسپ پروجیکٹس تلاش کر سکتے ہیں۔

## اکثر پوچھے گئے سوالات {#frequently-asked-questions}

### اگر میرے پاس ایک <span dir="ltr">ETH</span> پتہ ہے، تو کیا میرے پاس دیگر بلاک چینز پر بھی وہی پتہ ہے؟ {#if-i-own-an-eth-address-do-i-own-the-same-address-on-other-blockchains}

آپ تمام <span dir="ltr">EVM</span> ہم آہنگ بلاک چینز پر ایک ہی پتہ استعمال کر سکتے ہیں (اگر آپ کے پاس ریکوری فریز والا والیٹ ہے)۔ یہ [فہرست](https://chainlist.org/) آپ کو دکھائے گی کہ آپ کن بلاک چینز کو ایک ہی پتے کے ساتھ استعمال کر سکتے ہیں۔ کچھ بلاک چینز، جیسے بٹ کوائن، نیٹ ورک کے قواعد کا ایک بالکل الگ سیٹ نافذ کرتی ہیں اور آپ کو ایک مختلف فارمیٹ کے ساتھ ایک مختلف پتے کی ضرورت ہوگی۔ اگر آپ کے پاس سمارٹ کنٹریکٹ والیٹ ہے تو آپ کو اس کی پروڈکٹ ویب سائٹ چیک کرنی چاہیے تاکہ مزید معلومات مل سکیں کہ کون سی بلاک چینز تعاون یافتہ ہیں۔

### کیا میں ایک سے زیادہ آلات پر ایک ہی پتہ استعمال کر سکتا ہوں؟ {#can-i-use-the-same-address-on-multiple-devices}

جی ہاں، آپ ایک سے زیادہ آلات پر ایک ہی پتہ استعمال کر سکتے ہیں۔ والیٹس تکنیکی طور پر صرف ایک انٹرفیس ہیں جو آپ کو آپ کا بیلنس دکھاتے ہیں اور ٹرانزیکشنز کرنے میں مدد دیتے ہیں، آپ کا اکاؤنٹ والیٹ کے اندر محفوظ نہیں ہوتا، بلکہ بلاک چین پر ہوتا ہے۔

### مجھے کرپٹو موصول نہیں ہوئی، میں ٹرانزیکشن کی صورتحال کہاں چیک کر سکتا ہوں؟ {#i-have-not-received-the-crypto-where-can-i-check-the-status-of-a-transaction}

آپ کسی بھی ٹرانزیکشن کی صورتحال کو ریئل ٹائم میں دیکھنے کے لیے [بلاک ایکسپلوررز](/developers/docs/data-and-analytics/block-explorers/) کا استعمال کر سکتے ہیں۔ آپ کو بس اپنا والیٹ کا پتہ یا ٹرانزیکشن کی <span dir="ltr">ID</span> تلاش کرنے کی ضرورت ہے۔

### کیا میں ٹرانزیکشنز منسوخ یا واپس کر سکتا ہوں؟ {#can-i-cancel-or-return-transactions}

نہیں، ایک بار جب ٹرانزیکشن کی تصدیق ہو جاتی ہے، تو آپ ٹرانزیکشن کو منسوخ نہیں کر سکتے۔
