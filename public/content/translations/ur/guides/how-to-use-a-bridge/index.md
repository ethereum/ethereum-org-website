اگر ایتھیریم پر بہت زیادہ ٹریفک ہو، تو یہ مہنگا ہو سکتا ہے۔ اس کا ایک حل نئی "لیئرز" بنانا ہے: یعنی، مختلف نیٹ ورکس جو خود ایتھیریم کی طرح کام کرتے ہیں۔ یہ نام نہاد لیئر ۲ (<span dir="ltr">l2</span>) کم فیس پر بہت زیادہ ٹرانزیکشنز پروسیس کر کے، اور صرف ان کا نتیجہ کبھی کبھار ایتھیریم پر اسٹور کر کے ایتھیریم پر ہجوم اور لاگت کو کم کرنے میں مدد کرتی ہیں۔ اس طرح، یہ لیئر ۲ (<span dir="ltr">l2</span>) ہمیں تیز رفتاری اور کم لاگت کے ساتھ ٹرانزیکشن کرنے کے قابل بناتی ہیں۔ ان فوائد کی وجہ سے بہت سے مشہور کرپٹو پروجیکٹس لیئر ۲ (<span dir="ltr">l2</span>) کی طرف منتقل ہو رہے ہیں۔ ٹوکنز کو ایتھیریم سے لیئر ۲ (<span dir="ltr">l2</span>) پر منتقل کرنے کا سب سے آسان طریقہ پل کا استعمال کرنا ہے۔

**پیشگی شرط:** 

- ایک کرپٹو والیٹ رکھیں—اگر آپ کے پاس نہیں ہے، تو [ایک ایتھیریم اکاؤنٹ بنانے](/guides/how-to-create-an-ethereum-account/) کے لیے اس گائیڈ پر عمل کریں
- اپنے والیٹ میں فنڈز شامل کریں

## 1. فیصلہ کریں کہ آپ کون سا لیئر ۲ (<span dir="ltr">l2</span>) نیٹ ورک استعمال کرنا چاہتے ہیں {#1-determine-which-layer-2-network-you-want-to-use}

آپ ہمارے [لیئر ۲ (<span dir="ltr">l2</span>) صفحہ](/layer-2/) پر مختلف پروجیکٹس اور اہم لنکس کے بارے میں مزید جان سکتے ہیں۔

## 2. منتخب کردہ پل پر جائیں {#2-go-to-the-selected-bridge}

کچھ مشہور لیئر ۲ (<span dir="ltr">l2</span>) یہ ہیں:

- [آربٹرم پل](https://portal.arbitrum.io/bridge?l2ChainId=42161)
- [آپٹیمزم پل](https://app.optimism.io/bridge/deposit)
- [<span dir="ltr">Boba</span> نیٹ ورک پل](https://hub.boba.network/)

## 3. اپنے والیٹ کے ساتھ پل سے جڑیں {#3-connect-to-the-bridge-with-your-wallet}

یقینی بنائیں کہ آپ کا والیٹ ایتھیریم مین نیٹ نیٹ ورک سے جڑا ہوا ہے۔ اگر ایسا نہیں ہے، تو ویب سائٹ خود بخود آپ کو نیٹ ورکس تبدیل کرنے کا کہے گی۔

![Common interface for bridging tokens](./bridge1.png)

## 4. رقم بتائیں اور فنڈز منتقل کریں {#4-specify-the-amount-and-move-the-funds}

ناخوشگوار حیرت سے بچنے کے لیے اس رقم کا جائزہ لیں جو آپ کو لیئر ۲ (<span dir="ltr">l2</span>) نیٹ ورک پر بدلے میں ملے گی اور فیس چیک کریں۔

![Common interface for bridging tokens](./bridge2.png)

## 5. اپنے والیٹ میں ٹرانزیکشن کی تصدیق کریں {#5-confirm-the-transaction-in-your-wallet}

آپ کو ٹرانزیکشن پروسیس کرنے کے لیے <span dir="ltr">ETH</span> کی صورت میں ایک فیس (جسے [گیس](/glossary/#gas) کہا جاتا ہے) ادا کرنی ہوگی۔

![Common interface for bridging tokens](./bridge3.png)

## 6. اپنے فنڈز منتقل ہونے کا انتظار کریں {#6-wait-for-your-funds-to-be-moved}

اس عمل میں ۱۰ منٹ سے زیادہ وقت نہیں لگنا چاہیے۔

## 7. منتخب کردہ لیئر ۲ (<span dir="ltr">l2</span>) نیٹ ورک کو اپنے والیٹ میں شامل کریں (اختیاری) {#7-add-the-selected-layer-2-network-to-your-wallet-optional}

آپ نیٹ ورک کی <span dir="ltr">RPC</span> تفصیلات تلاش کرنے کے لیے [<span dir="ltr">chainlist.org</span>](https://chainlist.org) استعمال کر سکتے ہیں۔ ایک بار نیٹ ورک شامل ہونے اور ٹرانزیکشن مکمل ہونے کے بعد، آپ کو اپنے والیٹ میں ٹوکنز نظر آنے چاہئیں۔

## اکثر پوچھے گئے سوالات {#frequently-asked-questions}

### اگر میرے پاس ایکسچینج پر فنڈز ہوں تو کیا ہوگا؟ {#what-if-i-have-funds-on-an-exchange}

آپ شاید کسی ایکسچینج سے براہ راست کچھ لیئر ۲ (<span dir="ltr">l2</span>) پر رقم نکلوا سکیں۔ مزید معلومات کے لیے ہمارے [لیئر ۲ (<span dir="ltr">l2</span>) صفحہ](/layer-2/) کا "لیئر ۲ (<span dir="ltr">l2</span>) پر منتقل ہوں" سیکشن دیکھیں۔

### کیا میں اپنے ٹوکنز کو لیئر ۲ (<span dir="ltr">l2</span>) پر پل کرنے کے بعد واپس ایتھیریم مین نیٹ پر جا سکتا ہوں؟ {#can-i-go-back-to-ethereum-mainnet-after-i-bridge-my-tokens-to-l2}

جی ہاں، آپ ہمیشہ اسی پل کا استعمال کرتے ہوئے اپنے فنڈز کو واپس مین نیٹ پر منتقل کر سکتے ہیں۔
