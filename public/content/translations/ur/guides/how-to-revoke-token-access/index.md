# اسمارٹ کانٹریکٹ کی اپنے کرپٹو فنڈز تک رسائی کیسے منسوخ کریں

یہ گائیڈ آپ کو ان تمام [اسمارٹ کنٹریکٹس](/glossary/#smart-contract) کی فہرست دیکھنے کا طریقہ سکھائے گا جنہیں آپ نے اپنے فنڈز تک رسائی کی اجازت دی ہے اور انہیں کیسے منسوخ کرنا ہے۔

کبھی کبھی بدنیتی پر مبنی ڈیولپرز اسمارٹ کنٹریکٹس میں بیک ڈورز بناتے ہیں جو ان انجان صارفین کے فنڈز تک رسائی کی اجازت دیتے ہیں جو اسمارٹ کنٹریکٹ کے ساتھ تعامل کرتے ہیں۔ اکثر ایسا ہوتا ہے کہ ایسے پلیٹ فارمز مستقبل میں [گیس](/glossary/#gas) کی تھوڑی مقدار بچانے کی کوشش میں صارف سے **لامحدود تعداد میں ٹوکنز** خرچ کرنے کی اجازت مانگتے ہیں، لیکن اس سے خطرہ بڑھ جاتا ہے۔

ایک بار جب کسی پلیٹ فارم کو آپ کے [والیٹ](/glossary/#wallet) پر موجود ٹوکن تک لامحدود رسائی کے حقوق مل جاتے ہیں، تو وہ ان تمام ٹوکنز کو خرچ کر سکتے ہیں چاہے آپ نے ان کے پلیٹ فارم سے اپنے فنڈز اپنے والیٹ میں نکال لیے ہوں۔ بدنیتی پر مبنی اداکار اب بھی آپ کے فنڈز تک رسائی حاصل کر سکتے ہیں اور انہیں اپنے والیٹس میں نکال سکتے ہیں اور آپ کے لیے کوئی ریکوری کا آپشن باقی نہیں رہتا۔

واحد تحفظ یہ ہے کہ غیر آزمودہ نئے پروجیکٹس کا استعمال کرنے سے گریز کریں، صرف اسی چیز کی منظوری دیں جس کی آپ کو ضرورت ہے، یا باقاعدگی سے رسائی کو منسوخ کریں۔ تو، آپ یہ کیسے کرتے ہیں؟

## مرحلہ 1: رسائی منسوخ کرنے والے ٹولز کا استعمال کریں

کئی ویب سائٹس آپ کو اپنے پتے سے منسلک اسمارٹ کنٹریکٹس کو دیکھنے اور منسوخ کرنے کی سہولت دیتی ہیں۔ ویب سائٹ پر جائیں اور اپنا والیٹ جوڑیں:

- [Etherscan](https://etherscan.io/tokenapprovalchecker) (Ethereum)
- [Blockscout](https://eth.blockscout.com/apps/revokescout) (Ethereum)
- [Revoke](https://revoke.cash/) (متعدد نیٹ ورکس)
- [Unrekt](https://app.unrekt.net/) (متعدد نیٹ ورکس)
- [EverRevoke](https://everrise.com/everrevoke/) (متعدد نیٹ ورکس)

## مرحلہ 2: اپنا والیٹ جوڑیں

ایک بار جب آپ ویب سائٹ پر ہوں، "والیٹ جوڑیں" پر کلک کریں۔ ویب سائٹ کو آپ سے اپنا والیٹ جوڑنے کا اشارہ دینا چاہیے۔

یقینی بنائیں کہ آپ اپنے والیٹ اور ویب سائٹ میں ایک ہی نیٹ ورک استعمال کرتے ہیں۔ آپ کو صرف منتخب کردہ نیٹ ورک سے متعلق اسمارٹ کنٹریکٹس نظر آئیں گے۔ مثال کے طور پر، اگر آپ Ethereum Mainnet سے جڑتے ہیں، تو آپ کو صرف Ethereum کنٹریکٹس نظر آئیں گے، نہ کہ دیگر چینز جیسے Polygon کے کنٹریکٹس۔

## مرحلہ 3: ایک اسمارٹ کنٹریکٹ منتخب کریں جسے آپ منسوخ کرنا چاہتے ہیں

آپ کو وہ تمام کنٹریکٹس نظر آنے چاہئیں جنہیں آپ کے ٹوکنز تک رسائی کی اجازت ہے اور ان کی خرچ کرنے کی حد۔ وہ کنٹریکٹ تلاش کریں جسے آپ ختم کرنا چاہتے ہیں۔

اگر آپ نہیں جانتے کہ کون سا کنٹریکٹ منتخب کرنا ہے، تو آپ ان سب کو منسوخ کر سکتے ہیں۔ اس سے آپ کے لیے کوئی مسئلہ پیدا نہیں ہوگا، لیکن اگلی بار جب آپ ان میں سے کسی بھی کنٹریکٹ کے ساتھ تعامل کریں گے تو آپ کو اجازتوں کا ایک نیا سیٹ دینا ہوگا۔

## مرحلہ 4: اپنے فنڈز تک رسائی منسوخ کریں

ایک بار جب آپ منسوخ پر کلک کریں گے، تو آپ کو اپنے والیٹ میں ایک نئی ٹرانزیکشن کی تجویز نظر آنی چاہیے۔ یہ متوقع ہے۔ منسوخی کو کامیاب بنانے کے لیے آپ کو فیس ادا کرنی ہوگی۔ نیٹ ورک کے لحاظ سے اس پر کارروائی ہونے میں ایک منٹ سے کئی منٹ لگ سکتے ہیں۔

ہم آپ کو مشورہ دیتے ہیں کہ چند منٹ کے بعد منسوخ کرنے والے ٹول کو ریفریش کریں اور اپنے والیٹ کو دوبارہ جوڑ کر یہ یقینی بنائیں کہ منسوخ شدہ کنٹریکٹ فہرست سے غائب ہو گیا ہے۔

<mark>ہم تجویز کرتے ہیں کہ آپ پروجیکٹس کو اپنے ٹوکنز تک کبھی بھی لامحدود رسائی نہ دیں اور ٹوکن الاؤنس کی تمام رسائی کو باقاعدگی سے منسوخ کریں۔ ٹوکن تک رسائی کو منسوخ کرنے کے نتیجے میں کبھی بھی فنڈز کا نقصان نہیں ہونا چاہیے، خاص طور پر اگر آپ اوپر دیے گئے ٹولز کا استعمال کرتے ہیں۔</mark>

## اکثر پوچھے جانے والے سوالات

### کیا ٹوکن تک رسائی کو منسوخ کرنے سے اسٹیکنگ، پولنگ، قرض دینا وغیرہ بھی ختم ہو جاتا ہے؟

نہیں، یہ آپ کی کسی بھی [DeFi](/glossary/#defi) حکمت عملی کو متاثر نہیں کرے گا۔ آپ اپنی پوزیشنوں میں رہیں گے اور انعامات وغیرہ حاصل کرتے رہیں گے۔

### کیا کسی پروجیکٹ سے والیٹ کو منقطع کرنا میرے فنڈز استعمال کرنے کی اجازت ہٹانے کے مترادف ہے؟

نہیں، اگر آپ اپنے والیٹ کو پروجیکٹ سے منقطع کر دیتے ہیں، لیکن آپ نے ٹوکن الاؤنس کی اجازت دے رکھی ہے، تو وہ اب بھی ان ٹوکنز کا استعمال کر سکتے ہیں۔ آپ کو اس رسائی کو منسوخ کرنے کی ضرورت ہے۔

### کنٹریکٹ کی اجازت کب ختم ہوگی؟

کنٹریکٹ کی اجازتوں پر کوئی میعاد ختم ہونے کی تاریخ نہیں ہوتی ہے۔ اگر آپ کنٹریکٹ کی اجازت دیتے ہیں، تو ان کا استعمال کیا جا سکتا ہے، یہاں تک کہ اجازت دیے جانے کے سالوں بعد بھی۔

### پروجیکٹس لامحدود ٹوکن الاؤنس کیوں سیٹ کرتے ہیں؟

پروجیکٹس اکثر ایسا مطلوبہ درخواستوں کی تعداد کو کم کرنے کے لیے کرتے ہیں، جس کا مطلب ہے کہ صارف کو صرف ایک بار منظوری دینی ہوگی اور صرف ایک بار ٹرانزیکشن فیس ادا کرنی ہوگی۔ اگرچہ یہ آسان ہے، لیکن یہ صارفین کے لیے ان سائٹس پر لاپرواہی سے منظوری دینا خطرناک ہو سکتا ہے جو وقت کے ساتھ ثابت شدہ یا آڈٹ شدہ نہیں ہیں۔ کچھ والیٹس آپ کو اپنے خطرے کو محدود کرنے کے لیے منظور کیے جانے والے ٹوکنز کی مقدار کو دستی طور پر محدود کرنے کی اجازت دیتے ہیں۔ مزید معلومات کے لیے اپنے والیٹ فراہم کنندہ سے چیک کریں۔
