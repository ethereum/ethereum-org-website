یہ گائیڈ آپ کو سکھائے گی کہ ان تمام [سمارٹ کنٹریکٹس](/glossary/#smart-contract) کی فہرست کیسے دیکھیں جنہیں آپ نے اپنے فنڈز تک رسائی کی اجازت دی ہے اور انہیں کیسے منسوخ کریں۔

بعض اوقات بدنیتی پر مبنی ڈیولپرز سمارٹ کنٹریکٹس میں پچھلے دروازے (backdoors) بناتے ہیں جو ان بے خبر صارفین کے فنڈز تک رسائی کی اجازت دیتے ہیں جو سمارٹ کنٹریکٹ کے ساتھ تعامل کرتے ہیں۔ اکثر ایسا ہوتا ہے کہ ایسے پلیٹ فارمز مستقبل میں [گیس](/glossary/#gas) کی تھوڑی مقدار بچانے کی کوشش میں صارف سے **لامحدود تعداد میں ٹوکنز** خرچ کرنے کی اجازت مانگتے ہیں، لیکن اس سے خطرہ بڑھ جاتا ہے۔

ایک بار جب کسی پلیٹ فارم کو آپ کے [والیٹ](/glossary/#wallet) پر موجود ٹوکن تک لامحدود رسائی کے حقوق مل جاتے ہیں، تو وہ ان تمام ٹوکنز کو خرچ کر سکتے ہیں یہاں تک کہ اگر آپ نے اپنے فنڈز ان کے پلیٹ فارم سے اپنے والیٹ میں نکال لیے ہوں۔ بدنیتی پر مبنی عناصر اب بھی آپ کے فنڈز تک رسائی حاصل کر سکتے ہیں اور انہیں اپنے والیٹس میں نکال سکتے ہیں جس کے بعد آپ کے پاس بحالی کا کوئی آپشن نہیں بچتا۔

اس سے بچنے کا واحد طریقہ یہ ہے کہ غیر آزمودہ نئے پروجیکٹس استعمال کرنے سے گریز کریں، صرف اسی چیز کو منظور کریں جس کی آپ کو ضرورت ہو، یا باقاعدگی سے رسائی کو منسوخ کریں۔ تو، آپ یہ کیسے کرتے ہیں؟

## مرحلہ 1: رسائی منسوخ کرنے والے ٹولز استعمال کریں {#step-1-use-revoke-access-tools}

کئی ویب سائٹس آپ کو اپنے پتہ سے جڑے سمارٹ کنٹریکٹس دیکھنے اور منسوخ کرنے کی سہولت دیتی ہیں۔ ویب سائٹ پر جائیں اور اپنا والیٹ منسلک کریں:

- [<span dir="ltr">Etherscan</span>](https://etherscan.io/tokenapprovalchecker) (ایتھیریم)
- [<span dir="ltr">Blockscout</span>](https://eth.blockscout.com/essential-dapps/revoke) (ایتھیریم)
- [<span dir="ltr">Revoke</span>](https://revoke.cash/) (متعدد نیٹ ورکس)
- [<span dir="ltr">Unrekt</span>](https://app.unrekt.net/) (متعدد نیٹ ورکس)
- [<span dir="ltr">EverRevoke</span>](https://everrise.com/everrevoke/) (متعدد نیٹ ورکس)

## مرحلہ 2: اپنا والیٹ منسلک کریں {#step-2-connect-your-wallet}

ایک بار جب آپ ویب سائٹ پر آ جائیں، تو "والیٹ منسلک کریں" پر کلک کریں۔ ویب سائٹ آپ کو اپنا والیٹ منسلک کرنے کا اشارہ دے گی۔

یقینی بنائیں کہ آپ اپنے والیٹ اور ویب سائٹ میں ایک ہی نیٹ ورک استعمال کر رہے ہیں۔ آپ کو صرف منتخب کردہ نیٹ ورک سے متعلق سمارٹ کنٹریکٹس نظر آئیں گے۔ مثال کے طور پر، اگر آپ ایتھیریم مین نیٹ سے منسلک ہوتے ہیں، تو آپ کو صرف ایتھیریم کنٹریکٹس نظر آئیں گے، نہ کہ پولی گون جیسی دیگر چینز کے کنٹریکٹس۔

## مرحلہ 3: وہ سمارٹ کنٹریکٹ منتخب کریں جسے آپ منسوخ کرنا چاہتے ہیں {#step-3-select-a-smart-contract-you-wish-to-revoke}

آپ کو وہ تمام کنٹریکٹس نظر آنے چاہئیں جنہیں آپ کے ٹوکنز تک رسائی کی اجازت ہے اور ان کی خرچ کرنے کی حد بھی۔ وہ تلاش کریں جسے آپ ختم کرنا چاہتے ہیں۔

اگر آپ نہیں جانتے کہ کون سا کنٹریکٹ منتخب کرنا ہے، تو آپ ان سب کو منسوخ کر سکتے ہیں۔ اس سے آپ کے لیے کوئی مسئلہ پیدا نہیں ہوگا، لیکن اگلی بار جب آپ ان میں سے کسی بھی کنٹریکٹ کے ساتھ تعامل کریں گے تو آپ کو اجازتوں کا ایک نیا سیٹ دینا ہوگا۔

## مرحلہ 4: اپنے فنڈز تک رسائی منسوخ کریں {#step-4-revoke-access-to-your-funds}

ایک بار جب آپ منسوخ کرنے پر کلک کرتے ہیں، تو آپ کو اپنے والیٹ میں ایک نئی ٹرانزیکشن کی تجویز نظر آنی چاہیے۔ یہ متوقع ہے۔ منسوخی کے کامیاب ہونے کے لیے آپ کو فیس ادا کرنی ہوگی۔ نیٹ ورک پر منحصر ہے، اس پر کارروائی ہونے میں ایک منٹ سے لے کر کئی منٹ لگ سکتے ہیں۔

ہم آپ کو مشورہ دیتے ہیں کہ چند منٹ کے بعد منسوخ کرنے والے ٹول کو ریفریش کریں اور دوبارہ اپنا والیٹ منسلک کر کے دوہری جانچ کریں کہ آیا منسوخ شدہ کنٹریکٹ فہرست سے غائب ہو گیا ہے۔

<mark>ہم تجویز کرتے ہیں کہ آپ کبھی بھی پروجیکٹس کو اپنے ٹوکنز تک لامحدود رسائی کی اجازت نہ دیں اور باقاعدگی سے تمام ٹوکن الاؤنس کی رسائی کو منسوخ کریں۔ ٹوکن تک رسائی منسوخ کرنے کے نتیجے میں کبھی بھی فنڈز کا نقصان نہیں ہونا چاہیے، خاص طور پر اگر آپ اوپر دیے گئے ٹولز استعمال کرتے ہیں۔</mark>

## اکثر پوچھے گئے سوالات {#frequently-asked-questions}

### کیا ٹوکن تک رسائی منسوخ کرنے سے اسٹیکنگ، پولنگ، قرض دینا وغیرہ بھی ختم ہو جاتا ہے؟ {#does-revoking-token-access-also-terminate-staking-pooling-lending-etc}

نہیں، یہ آپ کی کسی بھی [غیر مرکزی مالیات (<span dir="ltr">DeFi</span>)](/glossary/#defi) حکمت عملی کو متاثر نہیں کرے گا۔ آپ اپنی پوزیشنز پر برقرار رہیں گے اور انعامات وغیرہ حاصل کرتے رہیں گے۔

### کیا کسی پروجیکٹ سے والیٹ کو منقطع کرنا میرے فنڈز استعمال کرنے کی اجازت ہٹانے کے مترادف ہے؟ {#is-disconnecting-a-wallet-from-a-project-the-same-as-removing-permission-to-use-my-funds}

نہیں، اگر آپ اپنے والیٹ کو پروجیکٹ سے منقطع کرتے ہیں، لیکن آپ نے ٹوکن الاؤنس کی اجازت دے رکھی ہے، تو وہ اب بھی ان ٹوکنز کو استعمال کر سکتے ہیں۔ آپ کو اس رسائی کو منسوخ کرنے کی ضرورت ہے۔

### کنٹریکٹ کی اجازت کب ختم ہوگی؟ {#when-will-the-contract-permission-expire}

کنٹریکٹ کی اجازتوں کی کوئی میعاد ختم ہونے کی تاریخ نہیں ہوتی۔ اگر آپ کنٹریکٹ کی اجازت دیتے ہیں، تو انہیں استعمال کیا جا سکتا ہے، یہاں تک کہ اجازت دیے جانے کے برسوں بعد بھی۔

### پروجیکٹس لامحدود ٹوکن الاؤنس کیوں سیٹ کرتے ہیں؟ {#why-do-projects-set-unlimited-token-allowance}

پروجیکٹس اکثر ایسا مطلوبہ درخواستوں کی تعداد کو کم کرنے کے لیے کرتے ہیں، جس کا مطلب ہے کہ صارف کو صرف ایک بار منظور کرنا پڑتا ہے اور لین دین کی فیس صرف ایک بار ادا کرنی پڑتی ہے۔ اگرچہ یہ آسان ہے، لیکن صارفین کے لیے ان سائٹس پر لاپرواہی سے منظوری دینا خطرناک ہو سکتا ہے جو وقت کے ساتھ ثابت شدہ نہیں ہیں یا جن کا آڈٹ نہیں کیا گیا ہے۔ کچھ والیٹس آپ کو اپنے خطرے کو محدود کرنے کے لیے منظور کیے جانے والے ٹوکنز کی مقدار کو دستی طور پر محدود کرنے کی اجازت دیتے ہیں۔ مزید معلومات کے لیے اپنے والیٹ فراہم کنندہ سے رجوع کریں۔
