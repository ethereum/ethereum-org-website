사람들이 자신감을 가지고 개발하고 필요한 지원을 받을 수 있도록 가능한 최고의 개발자 리소스를 나열하고자 합니다.

저희가 놓친 유용한 개발자 도구가 있다면, 적절한 곳에 자유롭게 제안해 주세요.

현재 [개발자 포털](/developers/) 전반에 걸쳐 개발자 도구를 나열하고 있습니다.

**적절한 페이지에 새로운 도구 추가를 자유롭게 제안해 주세요.**

## 결정 방법 {#ways-to-contribute}

개발자 도구 제출은 다음 기준에 따라 평가됩니다:

**이미 나열된 도구들과 의미 있게 차별화되는가?**

- 새로운 범주 또는 유형의 도구
- 기존의 유사한 도구와 비교하여 새로운 기능 제공
- 기존의 유사한 도구가 다루지 않는 뚜렷한 사용 사례를 대상으로 함

**도구에 대한 문서화가 잘 되어 있는가?**

- 문서가 존재하는가?
- 도구를 사용하기에 충분한가?
- 최근에 업데이트되었는가?

**도구가 널리 사용되는가?**

- GitHub 별표 수, 다운로드 통계, 알려진 회사나 프로젝트에서 사용하는지 여부 등의 지표를 고려합니다.

**도구의 품질이 충분한가?**

- 반복적으로 발생하는 버그가 있는가?
- 도구가 신뢰할 수 있는가?
- 도구가 활발하게 유지 관리되고 있는가?

**도구가 오픈 소스인가?**

이더리움 생태계의 많은 프로젝트는 오픈 소스입니다. 커뮤니티 개발자가 코드를 검사하고 기여할 수 있도록 허용하는 오픈 소스 프로젝트를 나열할 가능성이 더 높습니다.

## 제품 정렬 순서 {#product-ordering}

알파벳순과 같이 특별히 다른 순서로 정렬되지 않는 한, 제품은 페이지에 추가된 순서대로(가장 오래된 것부터 가장 최근 것 순으로) 표시됩니다. 즉, 가장 최신 제품이 목록의 맨 아래에 추가됩니다.
