# Ethereum Leitfäden

Möchten Sie Ihre Ethereum Reise starten? Unsere praktischen Leitfäden führen Sie Schritt für Schritt durch den Einstieg und erleichtern Ihnen den Umgang mit dieser neuen Technologie.

## Erste Schritte

1. [Wie Sie ein Ethereum-Konto „erstellen“](/guides/how-to-create-an-ethereum-account/) – Jeder kann kostenlos eine Wallet erstellen. Dieser Leitfaden zeigt Ihnen, wo Sie anfangen sollen.

2. [Wie Sie eine Wallet benutzen](/guides/how-to-use-a-wallet/) – Lernen Sie, wie Sie Token in Ihrer Wallet senden und empfangen und Ihre Wallet mit Projekten verbinden.

## Grundlagen zur Sicherheit

1. [Wie Sie den Zugriff von Smart Contracts auf Ihre Kryptogelder widerrufen](/guides/how-to-revoke-token-access/) – Falls Sie plötzlich eine Transaktion in Ihrer Wallet sehen, die Sie nicht initiiert haben, finden Sie in diesem Leitfaden eine Anleitung, wie sich das in Zukunft verhindern lässt.

2. [Wie Sie Scam-Tokens erkennen](/guides/how-to-id-scam-tokens/) – Was sind Scam-Tokens? Wie tarnen sie sich als legitim, und wie können Sie sie erkennen, um sich zu schützen und Betrug zu vermeiden?

## Ethereum verwenden

1. [Wie Sie Token in Layer 2 überführen](/guides/how-to-use-a-bridge/) – Sind Ethereum-Transaktionen zu teuer? Betrachten Sie den Wechsel auf sogenannte Layer-2 Skalierungslösungen.

2. [Wie Sie Token tauschen](/guides/how-to-swap-tokens/) – Möchten Sie Ihre Token gegen andere austauschen? Dieser simple Leitfaden zeigt Ihnen, wie Sie das machen.
