<FeaturedText>जानें कि .NET-आधारित प्रोजेक्ट्स और टूलिंग का उपयोग करके इथेरियम के लिए कैसे विकास करें</FeaturedText>

क्रिप्टोकरेंसी और ब्लॉकचेन तकनीक के लाभों का उपयोग करने वाले विकेंद्रीकृत एप्लिकेशन (dapp) बनाने के लिए इथेरियम का उपयोग करें। ये dapp भरोसेमंद हो सकते हैं, जिसका अर्थ है कि एक बार जब वे इथेरियम पर तैनात हो जाते हैं, तो वे हमेशा प्रोग्राम किए गए अनुसार ही चलेंगे। वे नए प्रकार के वित्तीय एप्लिकेशन बनाने के लिए डिजिटल संपत्तियों को नियंत्रित कर सकते हैं। वे विकेंद्रीकृत हो सकते हैं, जिसका अर्थ है कि कोई भी एकल संस्था या व्यक्ति उन्हें नियंत्रित नहीं करता है और उन्हें सेंसर करना लगभग असंभव है।

इथेरियम के ऊपर विकेंद्रीकृत एप्लिकेशन बनाएं और माइक्रोसॉफ्ट टेक्नोलॉजी स्टैक के टूल और भाषाओं का उपयोग करके स्मार्ट अनुबंधों के साथ इंटरैक्ट करें - .NET Framework/.NET Core/.NET Standard में VSCode और Visual Studio जैसे टूलिंग पर C#, # Visual Basic .NET, F# का समर्थन करते हुए। Microsoft Azure Blockchain का उपयोग करके मिनटों में Azure पर इथेरियम ब्लॉकचेन तैनात करें। .NET के प्यार को इथेरियम में लाएं!

## .NET समुदाय के योगदानकर्ता {#dot-net-community-contributors}

Nethereum में, हम ज़्यादातर [Gitter](https://gitter.im/Nethereum/Nethereum) पर समय बिताते हैं जहाँ सवाल पूछने/जवाब देने, मदद पाने या बस आराम करने के लिए सभी का स्वागत है। बेझिझक PR करें या [Nethereum GitHub रिपॉजिटरी](https://github.com/Nethereum) पर कोई समस्या (issue) खोलें, या हमारे पास मौजूद कई साइड/सैंपल प्रोजेक्ट्स को ब्राउज़ करें। आप हमें [डिस्कॉर्ड](https://discord.gg/jQPrR58FxX) पर भी पा सकते हैं!

यदि आप नेदरमाइंड में नए हैं और शुरुआत करने में मदद की आवश्यकता है, तो हमारे [डिस्कॉर्ड](https://discord.gg/PaCMRFdvWT) से जुड़ें। हमारे डेवलपर्स आपके सवालों के जवाब देने के लिए उपलब्ध हैं। PR खोलने या [नेदरमाइंड GitHub रिपॉजिटरी](https://github.com/NethermindEth/nethermind) पर कोई समस्या उठाने में संकोच न करें।
