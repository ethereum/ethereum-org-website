# Gemini Feedback: Context-Dependent Translation Challenges (Round 2)

Top 30 terms with significant context-dependent translation challenges.

## Terms requiring multiple translation forms

1. **Stake** - noun/verb/gerund/compound diverge; Slavic morphological cases, Romance root divergence
2. **Gas** - fuel metaphor vs unit vs limit; Arabic/Chinese gas/vapor vs fuel distinction
3. **Node** - hardware/software/network entity; JP/KO transliteration vs kanji debate
4. **Block** - data unit vs verb vs physical; FR/ES collision between bloc/bloquer
5. **Smart Contract** - code vs legal agreement; DE Vertrag, CN heyue/hetong distinction
6. **Oracle** - data provider vs mythological; Greek/Romance religious connotations
7. **Address** - hex string vs destination vs ENS name; Turkish/Hindi
8. **Token** - asset vs representation vs symbol; RU/PL default to "coin"
9. **Mint** - verb vs noun vs place; Romance/Germanic
10. **Burn** - verb vs mechanism; Semitic/East Asian
11. **Bridge** - infrastructure vs verb vs protocol; Hungarian/Finnish agglutinative
12. **Mainnet/Testnet** - noun vs adjective; transliteration vs translation debate
13. **Validator** - software vs person vs role; Romance gender, Slavic gendered suffixes
14. **Client** - software vs person; DE Client/Kunde, FR
15. **Wallet** - app vs hardware vs keys; JP saifu/woretto distinction
16. **Slashing** - penalty mechanism vs "cutting"; Vietnamese/Thai
17. **Rollup** - technology vs action; IT/PT literal translation sounds absurd
18. **Gwei** - denomination; AR/Cyrillic phonetic handling
19. **Peer** - network node vs social equal; CN duiding/tonghang
20. **State** - global database vs status vs political; all languages
21. **Epoch/Slot** - time period vs consensus cycle; Greek source, Romance
22. **Liquidity** - market depth vs fluidity; Arabic/Hebrew
23. **Yield** - financial return vs produce/give way; East Asian agricultural metaphors
24. **Hard Fork** - protocol split vs event; FR fourche/embranchement
25. **DeFi** - ecosystem vs tag vs category; acronym adoption vs linguistic purism
26. **Beacon Chain** - coordination layer; RU/UK finding non-poetic "guiding signal"
27. **Shard/Sharding** - database segment vs scaling; ES/IT
28. **Governance** - DAO voting vs protocol rules; PT/ES same word as "government"
29. **Nonce** - crypto term vs derogatory slang in some English dialects
30. **Merkle Tree** - proper name + structure; East Asian naming conventions
